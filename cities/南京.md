# 南京旅行攻略（快速可用版）

- 更新时间：2026-03-15
- 目的地：江苏省南京市
- 推荐停留：2–3 天最合适；1 天可刷核心；4 天适合把钟山、老城、河西和近郊都补全
- 适合人群：历史人文爱好者、博物馆控、近代史/六朝文化爱好者、亲子研学、城市摄影、周末短途旅行者
- 一句话总结：南京最强的地方，不是某一个单点景区，而是**帝王陵寝、近代中国、博物馆、古城墙、秦淮夜景和梧桐民国街景**能在一座地铁城市里高密度串起来，第一次去通常不会失望。

---

## 城市概览
南京是少数能把“古都感”和“现代城市便利度”同时做得很好的城市。它既有六朝、明代和民国留下来的强烈历史层次，也有成熟的地铁系统、稳定的酒店供给和适合周末出行的动线密度。

对第一次来的人，南京最大的优势有几条：
- **核心景点密度高**：钟山风景区、中山陵、明孝陵、南京博物院、总统府、夫子庙、秦淮河、城墙、玄武湖都能高效串联。
- **地铁好用**：绝大多数热门点位可通过地铁 + 短距离步行/打车解决。
- **题材丰富**：帝王陵寝、近代史、抗战纪念、寺庙、博物馆、动物园、夜景、书店都能安排。
- **2–3 天成功率很高**：不像有些城市要跑很散，南京很适合短假和周末。

但南京也有几个典型“踩坑点”：
- 钟山风景区看着集中，实际很大，**一天别贪太多点位**；
- 夫子庙、老门东、鸡鸣寺樱花季、秦淮夜游、红山动物园节假日都可能非常挤；
- 夏季闷热、暴晒明显，城墙和钟山爬坡体感会比地图看起来累得多；
- 很多热门场馆/景区都存在**预约、限流、分时入场**，先锁预约再排路线更稳。

如果是第一次来，建议把精力优先给：
1. **钟山风景区（中山陵 + 明孝陵）**
2. **南京博物院 / 总统府**
3. **夫子庙秦淮风光带 + 老门东夜游**
4. **南京城墙 / 玄武湖 / 鸡鸣寺**

## 行前速览

### 最佳旅行时间
- **3 月—5 月**：适合 Citywalk、樱花/梧桐/新绿，整体最舒服。
- **9 月—11 月**：秋高气爽，钟山、梧桐大道、明孝陵石象路尤其好看。
- **6 月—8 月**：非常热，湿度高，白天暴走容易崩；建议早出晚归，中午进博物馆。
- **12 月—2 月**：冷但可玩，人相对少；若遇雪景，城墙和钟山会很出片。

### 建议停留天数
- **1 天**：中山陵 / 明孝陵二选一 + 南京博物院 / 总统府二选一 + 夫子庙夜景
- **2 天**：第一天钟山，第二天老城 + 博物馆，适合大多数第一次来的人
- **3 天**：补城墙、玄武湖、纪念馆、牛首山、红山动物园或河西夜景，节奏最从容

### 大交通
- **高铁优先**：
  - **南京南站**：班次最多，地铁换乘方便，适合大多数游客。
  - **南京站**：离玄武湖、鸡鸣寺、老城更近，若住鼓楼/玄武一带很顺。
- **飞机**：南京禄口国际机场，进城可用地铁 S1 线换乘。
- **自驾**：不算最优方案。南京核心景区节假日停车压力大，老城和夫子庙周边体验一般。

### 市内交通
- **首选地铁**：2 号线、3 号线、1 号线和 4 号线对游客最关键。
- **钟山片区**：地铁只能解决到外围，景区内部常常需要接驳车、步行或短打车。
- **老城景点**：步行 + 地铁最高效。
- **节假日晚间**：夫子庙、老门东打车容易堵，地铁更稳。

### 穿衣建议
- 春秋：轻薄外套 + 舒适步行鞋。
- 夏季：防晒帽、墨镜、防晒霜、补水装备必须带；钟山和城墙非常晒。
- 秋冬：早晚温差明显，江边和城墙风大，建议带挡风外套。

### 住宿速选
- **第一次来 2 天**：住新街口 / 大行宫 / 夫子庙周边最省心。
- **喜欢安静与湖景**：住玄武湖、鼓楼、鸡鸣寺一带。
- **亲子 / 酒店条件优先**：河西、奥体、新街口更稳。
- **只玩钟山和南博**：明故宫 / 苜蓿园附近也可，但综合便利性不如新街口。

### 预约原则
优先处理以下几类：
- **中山陵**：免费但常需预约/分时
- **南京博物院**：免费但热门时段容易满
- **南京总统府**：实名预约购票更稳
- **侵华日军南京大屠杀遇难同胞纪念馆**：常见需提前预约
- **部分特展/夜游/节庆活动**：临近出发再核对公告

> 下面门票为公开资料与常见执行口径下的**参考价格**，实际以景区官方当日票务页面为准。

## 1-3 天行程建议

### 1 天精华版
**适合：第一次来南京、时间紧、但想看代表性内容的人**

**方案 A：历史人文优先**
- 上午：中山陵 → 明孝陵
- 下午：南京博物院
- 晚上：夫子庙 → 老门东 → 秦淮河夜景

**方案 B：老城 + 近代史优先**
- 上午：总统府
- 中午：新街口 / 长江路一带吃饭
- 下午：南京博物院 或 纪念馆
- 晚上：夫子庙 + 秦淮河夜游

> 1 天不要同时贪“钟山全套 + 老城全套 + 城墙”，会很赶。

### 2 天经典版
**适合：大多数第一次到南京的人**

#### Day 1：钟山主线
- 中山陵
- 明孝陵
- 美龄宫 / 灵谷寺 二选一
- 晚上回市区休息，或者去新街口吃饭

#### Day 2：城市核心线
- 总统府
- 南京博物院
- 夫子庙
- 老门东
- 秦淮河夜景 / 画舫

> 这是最稳的两天排法，几乎不会出大错。

### 3 天完整体验版
**适合：想把南京代表性内容尽量看全的人**

#### Day 1：钟山深度线
中山陵 → 明孝陵 → 美龄宫 → 灵谷寺 / 音乐台

#### Day 2：人文博物馆线
总统府 → 南京博物院 → 南京城墙（中华门或玄武门段）→ 夫子庙 / 老门东夜游

#### Day 3：按兴趣二选一
- **纪念与城市景观线**：侵华日军南京大屠杀遇难同胞纪念馆 → 玄武湖 → 鸡鸣寺 → 南京眼 / 鱼嘴湿地看日落
- **亲子与轻松线**：红山森林动物园 → 先锋书店 → 河西商圈 / 南京眼夜景

## 景点推荐（20 个，首次到访建议优先看前 10 个）

> 说明：每个景点都给出门票参考、建议时长、玩法、交通、预约要点和来源链接。票价、开放时间、预约规则可能随季节、活动、维护与政策调整，请在出发前以官方公告为准。

### 1. 中山陵
- **定位**：南京最具代表性的城市名片，钟山风景区核心点位
- **门票参考**：主体参观通常免费，常见需实名预约/分时入园
- **建议时长**：2–3 小时
- **推荐玩法**：走完整中轴线，感受台阶和纪念性空间；天气凉爽时体验最好；建议安排在钟山半天或一天主行程开头。
- **交通**：地铁 2 号线到苜蓿园/下马坊附近后换乘公交、景区接驳或打车
- **预约要点**：节假日和春秋旺季建议提前预约；迟到可能影响按时入园
- **来源链接**：
  - https://zschina.nanjing.gov.cn/fjms/fwzn/mpxx/
  - https://zschina.nanjing.gov.cn/

### 2. 明孝陵
- **定位**：南京最重要的帝王陵寝景区之一，世界文化遗产组成部分
- **门票参考**：成人票常见约 ¥70
- **建议时长**：2–3 小时
- **推荐玩法**：看神道石刻、陵寝格局、石象路；秋天尤其适合拍照；可与中山陵同日安排。
- **交通**：位于钟山风景区内，和中山陵可通过步行或景区接驳串联
- **预约要点**：旺季建议提前线上购票；石象路和核心区步行量不小，穿舒适鞋
- **来源链接**：
  - https://zschina.nanjing.gov.cn/
  - https://nj.bendibao.com/jingdian/mingxiaoling/

### 3. 南京总统府
- **定位**：近代中国历史叙事最集中的景点之一
- **门票参考**：成人票常见约 ¥35
- **建议时长**：2–3 小时
- **推荐玩法**：重点看煦园、办公旧址和近代史展陈；如果你对清末、民国、近代政务空间感兴趣，这里是必看。
- **交通**：地铁 2/3 号线大行宫站步行可达
- **预约要点**：建议通过官方微信小程序实名预约购票；节假日尽量早场
- **来源链接**：
  - http://www.njztf.cn/index.html
  - http://nj.bendibao.com/tour/2025425/172188.shtm

### 4. 南京博物院
- **定位**：华东地区最强博物馆之一，南京必看室内项目
- **门票参考**：常设展通常免费，常见需实名预约；特展另计
- **建议时长**：3–6 小时
- **推荐玩法**：第一次来建议优先看历史馆、特展和民国馆；如果想认真逛，半天只是起步。
- **交通**：地铁 2 号线明故宫站附近步行或换乘
- **预约要点**：热门时段建议提前预约；周一闭馆与否、特展安排以馆方公告为准
- **来源链接**：
  - https://activity.wisdommuseum.cn/reservation/activity/out/activityOutList.do
  - http://nj.bendibao.com/tour/2023417/152394.shtm

### 5. 夫子庙—秦淮风光带
- **定位**：南京最经典的夜游片区
- **门票参考**：街区免费；秦淮河游船常见日游约 ¥45 起、夜游约 ¥65 起
- **建议时长**：2–4 小时
- **推荐玩法**：傍晚后进入状态最好，看灯彩、逛小吃、拍河岸夜景；第一次来建议把夫子庙和老门东一起排。
- **交通**：地铁 3 号线夫子庙站
- **预约要点**：街区无需预约；游船节假日可能需要排队或提前购票
- **来源链接**：
  - https://www.fuzimiao.com/
  - https://piaojia.cn/jingdian/detail_5400.html

### 6. 老门东历史文化街区
- **定位**：南京老城夜游与吃喝氛围最好的片区之一
- **门票参考**：街区免费
- **建议时长**：2–3 小时
- **推荐玩法**：比夫子庙主街更适合慢逛，适合吃小吃、看院落、拍夜景；如果只想二选一，我更偏向把晚上的重心放这里。
- **交通**：地铁 3 号线武定门站步行可达
- **预约要点**：街区无需预约；热门餐饮店需排队
- **来源链接**：
  - https://www.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E8%80%81%E9%97%A8%E4%B8%9C

### 7. 侵华日军南京大屠杀遇难同胞纪念馆
- **定位**：南京最重要的纪念场馆之一
- **门票参考**：通常免费
- **建议时长**：1.5–3 小时
- **推荐玩法**：这是严肃纪念场所，不建议走马观花；看完最好留一点情绪缓冲时间，不要立刻安排太喧闹的项目。
- **交通**：地铁 2 号线云锦路站附近步行可达
- **预约要点**：常见需实名预约，热门日期建议提前数日查看名额
- **来源链接**：
  - http://www.19371213.net/
  - https://www.nanjing.gov.cn/hdjl/dwzsk/whjy/202410/t20241023_4991853.html

### 8. 南京城墙（推荐中华门段 / 玄武门段）
- **定位**：明城墙代表，南京最值得上的城市制高点之一
- **门票参考**：分段收费，中华门瓮城常见为收费核心点位
- **建议时长**：2–3 小时
- **推荐玩法**：第一次来建议优先中华门段，城防结构更强；想拍湖景就选玄武门段；傍晚时分最好看。
- **交通**：
  - 中华门段：地铁 1 号线中华门站
  - 玄武门段：地铁 1 号线玄武门站
- **预约要点**：一般可现场或线上购票；雨天城墙砖面可能湿滑
- **来源链接**：
  - http://www.njcitywall.com/
  - https://www.nanjing.gov.cn/

### 9. 玄武湖公园
- **定位**：南京市中心最舒服的大型湖景公园
- **门票参考**：公园主体免费
- **建议时长**：2–4 小时
- **推荐玩法**：适合散步、骑行、看日落，也适合把鸡鸣寺、城墙、南京站一线串起来；如果行程太满，这里可以作为“放松缓冲位”。
- **交通**：地铁 1 号线玄武门站，或南京站步行接入
- **预约要点**：一般无需预约；节庆活动以现场公告为准
- **来源链接**：
  - https://www.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E7%8E%84%E6%AD%A6%E6%B9%96

### 10. 鸡鸣寺
- **定位**：南京春季最热门的樱花和古寺打卡点之一
- **门票参考**：常见香花券/入寺票，价格以现场为准
- **建议时长**：1–2 小时
- **推荐玩法**：樱花季最火，但也最挤；非樱花季反而更适合慢看寺院本身；可与玄武湖、城墙同半天安排。
- **交通**：地铁 3/4 号线鸡鸣寺站
- **预约要点**：高峰期注意排队与限流，尽量早去；宗教场所注意礼仪
- **来源链接**：
  - https://www.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E9%B8%A1%E9%B8%A3%E5%AF%BA

### 11. 牛首山文化旅游区
- **定位**：南京近郊最成熟的大体量文化景区之一
- **门票参考**：成人票常见约 ¥98 左右，具体以当日票务为准
- **建议时长**：4–6 小时
- **推荐玩法**：核心看佛顶宫建筑空间，整体更偏“视觉震撼型”体验；如果你对宗教建筑空间、现代景区打造感兴趣，很值得。
- **交通**：地铁 + 公交/打车接驳更稳
- **预约要点**：节假日尽量提前购票，园区大、步行量不小
- **来源链接**：
  - https://www.niushoushan.net/
  - https://baike.baidu.com/item/%E7%89%9B%E9%A6%96%E5%B1%B1%E6%96%87%E5%8C%96%E6%97%85%E6%B8%B8%E5%8C%BA

### 12. 美龄宫
- **定位**：钟山风景区内最适合拍民国建筑的一站
- **门票参考**：收费点位，常见约 ¥30 左右
- **建议时长**：1–1.5 小时
- **推荐玩法**：秋天梧桐最出片；本体参观时间不长，适合作为中山陵/明孝陵之间的补充点。
- **交通**：位于钟山风景区内，依赖景区接驳或步行串联
- **预约要点**：旺季看钟山景区票务安排；如果当天体力一般，可以优先中山陵与明孝陵，把这里当可选项
- **来源链接**：
  - https://zschina.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E7%BE%8E%E9%BE%84%E5%AE%AB

### 13. 灵谷寺 / 灵谷景区
- **定位**：钟山风景区内适合慢走和看林荫的安静线路
- **门票参考**：收费景区，票价以钟山景区当日为准
- **建议时长**：2–3 小时
- **推荐玩法**：看无梁殿、松风阁、林荫步道；比中山陵、明孝陵更清静，适合不想一直在人潮里的人。
- **交通**：钟山风景区内接驳或步行
- **预约要点**：和钟山其他景点一起规划最顺；尽量不要在一天内把钟山所有收费点全部刷完
- **来源链接**：
  - https://zschina.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E7%81%B5%E8%B0%B7%E5%AF%BA

### 14. 音乐台
- **定位**：钟山风景区里最轻松好拍的补充点
- **门票参考**：收费点位，常见约 ¥10 左右
- **建议时长**：30–60 分钟
- **推荐玩法**：看白鸽、半圆形建筑和草坪视角；更适合轻松停留，不是重展陈型景点。
- **交通**：钟山风景区内步行/接驳可达
- **预约要点**：一般不复杂，现场购票常见；适合作为钟山日的小补丁
- **来源链接**：
  - https://zschina.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E9%9F%B3%E4%B9%90%E5%8F%B0

### 15. 大报恩寺遗址景区
- **定位**：南京佛教文化与遗址展示结合度很高的景区
- **门票参考**：成人票常见约 ¥90 左右
- **建议时长**：2–3 小时
- **推荐玩法**：看遗址、地宫、塔文化展示；夜景也不错；如果你想补一条“宗教与遗址”线，它很值得。
- **交通**：中华门、三山街一带换乘公交/打车较方便
- **预约要点**：节假日建议提前购票；适合和城墙/老门东同一天串联
- **来源链接**：
  - https://www.dbaensi.com/
  - https://baike.baidu.com/item/%E5%A4%A7%E6%8A%A5%E6%81%A9%E5%AF%BA

### 16. 朝天宫（南京市博物馆相关片区）
- **定位**：古建筑质感很强的地方史和建筑观赏点
- **门票参考**：常见约 ¥25 左右
- **建议时长**：1.5–2.5 小时
- **推荐玩法**：如果你喜欢古建院落和地方史展，这里体验很好；游客密度通常低于热门头部景点。
- **交通**：地铁 2 号线上海路附近换乘或步行
- **预约要点**：一般规则不复杂，按馆方公告执行
- **来源链接**：
  - https://www.njmuseum.com/
  - https://baike.baidu.com/item/%E6%9C%9D%E5%A4%A9%E5%AE%AB

### 17. 瞻园
- **定位**：老城里最适合补一个“小而精园林”的点位
- **门票参考**：常见约 ¥30 左右
- **建议时长**：1–2 小时
- **推荐玩法**：适合和夫子庙、老门东一起串联；相比苏州园林规模不算大，但位置好、补充性强。
- **交通**：夫子庙片区步行或短打车可达
- **预约要点**：常见现场或线上购票；夜间活动以公告为准
- **来源链接**：
  - https://www.nanjing.gov.cn/
  - https://baike.baidu.com/item/%E7%9E%BB%E5%9B%AD

### 18. 红山森林动物园
- **定位**：国内口碑很高的城市动物园，亲子和成年游客都适合
- **门票参考**：成人票常见约 ¥40
- **建议时长**：3–5 小时
- **推荐玩法**：不是单纯“带娃景点”，整体动线和动物展示都做得不错；如果同行有人不想一直看古迹，这里是很好的平衡项。
- **交通**：地铁 1 号线红山动物园站
- **预约要点**：节假日建议提前购票；园区起伏较多，推车和体力安排要提前考虑
- **来源链接**：
  - https://www.hszoo.cn/
  - https://baike.baidu.com/item/%E7%BA%A2%E5%B1%B1%E6%A3%AE%E6%9E%97%E5%8A%A8%E7%89%A9%E5%9B%AD

### 19. 先锋书店（五台山总店）
- **定位**：南京最知名的文化地标之一
- **门票参考**：免费
- **建议时长**：1–2 小时
- **推荐玩法**：更适合做行程里的“气质型休息点”，雨天尤其友好；如果你喜欢书店、文创、摄影，这里通常会喜欢。
- **交通**：上海路、汉中门一带步行/短打车可达
- **预约要点**：一般无需预约；周末人不少
- **来源链接**：
  - https://weibo.com/xianfengshudian
  - https://baike.baidu.com/item/%E5%85%88%E9%94%8B%E4%B9%A6%E5%BA%97

### 20. 南京眼步行桥 / 鱼嘴湿地公园
- **定位**：南京现代城市景观与长江日落代表线
- **门票参考**：多为免费公共空间
- **建议时长**：2–3 小时
- **推荐玩法**：如果前三天你已经看够古迹，这条线很适合收尾；傍晚看日落、夜里看现代天际线，能把南京“不是只有历史”的另一面补出来。
- **交通**：河西片区，地铁后接公交或打车更方便
- **预约要点**：一般无需预约；看天气和日落时间安排体验最好
- **来源链接**：

## 美食
南京不是那种“全国统一认知里最炸裂”的美食城市，但**本地早餐、鸭子体系、秦淮小吃和老馆子稳定度很高**。第一次去，不要试图一口气打卡 20 样，按场景吃最舒服。

### 必吃清单
- **盐水鸭 / 烤鸭**：来南京不吃鸭子几乎等于没来；适合打包当伴手礼。
- **鸭血粉丝汤**：最典型的南京快餐之一，适合作为早午餐或轻晚餐。
- **小笼包 / 汤包**：南京也有自己的早餐体系，不一定惊艳，但很稳。
- **牛肉锅贴**：外脆里汁多，适合配鸭血粉丝汤一起吃。
- **皮肚面**：南京本地面食代表，口味偏浓，适合重口一点的人。
- **赤豆元宵 / 糖芋苗 / 桂花糖芋苗**：甜口代表，适合夜游后收尾。
- **梅花糕、鸡汁汤包、蒸饺**：适合逛街时补充体力。

### 推荐觅食区域
- **夫子庙 / 老门东**：游客密度高，但第一次来最方便；适合边逛边吃。
- **新街口 / 大行宫**：选择多，连锁与本地馆子都集中，适合住这里的人。
- **科巷**：年轻人爱去的小吃密集区，适合多样化扫街。
- **湖南路 / 鼓楼周边**：老城区日常吃饭选择多。

### 吃饭建议
- 南京热门游客区的小吃有“氛围大于绝对味道”的情况，**别盲信排队长度**。
- 鸭血粉丝汤和鸭子类店铺在居民区附近往往更稳，没必要全部挤在景区里吃。
- 如果只安排一顿正式正餐，建议找一家评价稳定的**淮扬菜 / 金陵菜馆子**。

## 住宿

### 最推荐的住宿片区

#### 1. 新街口 / 大行宫
- **适合**：第一次来南京、只住 2 晚、想省时间的人
- **优点**：城市正中心，去总统府、南博、夫子庙、南京南站都方便；吃饭选择多
- **缺点**：热门商圈，房价可能偏高，部分老酒店房龄大

#### 2. 夫子庙 / 老门东周边
- **适合**：想把夜游体验拉满的人
- **优点**：晚上逛完直接回酒店最舒服；拍夜景和觅食方便
- **缺点**：节假日周边更吵，打车拥堵明显

#### 3. 玄武湖 / 鸡鸣寺 / 鼓楼一带
- **适合**：喜欢相对安静、偏城市景观的人
- **优点**：临湖和老城氛围好，去玄武湖、鸡鸣寺、南京站方便
- **缺点**：到夫子庙、老门东夜游结束后回程稍远

#### 4. 河西 / 奥体
- **适合**：带娃、商务型、对酒店品质要求更高的人
- **优点**：酒店普遍更新，房间条件更稳，商场配套好
- **缺点**：离传统老城景点稍远，不适合暴走式旅行

### 订房建议
- 第一次来南京，不折腾的话，**新街口/大行宫就是最稳解**。
- 如果行程重点是夫子庙和老门东夜游，住景区外围比住景区内部更实用。
- 樱花季、国庆、跨年、周末大型演唱会期间，南京房价会明显上浮，建议提前订。

## 交通

### 如何到南京
- **高铁**：
  - 南京南站：全国联通度最高，适合绝大多数游客
  - 南京站：更适合住玄武湖/鼓楼一带的人
- **飞机**：禄口机场进城时间一般要预留 1–1.5 小时

### 市内怎么走最省心
- **地铁优先**：对游客最友好的方式，没有之一。
- **老城步行**：总统府—长江路—新街口、夫子庙—老门东、鸡鸣寺—玄武湖都适合步行串联。
- **钟山片区**：靠地铁到外围，再结合接驳车/步行。
- **河西看夜景**：晚间打车比公交省时间，但散场高峰可能会堵。

### 常见动线建议
- **南京南站 → 新街口/大行宫**：地铁换乘最方便
- **新街口 → 总统府 / 南博 / 夫子庙**：地铁 2/3 号线非常高效
- **新街口 → 钟山**：地铁 + 接驳/打车常见
- **玄武湖 / 鸡鸣寺 → 南京站**：地铁或短距离步行都方便

## 避坑

### 1. 别把钟山风景区想得太“小”
地图上看着像一片，实际内部步行量大、坡度也有，**中山陵 + 明孝陵 + 美龄宫 + 灵谷寺 + 天文台**这类“全都要”的排法，很容易后半天直接废掉。

### 2. 先约热门点，再订死行程
中山陵、南京博物院、总统府、纪念馆这些一旦约不上，原本规划就会被打乱。**先锁票/锁预约，再排吃饭和夜游**，是最稳的顺序。

### 3. 夫子庙一定要管理预期
它是经典游客区，氛围和夜景有价值，但“全是隐藏神店”的概率不高。想吃得更稳，可以把正餐安排在新街口、科巷、居民区周边。

### 4. 南京夏天真的会把人晒蔫
尤其是城墙、钟山、牛首山、玄武湖这种大室外场景，中午暴走非常消耗体力。夏季建议把室内博物馆放在下午。

### 5. 樱花季鸡鸣寺和热门周末很容易失控
如果专门为樱花来，尽量工作日清晨去；如果只是顺路打卡，反而可以避开最火的时段。

### 6. 夜游别排太满
夫子庙、老门东、秦淮河、城墙夜景都想看很正常，但如果白天已经暴走一整天，晚上最好只抓一个重点，不然体验会从“夜游”变成“硬撑”。

### 7. 纪念馆不是“顺手打卡点”
侵华日军南京大屠杀遇难同胞纪念馆是非常严肃的场所，建议留足时间，遵守礼仪，不要把它塞在一个极度赶时间的流水线行程里。
