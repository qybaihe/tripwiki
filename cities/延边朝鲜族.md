# 延边朝鲜族

> 这是一个**说明型占位页**，不是常规“城市攻略”页。

## 这是什么

“延边朝鲜族”并不是一个边界清晰、可直接出发的标准城市或自治州名称。对普通旅行者来说，这个写法通常只是把**“延边朝鲜族自治州”截短了**，或者把“朝鲜族风情”这个主题词误当成了目的地名。

在真实出行语境里，游客会去的是：

- **延边朝鲜族自治州**：完整、正式、可执行的州级目的地；
- **延吉**：最常见的住宿、吃喝与中转中心；
- **图们 / 龙井 / 珲春 / 敦化 / 安图**：延边内部按边境线、长白山线、人文线展开的真实落点。

也就是说，“延边朝鲜族”更像一个**截断别名/不完整检索词**，而不是一个应该单独写成完整城市攻略的新对象。

## 为什么不写成普通城市攻略

因为它不对应新的独立目的地。

如果硬把“延边朝鲜族”写成正常城市页，会出现几个问题：

- **与《延边朝鲜族自治州》重复**：同一旅行对象被拆成两篇；
- **误导读者**：让人以为“延边朝鲜族”是一个正式行政区或独立城市；
- **后续维护分叉**：同一套景点、交通、住宿、美食信息会在两页分别更新，容易失真；
- **破坏检索导航**：别名页冒充主页面，会让真正该看的州级页面被稀释。

TripWiki 适合保留一个稳定主入口，而不是让截断名称和正式名称并存为两篇完整攻略。
