- 推荐指数：⭐4.3 / 5
- 适合天数：2–3 天
- 关键词：渭河源、陇西李氏文化、马家窑彩陶、贵清山、遮阳山、通渭温泉、县域串线

## 城市概览

定西位于甘肃中部偏南，是一座**非常典型的“要按县来玩”的城市**。如果只在市区停留，你大概率会觉得它普通；但把安定、陇西、渭源、漳县、通渭、临洮、岷县这些县区串起来看，定西会立刻变成一条内容密度很高的旅行带：一头是**渭河源与秦岭西端山地**，一头是**马家窑文化、李氏文化、红色长征记忆、温泉康养和书画之乡**。

它最适合的打开方式，不是“刷一座城市地标”，而是按兴趣选线：
- 想看山水避暑，优先 **渭源 + 漳县**；
- 想看历史人文，优先 **陇西 + 临洮**；
- 想休闲泡汤，优先 **通渭**；
- 想做小众西北县域串游，定西反而会比许多热门城市更耐玩。

**定西适合谁：**
- 喜欢小众目的地、县域慢游的人
- 对彩陶、考古、宗族文化、红色历史感兴趣的人
- 想自驾走山地 + 古建 + 温泉组合的人
- 夏秋季想找西北避暑线的人

**不太适合谁：**
- 只想在主城区步行逛 1 天的人
- 不愿意坐车、不接受景点分散的人
- 只偏爱顶流网红景区、对地方文化没兴趣的人

一句话判断：**定西不是“城市型旅游目的地”，而是“甘肃中部县域旅行集合体”。**

## 行前速览

### 最佳旅行时间
- **4–6 月**：山体返青，气温舒服，适合渭源、漳县线
- **7–8 月**：渭河源、首阳山、贵清山、遮阳山体感明显优于平原城市，适合避暑
- **9–10 月**：山地秋景和县城人文线都在线，是综合体验最稳的时候
- **11 月–次年 3 月**：能去，但山区可能遇结冰、封闭、风雪天气，体验波动较大

### 建议停留时间
- **1 天**：选 1 个县深挖，最推荐陇西或渭源
- **2 天**：可以做“山水线”或“人文线”中的一条完整组合
- **3 天**：比较舒服，能把渭源 / 漳县 / 陇西 / 临洮或通渭串起来
- **4 天以上**：适合自驾做定西全域慢游

### 第一次去怎么选
- **第一次、只想稳妥不踩空**：陇西 + 临洮
- **第一次、偏自然**：渭源 + 漳县
- **第一次、偏休闲**：通渭温泉 + 悦心国际书画村
- **时间极少**：安定区博物馆 + 西岩山 / 城区休闲线

### 气候与穿衣
定西整体海拔不算极端，但**昼夜温差、山地微气候、风感和紫外线**都比较明显。

建议常备：
- 防风外套 / 薄冲锋衣
- 防滑运动鞋或徒步鞋
- 防晒霜、帽子、墨镜
- 秋季和夏季进山都建议带一件长袖

### 预算参考（按 2 天游）
- **经济型**：500–900 元 / 人
- **常规型**：900–1800 元 / 人
- **舒适型**：1800–3200 元 / 人

主要差异在于：是否自驾、是否住温泉酒店、是否多县移动。

### 交通判断
- **最推荐：自驾**
- **次推荐：高铁 / 火车到节点县城 + 打车 / 包车**
- **不太推荐：纯公交硬串自然景区**

定西真正值得去的自然点大多在县域和山区，公共交通能到县城，但**末端到景区普遍不如自驾省心**。

## 1–3 天推荐行程

### 1 天玩法

#### 方案 A：陇西人文精华线
**李家龙宫 → 威远楼 → 陇西堂 → 古莱坞文化旅游度假区**

适合第一次到定西、喜欢历史与文化地标的人。整体移动短、容错高，是不自驾也相对好操作的一条线。

#### 方案 B：渭源山水轻徒步线
**渭河源景区 → 首阳山 / 莲峰山（二选一）**

适合想看源头地理景观、森林、峡谷和山地步道的人。夏季尤其舒服。

#### 方案 C：安定区中转半日线
**定西市博物馆 → 西岩山公园 → 市区小吃**

适合高铁 / 公路中转、时间有限的人，重点是建立对定西历史的基础认知。

### 2 天玩法

#### 方案 A：渭源 + 漳县自然线（最推荐）
**D1：渭河源景区 → 首阳山，住渭源**  
**D2：贵清山 / 遮阳山（二选一或体力强可连游），返程**

#### 方案 B：陇西 + 临洮人文线
**D1：李家龙宫 → 威远楼 → 古莱坞，住陇西**  
**D2：马家窑遗址 → 岳麓山 → 洮河风情线，返程**

#### 方案 C：通渭休闲线
**D1：榜罗会议纪念馆 → 悦心国际书画村，住通渭**  
**D2：通渭温泉 → 县城慢逛 / 返程**

### 3 天玩法

#### 方案 A：定西王牌山水线
**D1：渭河源景区 → 莲峰山，住渭源**  
**D2：首阳山 → 贵清山，住漳县**  
**D3：遮阳山 → 返回定西 / 兰州方向**

#### 方案 B：人文考古线
**D1：定西市博物馆 → 陇西李家龙宫 → 威远楼，住陇西**  
**D2：马家窑遗址 → 岳麓山 → 洮河风情线，住临洮**  
**D3：榜罗会议纪念馆 / 通渭温泉，返程**

#### 方案 C：休闲慢游线
**D1：通渭温泉 → 悦心国际书画村，住通渭**  
**D2：榜罗会议纪念馆 → 安定区市博物馆，住定西**  
**D3：西岩山公园 → 城区美食 → 返程**

## 景点推荐（22 个）

> 说明：以下门票、开放时间、交通方式均按公开资料与常见平台信息整理，主要用于出发前快速判断。**最终以景区 / 场馆当日公示为准。** 定西部分县域景点会因天气、养护、淡旺季调整票价或开放方式。

### 1. 渭河源景区（渭源县）
- **门票参考**：约 50–70 元
- **建议时长**：3–5 小时
- **怎么玩**：这是定西自然线最具代表性的目的地之一，核心看点是渭河源头意象、溪谷森林和避暑步道。第一次去定西，偏自然的人大多会把它排第一位。
- **交通**：从渭源县城出发更方便；自驾最佳，不自驾建议包车 / 打车往返
- **预约要点**：旺季周末建议早点出发；雨后路面湿滑，穿防滑鞋
- **来源链接**：
  - https://www.dingxi.gov.cn/
  - https://baike.baidu.com/item/甘肃渭河源国家森林公园/23475101

### 2. 甘肃渭河源国家森林公园（渭源县）
- **门票参考**：常见与相关片区联动，参考 50–80 元
- **建议时长**：4–6 小时
- **怎么玩**：这是理解渭源县山地生态最完整的入口，包含莲峰山、首阳山、天井峡、石门水库等片区。适合想把“渭河源”玩得更深的人。
- **交通**：自驾最省心
- **预约要点**：园区范围较大，不要低估移动时间；山里温差和天气变化明显
- **来源链接**：
  - https://baike.baidu.com/item/甘肃渭河源国家森林公园/23475101
  - https://www.dingxi.gov.cn/

### 3. 首阳山景区（渭源县）
- **门票参考**：约 30–60 元
- **建议时长**：3–5 小时
- **怎么玩**：首阳山兼具山景和伯夷叔齐文化传说，适合登山、拍山体层次和做半日轻徒步。它的辨识度在渭源非常高。
- **交通**：从渭源县城自驾前往最方便
- **预约要点**：体力一般的话别和渭河源重度步行全叠在同一天；雨后石阶会滑
- **来源链接**：
  - https://baike.baidu.com/item/甘肃渭河源国家森林公园/23475101
  - https://baike.baidu.com/item/首阳山

### 4. 莲峰山（渭源县）
- **门票参考**：约 20–50 元，部分时段可能按公园方式管理
- **建议时长**：2–4 小时
- **怎么玩**：莲峰山是渭源县山体地貌代表之一，适合轻登山、拍峰林、看县域山景，和首阳山相比节奏会更轻一些。
- **交通**：自驾最佳
- **预约要点**：适合搭配渭河源一起做 1 天；不要把它想成完全平缓的城市公园
- **来源链接**：
  - https://baike.baidu.com/item/甘肃渭河源国家森林公园/23475101
  - https://www.dingxi.gov.cn/

### 5. 天井峡片区（渭源县）
- **门票参考**：约 0–40 元，或包含在联票内
- **建议时长**：1.5–3 小时
- **怎么玩**：适合作为渭河源国家森林公园深度玩法的补充，看峡谷、丹霞感地貌和山间水系。
- **交通**：更适合自驾串联
- **预约要点**：不是所有游客都需要专门去，适合给 2–3 天深度线加料
- **来源链接**：
  - https://baike.baidu.com/item/甘肃渭河源国家森林公园/23475101
  - https://www.dingxi.gov.cn/

### 6. 石门水库片区（渭源县）
- **门票参考**：约 0–40 元，常见为联游片区
- **建议时长**：1–2 小时
- **怎么玩**：适合轻松散步、看山水倒影和补一个节奏较慢的停留点，不算震撼型景区，但很适合作为渭源自然线的缓冲段。
- **交通**：自驾串联最方便
- **预约要点**：更适合顺路，不建议单独远程专程去
- **来源链接**：
  - https://baike.baidu.com/item/甘肃渭河源国家森林公园/23475101
  - https://www.dingxi.gov.cn/

### 7. 双石门（渭源 / 漳县周边山地小众点）
- **门票参考**：约 0–40 元，以现场为准
- **建议时长**：2–4 小时
- **怎么玩**：偏小众户外和摄影向，适合看天然石门、山地谷地和高原夏季景观。更像“会玩的人加的一站”，不是大众必刷点。
- **交通**：基本建议自驾
- **预约要点**：雨后泥地和碎石较多，鞋子一定要靠谱
- **来源链接**：
  - https://baike.baidu.com/item/双石门/1744665
  - https://www.dingxi.gov.cn/

### 8. 贵清山（漳县）
- **门票参考**：约 50–80 元
- **建议时长**：4–6 小时
- **怎么玩**：这是定西最强的山岳景区之一。看点包括贵清峡、峰林、古刹、断涧仙桥、森林与山谷层次。夏秋去最稳，景观完成度很高。
- **交通**：距漳县县城较远，自驾最方便；也可县城包车
- **预约要点**：景区不小，别按“2 小时速通”估算；遇庙会和旺季人流会明显增加
- **来源链接**：
  - https://baike.baidu.com/item/贵清山/9614459
  - https://www.dingxi.gov.cn/

### 9. 遮阳山（漳县）
- **门票参考**：约 40–70 元
- **建议时长**：4–6 小时
- **怎么玩**：遮阳山以岩溶、峡谷、洞穴、丹霞感地貌见长，是漳县另一张硬牌。相比贵清山，它更偏“奇险幽深”的路线。
- **交通**：位于漳县大草滩乡，距县城约 29 公里，自驾 / 包车更现实
- **预约要点**：雨后景观更好，但山道更滑；若赶上自然岩壁攀岩活动或赛事，注意景区分区安排
- **来源链接**：
  - https://baike.baidu.com/item/遮阳山/8539505
  - https://baike.baidu.com/item/遮阳山旅游风景区/185065

### 10. 李家龙宫（陇西县）
- **门票参考**：约 20–40 元
- **建议时长**：1.5–3 小时
- **怎么玩**：陇西最核心的人文地标，也是“天下李氏出陇西”文化认知的关键点。适合看祠庙建筑、李氏宗族文化、古建筑细节和地方认同。
- **交通**：陇西县城内或近县城区域，打车 / 自驾都方便
- **预约要点**：遇祭祖活动、节庆时人流会增加；注意礼仪，不要把祠堂当成纯娱乐场所
- **来源链接**：
  - https://baike.baidu.com/item/李家龙宫/7472699
  - https://www.dingxi.gov.cn/

### 11. 李家龙宫遗址（陇西县）
- **门票参考**：常与主景区联动，参考 0–30 元
- **建议时长**：40 分钟–1 小时
- **怎么玩**：更偏遗址与修复史视角，适合对文保、宗族文化沿革有兴趣的人。普通游客可以和李家龙宫连看，不必单独安排太久。
- **交通**：与李家龙宫一并到达最省事
- **预约要点**：适合配合讲解看，否则信息密度不够直观
- **来源链接**：
  - https://baike.baidu.com/item/李家龙宫/7472699
  - https://www.dingxi.gov.cn/

### 12. 陇西堂（陇西县）
- **门票参考**：通常免费或低门槛开放
- **建议时长**：30–60 分钟
- **怎么玩**：这是李氏文化的符号化地标，适合拍照、补全“陇西堂号”的文化线索。更适合作为李家龙宫串线点。
- **交通**：县城内连游方便
- **预约要点**：顺路看即可，不建议单独专程跑一趟
- **来源链接**：
  - https://baike.baidu.com/item/陇西堂/1005969
  - https://www.dingxi.gov.cn/

### 13. 威远楼（陇西县）
- **门票参考**：约 0–30 元，以是否开放登楼为准
- **建议时长**：30–60 分钟
- **怎么玩**：陇西县城最醒目的老城地标之一。适合拍古楼、看老城轴线，傍晚和夜间外观最好拍。
- **交通**：县城中心到达方便
- **预约要点**：登楼开放时间可能调整，想上楼最好白天去
- **来源链接**：
  - https://www.dingxi.gov.cn/
  - https://baike.baidu.com/item/威远楼

### 14. 古莱坞文化旅游度假区（陇西县）
- **门票参考**：园区 / 项目票差异较大，参考 30–120 元
- **建议时长**：2–4 小时
- **怎么玩**：它更像大型文旅综合区，偏仿古建筑、影视街区、亲子游乐、戏水和休闲拍照，不是传统文保类景点。带孩子或想轻松娱乐的人会更容易觉得值。
- **交通**：自驾或打车前往方便
- **预约要点**：不同票种差很多，先确认自己是去逛街区、玩乐园还是玩水
- **来源链接**：
  - https://baike.baidu.com/item/古莱坞/18470991
  - https://www.dingxi.gov.cn/

### 15. 定西市博物馆（安定区）
- **门票参考**：通常免费，常见需实名入馆
- **建议时长**：1.5–2.5 小时
- **怎么玩**：想快速建立对定西整体认知，这里优先级很高。馆内重点是史前彩陶、地方历史、人文定西、红色定西等内容。新馆位于定西湖畔，整体观展条件比老印象好很多。
- **交通**：安定区城区内打车方便
- **预约要点**：注意周一闭馆 / 特定活动日调整；节假日前最好先查馆方通知
- **来源链接**：
  - https://baike.baidu.com/item/定西市博物馆/20111339
  - https://www.dingxi.gov.cn/

### 16. 西岩山公园（安定区）
- **门票参考**：通常免费
- **建议时长**：1–2 小时
- **怎么玩**：定西城区最适合“顺手爬个山、看看城景”的地方。更适合中转和本地休闲，不是需要专门大老远去打卡的重磅景点。
- **交通**：市区打车 / 公交都相对方便
- **预约要点**：傍晚视野更好，雨天台阶注意打滑
- **来源链接**：
  - https://www.dingxi.gov.cn/
  - https://baike.baidu.com/item/西岩山

### 17. 榜罗会议纪念馆（通渭县）
- **门票参考**：免费
- **建议时长**：1–2 小时
- **怎么玩**：定西最重要的红色旅游点之一，1935 年“榜罗会议”在中国革命史上地位很高。适合看展陈、旧址、了解长征路线转折节点。
- **交通**：位于通渭县榜罗镇，从县城自驾 / 包车更方便
- **预约要点**：常见开放时段为白天；团队参观最好提前确认讲解安排
- **来源链接**：
  - https://baike.baidu.com/item/榜罗镇会议纪念馆/13778807
  - https://www.dingxi.gov.cn/

### 18. 悦心国际书画村（通渭县）
- **门票参考**：基础参观约 0–60 元，展馆 / 温泉 / 酒店项目另计
- **建议时长**：2–4 小时
- **怎么玩**：这是“书画通渭”最直观的落地点。书画长廊、美术馆、画廊集群、园林式度假配套都集中在这里，适合慢逛和休闲拍照。
- **交通**：位于通渭西城片区，离高铁站和县城都不算太远，自驾最佳
- **预约要点**：如果想看展览、住园区酒店或泡温泉，最好提前分清票种与营业时间
- **来源链接**：
  - https://baike.baidu.com/item/悦心国际书画村/67365754
  - https://www.dingxi.gov.cn/

### 19. 通渭温泉（通渭县）
- **门票参考**：大众温泉票常见约 68–158 元，私汤 / 酒店套票另算
- **建议时长**：2–5 小时
- **怎么玩**：通渭温泉是“陇上神泉”代表之一，地热资源成熟，适合做康养、亲子和旅途休整。冬天和秋末尤其讨喜。
- **交通**：位于通渭县城西南约 8 公里处，自驾 / 打车都方便
- **预约要点**：周末和节假日晚间最热；自带泳衣拖鞋通常更省事
- **来源链接**：
  - https://baike.baidu.com/item/通渭温泉/2519018
  - https://baike.baidu.com/item/通渭县温泉/11055225

### 20. 马家窑遗址（临洮县）
- **门票参考**：约 0–50 元，以现场开放方式为准
- **建议时长**：1.5–3 小时
- **怎么玩**：这是定西历史含金量最高的点位之一，中国彩陶史绕不开这里。适合历史迷、考古爱好者和亲子科普。它的价值更多在“文明线索”，不是看视觉奇观。
- **交通**：临洮方向自驾最佳
- **预约要点**：遗址类景点强烈建议结合讲解或先看博物馆背景再去
- **来源链接**：
  - https://baike.baidu.com/item/马家窑遗址/7043410
  - https://www.dingxi.gov.cn/

### 21. 岳麓山（临洮县）
- **门票参考**：约 0–30 元
- **建议时长**：1.5–3 小时
- **怎么玩**：临洮县城边上的代表性山体景观，适合登高望城、看亭台古迹、做半日轻徒步。它更偏地方名山，不是超级大景区路线。
- **交通**：从临洮县城过去方便
- **预约要点**：适合和马家窑遗址搭配，不建议单独放太高预期
- **来源链接**：
  - https://baike.baidu.com/item/岳麓山/24167760
  - https://www.dingxi.gov.cn/

### 22. 洮河风情线（临洮县）
- **门票参考**：免费
- **建议时长**：1–2.5 小时
- **怎么玩**：这是临洮县相对轻松、适合夜景和顺路散步的河谷景观带。看点是滨河廊道、洮河水面、地方文化装置，适合做一天行程的收尾。
- **交通**：自驾或县城打车都方便
- **预约要点**：开放式景观带，无需预约；更适合作为补充点，不是终极目的地
- **来源链接**：
  - https://baike.baidu.com/item/洮河风情线/12601335
  - https://www.dingxi.gov.cn/

### 23. 秦长城遗址（岷县段）
- **门票参考**：多为免费或低门槛开放
- **建议时长**：1–2 小时
- **怎么玩**：适合历史遗址爱好者。它更偏文献考据和地理现场感，不是成熟景区型玩法。放在岷县 / 洮河一线顺访更合理。
- **交通**：自驾更现实
- **预约要点**：遗址分布较散，导航和乡道判断要提前做好功课
- **来源链接**：
  - https://baike.baidu.com/item/秦长城遗址/17780136
  - https://www.dingxi.gov.cn/

### 24. 岷州古城 / 岷县老城片区（岷县）
- **门票参考**：开放式街区多为免费
- **建议时长**：1–2 小时
- **怎么玩**：如果路线做到岷县，可以顺手感受一下岷州古城历史氛围、老街尺度和县城烟火。它不是巨型景区，更像“补一层地方气质”。
- **交通**：县城内到达方便
- **预约要点**：适合顺路散步和吃饭，不建议专程跑很远只看这一处
- **来源链接**：
  - https://www.dingxi.gov.cn/
  - https://baike.baidu.com/item/岷州

## 美食

定西的吃法，不靠“网红名店”，而靠**县城馆子 + 面食体系 + 土豆 + 牛羊 + 浆水**。

### 值得优先吃
- **定西宽粉**：定西非常有辨识度的本地名片，煮火锅、酸辣粉、炒粉都能试
- **洋芋搅团 / 洋芋擦擦 / 洋芋丸子**：定西是“马铃薯之乡”，土豆系小吃别跳过
- **浆水面**：夏季非常友好，酸爽解腻
- **荞面饸饹**：很多县城都能找到靠谱店
- **牛肉面 / 牛肉小饭**：在县城吃往往比景区更稳
- **臊子面**：临洮、陇西、通渭都常见
- **手抓羊肉 / 暖锅**：适合多人、秋冬更合适
- **酿皮、凉粉**：当加餐很好用
- **杂粮面点**：很能体现陇中饮食结构

### 觅食建议
- **安定区**：适合中转时找综合馆子、小吃店
- **陇西县城**：老城周边吃饭更有地方感
- **渭源 / 漳县**：景区边多是农家乐，讲究环境就留在县城吃
- **通渭**：温泉区适合休闲餐，想吃地道还是回县城更稳
- **临洮**：人文线做完后在县城收一顿家常馆子很舒服

## 住宿

### 住哪里最合理
- **安定区**：适合到达 / 离开日中转，酒店选择最多
- **陇西县城**：适合李家龙宫、威远楼、古莱坞线路
- **渭源县城**：适合渭河源、首阳山、莲峰山线，推荐自然线住这里
- **漳县县城**：适合贵清山、遮阳山两日线
- **通渭县城 / 温泉度假区**：适合休闲泡汤
- **临洮县城**：适合马家窑遗址、岳麓山线

### 住宿选择建议
- **预算型**：县城连锁快捷或本地商务酒店即可
- **舒适型**：优先选近主干道、停车方便、近两年新开的酒店
- **度假型**：通渭温泉配套酒店更有休闲感

### 订房提示
- 节假日、庙会、研学团、红色旅游节点时，部分县城价格会明显上浮
- 山地景区周边住宿别抱太高预期，住县城通常更稳

## 交通

### 外部交通
- **高铁 / 火车**：可优先到定西、陇西、临洮、通渭等节点
- **自驾**：最适合定西，尤其是渭源、漳县、岷县这类自然线
- **长途汽车**：可补足县际移动，但景区末端接驳会拉长时间

### 市内与县际移动
- **安定区 / 陇西 / 临洮县城内**：打车基本够用
- **县城到景区**：很多没有高频直达接驳，建议包车 / 拼车 / 自驾
- **山地驾驶提醒**：夜间山路、雨后湿滑、冬春积雪结冰都要保守驾驶

### 推荐交通策略
- **只有 1 天**：别贪多，只做一个县
- **2–3 天**：一条主线 + 一个补充点最合理
- **不自驾**：优先陇西 / 通渭 / 安定 / 临洮；渭源和漳县会更考验接驳

## 避坑指南

### 1. 不要只逛定西市区就下结论
定西真正的亮点很多在县域。只留在安定区，很容易得出“没啥可玩”的结论。

### 2. 不自驾时不要硬串多个自然景区
地图看着近，实际县城到景区、景区到景区耗时并不短。没车时，一天一个大点已经很满。

### 3. 山地景区一定穿防滑鞋
贵清山、遮阳山、首阳山、双石门这些地方，雨后、阴坡、石阶都很容易滑，鞋子比拍照更重要。

### 4. 遗址类景点要带着背景去看
马家窑遗址、秦长城遗址这类地方，不是那种“到现场就惊艳”的类型。提前看一点资料，体验差别会很大。

### 5. 别迷信旧门票信息
定西不少景区票价、摆渡、联票、开放时间会根据淡旺季和维护期调整，出发前一定复核。

### 6. 通渭温泉和书画村不是默认一票全通
不同景区、展馆、泡池、酒店常常分开售卖，先确认票种再去。

### 7. 冬季别低估风和干冷
定西冬季不是简单“数字不低就不冷”，很多山口和高地体感会比预期硬很多。

### 8. 景区餐饮不要期待过高
自然景区周边以农家乐和简餐为主，吃得稳这件事，往往还是县城馆子更靠谱。
