## 城市概览

尉犁最容易被误解成“罗布人村寨那个县”。

这种理解不算全错，但太浅。真正好用的打开方式，不是只记一个景区名，而是先看懂它的**三层结构**：

1. **罗布人村寨主场层**：第一次到尉犁，最完整、最省力、最容易建立直觉的主场；
2. **塔里木河—胡杨—湿地自然层**：离开景区后，继续沿塔里木河、胡杨带、罗布淖尔湿地去看“为什么这里能活下来”；
3. **县城补给与公路节点层**：尉犁不是看完就撤的壳，而是一个可以住宿、加油、补货、吃热饭、整备第二天长线的现实节点。

这三层叠在一起，尉犁才成立。

公开资料常见口径显示：尉犁位于天山南麓、塔里木盆地东北缘，县城距库尔勒约 20 公里，距库尔勒机场约 15—25 公里，县城到罗布人村寨约 28 公里；塔里木河在尉犁境内流域约 254 公里。也就是说，尉犁并不是那种“必须深入腹地很多天才能摸到边”的地方，而是**从库尔勒极易切入、但地理气质已经明显转硬**的那个边界节点。

### 为什么尉犁值得单独写一篇

- **它是库尔勒东南向最容易成立的一段短线。** 落地库尔勒后很快就能切进真正的荒漠河流地理现场。
- **它能把“河流如何缝住沙漠”讲得很清楚。** 在很多地方你只看到胡杨，或者只看到沙漠；到了尉犁，你能把河流、湿地、胡杨、聚落、人类活动放在一张图里看。
- **它是巴州东线非常实用的换挡点。** 再往东南，距离更长、点位更散、补给更依赖判断；尉犁正好是从州府舒适区切到荒漠公路线之前的缓冲站。
- **它天然适合住 1 晚。** 不是宣传口号，而是旅行效率问题：住下以后，主线能玩完整，副线能补上，第二天继续跑长线也更从容。

### v6 核心纠偏：尉犁不能只写成“一个景区县”

这版最重要的纠偏是三件事：

#### 1）罗布人村寨必须作为主场，而不是孤零零一个打卡点
第一次来尉犁，主线一定是罗布人村寨。这里最值钱的不是“木屋 + 沙丘”本身，而是它把**罗布文化、塔里木河、胡杨、沙地、聚落**用一个对游客最友好的方式收束到了同一套游览动线上。

#### 2）塔里木河—胡杨—湿地线必须补上
如果只去村寨而不去看塔河湿地、胡杨带和周边开放式生态线，你对尉犁的理解会天然短半截。因为尉犁最重要的不是“罗布人曾经在这里生活”，而是**为什么这种生活能在荒漠边缘持续成立**。

#### 3）要把它和库尔勒、博斯腾湖西南缘放在一个巴州东线框架里理解
尉犁并不是与库尔勒、博湖割裂的点。真实旅行里，经常出现这样的组合：

- **库尔勒负责大交通、母港住宿、大采购；**
- **博斯腾湖西南缘 / 阿洪口 / 大河口负责水域休闲、湖区度假、湿地轻松线；**
- **尉犁负责罗布文化、塔里木河胡杨线、东南向换挡。**

所以如果你在做巴州 3—5 天短线，合理的理解不是“博湖和尉犁二选一”，而是：

> **博斯腾湖西南缘解决“湖区休闲与水域生态”，尉犁解决“荒漠河流与罗布文化”，二者气质不同，完全可以前后串联。**

### 尉犁的核心判断：不是“值不值得来”，而是“住几晚才对”

如果你只想完成一个最基础的巴州近郊体验，确实可以从库尔勒当天往返；但如果你想把罗布人村寨玩完整、想补塔河湿地、想把县城补给层真正用起来，或者后面还准备继续往营盘 / 兴地方向推进，那么**尉犁这一晚几乎总是值得的**。

一句话概括：

> **尉犁不是只有一个景区，而是巴州东线中一个兼具内容密度、补给效率和路线分发能力的节点型目的地。**

## 行前速览

### 先记住这 14 件事

1. **第一次来尉犁，优先保住罗布人村寨 + 塔河 / 湿地，不要乱撒网。**
2. **尉犁最强的是三层结构，不是单点爆款。**
3. **罗布人村寨至少留半天，认真玩建议 4—6 小时。**
4. **只去村寨不看塔里木河胡杨 / 湿地，会少一半理解。**
5. **县城补给层不能省。** 尤其第二天要跑长一点线路时。
6. **尉犁默认比库尔勒更适合做“换挡点”，但不一定比库尔勒更适合做“大采购点”。**
7. **尉犁和博斯腾湖西南缘不是平替关系。** 一个偏荒漠河流与罗布文化，一个偏湖区湿地与度假节奏。
8. **如果巴州行程超过 3 天，常见正确组合是“库尔勒母港 + 博湖 1—2 天 + 尉犁 1—2 天”。**
9. **远线如营盘古城、兴地岩画不要跟村寨硬拼同一天。**
10. **地图上看着不远，不代表现场轻松。** 巴州东线的距离感是真实存在的。
11. **胡杨、湿地、水位、鸟况都有季节性。** 不要拿胡杨季想象套全年。
12. **如果后面还要继续跑公路长线，优先住一晚再出发。**
13. **如果你已经在博斯腾湖玩了 2 天，再进尉犁时更应该控制节奏，不要再硬塞高强度长线。**
14. **对第一次来的人，尉犁最常见错误不是“玩少了”，而是“把主线、副线、远线写成一天乱冲”。**

### 建议停留天数

- **1 天：** 适合库尔勒往返，保罗布人村寨 + 一条湿地 / 县城副线；
- **2 天：** 最推荐，第一天主线、第二天副线或补给线；
- **3 天：** 适合自驾、摄影或历史兴趣更强的人，可补营盘 / 兴地方向；
- **2 晚：** 适合把尉犁作为南疆东线出发节点，或需要恢复体力、补拍晨昏的人。

### 最佳旅行季节

- **4—6 月：** 气温相对舒服，适合主线慢玩；
- **7—9 月：** 水线与家庭玩法更完整，但中午暴晒明显；
- **10 月中旬—11 月上旬：** 胡杨季最佳，河岸、芦苇、金黄胡杨层次最完整；
- **冬季：** 游客少、氛围更荒寒，更适合喜欢空旷感和极简摄影的人。

### 气候与装备

公开资料常见描述显示，尉犁属暖温带大陆性荒漠气候，年均日照约 2975 小时，年均降水约 43 毫米、蒸发量约 2700 毫米，空气干燥、风沙天不少，昼夜温差明显。

**建议必带：**
- 防晒霜、墨镜、帽子、面巾
- 润唇膏、护手霜、保湿用品
- 充电宝、车充线、纸巾、湿巾
- 轻薄长袖 + 防风外套
- 足量饮水与电解质饮料

**自驾额外建议：**
- 高热量零食
- 线下地图或缓存导航
- 车载手机支架
- 胎压和油量提前检查
- 如次日跑远线，前一晚直接把油补满

### 到达方式

- **飞机：** 先飞库尔勒梨城机场，再转尉犁；
- **火车：** 先到库尔勒站，再租车 / 包车 / 打车；
- **自驾：** 尉犁最优解。县域内内容并不都在一个门楼里，自驾机动性越强，体验越完整。

### 公开距离参考

- **尉犁县城—库尔勒：** 约 20 公里
- **尉犁县城—库尔勒机场：** 约 15—25 公里
- **尉犁县城—罗布人村寨：** 约 28 公里
- **库尔勒—罗布人村寨：** 常见口径约 80—85 公里
- **尉犁县城—营盘古城及古墓群：** 约 112.5 公里
- **尉犁县城—兴地岩画：** 常见口径约 180 公里

### 尉犁、库尔勒、博斯腾湖西南缘怎么联动最顺

这是这版最重要的执行问题之一。

#### 组合 A｜库尔勒 1 晚 + 尉犁 1 晚
适合：第一次来巴州、只想保一个州府夜晚和一个尉犁夜晚的人。

- **库尔勒负责：** 机场 / 火车、租车、商超、药店、州府级酒店；
- **尉犁负责：** 罗布人村寨主线、塔里木河胡杨湿地线、第二天东南向继续推进前的整备。

#### 组合 B｜博斯腾湖西南缘 1 晚 + 尉犁 1 晚
适合：巴州 3—4 天短线，既想看湖区，又想看荒漠河流的人。

- **博斯腾湖西南缘 / 阿洪口 / 大河口：** 解决湖区水面、睡莲湿地、鱼宴、落日和度假感；
- **尉犁：** 解决罗布文化、塔河、胡杨、湿地和公路节点感。

这条线的本质不是重复，而是**从湖区柔性水域切换到荒漠硬边界**。

#### 组合 C｜库尔勒母港 + 博湖 2 天 + 尉犁 2 天
适合：巴州东线认真玩的人。

- 前段用博斯腾湖西南缘把“湖区—湿地—度假”玩透；
- 后段进入尉犁，把“罗布文化—塔河胡杨—东南向公路节点”补齐；
- 整体节奏明显比“每天回库尔勒”更完整。

### 县城补给怎么做最稳

#### 到尉犁前，优先在库尔勒完成的大项
- 租车、取大件行李
- 药店采购、防晒 / 保湿 / 晕车药补齐
- 大件商超采购
- 对住宿要求高的话，先把首晚酒店定好

#### 到尉犁县城后，建议完成的次级补给
- 晚饭吃稳，不要把所有餐食都赌在景区里
- 补水、补早餐、补零食
- 车加满油，尤其第二天要跑营盘 / 兴地方向的人
- 检查导航、证件、相机电量、充电线

### 是否住一晚 / 两晚：快速判断

#### 可以不住
- 只有 1 天时间；
- 只打算完成村寨主线；
- 当天返库尔勒的车、人、时间都很稳。

#### 默认应该住 1 晚
- 想把罗布人村寨玩完整；
- 想补塔河湿地 / 达西村 / 罗布湖；
- 带老人 / 小孩，不想打卡式赶路；
- 想把尉犁真正玩成“县域旅行”而不是“单景点往返”。

#### 建议住 2 晚
- 想拍晨昏胡杨 / 湿地；
- 前一站已经在博斯腾湖西南缘或库尔勒连玩两天，需要恢复体力；
- 第三天还要跑营盘 / 兴地方向；
- 想把罗布人村寨、塔河湿地、达西村 / 罗布湖、东南向长线拆开玩。

**直接结论：**

> **第一次来尉犁，默认住 1 晚；如果把它当巴州东线换挡点或长线前站，住 2 晚会更稳。**

## 1-3天行程

### 1 天快游版：第一次来尉犁最稳的骨架

**适用人群：** 库尔勒往返、第一次到访、只想保住主线的人

**路线：**
库尔勒 / 尉犁县城出发 → 罗布人文化展厅 → 罗布人民俗文化区 → 塔河吊桥与胡杨段 → 阿不旦美食广场简单补给 → 罗布淖尔国家湿地公园 → 回县城晚饭 / 返回库尔勒

**执行逻辑：**
- 上午把精力放在罗布人村寨主场；
- 下午补塔河湿地主线；
- 若当天回库尔勒，尽量不要把返程压到太晚。

### 2 天经典版：最推荐的住一晚方案

#### D1：罗布人村寨主场 + 塔里木河—胡杨—湿地
库尔勒抵达或尉犁县城早餐 → 罗布人文化展厅 → 罗布人民俗文化区 → 塔河吊桥 → 胡杨林保护区 → 罗布淖尔国家湿地公园 → 县城晚饭 → 住尉犁

**重点：**
- 第一天先把第一层和第二层结构看明白；
- 不要贪营盘 / 兴地这种更远的内容；
- 晚上回县城吃热饭、洗澡、补给，第二天节奏会轻松很多。

#### D2：县城补给层 + 达西村 / 罗布湖副线
县城早餐 → 达西村红色旅游区 → 达西村纪念馆 → 达西村民宿 / 农家乐片区 → 罗布湖 → 葫芦岛生态休闲区 → 孔雀河县城段散步 → 回县城或返库尔勒

**这一天的意义：**
把尉犁从“景区打卡”升级成“县域旅行”。你会看到今天的乡村、休闲湖区和县城生活，补齐第三层结构。

### 3 天完整版：主线 + 副线 + 长线前奏

#### 方案 A｜家庭 / 大多数游客版
- **D1：** 罗布人村寨主线 + 湿地，住县城
- **D2：** 达西村 + 罗布湖 + 孔雀河县城段散步
- **D3：** 胡杨生态园 / 县城补给后返库尔勒

#### 方案 B｜摄影 / 自驾版
- **D1：** 罗布人村寨深玩 + 塔河吊桥 + 胡杨逆光段
- **D2：** 罗布淖尔湿地清晨 + 塔里木河观景线 + 百里生态画廊
- **D3：** 营盘古城及古墓群 或 兴地岩画，二选一

#### 方案 C｜公路长线版
- **D1：** 库尔勒抵达后直接进尉犁，住县城，完成大补给
- **D2：** 罗布人村寨主线 + 塔河湿地
- **D3：** 视天气、路况和同行状态，选择营盘方向、兴地方向或返库尔勒

#### 方案 D｜博斯腾湖西南缘 + 尉犁联动版
- **D1：** 博湖 / 大河口 / 阿洪口 / 西海渔村，住湖边或博湖县城
- **D2：** 上午收尾博斯腾湖西南缘，下午转尉犁，住县城
- **D3：** 罗布人村寨主线 + 塔河胡杨湿地
- **D4：** 达西村 / 罗布湖 或继续往东南长线

**这条线为什么成立：**
前两天是湖区节奏，后两天切到荒漠河流节奏，内容不重，反而互相抬高完成度。

## 景点

> 说明：尉犁很多内容并非统一票务封闭景区，也包括开放式生态带、县城景观节点、乡村旅游区和文保认知线。以下“门票参考 / 建议时长 / 玩法 / 交通 / 预约要点 / 来源链接”综合地方公开资料、百科资料与常见出行口径整理；最终仍请以现场公示、道路情况、文保管理和生态保护要求为准。

### 1. 罗布人村寨风景名胜区
- **门票参考：** 全价约 ¥35，半价约 ¥17.5；区间车约 ¥15；骑骆驼、滑沙、越野等项目另收费
- **建议时长：** 4—6 小时
- **玩法：** 尉犁第一核心主场。建议按“展厅—民俗区—塔河—胡杨—沙丘”的顺序完整看，不要只在木屋区拍两张照就撤。
- **交通：** 距县城约 28 公里，自驾最方便，也可包车前往
- **预约要点：** 夏季常见开放时段约 9:30—21:30，冬季约 10:00—18:30；旺季建议提前确认区间车和自驾车辆入园限制，公开资料提到自驾车辆每日有数量控制
- **来源链接：**
  - https://baike.baidu.com/item/罗布人村寨风景名胜区/12504033
  - https://www.yuli.gov.cn/

- **预约要点：** 如果第二天要跑营盘 / 兴地，建议前一晚就完成而不是清晨临时找
- **来源链接：**
  - https://baike.baidu.com/item/尉犁县
  - https://www.yuli.gov.cn/

### 34. 库尔勒—尉犁联动补给节点
- **门票参考：** 无门票
- **建议时长：** 20—60 分钟
- **玩法：** 这同样不是传统景点，但它是尉犁能否成立的执行前提：机场、火车站、租车、药店、进县城住宿如何衔接，直接决定全程顺手程度。
- **交通：** 自驾、打车、包车均可
- **预约要点：** 这是效率节点，不是出片点；适合在抵达日先完成大项补给再进尉犁
- **来源链接：**
  - https://baike.baidu.com/item/尉犁县

### 35. 营盘古城及古墓群
- **门票参考：** 文保遗址参观条件以现场管理要求为准，不一定按常规景区票务运营
- **建议时长：** 2—4 小时，连往返至少半天
- **玩法：** 尉犁最硬核的历史线之一。它把楼兰道、佛寺、墓地、渠道和防御体系真正落到地面上，不是普通游客的一脚油门顺路点。
- **交通：** 距县城约 112.5 公里，建议自驾或包车
- **预约要点：** 出发前先确认开放程度；严禁翻越、攀爬和踩踏文保遗址；不要和罗布人村寨硬塞同一天
- **来源链接：**
  - https://baike.baidu.com/item/营盘古城及古墓群/22937675
  - https://baike.baidu.com/item/尉犁县

### 36. 营盘佛寺遗址及佛塔认知点
- **门票参考：** 多随文保区整体管理，以现场为准
- **建议时长：** 30—60 分钟
- **玩法：** 适合与营盘古城联动理解，可继续看丝路边缘佛教传播与聚落组织。
- **交通：** 一般与营盘线同游
- **预约要点：** 更适合历史兴趣明确的人，不适合抱网红废墟心态来
- **来源链接：**
  - https://baike.baidu.com/item/营盘古城及古墓群/22937675

### 37. 营盘灌溉渠遗址认知线
- **门票参考：** 多按文保区整体管理，以现场为准
- **建议时长：** 20—40 分钟
- **玩法：** 它告诉你营盘不是一座孤城，而是一整套农业与聚落系统。
- **交通：** 与营盘古城同线
- **预约要点：** 优先从合法可达范围观察，别擅自离道深入

- **建议时长：** 1—2 小时
- **玩法：** 适合和营盘线一起理解，说明尉犁同时也是丝路交通与防御边缘节点。
- **交通：** 通常需自驾或包车
- **预约要点：** 很多遗址并不适合普通游客随意深入，优先看合法可达点

### 39. 兴地岩画
- **门票参考：** 文保点参观安排以现场为准，常见为无常规商业门票
- **建议时长：** 2—3 小时；加往返建议预留全天或大半天

- **交通：** 越野条件更稳；普通车能否顺利进入取决于当日路况

## 美食

尉犁的吃法重点不是“搜一家最红的店”，而是**景区里轻补给，正餐回县城吃稳，本地特色抓重点**。

### 最值得优先吃的几类
- **罗布羊 / 尉犁羊肉：** 优先烤串、手抓肉、烤羊排、炖羊排
- **抓饭：** 县城最稳主食之一，适合赶路日
- **拌面：** 出餐快、容错率高
- **烤包子 / 薄皮包子：** 适合早餐或路餐
- **罗布烤鱼：** 在景区、水边或地方风味馆碰到可以优先试
- **椒麻鸡 / 大盘鸡：** 同行不想顿顿吃羊时的稳定备选
- **罗布麻茶、罗布麻茶蜜：** 有地方辨识度，适合买伴手礼
- **香梨、红枣、葡萄干：** 适合在县城集中采购

### 吃饭策略
- 想认真吃一顿，优先回县城；
- 景区餐饮更偏补给，不是性价比担当；
- 如果第二天要跑远线，晚饭别拖太晚，早餐和零食最好前一晚一起准备；
- 连续自驾时，主张“中午简、晚上稳”，比一路找网红店靠谱得多。

### 如果前一站刚从博斯腾湖西南缘过来
- 在博湖已经吃过一轮湖鲜鱼宴的话，到了尉犁建议换成羊肉、抓饭、拌面，不要继续沿用湖边度假节奏；
- 尉犁更适合“吃饱、睡稳、补给到位”，不是继续把重心放在餐饮体验感上。

## 住宿

### 结论先说

**第一次来尉犁，默认住县城。**

除非你明确要拍晨昏、蹲景区边缘、住营地，或者次日一早就要在景区附近开机，否则县城住宿的综合效率明显更高。

### 为什么县城住宿是尉犁成立的关键

1. **它把尉犁从一次性打卡变成可展开的县域旅行。**
2. **它能承担补给功能。** 晚饭、早餐、加油、药店、停车、洗澡、热水都更稳。
3. **它让第二天更自由。** 你可以补达西村 / 罗布湖，也可以直接切长线。
4. **它是公路长线的重要恢复点。** 前一天赶过路的人，在这里睡一晚状态会完全不一样。

### 适合住 1 晚的人
- 首次来尉犁
- 只有 2 天左右时间
- 想完成“主线 + 一条副线”
- 想让行程舒服，不想太碎

### 适合住 2 晚的人
- 想拍晨昏胡杨 / 湿地
- 想把村寨、湖区、达西村、营盘 / 兴地拆开玩
- 连续自驾，想把体力与风险控制得更稳
- 前序已经玩过博斯腾湖西南缘 / 阿洪口 / 大河口，希望在尉犁做一次真正的换挡恢复

### 如果在“住库尔勒还是住尉犁”之间犹豫

#### 继续住库尔勒更适合：
- 只安排尉犁 1 天快刷；
- 对酒店品质、夜生活、连锁品牌稳定性要求明显更高；
- 第二天并不准备继续往东南推进。

#### 住尉犁更适合：
- 想把主线玩完整；
- 想补塔河湿地 / 达西村 / 罗布湖；
- 第二天还要继续开往营盘 / 兴地方向；
- 想减少来回折返，把尉犁真正当目的地而不是外圈景点。

### 订房关注点
- 是否方便停车
- 热水与供暖 / 空调评价是否稳定
- 早餐开始时间够不够早
- 是否靠主路、夜间是否偏吵
- 胡杨季、国庆、周末是否明显涨价

### 是否住一晚：我的直接结论

- **只想去过：** 可以当天往返；
- **想玩对：** 通常应该住一晚；
- **后面还要继续跑巴州东线 / 南疆东线：** 这晚尤其值；
- **如果前面已经在博斯腾湖西南缘玩了 1—2 晚：** 尉犁最好别再赶成当天往返，住一晚更合理。

## 交通

### 外部交通
- **飞机：** 飞库尔勒梨城机场，再转尉犁最快
- **火车：** 到库尔勒站后换乘
- **公路：** 通过库尔勒—尉犁公路体系、218 国道及县域公路线连接主要点位

### 县内交通结论
- **县城内：** 打车够用
- **县域游：** 自驾最佳，包车次之
- **无车游客：** 优先保罗布人村寨 + 湿地 / 达西村，不要追远线

### 公路长线补给逻辑

这也是尉犁最容易被写浅的地方。

#### 从库尔勒进尉犁时
- 库尔勒负责大件补给、租车和标准化采购
- 尉犁负责住、吃、睡和第二天整备

#### 从博斯腾湖西南缘转进尉犁时
- 湖边负责风景和慢节奏，不负责东南向长线整备
- 真正准备往营盘 / 兴地方向推进时，补给、睡眠、加油和节奏整理最好放在尉犁完成
- 也就是说，**博湖更像休闲湖区站，尉犁更像公路换挡站**

#### 从尉犁继续往更深处跑时
- 出发前把油、饮水、零食、电量补满
- 不要以为导航里“有条线”就代表轻松好走
- 跑营盘 / 兴地方向，尽量上午出发，不要把返程压到傍晚以后

#### 为什么公路玩家更该住一晚
- 能把“赶路补给”和“看景体验”拆开
- 第二天状态更好，判断力更稳
- 万一天气、风沙、光线变化，也有调整余地

### 与库尔勒、博斯腾湖的关系
- **库尔勒 → 尉犁：** 很适合落地当天直接转场
- **尉犁 → 库尔勒：** 是最轻松的返程回收段
- **博斯腾湖西南缘 → 尉犁：** 适合做巴州 3—4 天短线里的气质切换段
- **尉犁 + 博斯腾湖：** 适合 3—5 天巴州短线，但别把两边都塞太满

## 避坑

1. **别把尉犁只当一个景区县城。** 它最强的是“主场景区 + 塔里木河胡杨湿地 + 县城补给”的整体结构。
2. **罗布人村寨不要只留 1 小时。** 认真玩至少 4 小时。
3. **不要忽略塔里木河本身。** 尉犁真正的灵魂是塔河，而不是某个单独打卡点。
4. **只去村寨不看湿地，会少一半理解。**
5. **县城补给层不能省。** 尤其第二天要跑远线的人，住一晚几乎总是更对。
6. **不要低估巴州东线后段公路线的长度感。** 地图上不远，不代表现场轻松。
7. **公路长线别临时起意。** 营盘、兴地、公路体验段都要看天气、油量、返程和同行体力。
8. **普通轿车不要盲下沙地、滩地、陌生土路。**
9. **文保点别按“网红废墟”玩法来。** 营盘古城、烽燧、岩画都是真文物。
10. **胡杨、湿地、水位、鸟况很吃季节。** 高光并非全年满配。
11. **无车游客不要贪多。** 保住主线比硬拼数量更重要。
12. **旺季别临时冲。** 胡杨季、国庆、活动日都可能抬高房价和排队时间。
13. **家庭游别照搬摄影路线。** 摄影党可以为光线反复折返，家庭游更适合稳节奏。
14. **晚上别空着肚子继续上路。** 县城把正餐吃稳，比景区里凑合强很多。
15. **不要误判“看起来能去”的边界。** 河滩、封控边、遗址外围、无名岔路都要谨慎。
16. **如果前一段刚玩完博斯腾湖西南缘，别误以为尉犁也该按“湖边慢游”处理。** 尉犁的正确打开方式是主线明确、补给扎实、住晚判断清楚。
17. **不要把罗布人村寨、湿地、达西村、营盘、兴地全塞同一版 2 天行程。** 这是最典型的纸面好看、现场崩盘。
18. **如果后面还有长线，尉犁最好承担恢复和整备功能。** 这通常比省一晚房费更值。
