# 黔南布依族苗族自治州（贵州）

- 更新时间：2026-03-12
- 推荐指数（整体）：⭐4.7（贵州最适合做“州域型深度旅行”的目的地之一：世界自然遗产喀斯特山水、FAST中国天眼、布依苗水等民族文化、州府都匀的山水城市气质，以及从荔波到平塘、从瓮安到三都的多样线路都很能打）

## 城市概览

这里写的是**贵州省黔南布依族苗族自治州**，不是模糊的“黔南”泛称，也不是单指都匀。它位于贵州省中南部，州府在**都匀市**，下辖都匀市、福泉市、荔波县、贵定县、瓮安县、独山县、平塘县、罗甸县、长顺县、龙里县、惠水县和三都水族自治县。

黔南布依族苗族自治州的旅行气质很特别：
- **荔波**负责世界级喀斯特山水与森林；
- **平塘**负责“中国天眼”与天坑峡谷、探地科普；
- **都匀**负责州府城市休闲、山水公园和黔南风味；
- **瓮安、福泉、贵定**有人文、古城、峡谷、温泉与茶山；
- **三都**则带来全国唯一水族自治县的人文识别度。

它不是那种“住在一个市中心，步行刷完全部景点”的城市，而是典型的**大范围州域旅行目的地**。如果你把黔南布依族苗族自治州只理解成“去荔波小七孔一天”，会低估它；但如果你第一次来就试图把荔波、平塘、中国天眼、都匀、瓮安、三都全扫一遍，又大概率会玩得很赶。

第一次来，最稳妥的思路通常有三条：
1. **荔波世界遗产线**：小七孔 + 大七孔 + 茂兰，自然风景优先；
2. **平塘探地科普线**：中国天眼 + 天坑群/掌布 + 甲茶，适合亲子、科普、轻度自驾；
3. **都匀州府休闲线**：斗篷山 + 剑江 + 秦汉影视城 + 周边县市补景点，适合时间不多的人。

一句话判断：**黔南布依族苗族自治州更像“贵州南线高质量目的地集合”，适合2—5天分主题游，不适合极限打卡式赶路。**

## 行前速览

- **适合谁**：第一次去贵州但不想只玩贵阳/安顺的人；自然风景控；亲子科普党；自驾用户；摄影爱好者；民族文化爱好者；想避开过度商业化古镇的人。
- **不太适合谁**：极其抗拒长距离州内转场的人；只想夜生活和城市逛街的人；完全不愿做路线取舍的人。
- **建议天数**：
  - 1 天：只适合都匀快游，或荔波小七孔单景区深度；
  - 2 天：荔波双景区 / 平塘探地线 / 都匀+周边；
  - 3 天：最实用，能形成一条像样主线；
  - 4—5 天：可以把荔波和平塘串起来，再补都匀或瓮安；
  - 6 天以上：才适合把州内多个县市拉开做较完整州域旅行。
- **最佳季节**：
  - 4—6 月：水量足、山色好，小七孔观感强；
  - 7—8 月：避暑属性突出，但热门景区人多；
  - 9—11 月：天气通常更稳，适合自驾和综合旅行；
  - 冬季：适合都匀/平塘/温泉类慢游，但荔波水色和体感不如暖季。
- **气候体感**：整体属湿润季风气候，州内多山，雨后很容易变成“雾气+森林+瀑布”氛围。夏天相对舒服，但山区和景区步道湿滑感明显，鞋子比衣服更关键。
- **核心交通**：
  - 高铁常用站：**都匀东站、都匀站、贵定北站、福泉站、独山东站、荔波站、三都县站**；
  - 航空：可用**荔波机场**，也常从贵阳进入后高铁/自驾分流；
  - 自驾：黔南非常适合自驾，但州内不是“同城通勤”，县与县之间转场仍需算足时间。
- **住宿建议**：
  - 玩荔波：住**荔波县城 / 小七孔景区东门附近**；
  - 玩平塘：住**平塘县城 / 克度镇（天眼周边）**；
  - 玩都匀：住**都匀市区剑江沿线 / 高铁站通勤方便区域**；
  - 玩瓮安：住**瓮安县城**；
  - 玩三都：住**三都县城**更稳。
- **预算感受**：
  - 荔波线门票和景交是州内支出大头；
  - 中国天眼相关参观不是纯“看一眼”那么简单，车接驳和配套时间要算进去；
  - 县城住宿普遍比省会友好，节假日荔波是例外。
- **预约提醒**：
  - **荔波小七孔、中国天眼景区**旺季/节假日建议提前订；
  - 涉及演出、夜游、漂流、景交、景区直通车时，尽量同步锁定；
  - 中国天眼周边对电子设备、参观动线可能有管理要求，出发前要看当日公告。
- **一句提醒**：黔南布依族苗族自治州最容易踩的坑不是“景点不好看”，而是**贪多、转场过密、对车程和步行量估计过低**。

## 1-3天行程

### 1 天快览版：都匀州府轻松线
- 上午：**秦汉影视城**
- 中午：都匀市区吃**酸汤鱼、冲冲糕、毛尖茶相关小食**
- 下午：**剑江河畔 + 石板古街 + 南沙洲公园 / 西山公园**
- 傍晚：天气好可去**斗篷山景区**或改成市区夜景散步
- 晚上：住都匀市区

适合：高铁落地、时间不多、先感受州府气质的人。

### 2 天经典版：荔波世界遗产线
- **D1**：抵达荔波 → **小七孔景区**全天 → 住荔波
- **D2**：**大七孔景区**半天 + **荔波古镇/荔波县城慢逛** → 时间够再补**瑶山古寨**或前往茂兰方向

适合：第一次来黔南布依族苗族自治州、只想抓住最强自然名片的人。

### 3 天进阶版：荔波 + 茂兰
- **D1**：小七孔景区深度
- **D2**：大七孔景区 + 荔波古镇夜晚散步
- **D3**：**茂兰国家级自然保护区**徒步/轻户外线 → 返程

适合：喜欢森林、步道和喀斯特生态，不满足于只打卡小七孔的人。

### 3 天探地科普版：平塘线
- **D1**：平塘县城 → **中国天眼景区** → 住克度或平塘
- **D2**：**平塘天坑群 / 天文小镇 / 探地体验** → **掌布景区**
- **D3**：**甲茶风景名胜区**慢游 → 返回都匀/贵阳方向

适合：亲子家庭、科普党、对“中国天眼”有兴趣的人。

### 3 天州府 + 周边综合版
- **D1**：都匀（秦汉影视城 + 剑江 + 石板古街）
- **D2**：斗篷山景区 / 螺丝壳草场二选一
- **D3**：贵定（金海雪山季节限定）或福泉（古城墙、洒金谷）或瓮安（草塘千年古邑）三选一

适合：不想跑太远、想以都匀为中心辐射玩的游客。

## 必去景点（精选 20 个）

> 说明：以下景点覆盖**贵州省黔南布依族苗族自治州**州域范围。门票价格以公开资料、景区常见票价口径、旅游平台常见区间及景区历史公示为参考，节假日、联票、摆渡车、索道、演艺、漂流、保险等可能会调整；**请务必以景区官方购票页、官方公众号或现场公示为准**。

### 1. 荔波小七孔景区（荔波县）
- **门票参考**：旺季常见约 120 元左右，景区观光车常见约 40 元左右；联票政策和淡旺季会调整。
- **建议时长**：5—8 小时
- **玩法**：看小七孔古桥、68级跌水瀑布、水上森林、翠谷瀑布、卧龙潭、鸳鸯湖，是黔南乃至贵州最稳的“第一次必去景点”。
- **交通**：从荔波县城或荔波站前往最方便，打车、包车、景区直通车都常见。
- **预约要点**：节假日和暑期强烈建议提前订票并尽量早入园；景区很大，不要以为“名字叫小七孔就很小”。
- **来源链接**：
  - <https://www.gzlibo.com/>
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>

### 2. 荔波大七孔景区（荔波县）
- **门票参考**：常见约 50—60 元，部分时期与小七孔联票售卖。
- **建议时长**：2—4 小时
- **玩法**：主打峡谷、天生桥、河谷步道和相对更野一点的喀斯特景观，适合和小七孔形成“细腻水景 + 开阔峡谷”的组合。
- **交通**：与小七孔同线，通常从荔波县城或小七孔方向前往。
- **预约要点**：如果时间只够一个景区，优先小七孔；大七孔更适合有半天余量、想拉开层次的人。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>
  - <https://www.gzlibo.com/>

### 3. 卧龙潭（荔波小七孔核心点位）
- **门票参考**：包含在小七孔景区门票内。
- **建议时长**：30—50 分钟
- **玩法**：看小七孔最有辨识度的蓝绿水色与堤坝弧线，是摄影党最常出片的点位之一。
- **交通**：乘景区观光车后步行进入。
- **预约要点**：晴天和雨后水色都能打，但人流高峰时拍空镜很难，建议早进或反向错峰。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>

### 4. 水上森林（荔波小七孔核心点位）
- **门票参考**：包含在小七孔景区门票内。
- **建议时长**：40—70 分钟
- **玩法**：走林间栈道和浅水步道，看“树在水中生，水从林中走”的经典喀斯特生态画面。
- **交通**：景区内步行可达，通常与翠谷瀑布、鸳鸯湖连看。
- **预约要点**：涉水路段和石面较滑，穿防滑鞋；不建议为了拍照硬踩非步道区域。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>

### 5. 68级跌水瀑布（荔波小七孔核心点位）
- **门票参考**：包含在小七孔景区门票内。
- **建议时长**：20—40 分钟
- **玩法**：层层叠叠的跌水台地非常适合慢看，是小七孔“柔和水景感”最强的路段之一。
- **交通**：景区步行线串联。
- **预约要点**：雨后更好看，但步道更湿；拍照注意镜头防水雾。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>

### 6. 鸳鸯湖（荔波小七孔核心点位）
- **门票参考**：通常包含在小七孔门票中，游船项目可能另收费。
- **建议时长**：30—60 分钟
- **玩法**：看树林环抱的湖泊，可搭配乘船项目，适合亲子与轻松型游客。
- **交通**：景区观光车 + 步行。
- **预约要点**：如想坐船，节假日需预留排队时间；只打卡不坐船也值得停留。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>

### 7. 小七孔古桥（荔波县）
- **门票参考**：包含在小七孔景区门票内。
- **建议时长**：20—30 分钟
- **玩法**：这是景区名字来源，桥体本身不算巨大，但和碧水、藤蔓、古意组合在一起辨识度很高。
- **交通**：景区内步行可达。
- **预约要点**：很多人拍完桥就走，其实周边水色和林木才是完整观感的一半。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波樟江风景名胜区>

### 8. 茂兰国家级自然保护区（荔波县）
- **门票参考**：公开资料常见约 50 元/人，部分线路、向导、接驳另计。
- **建议时长**：半天—1 天
- **玩法**：看喀斯特原始森林、漏斗森林、洼地森林、槽谷森林，是比小七孔更偏生态与徒步的一张王牌。
- **交通**：从荔波县城前往较方便，常见方式是自驾、包车或景区转乘。
- **预约要点**：景区很大，路线选择差异非常大；如果想走徒步线，建议提前确认开放线路和向导需求。
- **来源链接**：
  - <https://baike.baidu.com/item/贵州茂兰国家级自然保护区>

### 9. 瑶山古寨（荔波县）
- **门票参考**：常见几十元区间，演艺/体验项目另计。
- **建议时长**：1.5—3 小时
- **玩法**：看白裤瑶相关文化、寨门、民居、服饰和民族演艺，适合把荔波从纯自然景观升级为“自然 + 民族风情”组合。
- **交通**：从荔波县城或小七孔方向可串联。
- **预约要点**：演出有时段；如果你只对自然感兴趣可压缩，但带长辈或亲子通常体验不错。
- **来源链接**：
  - <https://www.gzlibo.com/>
  - <https://baike.baidu.com/item/荔波县>

### 10. 荔波古镇（荔波县）
- **门票参考**：街区步行为主，多为免费开放。
- **建议时长**：1—2 小时
- **玩法**：不属于全国最顶级古镇，但适合晚上吃饭、买手信、补一段轻松步行，作为荔波住宿区配套很实用。
- **交通**：荔波县城内步行/打车可达。
- **预约要点**：别把预期拉成“超级古镇目的地”，它更适合当作荔波旅游夜间补充。
- **来源链接**：
  - <https://baike.baidu.com/item/荔波县>

### 11. 中国天眼景区 / FAST观景体验区（平塘县克度镇）
- **门票参考**：常见景区票务几十元到百元内，摆渡、天文体验馆等可能分项收费。
- **建议时长**：3—5 小时
- **玩法**：看世界级大科学装置 FAST，做射电天文科普体验，理解它为什么会在平塘的喀斯特洼地落地。
- **交通**：从平塘县城或克度镇前往最方便，自驾、包车、景区接驳都常见。
- **预约要点**：需提前看当天票务和电子设备管理规则；部分区域参观会对手机、相机、智能穿戴设备有要求，别到了现场才慌。
- **来源链接**：
  - <https://baike.baidu.com/item/500米口径球面射电望远镜>
  - <https://www.pingtang.gov.cn/>

### 12. 平塘天文小镇（平塘县克度镇）
- **门票参考**：镇区步行多为免费，部分展馆/体验项目另收费。
- **建议时长**：1.5—3 小时
- **玩法**：适合在看完中国天眼后补充住宿、散步、科普和拍照，整体氛围偏“航天/天文科普主题小镇”。
- **交通**：与中国天眼同线安排最方便。
- **预约要点**：它不是单一硬核景点，更适合作为 FAST 行程的配套延伸。
- **来源链接**：
  - <https://www.pingtang.gov.cn/>
  - <https://baike.baidu.com/item/500米口径球面射电望远镜>

### 13. 掌布景区（平塘县）
- **门票参考**：常见几十元区间，具体以景区公告为准。
- **建议时长**：2—3 小时
- **玩法**：看峡谷、瀑布、怪石和喀斯特地质景观，是平塘“探地旅游”里相对成熟的一站。
- **交通**：通常从平塘县城自驾或包车前往更合适。
- **预约要点**：州内公开资料口径不算统一，出发前最好查景区当日开放和道路情况。
- **来源链接**：
  - <https://www.pingtang.gov.cn/>
  - <https://baike.baidu.com/item/平塘县>

### 14. 甲茶风景名胜区（平塘县甲茶镇）
- **门票参考**：常见几十元区间，漂流、游船、露营等项目另计。
- **建议时长**：3—5 小时
- **玩法**：看甲茶瀑布、水上森林、燕子洞、九曲十八湾，是平塘最适合做“山水慢游”的景区之一。
- **交通**：距平塘县城约 45—47 公里，自驾最方便，也可在县城换乘。
- **预约要点**：路程不短，别把它误判成县城旁边的小景点；夏季和雨后观感更好，但也更湿滑。
- **来源链接**：
  - <https://baike.baidu.com/item/甲茶风景名胜区>

### 15. 甲茶瀑布（平塘县甲茶景区核心点位）
- **门票参考**：包含在甲茶景区门票体系内。
- **建议时长**：30—50 分钟
- **玩法**：37 米高、40 米宽的瀑布在竹林和亚热带风貌包裹下很有南国气质，是甲茶最强记忆点。
- **交通**：景区内步行可达。
- **预约要点**：雨季更壮观；如果只冲这个点来，要同时预留进出景区与山区道路时间。
- **来源链接**：
  - <https://baike.baidu.com/item/甲茶风景名胜区>

### 16. 水上森林（平塘甲茶景区点位）
- **门票参考**：包含在甲茶景区门票内。
- **建议时长**：20—40 分钟
- **玩法**：与荔波的小七孔水上森林气质不同，这里更偏亚热带村寨河谷感，适合轻松散步、戏水和拍环境感照片。
- **交通**：景区内步行线串联。
- **预约要点**：名称容易和荔波混淆，做攻略时别串景区。
- **来源链接**：
  - <https://baike.baidu.com/item/甲茶风景名胜区>

### 17. 斗篷山景区（都匀市）
- **门票参考**：公开资料常见约 50 元通票，保险可能另计。
- **建议时长**：4—6 小时
- **玩法**：看森林、峡谷、瀑布、高山天池和“黔南第一山”气场，是都匀周边最有代表性的山地自然景区。
- **交通**：距都匀市区约 22 公里，可自驾、包车，也有城乡公交口径资料。
- **预约要点**：山路弯多且部分路段较窄；如果天气不好或大雾，山顶远景会受影响，但林谷线仍可玩。
- **来源链接**：
  - <https://baike.baidu.com/item/斗篷山>

### 18. 剑江风景带（都匀市）
- **门票参考**：城市滨水开放空间为主，通常免费。
- **建议时长**：1—2 小时
- **玩法**：看都匀最舒服的城市山水气质，适合傍晚散步、看夜景、感受州府生活节奏。
- **交通**：都匀市区内步行、打车都方便。
- **预约要点**：它不是封闭式景区，而是都匀城市体验的重要组成；适合放在吃饭前后轻松安排。
- **来源链接**：

### 19. 都匀秦汉影视城（都匀市）

### 20. 石板古街（都匀市）

## 美食

黔南布依族苗族自治州的吃，不是单一“酸汤鱼”能概括的。州内饮食整体有几个关键词：**酸、鲜、山野感、民族风味、县城烟火气**。

### 必吃清单
- **都匀酸汤鱼**：第一次到州府很适合来一顿，酸香开胃，比一些景区餐靠谱得多。
- **毛尖茶宴 / 毛尖茶入菜**：都匀毛尖是地方名片，部分餐厅会做茶香虾、茶叶炒蛋、茶香排骨等。
- **冲冲糕**：都匀街头小吃，适合作为走街串巷时的垫肚子选择。
- **牛肉粉 / 羊肉粉**：贵州通杀型早餐，在都匀、福泉、瓮安都容易吃到稳定版本。
- **布依八大碗**：如果去布依村寨或特色餐馆，值得体验一次完整民族宴席感。
- **竹筒饭**：在荔波、甲茶、民族村寨线常见，适合游客口味。
- **烤河鱼 / 竹笋鱼**：平塘、甲茶等河谷景区周边常见。
- **瑶山土鸡 / 山野菌汤**：荔波和茂兰一线的自然景区周边常见。
- **瓮安黄粑**：当地特产型小吃，适合买来路上吃或带回去。
- **豌豆凉粉、辣鸡粉、腊肉炒折耳根**：瓮安和县城系夜市里比较能打。
- **水族鱼包韭菜、马尾绣伴手礼衍生食品**：去三都时可以顺带试。
- **酸肉、腊肉、血豆腐、糯米饭**：民族风味店常见，适合多人拼桌点。

### 觅食建议
- 都匀市区适合认真吃正餐；
- 荔波景区周边更适合“够用型”吃饭，真正想吃好一点可回县城；
- 平塘和甲茶一线更适合吃农家菜、河鲜、竹笋类；
- 瓮安和福泉小城馆子往往比景区餐厅更有惊喜。

## 住宿建议

### 1. 都匀市区
- **适合**：第一次进州府、依赖高铁、想吃得方便的人。
- **优点**：交通最稳、餐饮密度高、夜晚可散步、酒店选择多。
- **缺点**：如果次日一早要冲荔波或平塘，会多一段转场。

### 2. 荔波县城 / 小七孔景区附近
- **适合**：玩荔波双景区、茂兰的游客。
- **优点**：景区进出最方便，节省早起时间。
- **缺点**：旺季价格上涨明显；景区边住宿质量分化大，别只看照片。

### 3. 平塘县城 / 克度镇
- **适合**：玩中国天眼和甲茶线。
- **优点**：做 FAST 行程最省心。
- **缺点**：夜生活和餐饮选择不如都匀、荔波丰富。

### 4. 瓮安县城 / 福泉市区
- **适合**：人文向、县城慢游、州内过渡住宿。
- **优点**：性价比通常不错，吃饭也方便。
- **缺点**：更适合补充线路，不太适合作为第一次来黔南的唯一驻点。

### 住宿总建议
- 旺季优先锁定**荔波**，其次是**平塘克度/天眼周边**；
- 酒店选择别只看“离景区几公里”，还要看是否真的方便停车、是否在主干道、是否夜晚安静；
- 山区县城住宿对“新不新”影响体验很大，宁可少住民宿风、优先看近两年的连锁或评分稳定酒店。

## 交通

## 如何到达黔南布依族苗族自治州
- **高铁**：对大多数人最实用。常见站点有**都匀东站、都匀站、贵定北站、福泉站、独山东站、荔波站、三都县站**。
- **飞机**：可关注**荔波机场**；也常见先飞贵阳，再转高铁或自驾进入州内。
- **自驾**：非常推荐，尤其适合平塘、荔波、都匀、瓮安多点串联。

## 州内交通逻辑
- **都匀**：适合做州府中转枢纽；
- **荔波**：景区导向明显，建议住一晚，不建议当日从远处硬杀来回；
- **平塘**：县城到中国天眼、甲茶都需要专门转场，自驾效率远高于临时拼车；
- **瓮安/福泉/贵定**：更适合自驾顺路串联；
- **三都**：公共交通可达，但想深入村寨文化仍建议包车或自驾。

## 自驾提醒
- 山区道路弯道多、雨雾多；
- 景区之间看着不远，实际耗时可能高于地图直觉；
- 景区停车场有的离入口不止“下车即到”，要给步行和摆渡留时间；
- 如果你每天都安排 2 个以上跨县核心景区，行程大概率已经偏紧。

## 避坑指南

1. **别把“贵州省黔南布依族苗族自治州”理解成一个紧凑城市。** 这是州域旅行，不是市中心景点合集。
2. **别一天同时硬塞小七孔、大七孔、茂兰。** 纸面上能写，身体上很赶。
3. **中国天眼不是到门口拍张照就结束。** 参观规则、接驳、科普馆、电子设备管理都可能影响节奏。
4. **甲茶和掌布不要临时起意傍晚出发。** 山区返程时间、路况和光线都要考虑。
5. **不要低估小七孔步行量。** 虽然有观光车，但真正好看的点还是要靠自己走。
6. **花期景点别盲冲。** 金海雪山这种季节限定型景区，错峰错期就是两回事。
7. **景区民宿照片不等于真实体验。** 尤其荔波旺季，优先看新旧程度、空调热水、停车和隔音。
8. **不要把所有“水上森林”混为一谈。** 荔波和甲茶都有同类名称，做路线时容易串错。
9. **州内县城餐饮往往比景区餐更稳。** 真想吃好，尽量回县城或市区。
10. **雨季必须重视防滑和备换衣。** 黔南很多体验好看的前提就是“有水”，但有水也意味着更湿、更滑。
11. **带老人和小孩时，优先做减法。** 荔波双景区 + 平塘天眼已经够满，不必再加长距离跨县。
12. **民族村寨要看“体验预期”。** 有活动时很热闹，没活动时更偏安静日常，不要全按节庆宣传图想象。
