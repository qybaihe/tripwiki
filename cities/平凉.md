# 平凉旅游攻略（快速可用版｜TripWiki）

- 更新时间：2026-03-17
- 推荐停留：2–3 天
- 适合人群：第一次来陇东、喜欢名山/石窟/古建/县域慢游、自驾短线旅行者
- 一句话判断：如果你愿意把平凉当成“崆峒山 + 泾川石窟 + 庄浪/崇信/华亭县域线”的组合目的地，而不是只看一座山，平凉非常值得专门来一趟。

## 城市概览

平凉位于甘肃东部、六盘山东麓、陕甘宁交界地带，是陇东旅行里最容易被低估的一座城市。它最出名的招牌当然是**崆峒山**，但真正撑起旅行含金量的，不是单个景区，而是这座城市同时具备的几条线：

- **名山线**：崆峒山、莲花台、关山、唐帽山，适合登高、避暑、看红叶和山地风景。
- **文物线**：王母宫石窟、南石窟寺、云崖寺石窟、石拱寺石窟、武康王庙、延恩寺塔、西山遗址，内容密度很高。
- **县域慢游线**：泾川温泉、庄浪梯田、静宁苹果、界石铺与战国—秦长城遗存，适合自驾慢慢串。

平凉的优点是：**景观类型丰富、游客密度不夸张、2–3 天就能玩出层次**。缺点也很明显：**点位分散，跨县移动耗时，如果不做路线规划很容易玩得零散**。

### 平凉适合谁

- 喜欢道教名山、佛教石窟、古建、遗址；
- 能接受 2–3 小时一段的县域移动；
- 想找一座“不是超热门，但内容扎实”的西北城市；
- 自驾党、包车党会明显更舒服。

### 不太适合谁

- 只想在市区 walk 一走、不想折腾交通；
- 完全不喜欢文物和山地景区；
- 希望夜生活、商业街体验很强的人。

## 行前速览

### 最佳旅行时间

- **4–6 月**：天气稳定，适合登崆峒山、跑泾川和庄浪线。
- **7–8 月**：山地避暑季，莲花台、关山、崆峒山体感好，但周末本地客会增加。
- **9–10 月**：一年里综合体验最强，崆峒山红叶、静宁苹果季、县域层次最好。
- **11 月下旬–次年 3 月**：可玩，但山上风大偏冷，部分山地景区观感和舒适度下降。

### 建议停留

- **1 天**：只够崆峒山，或者泾川石窟/温泉线二选一。
- **2 天**：推荐“崆峒山 + 泾川人文线”。
- **3 天**：推荐“崆峒山 + 泾川 + 庄浪/崇信/华亭三选一”。
- **4 天及以上**：适合自驾把静宁、灵台也纳入。

### 预算参考

- **经济型**：220–400 元/天
- **舒适型**：450–800 元/天
- **自驾深度型**：700–1200 元/天

### 到达方式

- **火车/高铁中转**：平凉站是主要入口，通常从兰州、西安、银川方向中转进入。
- **长途客运**：市区与县城之间仍有班线，但班次未必友好。
- **自驾**：最推荐。平凉真正好玩的内容分布在崆峒区、泾川、庄浪、崇信、华亭、静宁、灵台等地，自驾效率远高于纯公共交通。

### 市内与跨县移动判断

- **市区 + 崆峒山**：打车/网约车可以解决。
- **泾川**：可班车 + 打车，但效率一般。
- **庄浪 / 华亭 / 崇信 / 静宁 / 灵台**：更建议自驾或包车。

### 出发前要知道

1. 平凉不是“一个景区的城市”，而是“多个县域拼起来”的旅行目的地。
2. 山地景区多，鞋子和体力安排比穿搭更重要。
3. 石窟、古建、纪念馆类点位开放政策变化较快，出发前要二次核对。
4. 文中门票均为**公开常见参考价/常见售票口径**，实际以官方售票页、景区公告和现场公示为准。

## 1–3天推荐行程

## Day 1｜第一次来平凉的标准解法

**崆峒山游客中心 → 崆峒山主线徒步/景交 → 回主城区 → 柳湖公园散步 → 市区晚餐**

这是最稳的第一天。上午尽早进山，按照体力选择中台、香山寺、问道宫、雷声峰等线路；晚上回城把节奏放慢，吃一顿羊肉或饸饹面。

## Day 2｜平凉的人文精华线

**平凉博物馆 → 延恩寺塔 → 前往泾川 → 王母宫石窟 → 大云寺·王母宫景区/大云寺博物馆 → 视时间补南石窟寺或温泉**

这一天会让人真正意识到平凉不是只有崆峒山。泾川一线是全市最强的人文组合之一。

## Day 3｜按喜好二选一

### 方案 A：庄浪自然 + 石窟线

**云崖寺石窟 → 关山大景区 → 庄浪梯田观景点 → 返程**

适合喜欢峡谷、石窟和乡野层次的人。

### 方案 B：崇信/华亭轻山地 + 古建线

**武康王庙 → 龙泉寺景区 → 唐帽山/海寨沟二选一 → 时间够再补石拱寺石窟或莲花台方向**

适合自驾，不想连续高强度登山，但还想看一点县域风景的人。

## 景点清单（22 个，含门票参考 / 建议时长 / 玩法 / 交通 / 预约要点 / 来源链接）

> 说明：门票为公开渠道常见参考价，部分景区存在“门票 + 景交 + 索道/观光车”分开收费情况；文保点与纪念馆存在淡旺季、周一闭馆、保护性关闭等情形，出发前请二次核验。

### 1. 崆峒山风景名胜区
- **门票参考**：大门票常见约 110 元；景交、索道按线路另计。
- **建议时长**：4–8 小时。
- **玩法**：平凉第一王牌，道教名山 + 丹霞地貌 + 山门古刹 + 秋季红叶。第一次来不用追求“刷全山”，选一条主线走深更划算。
- **交通**：距离平凉主城区不远，打车、自驾、景区接驳均可。
- **预约要点**：节假日、红叶季建议提前购票；雨雪、大风天气体感差异很大。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_5dca027a02fd4045a5e6f2ff656004f1.html
  - https://baike.baidu.com/item/崆峒山/21534

### 2. 王母宫石窟
- **门票参考**：约 30–50 元；若与大云寺·王母宫景区联售，以现场为准。
- **建议时长**：1–1.5 小时。
- **玩法**：泾川最重要的人文点之一。北魏太和年间开凿，为中心塔柱式石窟，造像密集，外部飞阁也很有辨识度。
- **交通**：位于泾川县城西王母宫山下，自驾最方便，也可到泾川县城后打车。
- **预约要点**：文保区域遇维护可能有限流或局部关闭，尽量白天去看细节。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2024/art_5e5b08d149184981aa28cc789122d7b8.html
  - https://baike.baidu.com/item/王母宫石窟

### 3. 大云寺·王母宫景区
- **门票参考**：约 35–60 元，票型可能因展馆/联票不同而变化。
- **建议时长**：1.5–2.5 小时。
- **玩法**：和王母宫石窟一起看最顺。适合了解泾川佛教文化、地方历史和景区整体脉络。
- **交通**：位于泾川县，可与王母宫、温泉同日串联。
- **预约要点**：注意闭馆日、临展调整及团队高峰时段。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/大云寺

### 4. 南石窟寺
- **门票参考**：约 20–40 元。
- **建议时长**：1–2 小时。
- **玩法**：北朝石窟遗存，更适合对石窟、美术史、佛教传播史感兴趣的人。和王母宫一起看，信息量会比单看更高。
- **交通**：在泾川方向，适合自驾或包车联游。
- **预约要点**：部分洞窟可能因保护而限制近距离参观，提前查开放口径。
- **来源链接**：
  - https://baike.baidu.com/item/南石窟寺
  - https://www.pingliang.gov.cn/

### 5. 云崖寺石窟
- **门票参考**：约 30–60 元。
- **建议时长**：2–3 小时。
- **玩法**：位于庄浪关山峡谷，始凿于北魏晚期，沿悬崖分三层分布，兼具峡谷环境与石窟层次，是平凉最值得看的石窟之一。
- **交通**：庄浪县韩店镇郭漫村黄草东南约 2.7 公里关山峡谷中，建议自驾或包车。
- **预约要点**：山区道路受天气影响大，雨后路滑，不建议太晚进峡谷。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_9ab1339dac7a47af8f3f78274e82fd78.html
  - https://baike.baidu.com/item/云崖寺石窟

### 6. 石拱寺石窟
- **门票参考**：约 20–40 元。
- **建议时长**：1–1.5 小时。
- **玩法**：华亭方向的重要石窟，始凿于北魏晚期，现存窟龛 15 个、造像 250 余身，是陇山东段佛教艺术传播的重要节点。
- **交通**：华亭市城南上关乡半川村一带，适合自驾前往。
- **预约要点**：文保点位更适合白天去，局部区域可能因保护关闭。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_c8230ad6213945b08080b320069147bf.html
  - https://baike.baidu.com/item/石拱寺石窟

### 7. 武康王庙
- **门票参考**：免费或低价开放，以现场公示为准。
- **建议时长**：40–60 分钟。
- **玩法**：崇信县重要古建筑，全国重点文保。虽然规模不大，但木构层次和建筑史价值很高，适合古建爱好者细看。
- **交通**：崇信县锦屏镇东街村东街 6 号，县城内可打车前往。
- **预约要点**：一般无需预约，但开放时段要提前确认。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_ec553439638b4be09f222ed4fd3b8c94.html
  - https://baike.baidu.com/item/武康王庙

### 8. 延恩寺塔
- **门票参考**：免费或低价开放。
- **建议时长**：30–50 分钟。
- **玩法**：平凉主城区最值得顺路补的一处古塔，明代七级八角楼阁式空心砖塔，适合和博物馆、市区老城一起安排。
- **交通**：崆峒区东郊宝塔梁，市区打车方便。
- **预约要点**：若内部暂停开放，外观与周边拍照仍值得。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2024/art_a1376a8df3c4461e884a21cf8bb62609.html
  - https://baike.baidu.com/item/延恩寺塔

### 9. 平凉博物馆
- **门票参考**：通常免费。
- **建议时长**：1–2 小时。
- **玩法**：适合在跑县域前先建立平凉通史、崆峒文化、文物体系认知。不是“非去不可”的网红馆，但对理解全城很有帮助。
- **交通**：位于主城区，打车、公交都方便。
- **预约要点**：注意闭馆日与是否需要实名预约。
- **来源链接**：
  - https://baike.baidu.com/item/平凉市博物馆
  - https://www.pingliang.gov.cn/

### 10. 柳湖公园
- **门票参考**：免费。
- **建议时长**：40–90 分钟。
- **玩法**：主城区最适合晚上散步、调整节奏的公园。不是必须专程打卡，但很适合作为崆峒山返城后的收尾。
- **交通**：位于崆峒区城区内，步行、公交、打车都方便。
- **预约要点**：开放式公园，无需预约。
- **来源链接**：
  - https://baike.baidu.com/item/柳湖公园
  - https://www.pingliang.gov.cn/

### 11. 关山大景区（庄浪关山片区）
- **门票参考**：约 40–80 元。
- **建议时长**：3–5 小时。
- **玩法**：适合把云崖寺石窟和关山草甸、峡谷景观放在同一天。夏秋更舒服，属于庄浪自然线的主景区。
- **交通**：位于庄浪县关山区域，最好自驾。
- **预约要点**：雾天和雨天能见度变化大，提前看天气。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/关山

### 12. 庄浪梯田景观带
- **门票参考**：多为开放式免费。
- **建议时长**：1–2 小时。
- **玩法**：庄浪“梯田王国”是当地非常有辨识度的一张牌，适合在去关山或回程时选观景点停留拍照。
- **交通**：分布于庄浪多个乡镇，自驾最现实。
- **预约要点**：最佳观景位较分散，提前规划路线；雨后乡道谨慎驾驶。
- **来源链接**：
  - https://mp.weixin.qq.com/s/ZGDPCpWl6vTAIHuEaSTm6A
  - https://www.pingliang.gov.cn/

### 13. 龙泉寺景区（崇信）
- **门票参考**：约 30–60 元。
- **建议时长**：1.5–2.5 小时。
- **玩法**：崇信方向比较轻松的一站，寺院、水景、山谷氛围兼有，适合作为古建线之间的缓冲。
- **交通**：位于崇信县，适合与武康王庙、唐帽山联游。
- **预约要点**：周末午后客流通常更集中。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/龙泉寺

### 14. 唐帽山景区
- **门票参考**：约 30–60 元。
- **建议时长**：2–3 小时。
- **玩法**：崇信方向的轻徒步山地景区，视野开阔，适合想看点山景但不想再刷高强度线路的人。
- **交通**：更适合自驾前往。
- **预约要点**：阴雨天景观层次会打折，山路注意防滑。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/唐帽山

### 15. 海寨沟旅游度假区
- **门票参考**：约 30–80 元，部分娱乐项目另收费。
- **建议时长**：2–4 小时。
- **玩法**：偏生态休闲、亲子和度假属性，适合跟崇信/华亭方向其他点拼成一天，不太建议为了它单独专程跑远路。
- **交通**：自驾最方便，公共交通衔接一般。
- **预约要点**：假期家庭客较多，停车和项目排队会增加。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/海寨沟

### 16. 莲花台景区（华亭）
- **门票参考**：约 50–80 元；景交/索道另计。
- **建议时长**：3–5 小时。
- **玩法**：华亭代表性山地景区，适合夏季避暑和山岳风景爱好者。若已经刷过崆峒山，这里更像“加餐”，而不是必修课。
- **交通**：华亭市麻庵乡一带，建议自驾或包车。
- **预约要点**：山地天气变化快，旺季提前查景交和索道运行信息。
- **来源链接**：
  - https://baike.baidu.com/item/莲花台
  - https://www.pingliang.gov.cn/

### 17. 西山遗址
- **门票参考**：通常免费。
- **建议时长**：30–60 分钟。
- **玩法**：灵台县重要仰韶文化遗址，适合对史前聚落和考古遗存有兴趣的人顺路补充，不是成熟大景区，但很有“平凉时间深度”。
- **交通**：灵台县中台镇许家沟村一带，建议自驾。
- **预约要点**：遗址型点位更依赖背景知识，建议提前做功课。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_1f110a7722f442cf9787d3cc3cfe779c.html
  - https://baike.baidu.com/item/西山遗址

### 18. 战国—秦长城（静宁段）
- **门票参考**：多为免费开放。
- **建议时长**：1–2 小时。
- **玩法**：典型小众遗存，不是成熟旅游大景区，更适合长城遗址、地理和考古爱好者。看点在“遗存分布与地形关系”。
- **交通**：主要分布在静宁县四河镇、界石铺镇、三合镇、原安镇、红寺镇等地，自驾最现实。
- **预约要点**：不少点位远离居民区，土路、导航和天气都要提前判断。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2024/art_d3724c41e90d460089966ceff1d202e1.html
  - https://baike.baidu.com/item/秦长城

### 19. 界石铺红色旅游景区
- **门票参考**：多为免费或低价开放。
- **建议时长**：1–1.5 小时。
- **玩法**：如果本来就要跑静宁苹果、长城、乡村线，界石铺是很顺手的补充。单独专程意义一般，但放在线路里很合适。
- **交通**：静宁县界石铺镇，自驾更方便。
- **预约要点**：纪念馆类点位注意闭馆时间。
- **来源链接**：
  - https://baike.baidu.com/item/界石铺红军长征毛泽东旧居纪念馆
  - https://www.pingliang.gov.cn/

### 20. 皇甫谧文化园 / 皇甫谧针灸文化相关景区
- **门票参考**：约 20–50 元。
- **建议时长**：1–2 小时。
- **玩法**：灵台方向很有代表性的一条专题文化线，适合对中医文化、《针灸甲乙经》和人物纪念园感兴趣的人。
- **交通**：位于灵台县，适合与西山遗址联游。
- **预约要点**：留意展陈开放信息和讲解安排。
- **来源链接**：
  - https://baike.baidu.com/item/皇甫谧
  - https://www.pingliang.gov.cn/

### 21. 泾川温泉 / 大云寺温泉康养片区
- **门票参考**：常见约 68–168 元，住宿套餐另计。
- **建议时长**：2–4 小时；住一晚更舒服。
- **玩法**：如果不想把平凉玩成纯登山 + 文物之旅，泾川温泉是非常好的节奏缓冲，适合放在第二天下午或晚上。
- **交通**：位于泾川方向，适合与王母宫、大云寺一并安排。
- **预约要点**：周末和节假日建议提前订票或订房，自带泳衣更稳。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/泾川温泉

### 22. 田家沟生态风景区
- **门票参考**：约 20–50 元。
- **建议时长**：1.5–3 小时。
- **玩法**：偏近郊生态休闲、亲子和轻松散步，不是平凉“必须看”的头牌，但适合不想全程都在文物和山地里的人。
- **交通**：更适合自驾前往。
- **预约要点**：节假日以本地家庭客为主，最好提前确认园区开放情况。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/田家沟

## 美食

平凉的吃，核心不是“打卡感”，而是**西北日常碳水 + 羊肉系 + 县域农家菜**。吃对了会很舒服，吃错了会觉得平平无奇。

### 到平凉优先吃什么

- **羊肉泡馍 / 清炖羊肉 / 羊杂**：天冷时尤其舒服，陇东口味偏扎实。
- **饸饹面**：本地高频主食，筋道、顶饱，适合正餐。
- **锅盔 / 锅盔夹辣子**：早餐、路餐都很实用。
- **洋芋搅团 / 洋芋擦擦**：非常地方化，喜欢土豆的人会开心。
- **浆水面 / 臊子面**：夏天尤其适合。
- **庄浪暖锅、农家土鸡、山野菜**：跑县域时更值得点。
- **静宁苹果**：秋季来基本别空手走。

### 用餐建议

- 第一天和最后一天尽量在**平凉主城区**吃，选择更多、更稳。
- 跑庄浪、华亭、崇信、灵台时，**正餐尽量不要拖到太晚**。
- 本地分量普遍实在，点菜保守一点，尤其面食和锅盔。

## 住宿

### 1. 平凉主城区（最稳）
- **适合**：第一次来、公共交通为主、只住 1–2 晚的人。
- **优点**：吃饭方便，回城补给强，去崆峒山也不远。
- **缺点**：第二天跑泾川/庄浪还是要早起。

### 2. 崆峒山景区周边
- **适合**：想一早进山、拍晨景的人。
- **优点**：节省早晨通勤。
- **缺点**：餐饮和夜间便利性一般。

### 3. 泾川县城 / 温泉酒店
- **适合**：第二天主打泾川人文 + 温泉。
- **优点**：行程衔接顺，晚上舒服。
- **缺点**：如果隔天还要回主城或去庄浪，折返感会增加。

### 4. 庄浪 / 华亭 / 崇信县城
- **适合**：自驾深度游。
- **优点**：减少来回折返，第二天更轻松。
- **缺点**：酒店选择和稳定度不如主城区。

### 住宿一句话建议

- **不自驾 / 第一次来：住平凉主城区。**
- **自驾 + 深入跑县域：跟着第二天第一站住。**

## 交通

### 外部到达
- **铁路**：平凉站是主要节点，从兰州、西安、银川方向进入都可行。
- **公路**：自驾很适合，尤其适合把多个县串起来。

### 城内移动
- **市区**：公交 + 打车足够。
- **崆峒山**：市区往返方便。
- **泾川 / 庄浪 / 华亭 / 崇信 / 静宁 / 灵台**：公共交通能到县城，但真正到景区往往还要继续打车，效率有限。

### 是否要自驾
如果行程里同时出现以下两类及以上：
- 崆峒山
- 泾川石窟/温泉
- 庄浪云崖寺/关山
- 崇信龙泉寺/唐帽山
- 华亭莲花台/石拱寺

那**自驾几乎就是最优解**。

---

## 避坑

1. **别把平凉只当崆峒山一日游。** 真正值回票价的是组合线。
2. **不要一天塞太多跨县点位。** 地图距离不夸张，真实耗时很夸张。
3. **崆峒山别盲目逞强。** 山路、台阶和风会持续消耗体力。
4. **门票预算别只看“大门票”。** 山地景区常有景交/索道分开算。
5. **石窟和古建要带着“文物视角”看。** 如果只追求巨大视觉冲击，可能会觉得一般。
6. **县域餐饮和返程不要拖太晚。** 特别是庄浪、华亭、崇信方向。
7. **静宁、庄浪乡村线别完全依赖临场导航。** 观景点、遗址点位有时不够精确。
8. **旺季和秋季别低估停车压力。** 崆峒山红叶季、静宁苹果季的周末会更明显。
