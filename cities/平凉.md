# 平凉｜可发布的 mode2 城市攻略

> 版本说明：本稿按 TripWiki mode2 城市攻略重写，适合第一次做平凉行程的人直接拿来排路线。平凉的正确打开方式不是“只去一座崆峒山”，而是把崆峒山、泾川石窟线、庄浪/华亭/崇信县域线组合起来看。若时间只有 1 天，可以只看崆峒山；若有 2–3 天，平凉的旅行完整度会明显上一个台阶。

## 0. 城市概览

平凉位于甘肃东部、六盘山东麓，是陇东旅行里很容易被低估的一站。它的辨识度来自三件事：

- **名山**：崆峒山是全国知名度最高的招牌，适合登山、看古建、做西北山岳短线。
- **石窟与古迹**：王母宫石窟、南石窟寺、云崖寺石窟、石拱寺石窟、武康王庙、延恩寺塔等，决定了平凉并不只是“山景城市”。
- **县域慢游**：庄浪梯田、静宁苹果、泾川温泉、崇信/华亭山地景区，把它变成一座适合 2–4 天自驾慢慢串的城市。

平凉的优点是内容类型多、游客密度不算夸张、2–3 天就能玩得有层次；缺点是点位较分散，跨县交通耗时明显。如果不提前做取舍，很容易跑成“赶路大于旅行”。

## 1. 行前速览

### 1.1 适合谁

- 第一次想系统看陇东的人
- 喜欢名山、石窟、古建、遗址、县域风景的人
- 能接受自驾或包车跨县移动的人
- 不追求大城市夜生活，更看重内容密度的人

### 1.2 最佳旅行时间

- **4–6 月**：天气稳定，适合登山、跑泾川和庄浪线。
- **7–8 月**：山地避暑舒服，崆峒山、莲花台、关山体感较好。
- **9–10 月**：全年综合体验最好，红叶、果季、乡野层次都更强。
- **11 月–次年 3 月**：能玩，但山上风大偏冷，舒适度会下降。

### 1.3 推荐停留

- **1 天**：崆峒山一地深玩。
- **2 天**：崆峒山 + 泾川人文线。
- **3 天**：崆峒山 + 泾川 + 庄浪/华亭/崇信三选一。
- **4 天及以上**：可继续补静宁、灵台等县域内容。

### 1.4 预算参考

- **经济型**：约 220–400 元/人/天
- **舒适型**：约 450–800 元/人/天
- **自驾深度型**：约 700–1200 元/人/天

### 1.5 到达方式

- **铁路/火车**：可从兰州、西安、银川等方向中转进入平凉。
- **公路自驾**：这是平凉最推荐的方式，尤其适合串县域景点。
- **客运班线**：能到部分县城，但到景点往往还要继续打车，效率一般。

### 1.6 交通判断

- **平凉主城区 + 崆峒山**：公交 + 打车基本够用。
- **泾川线**：能去，但公共交通效率一般。
- **庄浪 / 华亭 / 崇信 / 静宁 / 灵台**：明显更适合自驾或包车。

### 1.7 出发前要知道

1. 平凉不是适合“临场随便走”的城市，提前规划路线会省很多时间。
2. 山地景区较多，鞋子、防晒、保暖比拍照穿搭更重要。
3. 石窟、古建、纪念馆的开放安排可能调整，出发前要再查一次。
4. 本文涉及门票、开放、预约的地方，如无稳定官方口径，统一按**“以现场/官方公告为准”**理解。

## 2. 景点 / 点位

> 阅读说明：本章按可落地排线来写。每个点都提供门票参考、建议时长、玩法、交通、预约要点、来源链接。若你时间有限，优先级通常是：**崆峒山 → 王母宫石窟 / 大云寺 → 云崖寺石窟 → 武康王庙 / 石拱寺 / 莲花台**。

### 2.1 崆峒山风景名胜区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：4–8 小时。
- **玩法**：平凉第一核心景区，道教名山 + 山岳步道 + 古建宫观是主要看点。第一次来不必执着“刷完全山”，按体力走一条主线更合理。
- **交通**：距平凉主城区不远，可从市区打车、自驾或按景区接驳安排前往。
- **预约要点**：节假日、红叶季建议提前看景区售票与入园安排；索道、景交等运行信息以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_5dca027a02fd4045a5e6f2ff656004f1.html
  - https://baike.baidu.com/item/崆峒山/21534

### 2.2 平凉博物馆
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：适合在正式跑县域之前先建立对平凉历史、崆峒文化、地方文物系统的基础认知。不是最炸裂的馆，但很适合作为总览入口。
- **交通**：位于平凉主城区，公交、打车前往都方便。
- **预约要点**：闭馆日、实名预约要求以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/平凉市博物馆
  - https://www.pingliang.gov.cn/

### 2.3 延恩寺塔
- **门票参考**：以现场/官方公告为准。
- **建议时长**：30–50 分钟。
- **玩法**：平凉城区内值得顺路补的一处古塔，适合与博物馆、市区散步组合。更适合喜欢古建筑和城市历史层次的人。
- **交通**：位于崆峒区东郊宝塔梁一带，市区打车前往较方便。
- **预约要点**：是否开放入内、开放时段以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2024/art_a1376a8df3c4461e884a21cf8bb62609.html
  - https://baike.baidu.com/item/延恩寺塔

### 2.4 柳湖公园
- **门票参考**：免费开放，具体以现场/官方公告为准。
- **建议时长**：40–90 分钟。
- **玩法**：更适合晚上散步、调整节奏，不属于必须专程打卡型景点，但很适合作为崆峒山回城后的收尾。
- **交通**：位于主城区，步行、公交、打车都方便。
- **预约要点**：开放式公园，一般无需预约，特殊管理安排以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/柳湖公园
  - https://www.pingliang.gov.cn/

### 2.5 王母宫石窟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–1.5 小时。
- **玩法**：泾川人文线的重要核心，属于平凉最值得专程看的石窟之一。适合与大云寺放在同一时段联看，信息密度会明显更高。
- **交通**：位于泾川县城西侧一带，建议自驾；也可先到泾川县城后再打车。
- **预约要点**：文保区域可能因保护、维护、限流调整参观方式，以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2024/art_5e5b08d149184981aa28cc789122d7b8.html
  - https://baike.baidu.com/item/王母宫石窟

### 2.6 大云寺·王母宫景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1.5–2.5 小时。
- **玩法**：适合和王母宫石窟打包看，重点在佛教文化、地方历史和景区整体脉络。若你只去一个泾川人文点，这里和王母宫优先级最高。
- **交通**：位于泾川县，可与王母宫石窟、泾川温泉同日串联。
- **预约要点**：景区联票、展馆开放时间、团队高峰安排以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/大云寺

### 2.7 南石窟寺
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：更适合对石窟、美术史、佛教传播史感兴趣的人。若和王母宫石窟联看，会更容易感受到平凉的人文深度。
- **交通**：位于泾川方向，建议自驾或包车前往。
- **预约要点**：部分洞窟可能因保护需要限制参观距离或开放范围，以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/南石窟寺
  - https://www.pingliang.gov.cn/

### 2.8 泾川温泉 / 大云寺温泉康养片区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2–4 小时；若住一晚会更舒服。
- **玩法**：适合给“登山 + 石窟”行程做节奏缓冲。若不想把平凉玩得太硬核，第二天晚上住温泉酒店会非常舒服。
- **交通**：位于泾川方向，适合和王母宫石窟、大云寺同日安排。
- **预约要点**：周末、节假日房态和温泉票情况以现场/官方公告为准；是否需自带泳衣也以商家要求为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/泾川温泉

### 2.9 云崖寺石窟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2–3 小时。
- **玩法**：位于关山峡谷中的高质量石窟点，兼具峡谷环境和石窟层次，是平凉最值得看的石窟之一。若你只能在平凉看第二组重磅石窟，优先考虑这里。
- **交通**：位于庄浪县韩店镇郭漫村附近关山峡谷中，建议自驾或包车。
- **预约要点**：山区天气、道路状况、局部开放安排以现场/官方公告为准；雨后山路谨慎慢行。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_9ab1339dac7a47af8f3f78274e82fd78.html
  - https://baike.baidu.com/item/云崖寺石窟

### 2.10 关山大景区（庄浪关山片区）
- **门票参考**：以现场/官方公告为准。
- **建议时长**：3–5 小时。
- **玩法**：适合与云崖寺石窟安排成同一天，看峡谷、山地、草甸等自然层次。夏秋体验更好，属于庄浪自然线的主菜。
- **交通**：位于庄浪关山区域，自驾最省时间。
- **预约要点**：景区开放范围、道路通行、观光车安排以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/关山

### 2.11 庄浪梯田景观带
- **门票参考**：多为开放式区域，以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：适合在关山或庄浪县域移动途中补观景点，拍梯田层次和乡野道路。更适合自驾党，不太适合把它当唯一目的地。
- **交通**：分布在庄浪多个乡镇，自驾最现实。
- **预约要点**：具体观景位、乡道通行和天气情况以现场/官方公告为准。
- **来源链接**：
  - https://mp.weixin.qq.com/s/ZGDPCpWl6vTAIHuEaSTm6A
  - https://www.pingliang.gov.cn/

### 2.12 石拱寺石窟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–1.5 小时。
- **玩法**：华亭方向的重要石窟点，体量不算最大，但在平凉石窟体系里很有代表性，适合和莲花台、周边山地线一起看。
- **交通**：位于华亭市上关乡半川村一带，建议自驾前往。
- **预约要点**：文保点开放范围可能随保护需要调整，以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_c8230ad6213945b08080b320069147bf.html
  - https://baike.baidu.com/item/石拱寺石窟

### 2.13 莲花台景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：3–5 小时。
- **玩法**：华亭代表性的山地景区，适合夏季避暑和轻度山岳旅行。如果已经去过崆峒山，这里更像升级版补充，不一定是首选必打卡。
- **交通**：位于华亭市麻庵乡一带，建议自驾或包车。
- **预约要点**：索道、景交、天气影响下的开放情况以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/莲花台
  - https://www.pingliang.gov.cn/

### 2.14 海寨沟旅游度假区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2–4 小时。
- **玩法**：偏生态休闲、亲子、度假属性，更适合做华亭或周边县域线里的轻松一站，不建议为了它单独专程跑特别远。
- **交通**：自驾最方便，公共交通衔接相对一般。
- **预约要点**：节假日项目开放、停车与排队情况以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/海寨沟

### 2.15 武康王庙
- **门票参考**：以现场/官方公告为准。
- **建议时长**：40–60 分钟。
- **玩法**：崇信的重要古建筑，全国重点文保。规模不算巨大，但木构层次和建筑史价值很强，古建爱好者会很喜欢。
- **交通**：位于崇信县锦屏镇东街村一带，可自驾或在县城内打车前往。
- **预约要点**：开放时段、讲解服务和是否开放内部区域以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_ec553439638b4be09f222ed4fd3b8c94.html
  - https://baike.baidu.com/item/武康王庙

### 2.16 龙泉寺景区（崇信）
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1.5–2.5 小时。
- **玩法**：寺院、水景、山谷氛围兼有，比较适合搭配武康王庙作为崇信半日或一日轻松线路。不是超级重磅，但胜在节奏舒服。
- **交通**：位于崇信县，建议自驾前往，也可先到县城再转车。
- **预约要点**：开放时间、客流高峰和园区临时活动安排以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/龙泉寺

### 2.17 唐帽山景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2–3 小时。
- **玩法**：适合做轻徒步和县域山景补充，比较适合“不想全天高强度爬山，但还想看一点山地风景”的人。
- **交通**：位于崇信方向，建议自驾前往。
- **预约要点**：山路、天气、景区开放状态以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/
  - https://baike.baidu.com/item/唐帽山

### 2.18 西山遗址
- **门票参考**：以现场/官方公告为准。
- **建议时长**：30–60 分钟。
- **玩法**：灵台重要的史前遗址，适合对考古、仰韶文化、聚落遗存有兴趣的人顺路去看。它不是典型大景区，但很能拉长平凉的时间维度。
- **交通**：位于灵台县中台镇许家沟村附近，建议自驾前往。
- **预约要点**：遗址类点位的开放展示形式和保护要求以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2025/art_1f110a7722f442cf9787d3cc3cfe779c.html
  - https://baike.baidu.com/item/西山遗址

### 2.19 皇甫谧文化园 / 皇甫谧针灸文化相关景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：更适合对中医文化、人物纪念园、《针灸甲乙经》相关内容感兴趣的人。作为灵台线的专题补充很合适。
- **交通**：位于灵台县，适合与西山遗址同日串联。
- **预约要点**：展陈开放、讲解安排、纪念馆开放口径以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/皇甫谧
  - https://www.pingliang.gov.cn/

### 2.20 战国—秦长城（静宁段）
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：更偏小众遗址型点位，适合长城遗存、地理和考古爱好者。看点不在“成熟景区设施”，而在遗存与地形关系。
- **交通**：主要分布于静宁县多地，建议自驾，并提前设好精确导航点。
- **预约要点**：乡道、土路、天气、现场可达性以现场/官方公告为准。
- **来源链接**：
  - https://www.pingliang.gov.cn/cxzx/lyxx/art/2024/art_d3724c41e90d460089966ceff1d202e1.html
  - https://baike.baidu.com/item/秦长城

### 2.21 界石铺红色旅游景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1–1.5 小时。
- **玩法**：如果本来就计划跑静宁苹果、长城、乡村线，它是非常顺路的补充；单独为它专程跑一趟则优先级一般。
- **交通**：位于静宁县界石铺镇，自驾更方便。
- **预约要点**：纪念馆开放时段、临时闭馆信息以现场/官方公告为准。
- **来源链接**：

### 2.22 田家沟生态风景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1.5–3 小时。
- **玩法**：偏近郊生态休闲，适合家庭出行或想降低行程强度的人。不是平凉最硬核的看点，但作为轻松半天还可以。
- **交通**：更适合自驾前往。
- **预约要点**：园区开放范围、活动项目、节假日客流以现场/官方公告为准。

## 3. 1–3 天推荐行程

### 3.1 1 天版｜第一次来平凉的最低成本正确解

**上午：崆峒山风景名胜区**  
尽量早进山，把核心体力放在白天。第一次来不建议为了“刷全”把自己走崩，挑一条主线把体验做完整更重要。

**下午：回主城区，补平凉博物馆或延恩寺塔**  
如果体力还行，可以补一个城区人文点；如果体力一般，直接回酒店休息也完全合理。

**晚上：柳湖公园散步 + 市区吃饭**  
适合把第一天节奏放慢，吃羊肉、饸饹面或锅盔类本地食物。

### 3.2 2 天版｜最推荐的新手行程

**Day 1：崆峒山 + 主城区**  
崆峒山 → 平凉市区住宿 → 晚上柳湖公园 / 市区觅食

**Day 2：泾川人文线**  
平凉出发 → 王母宫石窟 → 大云寺·王母宫景区 → 视时间补南石窟寺 → 若想放松则改去泾川温泉

这条线最稳，因为它把“名山”和“石窟/地方文化”都覆盖到了。

### 3.3 3 天版｜把平凉玩出层次

**Day 1：崆峒山深玩**  
集中刷平凉第一名片，不和别的县硬拼。

**Day 2：泾川人文 + 温泉缓冲**  
王母宫石窟 → 大云寺·王母宫景区 → 南石窟寺 / 温泉二选一。

**Day 3：按兴趣三选一**

- **庄浪线**：云崖寺石窟 → 关山大景区 → 庄浪梯田观景点  
  适合喜欢石窟 + 山地 + 乡野层次的人。
- **华亭线**：石拱寺石窟 → 莲花台 → 海寨沟  
  适合自驾、想再补一条山地和石窟线的人。
- **崇信线**：武康王庙 → 龙泉寺 → 唐帽山  
  适合喜欢古建、轻徒步、节奏不要太累的人。

## 4. 美食

平凉的吃，不是那种“为了某一道网红美食必须专程冲”的城市型美食目的地，而是很典型的西北日常碳水 + 羊肉系 + 县域农家菜组合。吃对了会很舒服，尤其适合山地和跨县旅行后补能量。

### 4.1 优先吃什么

- **羊肉泡馍 / 清炖羊肉 / 羊杂**：天气冷的时候尤其舒服。
- **饸饹面**：本地主食代表，筋道、顶饱、稳定。
- **锅盔 / 锅盔夹辣子**：适合早餐或路上带着吃。
- **浆水面 / 臊子面**：夏天吃会更顺口。
- **洋芋类做法**：像搅团、擦擦这类，更有地方感。
- **庄浪、华亭、崇信农家菜**：跑县域时比在市区更值得尝。
- **静宁苹果**：秋季来平凉时很适合买一点带走。

### 4.2 用餐建议

- 第一天和最后一天尽量安排在**平凉主城区**吃，稳。
- 跑县域时，午饭尽量在县城或成熟景区附近解决，不要拖到下午两三点。
- 当地分量普遍偏实在，点菜可以保守一点。

## 5. 住宿

### 5.1 平凉主城区

**适合人群**：第一次来、公共交通为主、只住 1–2 晚的人。  
**优点**：吃饭方便、补给方便、去崆峒山也顺手。  
**缺点**：第二天跑泾川或庄浪还是得早起。

### 5.2 崆峒山景区周边

**适合人群**：想一早进山、想节省通勤的人。  
**优点**：进景区省时间。  
**缺点**：晚上吃饭和便利性不如主城区。

### 5.3 泾川县城 / 温泉酒店

**适合人群**：第二天主打泾川人文线并想放松的人。  
**优点**：王母宫、大云寺、温泉衔接舒服。  
**缺点**：如果隔天还要回主城区或转庄浪，会有折返感。

### 5.4 庄浪 / 华亭 / 崇信县城

**适合人群**：自驾深度游。  
**优点**：减少折返，第二天更从容。  
**缺点**：酒店选择和稳定度不如平凉主城区。

### 5.5 一句话建议

- **不自驾 / 第一次来：优先住平凉主城区。**
- **自驾深度玩：住到第二天第一站附近。**

## 6. 交通

### 6.1 外部到达

- **火车/铁路进入**：平凉是可达的，但未必像大热门城市那样“一趟直达解决所有问题”。
- **自驾进入**：如果你本来就在甘肃东部、陕西西部、宁夏南部一线旅行，自驾把平凉放进环线会很合适。

### 6.2 市内和跨县移动

- **主城区内**：公交 + 打车够用。
- **崆峒山**：从市区往返方便。
- **泾川 / 庄浪 / 华亭 / 崇信 / 静宁 / 灵台**：真正耗时的是“到县城之后怎么去景点”，这也是为什么自驾优势明显。

### 6.3 是否值得自驾

如果你的行程同时包含下面任意两类以上：

- 崆峒山
- 泾川石窟 / 温泉
- 庄浪云崖寺 / 关山
- 华亭莲花台 / 石拱寺
- 崇信龙泉寺 / 唐帽山 / 武康王庙

那基本可以直接判断：**自驾就是更优解**。

## 7. 避坑

1. **别把平凉只玩成“崆峒山一日游”。** 如果只去一座山，你会低估这座城市。
2. **不要一天硬塞多个跨县点位。** 地图上看不远，真实耗时往往更长。
3. **崆峒山别盲目逞强。** 山路、台阶、风和高差会持续消耗体力。
4. **预算别只按大门票算。** 山地景区的景交、索道、停车等容易被忽略。
5. **石窟和古建更吃“看法”。** 如果你只追求强烈视觉刺激，部分点位可能会觉得朴素。
6. **县域正餐别拖太晚。** 特别是庄浪、华亭、崇信方向，返程和吃饭都容易受影响。
7. **别完全依赖临场导航找遗址和观景点。** 静宁、庄浪这类点位最好提前查精确位置。
8. **秋季和节假日别低估客流。** 崆峒山红叶季、苹果季周末，停车和排队压力会更明显。
9. **开放信息务必二次核验。** 本文已尽量保守处理，具体门票、开放、预约仍以现场/官方公告为准。
