## 一、城市概览

江门位于广东珠江西岸，是粤港澳大湾区里气质很特别的一座城市。很多人对它的第一印象是“开平碉楼”，但真到现场会发现：江门不是只靠一个世界遗产撑场面的城市，它真正好玩的地方在于 **侨乡文化、近代骑楼街区、海岛海湾、温泉休闲、圭峰山系的自然线** 可以拼成一趟节奏很舒服的 2–3 天旅行。

对第一次来江门的人，可以先这样理解这座城市：

- **蓬江 / 江海 / 新会**：适合玩城市休闲、老街、陈白沙、启超故里、圭峰山、小鸟天堂。
- **开平**：江门文旅王牌，重点是 **开平碉楼与村落**、赤坎华侨古镇，以及侨乡建筑群。
- **台山**：更偏海边和海岛度假，可玩 **上川岛、下川岛、那琴半岛、浪琴湾、黑沙湾**。
- **恩平**：主打温泉休闲，适合和江门主线拼成“文化 + 放松”的行程。
- **鹤山**：古劳水乡是最适合轻松半日游的县域补充点。

### 江门适合谁去
- 想看 **华侨文化 / 碉楼 / 骑楼街 / 民国风建筑** 的人
- 想做一趟 **广州 / 深圳 / 珠海 / 佛山 出发的 2–3 天短途旅行** 的人
- 既喜欢人文，也希望加一点 **海边 / 温泉 / 轻徒步** 的人
- 亲子、家庭、长辈同行，想避开过度硬核爬山的人

### 不太适合谁
- 只想在一个区步行刷完所有核心景点的人
- 对华侨史、建筑、古村、老街完全无感的人
- 不接受跨县移动，却又想一次性刷开平 + 台山 + 新会的人

### 江门旅行的核心判断
- **2 天能玩到精华，但 3 天会舒服很多。**
- **第一次去，最稳的主线是：开平碉楼 + 新会侨乡线。**
- **如果你偏度假，台山海岛 / 恩平温泉会比一味赶景点更值。**

## 二、行前速览

### 最建议的旅行天数
- **1 天快刷**：开平碉楼（立园 / 自力村 / 马降龙三选二）
- **2 天经典**：D1 开平世遗线；D2 新会圭峰山 + 小鸟天堂 + 梁启超故居
- **3 天完整**：在 2 天基础上，加台山海边 / 海岛 或 恩平温泉一条线

### 最佳旅行季节
- **10–次年 4 月最稳**：气温舒服，适合逛古村、老街、爬山、拍建筑。
- **5–9 月适合海岛和海边**：但天气更热，也更吃台风和雷雨情况。
- **暑期 / 国庆 / 春节**：热门景区、海岛和赤坎华侨古镇客流明显上升，建议提前订票订房。

### 城市分区怎么理解
- **蓬江区**：长堤历史文化街区、启明里、陈白沙纪念馆，城市人文散步线。
- **新会区**：圭峰山、玉台寺、小鸟天堂、梁启超故居，小而全。
- **开平市**：碉楼世遗主战场，适合整天安排。
- **台山市**：海岛、海湾、温泉、滨海自驾。
- **恩平市**：温泉度假型目的地。
- **鹤山市**：古劳水乡适合半天慢游。

### 到达方式
- **高铁**：可优先看江门站、新会站、开平南站、台山站等，视你的具体线路决定下车点。
- **自驾**：如果要串联开平、台山、恩平，自驾效率明显最高。
- **公共交通**：玩蓬江 / 新会问题不大；但去海岛、乡村碉楼、温泉度假区时，时间成本会明显上升。

### 市内与跨县交通判断
- **只玩市区 / 新会**：打车 + 网约车足够。
- **玩开平碉楼**：可以高铁到开平后打车串联，但更推荐自驾。
- **玩台山海岛**：要额外考虑码头、船班、天气、上岛车接驳。
- **玩恩平温泉**：自驾最省心。

### 预算参考（2 天 1 晚 / 1 人）
- **经济型**：500–900 元
- **舒适型**：900–1600 元
- **海岛 / 温泉度假型**：1600 元以上

主要变量在于：是否自驾、是否住古镇 / 温泉酒店、是否上岛、是否买演艺或套票。

## 三、1–3 天行程推荐

## 1 天快刷版（第一次去江门）
**开平碉楼世遗线**：立园 → 自力村碉楼群 → 赤坎华侨古镇（夜景）
- 适合：第一次到江门、最想看侨乡建筑与世遗的人
- 优点：江门辨识度最高的一条线
- 缺点：基本看不到江门城市面和海岛面

## 2 天经典版
**D1 开平**：立园 → 自力村 → 马降龙 / 锦江里（择一）→ 赤坎华侨古镇夜景  
**D2 新会**：圭峰山 → 玉台寺 → 小鸟天堂 → 梁启超故居纪念馆

这套组合最稳，因为把江门最有名的 **侨乡世遗 + 新会山水人文** 一次打包了。

## 3 天完整可用版
**D1 开平世遗线**：立园 → 自力村 → 马降龙 → 赤坎华侨古镇  
**D2 新会侨乡线**：圭峰山 → 玉台寺 → 小鸟天堂 → 梁启超故居  
**D3 二选一**：
- **海岛海边线**：上川岛 / 下川岛 或 那琴半岛 + 浪琴湾
- **休闲温泉线**：古兜温泉 或 恩平温泉度假区

### 如果你偏人文
优先顺序：**开平碉楼 > 赤坎华侨古镇 > 梁启超故居 > 长堤历史文化街区 / 启明里**。

### 如果你偏海边度假
优先顺序：**台山海岛 / 海湾 > 那琴半岛 > 浪琴湾 > 黑沙湾**。

### 如果你带长辈或孩子
优先组合：**小鸟天堂 + 梁启超故居 + 古劳水乡 + 温泉**，整体更轻松。

## 四、景点清单（20 个）

> 说明：
> - “门票参考”按景区官网、政府公开资料、长期稳定票务平台常见价格区间整理；如有联票、优惠票、景交、演艺票，以现场公示为准。  
> - “来源链接”尽量优先给政府站、景区官方或稳定资料页；若官方页面改版，建议出发前再搜景区官方公众号核实。  
> - 免费景点也要注意开放时间、预约制度、临时限流、船班与天气。

### 1. 开平立园【第一次去江门最稳的碉楼 / 园林代表】
- **门票参考**：常见联票体系内，单点或套票参考约 60–100 元
- **建议时长**：1.5–2.5 小时
- **玩法**：立园是开平碉楼里最容易“看懂”和“出片”的一站，融合中西建筑、园林、水系与侨乡审美，适合第一次来江门建立对侨乡建筑的整体印象。
- **交通**：位于开平市潭江流域周边，建议自驾或从开平城区打车前往
- **预约要点**：节假日建议提前购票；联票游玩时要留足各点之间通勤时间
- **来源链接**：<https://www.kaiping.gov.cn/>；<https://whc.unesco.org/en/list/1112/>；<https://baike.baidu.com/item/立园>

### 2. 自力村碉楼群【开平世遗核心观感最强之一】
- **门票参考**：常见联票体系内，单点 / 联票约 60–100 元区间
- **建议时长**：1.5–2 小时
- **玩法**：如果只能选一个“最像你想象中的开平碉楼”的地方，自力村通常不会出错。田园中的多座碉楼组合，非常适合拍照，也更容易理解碉楼的防御与侨汇建筑背景。
- **交通**：开平市内自驾 / 打车前往最方便
- **预约要点**：旺季尽量早到；夏季暴晒明显，村内步行要注意补水
- **来源链接**：<https://www.kaiping.gov.cn/>；<https://whc.unesco.org/en/list/1112/>；<https://baike.baidu.com/item/自力村碉楼群>

### 3. 马降龙碉楼群【世遗村落氛围最好的一档】
- **门票参考**：常见联票体系内，参考 50–80 元
- **建议时长**：1.5–2 小时
- **玩法**：相比更“标志性”的碉楼单体，马降龙的优势在于村落整体氛围，竹林、村道、民居和碉楼同框，适合慢慢走，不要当成纯打卡点。
- **交通**：开平市内建议自驾 / 打车串联立园、自力村
- **预约要点**：部分路段遮阴有限，午后偏热；节假日联票排队时间可能增加
- **来源链接**：<https://www.kaiping.gov.cn/>；<https://whc.unesco.org/en/list/1112/>；<https://baike.baidu.com/item/马降龙碉楼群>

### 4. 锦江里碉楼群（含瑞石楼）【看“碉楼高度感”的代表点】
- **门票参考**：常见参考 50–80 元，联票更划算
- **建议时长**：1–1.5 小时
- **玩法**：瑞石楼辨识度很高，适合喜欢建筑立面和单体观感的人。如果你不想全刷碉楼群，立园 + 自力村 + 锦江里就是比较高效的三点组合。
- **交通**：开平市内自驾 / 包车优先
- **预约要点**：若同天连刷多个碉楼点，建议控制节奏，不要把时间全压在拍照上
- **来源链接**：<https://www.kaiping.gov.cn/>；<https://whc.unesco.org/en/list/1112/>；<https://baike.baidu.com/item/锦江里碉楼群>

### 5. 南楼纪念公园【开平世遗线里的人文补充点】
- **门票参考**：常见免费或低收费开放
- **建议时长**：40–60 分钟
- **玩法**：这里更偏纪念性和历史性，不是最强视觉景点，但适合和碉楼世遗线一起理解地方抗争与侨乡近代史。
- **交通**：适合与开平碉楼片区串联，自驾更方便
- **预约要点**：通常无需复杂预约，注意现场开放时段
- **来源链接**：<https://www.kaiping.gov.cn/>；<https://baike.baidu.com/item/南楼纪念公园>

### 6. 赤坎华侨古镇【江门最热门的沉浸式古镇目的地】
- **门票参考**：日场 / 夜场 / 演艺票差异较大，常见参考 60–150+ 元
- **建议时长**：3–5 小时
- **玩法**：赤坎的核心不是“古镇小吃街”，而是大规模侨乡骑楼与水乡格局。适合看建筑、拍夜景、看演艺、住一晚。若你只白天匆匆逛一圈，会低估它的完成度。
- **交通**：位于开平赤坎镇，建议自驾或从开平城区打车前往
- **预约要点**：周末和节假日强烈建议提前订票；若想看演艺或住景区酒店，更要提前锁定
- **来源链接**：<https://www.kaiping.gov.cn/>；<https://baike.baidu.com/item/赤坎华侨古镇>

### 7. 梁启超故居纪念馆【新会最值得去的人文点】
- **门票参考**：常设展通常免费，部分场馆需实名登记
- **建议时长**：1–1.5 小时
- **玩法**：这是新会人文线的核心。故居、展陈和周边环境一起看，能更完整地理解梁启超与近代新会的关系。适合与小鸟天堂、圭峰山组成一天。
- **交通**：位于新会区茶坑村一带，自驾 / 打车前往方便
- **预约要点**：博物馆类场馆常见周一闭馆或分时开放，出发前查官方公告
- **来源链接**：<https://www.xinhui.gov.cn/>；<https://baike.baidu.com/item/梁启超故居纪念馆>

### 8. 小鸟天堂【江门亲子友好度最高的招牌景点之一】
- **门票参考**：常见参考 30–60 元，游船 / 套票另计
- **建议时长**：1.5–3 小时
- **玩法**：新会最经典的生态景点，以独木成林与观鸟闻名。它不是“硬核自然保护区徒步”，而是适合家庭、轻松散步、坐船看生态的一站。
- **交通**：位于新会区天马村附近，自驾 / 打车方便
- **预约要点**：观鸟效果受季节、天气、时段影响；节假日建议提前购票
- **来源链接**：<https://www.xinhui.gov.cn/>；<https://baike.baidu.com/item/小鸟天堂>

### 9. 圭峰山国家森林公园【新会最稳的自然线】
- **门票参考**：主景区公共区域多为免费或低收费；个别园中园项目另计
- **建议时长**：2–4 小时
- **玩法**：如果你想在江门加一点山景和步道，圭峰山是最实用的选择。强度不算太高，适合做新会半日游主轴。
- **交通**：位于新会城区附近，打车 / 自驾都方便
- **预约要点**：雨后山路偏滑；节假日停车位紧张，尽量早到
- **来源链接**：<https://www.xinhui.gov.cn/>；<https://baike.baidu.com/item/圭峰山国家森林公园>

### 10. 玉台寺【圭峰山线的宗教建筑补充】
- **门票参考**：常见免费或香火随喜
- **建议时长**：30–60 分钟
- **玩法**：适合和圭峰山一起安排，属于“顺路加分项”。如果你喜欢寺院、古木和山中清静氛围，会比单纯爬山更有层次。
- **交通**：位于圭峰山景区范围或周边，适合自驾 / 打车 / 徒步串联
- **预约要点**：通常无需预约；注意寺院礼仪和着装
- **来源链接**：<https://www.xinhui.gov.cn/>；<https://baike.baidu.com/item/玉台寺>

### 11. 古兜温泉旅游度假区【江门最成熟的温泉度假目的地之一】
- **门票参考**：温泉票常见 100–200+ 元；酒店套票差异较大
- **建议时长**：半天–1天
- **玩法**：如果你这趟想“少跑景点、多放松”，古兜是最省心的方案。可做纯泡汤，也可以住一晚，把旅行节奏放慢。
- **交通**：位于新会区崖门镇方向，自驾最佳
- **预约要点**：周末和节假日房价涨幅明显；温泉套票和住宿套餐常变化，建议提前订
- **来源链接**：<https://www.xinhui.gov.cn/>；<https://baike.baidu.com/item/古兜温泉>

### 12. 宋元崖门海战文化旅游区【江门历史感最强的海边人文景区】
- **门票参考**：常见参考 30–60 元
- **建议时长**：1.5–3 小时
- **玩法**：如果你对“崖山之后无中国”这段历史有兴趣，这里很值得专门去。景区本身兼具海边环境与纪念性叙事，适合和古兜温泉 / 新会南部线路同天安排。
- **交通**：新会区崖门镇一带，自驾更方便
- **预约要点**：天气炎热时室外暴晒明显；纪念类景区适合搭配讲解或先做简单功课
- **来源链接**：<https://www.xinhui.gov.cn/>；<https://baike.baidu.com/item/宋元崖门海战文化旅游区>

### 13. 长堤历史文化街区【蓬江最适合散步拍街景的地方】
- **门票参考**：免费
- **建议时长**：1–2 小时
- **玩法**：比起“景区化古镇”，长堤更像一段保留了侨乡城市纹理的街区。适合白天看骑楼，晚上吃饭散步，也适合顺手感受江门城市面。
- **交通**：位于蓬江区市区，打车方便，适合与启明里联动
- **预约要点**：无需预约；老街停车不算特别友好，建议网约车前往
- **来源链接**：<https://www.jiangmen.gov.cn/>；<https://baike.baidu.com/item/长堤历史文化街区>

### 14. 启明里【江门近代侨乡街巷感最好的城市点位之一】
- **门票参考**：免费
- **建议时长**：40–90 分钟
- **玩法**：如果你喜欢老街、民国感、街巷拍照，这里比纯商业街更值得慢慢走。很适合和长堤片区放在同一时段。
- **交通**：蓬江区内，适合打车或步行串联老城线
- **预约要点**：通常无需预约；夜间更有氛围，但部分店铺营业时间有限
- **来源链接**：<https://www.jiangmen.gov.cn/>；<https://baike.baidu.com/item/启明里>

### 15. 陈白沙纪念馆【江门老城里最稳的文化补充点】
- **门票参考**：常设展多为免费开放
- **建议时长**：45–90 分钟
- **玩法**：适合对岭南文化、理学人物、地方文脉感兴趣的人。不是游客最热门的点，但能让江门老城线更完整。
- **交通**：位于蓬江区，适合与长堤 / 启明里同天安排
- **预约要点**：注意周一闭馆或午间休息等博物馆常见规则
- **来源链接**：<https://www.jiangmen.gov.cn/>；<https://baike.baidu.com/item/陈白沙纪念馆>

### 16. 古劳水乡【想走轻松路线时最好用的半日目的地】
- **门票参考**：景区核心区 / 游船项目常见 30–80 元不等
- **建议时长**：2–4 小时
- **玩法**：鹤山方向最适合轻松慢游的一站。水网、村落、船游和田园感都比较适合家庭游客，不追求“震撼”，但体感轻松。
- **交通**：位于鹤山市古劳镇，自驾最佳
- **预约要点**：周末游客会明显增加；若要坐船，注意末班时间和天气变化
- **来源链接**：<https://www.heshan.gov.cn/>；<https://baike.baidu.com/item/古劳水乡>

### 17. 上川岛【江门最经典的海岛目的地之一】
- **门票参考**：上岛需叠加船票、景区交通、住宿等；综合成本通常高于普通陆地景点
- **建议时长**：1–2 天
- **玩法**：适合真正想做海岛度假的人。重点不是刷景点数量，而是海边、沙滩、骑行、发呆和住一晚的节奏感。
- **交通**：通常需先到台山相关码头，再乘船上岛；全程要预留充足时间
- **预约要点**：一定提前确认船班、天气、台风预警、上岛车辆与酒店接驳安排
- **来源链接**：<https://www.cnts.gov.cn/>；<https://baike.baidu.com/item/上川岛>

### 18. 下川岛【比“只去一片海滩”更完整的海岛选择】
- **门票参考**：与上川岛类似，需综合船票 / 接驳 / 住宿核算
- **建议时长**：1–2 天
- **玩法**：适合把江门当成周末海岛目的地来玩。和上川岛一样，更建议住一晚，不然大半时间会花在赶船和接驳上。
- **交通**：台山码头乘船上岛，适合自驾 + 船运组合
- **预约要点**：节假日船票和住宿都建议提前锁定；雷雨和海况会直接影响体验
- **来源链接**：<https://www.cnts.gov.cn/>；<https://baike.baidu.com/item/下川岛>

### 19. 那琴半岛地质海洋公园【台山最容易出片的海边之一】
- **门票参考**：常见参考 80–120 元
- **建议时长**：2–4 小时
- **玩法**：主打礁石海岸、海景栈道和拍照效果，属于“视觉很强”的台山滨海点。适合不想上岛、但又想要海边风景的人。
- **交通**：位于台山市北陡镇方向，建议自驾
- **预约要点**：晴天和蓝天加成很大；夏季要防晒，恶劣天气下部分区域可能关闭
- **来源链接**：<https://www.cnts.gov.cn/>；<https://baike.baidu.com/item/那琴半岛地质海洋公园>

### 20. 浪琴湾【江门滨海自驾线里很实用的一站】
- **门票参考**：常见免费开放或低收费管理，具体以现场为准
- **建议时长**：1.5–3 小时
- **玩法**：比起“重景区化”的海边，浪琴湾更适合看海、散步、拍日落。非常适合作为台山滨海自驾中的一段放松行程。
- **交通**：台山市沿海方向，自驾优先
- **预约要点**：看日落要提前确认天气；海边餐饮和配套不一定稳定，别把补给压得太死
- **来源链接**：<https://www.cnts.gov.cn/>；<https://baike.baidu.com/item/浪琴湾>

### 21. 黑沙湾【适合顺路补充的海滩型点位】
- **门票参考**：常见低收费或按现场公示
- **建议时长**：1–2 小时
- **玩法**：如果你已经在台山海边线里，不妨顺路看看。它更像是“补充型海滩”，不一定需要专门单独跑一趟。
- **交通**：台山滨海线，自驾最顺
- **预约要点**：海边天气和潮汐影响体验明显；留意安全警示
- **来源链接**：<https://www.cnts.gov.cn/>；<https://baike.baidu.com/item/黑沙湾>

### 22. 恩平温泉度假区（锦江温泉 / 山泉湾 / 帝都温泉等）【纯放松最合适】
- **门票参考**：单温泉票常见 80–200+ 元，住宿套餐浮动较大
- **建议时长**：半天–1天
- **玩法**：如果你不执着于刷更多“景点”，恩平温泉很适合拿来做旅行最后一天。泡汤、住酒店、慢慢吃饭，比再硬赶一个景区舒服得多。
- **交通**：恩平市内多个温泉度假区分散分布，自驾最方便
- **预约要点**：热门酒店周末房价明显上涨；注意区分不同温泉酒店的位置与配套差异
- **来源链接**：<https://www.enping.gov.cn/>；<https://baike.baidu.com/item/恩平温泉>

## 五、美食

江门吃的重点，不是“全国统一网红款”，而是很典型的 **侨乡 + 粤西 / 珠西口味**。第一次去，建议把重点放在下面这些方向：

### 必吃方向
- **新会陈皮相关菜 / 陈皮老鸭汤 / 陈皮红豆沙**：新会辨识度最高。
- **古井烧鹅**：江门本地很有代表性的烧味线。
- **台山黄鳝饭**：香气很足，适合当正餐。
- **台山生蚝**：海边线尤其适合吃。
- **开平豆腐角、濑粉、牛杂、小食**：适合在开平老街或古镇附近尝。
- **海鲜大排档**：台山沿海方向最适合安排一顿。
- **早茶**：蓬江 / 新会城区都可以找本地口碑茶楼。

### 推荐觅食区域
- **蓬江区老城 / 长堤周边**：适合吃本地馆子和夜宵。
- **新会城区**：适合找陈皮菜、烧味、家常粤菜。
- **开平赤坎 / 市区**：适合小吃、侨乡风味和游客友好餐厅。
- **台山海边**：优先海鲜。

### 点菜建议
- 2 人行不要一口气点太多硬菜，江门很多馆子分量偏实在。
- 如果想吃陈皮菜，优先找本地评价稳定的餐厅，不要只在景区里解决。

## 六、住宿建议

### 1. 第一次去，住哪里最省心？

#### 住开平（适合世遗主线）
- 适合：第一天想深玩碉楼 + 赤坎的人
- 优点：离核心景点近，晚上逛古镇方便
- 缺点：如果第二天转新会 / 台山，还得继续移动

#### 住新会区
- 适合：想兼顾圭峰山、小鸟天堂、梁启超故居、古兜温泉
- 优点：综合能力最强，吃住成熟
- 缺点：去开平和台山都还要车程

#### 住江门市区（蓬江 / 江海）
- 适合：偏城市休闲、老街散步、商务型出行
- 优点：配套成熟、餐饮多
- 缺点：去开平和海边都不是最近

#### 住赤坎华侨古镇 / 温泉酒店 / 海岛
- 适合：明确想做“住景区 / 度假型”体验的人
- 优点：氛围好，旅行感强
- 缺点：价格通常更高，需要提前预订

### 2. 住宿决策建议
- **人文党**：住开平 1 晚 + 新会 / 市区 1 晚
- **轻松党**：全程住新会或市区，打车 / 自驾出去玩
- **度假党**：住古镇 / 温泉 / 海岛，不要安排太满

## 七、交通

### 外部交通
- **高铁进江门**：适合从广州、深圳、珠海、佛山、中山等地出发。
- **自驾进江门**：如果你要跑开平 + 台山 + 恩平，自驾是体验差异最大的加分项。

### 市内交通
- **蓬江 / 新会 / 江海**：打车、网约车最方便。
- **开平碉楼群**：点位分散，打车能解决，但自驾效率更高。
- **台山海岛**：务必把码头接驳和船班一起算进去，不能只看地图直线距离。

### 交通实操建议
- 一天只安排 **一个主片区** 最合理，不要上午新会、下午开平、晚上台山硬切。
- 海岛行程要把“天气”和“最后一班船”当成核心变量。
- 节假日自驾去热门景区，宁愿早点到，也别卡着中午高峰进场。

## 八、避坑提醒

### 1. 不要低估跨县移动成本
江门地图上看不算夸张，但真玩起来，**开平 / 新会 / 台山 / 恩平** 不是一脚油门随便串的。一天一个片区，体验最好。

### 2. 碉楼不是越多越好
第一次去没必要把所有碉楼群刷完。**立园 + 自力村 + 马降龙 / 锦江里** 已经够完整，再多容易审美疲劳。

### 3. 海岛一定看天气与船班
上川岛、下川岛不是“说走就走”型景点，天气一变，体验和通行都会受影响。不要把返程时间压太死。

### 4. 赤坎华侨古镇更适合留到傍晚
如果你只能安排一次，优先 **下午进 + 夜景**，整体氛围比中午更好。

### 5. 温泉酒店周末溢价明显
如果预算敏感，尽量避开周六晚；很多套餐看着便宜，但房型、餐券、泡池次数差异很大，要看清楚。

### 6. 夏季海边和古村都很晒
江门夏天不是“走走不热”的地方，古镇、碉楼村、海边都容易暴晒，防晒、补水和中午留室内时间很重要。

### 7. 老街不一定适合开车硬钻
长堤、启明里这类点位，停车未必友好。老城线更适合打车到附近后步行解决。

## 十、怎么选最适合你的江门路线

### 只来一次、想看江门辨识度最高的内容
直接选：**开平碉楼 + 赤坎华侨古镇 + 梁启超故居 / 小鸟天堂**。

### 想轻松舒服，不想太赶
直接选：**新会（圭峰山 + 小鸟天堂）+ 古兜温泉 / 恩平温泉**。

### 想看海
直接选：**台山海边 / 海岛单独做 2 天**，不要再硬塞开平和新会。

### 想兼顾拍照和文化
直接选：**立园 + 自力村 + 赤坎华侨古镇 + 长堤 / 启明里**。

江门最怕的不是“景点不够”，而是 **贪多**。把它当成一座“侨乡文化很强、适合分片区慢慢玩”的城市，体验会比走马观花好很多。
