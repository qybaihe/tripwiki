# 江门｜mode2 城市攻略

> 更新时间：2026-03-18  
> 适用场景：第一次去江门、周末 2–3 天短途、想把侨乡建筑 / 碉楼世遗 / 老街古镇 / 山海温泉一次梳理清楚的人。  
> 阅读提示：江门适合“分片区玩”，不要贪多。第一次去，最稳的主线通常是 **开平碉楼 + 赤坎华侨古镇 + 新会圭峰山 / 小鸟天堂**；如果更偏度假，再加台山海边或恩平温泉。

---

## 0. 城市概览

江门位于广东珠江西岸，是粤港澳大湾区里辨识度很高的一座侨乡城市。它的旅行价值不只在“开平碉楼”这一个世界遗产，而在于几条完全不同的旅行线可以拼在一起：

- **开平**：碉楼、村落、华侨建筑，是江门最强的人文名片。
- **新会**：圭峰山、小鸟天堂、梁启超故居，适合做山水 + 人文轻松线。
- **蓬江 / 江海**：长堤、启明里、陈白沙纪念馆，适合看城市肌理与老街区。
- **台山**：上川岛、下川岛、那琴半岛、浪琴湾，偏海边度假。
- **恩平**：温泉休闲更突出，适合作为放松收尾。
- **鹤山**：古劳水乡是适合半日慢游的补充点。

一句话理解江门：**这是一座适合 2–3 天分区慢玩、兼具侨乡文化与度假气质的城市。**

### 江门适合谁
- 想看华侨史、碉楼、骑楼街、近代建筑的人
- 想做广州 / 深圳 / 珠海 / 佛山出发的周末短途的人
- 希望一趟里兼顾人文、自然、海边或温泉的人
- 亲子、家庭、长辈同行，希望整体节奏别太累的人

### 不太适合谁
- 想靠步行在一个核心区刷完全部精华的人
- 对建筑、古村、老街、地方文脉完全无感的人
- 不愿意跨区移动，却又想一次塞进开平、台山、新会的人

## 1. 行前速览

### 建议旅行天数
- **1 天**：只够刷开平世遗主线的一部分
- **2 天**：可以完成“开平 + 新会”经典第一次路线
- **3 天**：最舒服，可在经典线基础上加台山海边或温泉

### 最佳季节
- **10 月—次年 4 月**：最适合逛古镇、村落、老街、山线
- **5 月—9 月**：更适合海岛与海边，但炎热、雷雨和台风影响更大
- **暑期 / 国庆 / 春节**：热门景区和海岛价格、客流都会明显上升

### 到达方式
- **高铁**：可关注江门站、新会站、开平南站、台山站等，按线路下车
- **自驾**：如果要串开平、台山、恩平，自驾效率明显更高
- **公共交通 + 打车**：适合玩蓬江 / 新会；玩碉楼群、海岛、温泉则时间成本偏高

### 预算参考（2 天 1 晚 / 单人）
- **经济型**：约 500–900 元
- **舒适型**：约 900–1600 元
- **海岛 / 温泉度假型**：约 1600 元以上

### 出发前要确认的事
- 海岛线：**天气、船班、返程时间**
- 温泉线：**房价、套票、是否含餐**
- 古镇 / 热门景区：**节假日票务、停车与演艺场次**
- 碉楼线：**景点分布较散，当天交通衔接是否合理**

## 2. 1–3 天行程

### 2 天经典第一次路线

**D1｜开平侨乡世遗线**  
立园 → 自力村碉楼群 → 马降龙碉楼群 / 锦江里碉楼群（二选一）→ 赤坎华侨古镇夜景

**D2｜新会山水人文线**  
圭峰山国家森林公园 → 玉台寺 → 小鸟天堂 → 梁启超故居纪念馆

这条线最稳，基本把第一次来江门最值得看的内容打包了。

### 3 天完整可发布路线

**D1｜开平**  
立园 → 自力村碉楼群 → 马降龙碉楼群 → 赤坎华侨古镇

**D2｜新会**  
圭峰山国家森林公园 → 玉台寺 → 小鸟天堂 → 梁启超故居纪念馆

**D3｜二选一**
- **海边线**：那琴半岛地质海洋公园 → 浪琴湾 / 黑沙湾，或直接上川岛 / 下川岛住一晚
- **休闲线**：古兜温泉旅游度假区 / 恩平温泉度假区

### 1 天压缩版建议
- **只想看江门代表性**：立园 + 自力村 + 赤坎华侨古镇
- **只想轻松一点**：小鸟天堂 + 圭峰山 + 梁启超故居
- **只想看海**：台山海边单独做一天，不要再硬塞开平

## 3. 景点 / 点位

> 说明：以下门票、开放、预约、船班、演出与接驳等信息存在调整可能；如无把握，统一按 **“以现场/官方公告为准”** 执行。  
> 本章按可发布版 mode2 结构整理，每个点位均提供：门票参考、建议时长、玩法、交通、预约要点、来源链接。

### 2.1 立园
- **门票参考**：常见纳入开平碉楼联票体系，单点或联票价格以现场/官方公告为准。
- **建议时长**：1.5–2.5 小时。
- **玩法**：第一次到开平最值得优先看的点位之一。园林、水系、碉楼式建筑和中西合璧装饰融合得最好，适合建立对侨乡建筑审美的第一印象，也很适合拍照。
- **交通**：位于开平市，建议自驾前往；也可从开平城区打车到达。
- **预约要点**：节假日建议提前查看票务与客流信息；具体票型与是否联票，以现场/官方公告为准。
- **来源链接**：
  - https://whc.unesco.org/en/list/1112/
  - https://www.kaiping.gov.cn/
  - https://baike.baidu.com/item/%E7%AB%8B%E5%9B%AD

### 2.2 自力村碉楼群
- **门票参考**：通常纳入开平碉楼联票体系，价格以现场/官方公告为准。
- **建议时长**：1.5–2 小时。
- **玩法**：如果只能选一个最能体现“田园中的碉楼群”气质的地方，自力村基本不会出错。适合步行慢看，也适合广角拍摄村落与碉楼同框。
- **交通**：建议自驾或从开平城区打车前往，与立园串联最省心。
- **预约要点**：旺季最好早点到；门票、开放时段与讲解服务以现场/官方公告为准。
- **来源链接**：
  - https://whc.unesco.org/en/list/1112/
  - https://www.kaiping.gov.cn/
  - https://baike.baidu.com/item/%E8%87%AA%E5%8A%9B%E6%9D%91%E7%A2%89%E6%A5%BC%E7%BE%A4

### 2.3 马降龙碉楼群
- **门票参考**：常见为联票点位，价格以现场/官方公告为准。
- **建议时长**：1.5–2 小时。
- **玩法**：它的强项不是单一建筑，而是整体村落氛围。竹林、村道、传统民居和碉楼的组合很舒服，更适合慢慢走，不适合纯打卡式冲刺。
- **交通**：建议自驾串联立园、自力村；公共交通转乘会比较耗时。
- **预约要点**：中午偏晒，建议避开最热时段；门票、开放信息以现场/官方公告为准。
- **来源链接**：
  - https://whc.unesco.org/en/list/1112/
  - https://www.kaiping.gov.cn/
  - https://baike.baidu.com/item/%E9%A9%AC%E9%99%8D%E9%BE%99%E7%A2%89%E6%A5%BC%E7%BE%A4

### 2.4 锦江里碉楼群（含瑞石楼）
- **门票参考**：通常为联票或单点售票，价格以现场/官方公告为准。
- **建议时长**：1–1.5 小时。
- **玩法**：适合喜欢看标志性碉楼单体的人，瑞石楼辨识度高。若你不想刷太多同类型点位，立园 + 自力村 + 锦江里就是很高效的一组。
- **交通**：建议自驾或打车前往，和开平其他碉楼点一并安排。
- **预约要点**：票务、开放时间及可进入区域以现场/官方公告为准。
- **来源链接**：
  - https://whc.unesco.org/en/list/1112/
  - https://www.kaiping.gov.cn/
  - https://baike.baidu.com/item/%E9%94%A6%E6%B1%9F%E9%87%8C%E7%A2%89%E6%A5%BC%E7%BE%A4

### 2.5 南楼纪念公园
- **门票参考**：多见免费或低收费开放，以现场/官方公告为准。
- **建议时长**：40–60 分钟。
- **玩法**：偏纪念性与历史性，不是纯视觉型景点，更适合当作开平人文补充点来理解当地近代历史。
- **交通**：适合自驾顺路串联；从开平城区打车也可到达。
- **预约要点**：是否收费、具体开放安排以现场/官方公告为准。
- **来源链接**：
  - https://www.kaiping.gov.cn/
  - https://baike.baidu.com/item/%E5%8D%97%E6%A5%BC%E7%BA%AA%E5%BF%B5%E5%85%AC%E5%9B%AD

### 2.6 赤坎华侨古镇
- **门票参考**：日场、夜场、演艺票、套票差异较大，以现场/官方公告为准。
- **建议时长**：3–5 小时，想看夜景更建议半天起步。
- **玩法**：它不是普通“古镇小吃街”，核心价值是成片骑楼建筑与侨乡水乡格局。更推荐下午进入、顺带看夜景或演艺，整体氛围比中午更好。
- **交通**：位于开平赤坎镇，建议自驾；从开平城区打车也比较方便。
- **预约要点**：周末和节假日建议提前确认票务、演艺场次、停车与入园规则；以现场/官方公告为准。
- **来源链接**：
  - https://www.kaiping.gov.cn/
  - https://baike.baidu.com/item/%E8%B5%A4%E5%9D%8E%E5%8D%8E%E4%BE%A8%E5%8F%A4%E9%95%87

### 2.7 梁启超故居纪念馆
- **门票参考**：常设展多见免费开放，以现场/官方公告为准。
- **建议时长**：1–1.5 小时。
- **玩法**：新会人文线的核心点位之一，适合和小鸟天堂、圭峰山同天安排。这里更适合慢看展陈和故居环境，而不是匆匆到此一游。
- **交通**：位于新会区茶坑村一带，建议自驾或打车前往。
- **预约要点**：博物馆类场馆常见闭馆日、实名登记或分时段入馆安排，以现场/官方公告为准。
- **来源链接**：
  - https://www.xinhui.gov.cn/
  - https://baike.baidu.com/item/%E6%A2%81%E5%90%AF%E8%B6%85%E6%95%85%E5%B1%85%E7%BA%AA%E5%BF%B5%E9%A6%86

### 2.8 小鸟天堂
- **门票参考**：景区门票、游船或联票价格以现场/官方公告为准。
- **建议时长**：1.5–3 小时。
- **玩法**：江门亲子友好度很高的一站，以独木成林和生态观鸟闻名。适合轻松散步、坐船、看湿地生态，不是高强度徒步型自然景区。
- **交通**：位于新会区天马村附近，自驾或打车都方便。
- **预约要点**：观鸟效果受季节和天气影响较大；门票、开放时段与是否需预约，以现场/官方公告为准。
- **来源链接**：
  - https://www.xinhui.gov.cn/
  - https://baike.baidu.com/item/%E5%B0%8F%E9%B8%9F%E5%A4%A9%E5%A0%82

### 2.9 圭峰山国家森林公园
- **门票参考**：主景区多为免费或低收费，个别园中园项目以现场/官方公告为准。
- **建议时长**：2–4 小时。
- **玩法**：新会最稳的自然线，适合做轻徒步、山景步道、看湖景，也适合和玉台寺一起串成半日行程。
- **交通**：位于新会城区附近，自驾、打车前往都方便。
- **预约要点**：停车、园内接驳、个别项目收费情况以现场/官方公告为准；雨后山路偏滑要注意。
- **来源链接**：
  - https://www.xinhui.gov.cn/
  - https://baike.baidu.com/item/%E5%9C%AD%E5%B3%B0%E5%B1%B1%E5%9B%BD%E5%AE%B6%E6%A3%AE%E6%9E%97%E5%85%AC%E5%9B%AD

### 2.10 玉台寺
- **门票参考**：多见免费或香火随喜，以现场/官方公告为准。
- **建议时长**：30–60 分钟。
- **玩法**：适合搭配圭峰山顺路安排，寺院、古木和山间环境都比较舒服，是新会线里很稳的加分项。
- **交通**：位于圭峰山片区，适合自驾、打车或徒步串联。
- **预约要点**：通常无需复杂预约；开放安排与礼佛区域规定以现场/官方公告为准。
- **来源链接**：
  - https://www.xinhui.gov.cn/
  - https://baike.baidu.com/item/%E7%8E%89%E5%8F%B0%E5%AF%BA

### 2.11 古兜温泉旅游度假区
- **门票参考**：温泉票、酒店套餐、园区项目价格以现场/官方公告为准。
- **建议时长**：半天–1 天。
- **玩法**：适合把江门当作休闲度假目的地来玩。最好的打开方式不是匆匆泡个汤就走，而是直接住一晚，把节奏慢下来。
- **交通**：位于新会区崖门镇方向，自驾最方便。
- **预约要点**：周末和节假日房价波动明显，票型和套餐内容变化较大，以现场/官方公告为准。
- **来源链接**：
  - https://www.xinhui.gov.cn/
  - https://baike.baidu.com/item/%E5%8F%A4%E5%85%9C%E6%B8%A9%E6%B3%89

### 2.12 宋元崖门海战文化旅游区
- **门票参考**：门票及园内项目价格以现场/官方公告为准。
- **建议时长**：1.5–3 小时。
- **玩法**：适合对宋元历史、海战题材和纪念性景区感兴趣的人。可与古兜温泉或新会南部线路同天安排。
- **交通**：位于新会区崖门镇一带，自驾更方便。
- **预约要点**：门票、开放时段、讲解服务与园区活动安排以现场/官方公告为准。
- **来源链接**：
  - https://www.xinhui.gov.cn/
  - https://baike.baidu.com/item/%E5%AE%8B%E5%85%83%E5%B4%96%E9%97%A8%E6%B5%B7%E6%88%98%E6%96%87%E5%8C%96%E6%97%85%E6%B8%B8%E5%8C%BA

### 2.13 长堤历史文化街区
- **门票参考**：免费开放，以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：蓬江最适合散步拍街景的地方之一。适合白天看骑楼立面、晚上吃饭散步，也适合作为理解江门城市面的入门点。
- **交通**：位于蓬江区市区，打车最方便。
- **预约要点**：通常无需预约；街区停车条件一般，营业时间与局部开放情况以现场/官方公告为准。
- **来源链接**：
  - https://www.jiangmen.gov.cn/
  - https://baike.baidu.com/item/%E9%95%BF%E5%A0%A4%E5%8E%86%E5%8F%B2%E6%96%87%E5%8C%96%E8%A1%97%E5%8C%BA

### 2.14 启明里
- **门票参考**：免费开放，以现场/官方公告为准。
- **建议时长**：40–90 分钟。
- **玩法**：适合喜欢民国感街巷、侨乡城市肌理和轻拍照的人。和长堤一起安排，城市老街线就比较完整了。
- **交通**：位于蓬江区，可打车到附近后步行游览。
- **预约要点**：通常无需预约；商铺营业时间、街区活动安排以现场/官方公告为准。
- **来源链接**：
  - https://www.jiangmen.gov.cn/
  - https://baike.baidu.com/item/%E5%90%AF%E6%98%8E%E9%87%8C

### 2.15 陈白沙纪念馆
- **门票参考**：常设展多见免费开放，以现场/官方公告为准。
- **建议时长**：45–90 分钟。
- **玩法**：适合对岭南文化和地方文脉有兴趣的人。虽然不是最热门景点，但很适合作为蓬江老城线的文化补充。
- **交通**：位于蓬江区，适合与长堤、启明里同天安排。
- **预约要点**：博物馆类场馆常见闭馆日、午间休息或登记安排，以现场/官方公告为准。
- **来源链接**：
  - https://www.jiangmen.gov.cn/
  - https://baike.baidu.com/item/%E9%99%88%E7%99%BD%E6%B2%99%E7%BA%AA%E5%BF%B5%E9%A6%86

### 2.16 古劳水乡
- **门票参考**：景区门票、游船和套票价格以现场/官方公告为准。
- **建议时长**：2–4 小时。
- **玩法**：鹤山方向最适合轻松半日游的一站，重点是水网村落、船游和田园氛围，不追求震撼，但很适合家庭和慢节奏旅行。
- **交通**：位于鹤山市古劳镇，建议自驾前往。
- **预约要点**：是否需要购票入园、游船班次与天气影响情况，以现场/官方公告为准。
- **来源链接**：
  - https://www.heshan.gov.cn/
  - https://baike.baidu.com/item/%E5%8F%A4%E5%8A%B3%E6%B0%B4%E4%B9%A1

### 2.17 上川岛
- **门票参考**：涉及船票、上岛交通、住宿与景区消费，整体费用以现场/官方公告为准。
- **建议时长**：1–2 天。
- **玩法**：更适合真正想做海岛度假的人，而不是打卡式快闪。海边放空、沙滩、骑行、住一晚，体验会比赶景点舒服得多。
- **交通**：通常需先抵达台山相关码头，再换乘船只上岛；建议自驾配合船运。
- **预约要点**：船班、天气、海况、返程安排都非常关键，务必以现场/官方公告为准。
- **来源链接**：
  - https://www.cnts.gov.cn/
  - https://baike.baidu.com/item/%E4%B8%8A%E5%B7%9D%E5%B2%9B

### 2.18 下川岛
- **门票参考**：船票、接驳、住宿与景区项目价格以现场/官方公告为准。
- **建议时长**：1–2 天。
- **玩法**：适合周末海岛休闲，比单去一片海滩更完整。最好的玩法依然是住一晚，否则大量时间会消耗在赶船和接驳上。
- **交通**：从台山方向相关码头乘船上岛，建议预留充足换乘时间。
- **预约要点**：节假日船票、酒店和返程安排都要提前确认，以现场/官方公告为准。
- **来源链接**：
  - https://www.cnts.gov.cn/
  - https://baike.baidu.com/item/%E4%B8%8B%E5%B7%9D%E5%B2%9B

### 2.19 那琴半岛地质海洋公园
- **门票参考**：景区门票、停车及园内项目价格以现场/官方公告为准。
- **建议时长**：2–4 小时。
- **玩法**：主打礁石海岸、栈道和拍照效果，属于“视觉冲击力很强”的台山海边代表点。适合不想上岛、但又想看海景大片的人。
- **交通**：位于台山北陡镇方向，建议自驾前往。
- **预约要点**：天气对观感影响很大；门票、开放区域和恶劣天气下的管制信息，以现场/官方公告为准。
- **来源链接**：
  - https://www.cnts.gov.cn/
  - https://baike.baidu.com/item/%E9%82%A3%E7%90%B4%E5%8D%8A%E5%B2%9B%E5%9C%B0%E8%B4%A8%E6%B5%B7%E6%B4%8B%E5%85%AC%E5%9B%AD

### 2.20 浪琴湾
- **门票参考**：是否收费及管理费用以现场/官方公告为准。
- **建议时长**：1.5–3 小时。
- **玩法**：适合看海、散步、拍日落，是台山滨海自驾线里很实用的一站。它的优点是松弛，不是景点密度高。
- **交通**：位于台山沿海方向，自驾最方便。
- **预约要点**：海边天气、潮汐和道路通行情况会影响体验，具体以现场/官方公告为准。
- **来源链接**：
  - https://www.cnts.gov.cn/
  - https://baike.baidu.com/item/%E6%B5%AA%E7%90%B4%E6%B9%BE

### 2.21 黑沙湾
- **门票参考**：收费规则与现场管理安排以现场/官方公告为准。
- **建议时长**：1–2 小时。
- **玩法**：更适合作为台山海边线的顺路补充点，不一定值得专门单跑，但如果你本来就在沿海自驾，可以停下来看看海、吹吹风。
- **交通**：位于台山滨海线，建议自驾前往。
- **预约要点**：潮汐、天气和安全提示会直接影响体验，以现场/官方公告为准。
- **来源链接**：
  - https://www.cnts.gov.cn/
  - https://baike.baidu.com/item/%E9%BB%91%E6%B2%99%E6%B9%BE

### 2.22 恩平温泉度假区
- **门票参考**：不同温泉酒店、单次泡池票、住宿套餐价格差异较大，以现场/官方公告为准。
- **建议时长**：半天–1 天，最好住一晚。
- **玩法**：如果你不执着继续刷更多景点，这里很适合拿来做旅行最后一站。泡温泉、住酒店、慢慢吃饭，比继续赶路舒服很多。
- **交通**：恩平各温泉度假区分布较散，自驾最方便。
- **预约要点**：周末房价波动明显，套餐内容差异大，具体以现场/官方公告为准。
- **来源链接**：
  - https://www.enping.gov.cn/
  - https://baike.baidu.com/item/%E6%81%A9%E5%B9%B3%E6%B8%A9%E6%B3%89

## 4. 美食

江门好吃的重点，不在全国统一网红款，而在很典型的侨乡与珠西风味。

### 优先尝试的方向
- **新会陈皮菜**：陈皮老鸭汤、陈皮骨、陈皮甜品
- **古井烧鹅**：江门代表性烧味之一
- **台山黄鳝饭**：很适合作为正餐主菜
- **台山生蚝、海鲜**：走海边线时优先安排
- **开平小吃**：豆腐角、濑粉、牛杂等
- **粤式早茶**：蓬江、新会城区都比较容易找到稳定选择

### 觅食区域建议
- **蓬江老城 / 长堤周边**：适合本地馆子、夜宵
- **新会城区**：适合陈皮菜、烧味、家常粤菜
- **开平赤坎 / 市区**：适合边逛边吃和游客友好餐厅
- **台山沿海**：优先海鲜排档，但别太晚才找店

### 实用建议
- 很多本地馆子分量偏实在，2 人行别点太满
- 想吃陈皮菜，优先找本地评价稳定的店，不要全部在景区内解决
- 海边线餐饮选择可能不如城区稳定，补给别压得太极限

## 5. 住宿

### 第一次去怎么住最省心

#### 住开平
- **适合**：把碉楼世遗和赤坎古镇放在第一优先级的人
- **优点**：离核心景点近，晚上留在古镇更方便
- **缺点**：第二天若转新会或台山，仍需继续移动

#### 住新会
- **适合**：想兼顾圭峰山、小鸟天堂、梁启超故居、古兜温泉的人
- **优点**：综合能力强，吃住更成熟
- **缺点**：去开平或台山都还要车程

#### 住江门市区（蓬江 / 江海）
- **适合**：偏城市休闲、老街散步、商务型出行
- **优点**：配套成熟、餐饮密度高
- **缺点**：跑开平和海边都不算最近

#### 住景区型酒店
- **适合**：明确要做“古镇 / 温泉 / 海岛度假”的人
- **优点**：氛围感强，旅行感更完整
- **缺点**：价格通常更高，且节假日涨幅明显

### 住宿选择结论
- **第一次去、偏人文**：开平 1 晚 + 新会 / 市区 1 晚
- **第一次去、想轻松**：全程住新会或江门市区
- **度假优先**：直接住赤坎古镇、温泉酒店或海岛，不要再塞太多景点

## 6. 交通

### 外部交通
- **高铁**：适合从广州、深圳、佛山、珠海、中山等地出发
- **自驾**：如果要玩开平 + 新会 + 台山 / 恩平，自驾体验差异非常明显

### 市内交通判断
- **蓬江 / 新会 / 江海**：打车 + 网约车足够
- **开平碉楼群**：点位分散，自驾效率最高；打车也能玩，但衔接要算好
- **台山海岛**：必须把码头、船班、接驳一起算进去
- **温泉度假区**：自驾通常是最省心的选择

### 交通实操建议
- 一天尽量只玩 **一个主片区**
- 不要安排“上午新会、下午开平、晚上台山”的跨区硬切
- 海岛返程千万别把时间压在最后一班船上
- 节假日热门景区和老街停车都可能紧张，能早到就早到

## 7. 避坑

### 1）不要低估跨区移动成本
江门是“大市 + 多县级区域”的玩法结构，地图上看不夸张，真玩起来通勤成本不低。一天一个片区，体验最好。

### 2）第一次去别把碉楼刷成清单任务
立园、自力村、马降龙、锦江里里选 2–3 个已经很完整。刷太多反而容易审美疲劳。

### 3）海岛线不是临时起意就一定顺
上川岛、下川岛高度依赖天气、海况、船班和接驳，计划必须留余量。

### 4）赤坎华侨古镇更适合下午到夜间
如果只能去一次，尽量把时间放在傍晚到夜景时段，整体氛围明显更强。

### 5）温泉酒店周末价格容易跳
看似便宜的套餐，房型、餐券、泡池次数差异可能很大，预订前要看清。

### 6）夏季的江门真的会很晒
古村、古镇、碉楼和海边都不太遮阴。防晒、补水和避开正午暴晒很重要。

### 7）老街并不总适合开车硬钻
长堤、启明里这一类点位，通常打车到附近再步行更省事。

### 8）海边配套别想得太理想
沿海一些点位的餐饮、补给、洗手间和停车条件，未必像成熟城市景区那样稳定，提前准备更稳。

## 9. 怎么选最适合你的江门路线

### 只来一次，想看江门辨识度最高的内容
直接选：**立园 + 自力村 + 赤坎华侨古镇 + 小鸟天堂 / 梁启超故居**。

### 想轻松舒服，不想太赶
直接选：**新会（圭峰山 + 小鸟天堂）+ 古兜温泉 / 恩平温泉**。

### 想看海
直接选：**台山海边 / 海岛单独做 2 天**，不要硬塞开平和新会。

### 想兼顾建筑、文化和拍照
直接选：**立园 + 自力村 + 赤坎华侨古镇 + 长堤 / 启明里**。

江门最怕的不是景点不够，而是路线太贪。把它当成一座 **适合分区慢玩、侨乡气质非常鲜明** 的城市，体验会比走马观花好很多。
