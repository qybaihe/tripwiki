# 宜宾｜三江交汇、酒城古镇与竹海石海并行的川南代表城市

- 更新时间：2026-03-18
- 适合谁：第一次去川南、喜欢古镇与近代文化史、想把城市夜景和山水景区一起打包的人
- 推荐停留：2-3 天
- 玩法关键词：三江口夜景、李庄古镇、哪吒文化、五粮液、蜀南竹海、兴文石海、川南重口味早餐

## 0. 城市概览
宜宾是四川南部辨识度很高的一座城市。它最直观的标签是“万里长江第一城”——金沙江、岷江在这里汇合成长江；更深入一点，你会发现这座城市并不是单一景点型目的地，而是由三层内容叠出来的：主城三江口与老街夜景、李庄古镇与抗战文化旧址、再加上蜀南竹海和兴文石海两条自然线。

宜宾很适合做一次 2-3 天的小旅行：主城半天到一天能看见这座江城的气质，李庄能补足历史厚度，竹海或石海再给你一张真正意义上的“出片大景”。如果只把宜宾当成路过中转站，会低估它；如果按“江城 + 古镇 + 山水”来排，它会比很多只靠单一王牌景区的城市更完整。

一句话判断：宜宾不是那种一小时打卡完的网红城市，它更像一座内容密度很高、节奏不急但越玩越顺的川南综合样本。

## 1. 行前速览

### 1.1 城市分区怎么理解
- **翠屏区主城**：合江门、三江口、冠英古街、流杯池、翠屏山、哪吒行宫、博物馆等都集中在这里，第一次到宜宾基本从这片开始。
- **李庄古镇片区**：离主城不算太远，适合单独安排半天到一天。核心不是“逛商业古镇”，而是古镇街巷 + 抗战内迁文化旧址。
- **长宁方向**：蜀南竹海、七洞沟在这条线上，适合自然风景和避暑。
- **兴文方向**：兴文石海、僰王山属于另一条偏喀斯特和山地地貌线路。
- **南溪/叙州支线**：南溪古街、横江古镇、石城山、少娥湖更适合二刷或自驾补充。

### 1.2 最适合玩几天
- **1 天**：主城 + 李庄古镇精华版
- **2 天**：主城 + 李庄 + 蜀南竹海 / 兴文石海二选一
- **3 天**：主城 + 李庄 + 蜀南竹海 + 兴文石海二选一深玩，或自然景区外再补一条南溪/横江支线

### 1.3 什么时候去更舒服
- **3-5 月、9-11 月**：体感最好，适合古镇慢逛和山地步行
- **夏季**：主城偏闷热，但蜀南竹海、七洞沟会明显舒服，适合避暑
- **冬季**：城市人文线可以玩，自然景区观感更看天气和能见度

### 1.4 怎么到宜宾
- **高铁**：宜宾站、宜宾西站是主要铁路节点，川内联动很方便
- **飞机**：宜宾五粮液机场有一定航线覆盖，适合外地直接飞入
- **自驾**：如果你打算把竹海、石海、横江、江安等地一起串，自驾效率明显更高

### 1.5 市内怎么移动
- 主城最省心的方式是 **打车 / 网约车 + 步行**。
- 去李庄古镇可公共交通前往，但首刷更建议直接打车，节省时间。
- 去蜀南竹海、兴文石海、僰王山、横江古镇这类远郊点，优先考虑 **自驾 / 包车 / 一日团**。

### 1.6 预算大致怎么估
- **主城 + 李庄 1 天**：很多点免费，预算通常不高
- **加竹海或石海**：门票、景交、往返交通会明显增加
- **整体 2-3 天**：丰俭差异很大，但宜宾总体不是高消费旅游城市

### 1.7 出发前最该注意什么
- 博物馆、纪念馆类点位留意 **周一闭馆 / 临时布展公告**。
- 山地景区、溶洞景区受天气影响较大，开放情况 **以现场/官方公告为准**。
- 李庄、哪吒行宫、蜀南竹海、兴文石海在节假日都容易人多，能早到尽量早到。

## 2. 景点 / 点位
> 本章按可发布攻略写法整理。以下门票、开放、预约、景交、演出与活动信息存在动态调整，统一以现场/官方公告为准。

### 2.1 合江门广场·三江口
- **门票参考**：免费
- **建议时长**：0.5-1.5 小时
- **玩法**：这是理解宜宾最重要的城市点位。金沙江、岷江在此汇成长江，白天适合看江水交汇和城市轮廓，晚上适合沿江散步看灯光。第一次来宜宾，不到三江口基本等于没建立城市印象。
- **交通**：位于主城核心，打车最方便，也可与冠英古街步行串联。
- **预约要点**：一般无需预约；大型节庆活动或临时管制以现场/官方公告为准。
- **来源链接**：
  - https://www.yibin.gov.cn/
  - https://baike.baidu.com/item/宜宾

### 2.2 冠英古街
- **门票参考**：免费
- **建议时长**：1-2 小时
- **玩法**：适合傍晚或夜里来。这里不是那种全国最强古镇，但和三江口连起来后，会把宜宾的老城气质补完整：石板路、川南民居、小吃店、夜景氛围都不错。建议当作城市慢逛线，不要按“超大古镇景区”预期看它。
- **交通**：位于翠屏区老城，和合江门可步行串联。
- **预约要点**：一般无需预约；节假日夜间人流较大，以现场导流安排为准。
- **来源链接**：
  - https://baike.baidu.com/item/冠英古街
  - https://www.yibin.gov.cn/

### 2.3 翠屏山景区
- **门票参考**：通常免费，个别内部点位或项目以现场/官方公告为准
- **建议时长**：2-4 小时
- **玩法**：主城最值得上的山。它的价值不是“爬山”本身，而是把城市视野、哪吒文化、古建、纪念馆串到一起。若时间有限，可以把翠屏山视作宜宾主城的核心半日线。
- **交通**：位于主城，公交和打车都方便。
- **预约要点**：通常无需预约；节假日上山人流较多，开放和游览路线以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/翠屏山
  - https://www.yibin.gov.cn/

### 2.4 哪吒行宫
- **门票参考**：通常免费，以现场/官方公告为准
- **建议时长**：0.5-1.5 小时
- **玩法**：宜宾辨识度极高的文化地标，位于翠屏山上。对哪吒题材感兴趣的人会很有代入感；即使不是深度爱好者，也值得顺路看一眼，把“宜宾与哪吒文化”的连接补上。
- **交通**：位于翠屏山景区内，可步行上山或与周边点位串联游览。
- **预约要点**：通常无需预约；热门节假日、传统节庆期间参观秩序以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/哪吒行宫
  - https://www.yibin.gov.cn/

### 2.5 真武山古建筑群
- **门票参考**：通常免费或低票价，以现场/官方公告为准
- **建议时长**：1-2 小时
- **玩法**：如果你喜欢古建筑，这里值得看。明清道教建筑群保留得较好，和普通“山上寺庙打卡”不同，整体格局和文保价值都更强，适合与翠屏山同线安排。
- **交通**：位于主城山地片区，打车或与翠屏山步行联游都可。
- **预约要点**：一般无需预约；雨后石阶湿滑，部分殿宇开放情况以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/真武山古建筑群
  - https://www.yibin.gov.cn/

### 2.6 流杯池公园
- **门票参考**：通常免费或低票价，以现场/官方公告为准
- **建议时长**：1.5-3 小时
- **玩法**：这是宜宾最有文气的点位之一，适合喜欢园林、题刻、文人典故的人。它的节奏很慢，不是激烈型景点，但如果你喜欢黄庭坚、流觞曲水、传统园林氛围，这里会比想象中更耐看。
- **交通**：位于主城区，打车方便。
- **预约要点**：通常无需预约；局部展陈、园内活动安排以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/流杯池公园
  - https://www.yibin.gov.cn/

### 2.7 宜宾市博物院
- **门票参考**：通常免费开放，以现场/官方公告为准
- **建议时长**：1.5-3 小时
- **玩法**：第一次来宜宾、又怕信息散的人，建议先看博物馆。它能快速把宜宾的历史、僰文化、酒城线索和区域地理建立起来。天气不好时也很适合作为室内备选。
- **交通**：位于主城区，适合打车前往。
- **预约要点**：是否需要预约、开放时段、临展安排以现场/官方公告为准，周一闭馆风险需提前确认。
- **来源链接**：
  - https://www.yibin.gov.cn/
  - https://baike.baidu.com/item/宜宾市博物院

### 2.8 赵一曼纪念馆
- **门票参考**：通常免费开放，以现场/官方公告为准
- **建议时长**：1-2 小时
- **玩法**：适合对近现代史和抗战史感兴趣的人。展览内容偏扎实型，不是花哨互动路线，但和翠屏山、主城人文线组合后，会让宜宾这座城市的历史厚度更完整。
- **交通**：位于翠屏山片区，可与周边景点联游。
- **预约要点**：开放时间、团队接待、闭馆安排以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/赵一曼纪念馆
  - https://www.yibin.gov.cn/

### 2.9 李庄古镇
- **门票参考**：古镇街区通常免费，部分旧址/展馆联票以现场/官方公告为准
- **建议时长**：4-8 小时
- **玩法**：李庄不是只适合拍照的古镇，它真正厉害的是历史密度。建议至少留半天，把老街、码头、茶馆、白肉馆子和抗战内迁旧址一起看，不然很容易误判成“普通商业古镇”。
- **交通**：位于主城外侧，首刷建议打车前往。
- **预约要点**：古镇本体一般无需预约；展馆、讲解、活动安排以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/李庄古镇
  - https://www.yibin.gov.cn/

### 2.10 李庄同济大学旧址群
- **门票参考**：部分点位免费，联票或讲解服务以现场/官方公告为准
- **建议时长**：1.5-3 小时
- **玩法**：如果说李庄古镇是外壳，这条线就是灵魂。抗战时期同济大学等机构内迁到此，使李庄在中国近现代文化史上有了非常特殊的位置。建议专门留时间，不要只在街口拍拍照就走。
- **交通**：位于李庄古镇片区内，可步行串联。
- **预约要点**：展陈开放、讲解场次、团体接待规则以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/李庄古镇
  - https://www.yibin.gov.cn/

### 2.11 月亮田景区 / 李庄抗战文化核心点位
- **门票参考**：以现场/官方公告为准
- **建议时长**：0.5-1.5 小时
- **玩法**：月亮田常被作为李庄抗战文化代表空间之一来看，它的价值在于把“李庄为什么重要”这件事具体化。如果你准备深玩李庄，建议把它和旧址群一起纳入同一条步行线。
- **交通**：位于李庄古镇片区，步行可达。
- **预约要点**：具体开放与参观安排以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/李庄古镇
  - https://www.yibin.gov.cn/

### 2.12 蜀南竹海
- **门票参考**：门票通常为景区成人票，景交车、索道、游船等另计，具体以现场/官方公告为准
- **建议时长**：6-10 小时，若住一晚会更从容
- **玩法**：这是宜宾自然风景王牌。真正的玩法不是冲一个单点，而是享受整片竹海带来的连续景观：竹林、栈道、湖面、崖壁、云雾。夏天尤其舒服，适合避暑和慢走。
- **交通**：位于长宁县方向，离主城较远，建议自驾、包车或参加一日游。
- **预约要点**：节假日门票、景交与部分项目运营情况以现场/官方公告为准，建议尽量早入园。
- **来源链接**：
  - https://baike.baidu.com/item/蜀南竹海
  - https://www.lszhg.cn/

### 2.13 翡翠长廊
- **门票参考**：通常包含在蜀南竹海大景区内，具体以现场/官方公告为准
- **建议时长**：0.5-1 小时
- **玩法**：是竹海里最经典、最容易出片的路段之一。高密度竹林形成天然绿色通道，适合拍照、散步、感受竹海标志性氛围。若你时间有限，这一段基本属于“不能漏”。
- **交通**：位于蜀南竹海景区内，需结合景交或自驾分区游览。
- **预约要点**：景区内分区交通、临时管控与游览路线以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/蜀南竹海
  - https://www.lszhg.cn/

### 2.14 海中海
- **门票参考**：通常包含在蜀南竹海大景区内，游船等项目以现场/官方公告为准
- **建议时长**：1-2 小时
- **玩法**：竹林和水面组合得很舒服，节奏比很多山地景点更松弛。适合慢走、拍倒影、视情况体验水上项目。若同行人里有老人小孩，这里通常接受度也高。
- **交通**：位于蜀南竹海景区内，按景区路线前往。
- **预约要点**：游船、竹筏、项目运营情况以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/蜀南竹海
  - https://www.lszhg.cn/

### 2.15 仙寓洞
- **门票参考**：通常包含在蜀南竹海大景区内，具体以现场/官方公告为准
- **建议时长**：1-2 小时
- **玩法**：属于竹海里更有地形变化和宗教空间感的点位，既有洞、也有山壁和步道。相较于纯拍照型点位，这里更适合喜欢边走边看地貌与人文空间结合的人。
- **交通**：位于蜀南竹海景区内，需结合景区内部交通。
- **预约要点**：步道和洞内开放情况、天气影响以现场/官方公告为准。

  - https://www.lszhg.cn/

### 2.16 花溪十三桥
- **门票参考**：通常包含在蜀南竹海大景区内，具体以现场/官方公告为准
- **建议时长**：0.5-1.5 小时
- **玩法**：更适合慢游和轻摄影。桥、水、竹林叠在一起，视觉节奏舒服，不是冲击力极强的大景，但很适合作为竹海游览中的休闲段。
- **交通**：位于蜀南竹海景区内，按景区导览串联游览。
- **预约要点**：开放路线和临时维护情况以现场/官方公告为准。

### 2.17 兴文石海
- **门票参考**：门票、景交及洞内项目费用以现场/官方公告为准
- **建议时长**：5-8 小时
- **玩法**：这是宜宾另一张重量级自然名片，主打喀斯特地貌、石海、天坑、溶洞系统。如果你已经看过很多古镇，想要一个真正不同的自然景观日，兴文石海非常值得单独留一天。
- **交通**：位于兴文县方向，建议自驾、包车或一日游。
- **预约要点**：洞线开放、天气影响、安全管理要求以现场/官方公告为准。

### 2.18 天泉洞
- **门票参考**：通常按兴文石海景区整体规则执行，具体以现场/官方公告为准
- **建议时长**：1-2 小时
- **玩法**：是石海景区里非常核心的洞穴体验部分。适合看溶洞空间、钟乳石景观和地下地貌层次。对第一次系统看喀斯特洞穴的人来说，记忆点会很强。
- **交通**：位于兴文石海景区内，结合景区线路游览。
- **预约要点**：洞内开放、安全通行、讲解服务等以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/兴文石海
  - https://www.xwsh.cn/

### 2.19 僰王山风景区
- **门票参考**：门票及内部项目费用以现场/官方公告为准
- **建议时长**：4-7 小时
- **玩法**：如果你想在石海外再找一个更偏山地感、瀑布感、森林感的自然点，僰王山是不错的补位。它没有竹海那么轻松，也不像主城那样友好，更适合喜欢徒步和自然感的人。
- **交通**：位于兴文县方向，建议自驾或包车。
- **预约要点**：天气对游玩影响较大，步道、索桥、洞穴等开放情况以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/僰王山
  - https://www.yibin.gov.cn/

### 2.20 七洞沟旅游景区
- **门票参考**：门票及项目费用以现场/官方公告为准
- **建议时长**：3-6 小时
- **玩法**：长宁方向的补充自然线，偏峡谷、瀑布、森林环境。夏天尤其合适。如果你已经去过蜀南竹海但还想继续留在长宁一带玩一天，七洞沟是很顺的延展选择。
- **交通**：位于长宁县方向，建议自驾或包车。
- **预约要点**：雨季步道、水量与安全提示以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/七洞沟
  - https://www.yibin.gov.cn/

### 2.21 夕佳山民居
- **门票参考**：门票以现场/官方公告为准
- **建议时长**：1.5-3 小时
- **玩法**：川南大型庄园式民居代表之一，适合看院落格局、家族宅院尺度和传统木构细节。如果你对古宅、地方建筑比对城市公园更有兴趣，这里会比预期更值。
- **交通**：位于江安县方向，更适合自驾或包车。
- **预约要点**：开放区域、讲解服务安排以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/夕佳山民居
  - https://www.yibin.gov.cn/

### 2.22 南溪古街
- **门票参考**：免费，个别消费项目以现场/官方公告为准
- **建议时长**：1.5-3 小时
- **玩法**：更偏城市休闲型古街，夜景和水街氛围不错，适合作为轻松半日线。它不一定有李庄那种厚重感，但如果你在宜宾待得稍久，想找个好吃、好走、压力不大的地方，南溪古街很合适。
- **交通**：位于南溪区，建议打车或自驾前往。
- **预约要点**：一般无需预约；节假日停车、夜游人流情况以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/南溪古街
  - https://www.yibin.gov.cn/

### 2.23 横江古镇
- **门票参考**：古镇主体通常免费，以现场/官方公告为准
- **建议时长**：2-4 小时
- **玩法**：如果你不想只玩热门线，横江古镇很适合做一条偏冷门支线。整体气质比李庄更旧、更边镇化，适合喜欢古交通节点、老街巷和相对安静氛围的人。
- **交通**：位于叙州区方向，建议自驾前往。
- **预约要点**：通常无需预约；具体参观范围和周边遗址开放情况以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/横江古镇
  - https://www.yibin.gov.cn/

### 2.24 石城山森林公园
- **门票参考**：通常免费或低票价，以现场/官方公告为准
- **建议时长**：2.5-5 小时
- **玩法**：适合喜欢山地步行、古道和相对野趣环境的人。它不是第一次来宜宾必须打卡的绝对王牌，但若你自驾、时间够、又想避开热门人流，这里是不错的补充。
- **交通**：位于横江一带，建议自驾与横江古镇串联。
- **预约要点**：步道、天气、安全管理要求以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/石城山森林公园
  - https://www.yibin.gov.cn/

## 3. 1-3 天行程建议

### 3.1 1 天速玩版：主城 + 李庄
**适合人群**：第一次来宜宾、停留时间短、主要靠高铁往返的人。

- 上午：合江门广场·三江口 → 冠英古街 → 流杯池 / 宜宾市博物院二选一
- 中午：主城吃燃面、炖鸡面、抄手或家常川菜
- 下午：打车去李庄古镇 → 李庄同济大学旧址群 → 月亮田一线
- 晚上：回三江口看夜景，或者继续在冠英古街附近吃饭散步

**这条线的优点**：时间利用率高，能把宜宾“江城 + 古镇 + 历史”三层核心体验到。

### 3.2 2 天经典版：主城人文 + 竹海山水
**适合人群**：第一次来就想玩到宜宾招牌内容的人。

**Day 1**
- 翠屏山景区
- 哪吒行宫
- 真武山古建筑群
- 合江门 / 三江口夜景

**Day 2**
- 蜀南竹海一日
- 重点串联：翡翠长廊 → 海中海 → 仙寓洞 → 花溪十三桥

**这条线的优点**：兼顾城市辨识度和自然代表作，适合大多数第一次来宜宾的游客。

### 3.3 3 天深度版：主城 + 李庄 + 石海 / 竹海完整线
**适合人群**：有完整周末或小长假，希望把宜宾玩得更立体的人。

**Day 1**
- 合江门广场·三江口
- 冠英古街
- 流杯池公园
- 晚上主城夜游

**Day 2**
- 李庄古镇深度游
- 李庄同济大学旧址群
- 月亮田 / 抗战文化点位
- 晚上回主城休息

**Day 3 方案 A：自然景观优先**
- 兴文石海 → 天泉洞

**Day 3 方案 B：轻松避暑优先**
- 蜀南竹海一日，若体力足可加七洞沟

**这条线的优点**：不会赶，且每一天主题都清楚：城市、古镇、人文、自然。

## 4. 美食
宜宾是会让人早餐就感受到“川南口味不轻”的城市。

### 4.1 必吃清单
- **宜宾燃面**：本地最稳定的城市名片之一，香、辣、干拌感强
- **炖鸡面**：汤头和面搭起来很有地方特色
- **李庄白肉**：去李庄基本都会吃，关键在刀工和蘸水
- **李庄白糕**：适合买来边走边吃或当伴手礼
- **南溪豆腐干**：很适合买回去当零食或下酒菜
- **双河凉糕、黄粑、芽菜系小吃**：适合补充川南甜咸混搭的地方味道
- **江鲜 / 川菜 / 江湖菜**：晚上可以安排更正式的一顿

### 4.2 怎么吃更合理
- **早餐**：燃面 + 汤品 / 抄手，最能快速进入宜宾状态
- **李庄一线**：白肉、白糕、家常小炒都值得试
- **主城晚餐**：找本地评价稳定的川菜或江湖菜馆，比追景区店更稳

### 4.3 口味提醒
- 整体偏麻辣、偏重口、偏油香
- 如果带老人或小孩，建议搭配清汤菜、蒸蛋、蔬菜类，不要全桌硬菜

## 5. 住宿

### 5.1 第一次来住哪里最省心
- **翠屏区 / 三江口周边**：适合第一次来，晚上能散步，去主城景点最顺手
- **高铁站周边**：适合赶车型行程，酒店选择更新一些，但看夜景和老城氛围会弱一点

### 5.2 如果重点玩景区
- **蜀南竹海**：住景区内或景区口，会比从主城往返轻松得多
- **兴文石海**：住兴文县城或景区附近，能减少来回折返疲劳

### 5.3 订房提醒
- 节假日李庄、竹海、石海周边房价浮动会比较明显
- 主城老牌酒店位置通常不错，但设施新旧差异较大，订前看近一年评价更稳

## 6. 交通

### 6.1 外部交通
- **高铁**：宜宾站、宜宾西站
- **飞机**：宜宾五粮液机场
- **自驾**：适合做“主城 + 李庄 + 竹海 / 石海 + 支线古镇”组合

### 6.2 市内交通建议
- 主城：优先打车 / 网约车
- 主城老街区：适合步行慢逛
- 李庄：打车最稳
- 竹海、石海、僰王山、横江：自驾 / 包车效率最高

### 6.3 时间分配建议
- 主城景点彼此之间相对集中，可半天到一天解决
- 李庄建议单独留至少半天
- 蜀南竹海和兴文石海都属于“该留完整一天”的点，不建议挤在同一天

## 7. 避坑
- **不要把蜀南竹海和兴文石海塞进同一天。** 这不是高效，是纯赶路。
- **李庄别只逛街拍照。** 真正有价值的是抗战文化旧址线，不看这层会很亏。
- **主城不用排太满。** 合江门、冠英古街、翠屏山、流杯池任选 2-3 个核心点就够，不必把每个公园都硬刷完。
- **夏天主城会闷。** 白天多安排室内或古街慢逛，夜里再补夜景。
- **山地景区别穿纯拍照鞋。** 竹海、石海、僰王山、七洞沟都更需要舒适鞋、防滑和轻便雨具。
- **节假日早出发。** 哪吒行宫、李庄、竹海、石海这些点拖到中午后，体验很容易明显下降。
- **对古街预期要准。** 南溪古街更偏休闲型；冠英古街更偏城市老街；李庄才是宜宾古镇线的主角。
