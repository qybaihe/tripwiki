# 和硕

- 更新时间：2026-03-16
- 版本：快速可用版 v6
- 推荐指数（整体）：⭐4.6
- 适合人群：想把**和硕—博湖—焉耆—和静前山**做成 2–3 天高完成度小环线的人；想同时体验**博斯腾湖东段亲水度假、马兰红色军工遗址、危须古国遗址链、乌什塔拉—曲惠葡萄酒庄园**的人；以及带老人或孩子、自驾优先、希望路线清晰不折腾的家庭旅行者
- 建议游玩：2–3 天

## 城市概览

和硕县位于**新疆巴音郭楞蒙古自治州东北部、焉耆盆地东北缘、博斯腾湖北岸—东北岸过渡地带**。第一次做行程时，最容易犯的错，是把和硕简单理解成“博湖旁边的另一个县”或者“去马兰顺手经过的地方”。真正好用的理解方式，是把它看成**焉耆盆地东侧的结构型节点**：它一头连着博斯腾湖东段与东北岸的沙滩、湿地、湖滨度假带，一头连着曲惠—乌什塔拉—沙梁湾的绿洲灌区、葡萄园和古城遗址链，再向西南又能与博湖西岸 / 南岸、焉耆县城和和静前山方向形成小环线。

把和硕玩明白，关键不是“有几个景区”，而是看懂它的四层结构：

1. **金沙滩—博斯腾湖东段 / 东北岸**：负责亲水、度假、沙滩、湖面开阔感、家庭游和傍晚日落；
2. **马兰—红山军博园遗址群**：负责和硕最硬的人文厚度，是全县最有全国辨识度的一条主线；
3. **曲惠古城—兰城遗址—烽燧链**：负责把和硕从“湖边休闲县”拉到“危须古国—丝路防御体系”的历史维度；
4. **乌什塔拉—曲惠葡萄园—酒庄产业带**：负责现代和硕最鲜明的田园与产业层，解释这里为什么能形成葡萄酒地理标志产区。

### 和硕在湖盆联动里的真实位置

如果把焉耆盆地和博斯腾湖周边看成一张功能图，大致可以这样理解：

- **焉耆**：负责住宿、补给、夜间餐饮、县城总控和古焉耆文明入口；
- **博湖**：负责博斯腾湖西岸主入口、西南小湖区湿地、南岸补完线、鱼宴和成熟湖区体验；
- **和硕**：负责博斯腾湖东段 / 东北岸亲水度假、马兰红色遗址、危须古国遗址链与葡萄酒庄田园；
- **和静前山 / 巴仑台方向**：负责把焉耆盆地平原旅行往山地草原线继续推进。

所以，**和硕不是博斯腾湖的附属点，而是焉耆盆地东侧把湖区、遗址、酒庄和公路节点压缩在一起的结构型县城。**

### 金沙滩为什么必须和“博湖南岸 / 西岸”一起理解

很多人只会问：“金沙滩值不值得去？”真正该问的是：**金沙滩在整个博斯腾湖体系里负责什么？**

- **博湖大河口 / 西岸体系**：负责第一次见博斯腾湖的大场面、成熟码头、游船、湖鲜正餐和西岸主入口；
- **博湖南岸补完线**：负责白鹭洲、扬水站、海心山、浅滩湿地与湖盆结构补完；
- **和硕金沙滩 / 东段—东北岸**：负责沙滩亲水、天然浴场、东段湖景、相对更轻松的度假气质和“从湖东侧理解博斯腾湖”的视角。

也就是说，**大河口不是金沙滩的替代品，金沙滩也不是大河口的平替。**
一个偏西岸主场，一个偏东段 / 东北岸度假；一个擅长建立“大湖主入口”印象，一个擅长建立“沙滩—湿地—家庭游—轻松收尾”的体验。第一次来时，如果时间足够，最好的方法不是二选一，而是**博湖看西岸 / 南岸，和硕看东段 / 东北岸**，这样整个湖盆结构才真正成立。

### 乌什塔拉为什么是和硕最容易被低估的一段

乌什塔拉不是一个“随手开过去的乡镇名”，它是和硕内部很重要的**田园—遗址—公路转换带**：

- 往西南，可衔接焉耆与博湖方向；
- 往东 / 东南，可补曲惠古城、兰城遗址、烽燧和乡野灌区；
- 自身又串着葡萄园、合作社、酒庄和山前绿洲田园。

如果只去金沙滩和马兰，你会觉得和硕是“湖区 + 红色遗址”的双核心县；但把乌什塔拉—曲惠—兰城跑完后，才会发现和硕真正厉害的是：**它能在一天内把大湖、水系灌区、葡萄酒庄、危须古城、烽燧和红色遗址放进同一张地图。**

一句话总结：

> **和硕最值得的，不是单一爆款，而是“博斯腾湖东段 + 马兰 + 危须古城遗址链 + 葡萄酒庄田园”四层内容能被组织成一个顺路小系统。**

## 行前速览

### 和硕最适合谁去
- 想把 **焉耆—博湖—和硕** 做成 2–3 天闭环的人
- 想在新疆看一个同时拥有**湖区、遗址、红色人文、葡萄酒庄**的小县城的人
- 对“两弹一星”、马兰精神、核试验基地历史有明确兴趣的人
- 对危须国、曲惠古城、兰城遗址、烽燧系统感兴趣的人
- 愿意自驾 / 包车，希望把散点按逻辑跑顺的人
- 带老人、小孩，希望白天有轻松主线、晚上回县城稳住的家庭游客

### 不太适合谁
- 完全不接受县域散点结构，只想找一个大门进出式综合景区的人
- 完全依赖随机打车，还想把**金沙滩 + 马兰 + 曲惠 + 兰城**一天跑完的人
- 对遗址、酒庄、田园结构完全无感，只想玩高密度娱乐项目的人

### 建议停留时间
- **1 天**：只能押一条主线，优先“金沙滩东段湖区线”或“马兰线”
- **2 天**：最实用，能完成“湖区 + 马兰”双核心
- **3 天**：最完整，能把“曲惠—乌什塔拉—兰城—酒庄—烽燧”也补齐

### 最佳季节
- **5–10 月**：整体完成度最高，湖区、湿地、葡萄园、县城和古迹都比较完整
- **6–8 月**：金沙滩旺季，适合玩水和亲子，但也最晒、最容易遇到周末人流
- **9–10 月**：第一次来最推荐，跑马兰、兰城、烽燧和酒庄体感都更舒服
- **冬季**：能来，但湖区项目收缩明显，更适合路过短停或回访

### 气候与穿衣
- 和硕是典型干燥大陆性气候：**紫外线强、空气干、昼夜温差大**
- 靠湖不等于全天凉快，正午同样很晒
- 马兰、兰城、烽燧点位遮阴极少，风起来体感偏硬
- 建议固定携带：**帽子、墨镜、防晒霜、润唇膏、补水用品、防风外套、包脚鞋**

### 大交通怎么到
- **飞机**：通常先到库尔勒梨城机场，再自驾 / 包车 / 换乘进入和硕
- **火车**：可关注和硕站、焉耆站、库尔勒站，再转县城
- **公路**：和硕是巴州东线的重要路口，适合衔接焉耆、博湖、吐鲁番方向

### 县内交通怎么选
1. **自驾**：最优解。金沙滩、马兰、曲惠、乌什塔拉、兰城、烽燧都不在步行圈内。
2. **包车**：不想自己开但想把三条主线跑顺，包 1.5–2 天通常比临时打车更靠谱。
3. **打车 + 缩线**：只适合“县城 + 金沙滩半天”或“县城 + 马兰单线”，不适合完整 2–3 天行程。

### 门票与预约总原则
- 和硕不是统一总票城市，而是**景区 + 展陈 + 国保遗址 + 开放式文保点 + 酒庄预约点**的混合结构
- 湖区项目看**天气、风力、水位和旺淡季**
- 马兰线看**开放安排、实名要求、讲解、分区开放**
- 古城遗址与烽燧多为**开放式文保点思维**：可能免费，但配套弱
- 酒庄和采摘园通常不是标准售票景区，更依赖**电话确认、团队接待、季节开放**

### 和硕—博湖—焉耆—和静怎么联动最顺
不要按行政区死切，而要按功能理解：

- **焉耆**：负责住、吃、补给、夜间休整；
- **博湖**：负责博斯腾湖西岸 / 南岸成熟入口、大湖主视角、鱼宴和湿地主线；
- **和硕**：负责博斯腾湖东段 / 东北岸亲水度假、马兰主轴、危须古城与酒庄灌区；
- **和静**：负责把平原湖盆旅行继续往前山 / 草原线推进。

第一次联游最稳的排序：

1. **焉耆住 / 补给**
2. **博湖看西岸主场或南岸补完线**
3. **和硕看金沙滩东段湖区 + 马兰**
4. **曲惠—乌什塔拉—兰城补古迹与酒庄层次**
5. **若还要继续北上，再转和静**

如果从乌鲁木齐 / 吐鲁番南下，则反过来更顺：

1. **和硕先落地**
2. **金沙滩 + 东段湖区**
3. **马兰**
4. **曲惠—乌什塔拉—兰城**
5. **再转焉耆或博湖**

## 1-3天行程

### 1 天快刷版

#### 方案 A｜金沙滩东段湖区亲水线
- 和硕县城早餐后出发
- 金沙滩旅游度假区
- 天然浴场
- 鸟岛
- 芦苇荡观景带
- 博斯腾湖东段观景线
- 湖岸日落点
- 晚上回县城吃饭休息

**适合**：第一次路过和硕、带孩子、偏轻松、不想跑太远的人。

#### 方案 B｜马兰红色军工主轴线
- 县城出发去乃仁克尔乡 / 马兰方向
- 马兰红山军博园展陈区
- 红山核武器试爆指挥中心旧址
- 防空洞体系
- 将军楼、机关楼、政治部大楼、第三招待所旧址
- 返回县城

**适合**：对“两弹一星”、马兰精神、核基地历史有明确兴趣的人。

#### 方案 C｜曲惠—乌什塔拉—兰城田园古迹线
- 曲惠古城
- 曲惠葡萄园观景线
- 酒庄 / 葡萄酒品鉴点
- 乌什塔拉灌区与乡间公路线
- 兰城遗址
- 视时间补塔哈其烽火台或四十里大墩烽火台

**适合**：第二次来和硕，或已经玩过金沙滩和马兰的人。

### 2 天经典版

#### Day 1｜金沙滩 + 东段湖区 + 湖盆认知日
- 上午从和硕县城出发，不必太早
- 先到金沙滩旅游度假区
- 玩天然浴场、主沙滩、水上项目
- 补鸟岛、芦苇荡观景带
- 下午走博斯腾湖东段 / 东北岸观景带
- 有余力可加塔什汗国家湿地公园外围观景
- 傍晚看湖边日落
- 夜宿和硕县城

**这一天重点**：不是刷项目数，而是先把“和硕为什么是博斯腾湖东段入口”看清楚，同时把它和博湖西岸 / 南岸的分工关系建立起来。

#### Day 2｜马兰主轴 + 曲惠 / 酒庄补点
- 一早出发去马兰红山军博园
- 先看展陈馆，再看旧址本体
- 重点看红山核武器试爆指挥中心旧址、将军楼、防空洞、机关楼
- 下午回县域中段
- 视体力补**曲惠古城 + 葡萄园 / 酒庄**，或者只补其中一条
- 晚上住和硕县城，或顺势转焉耆

**这一天重点**：把和硕从“湖边度假点”升级成“有全国辨识度的人文节点”。

### 3 天完整初访版

#### Day 1｜金沙滩把旅行打开
- 和硕县城 → 金沙滩旅游度假区
- 天然浴场 → 鸟岛 → 芦苇荡观景带
- 博斯腾湖东段观景线 → 湖岸日落
- 夜宿和硕县城

#### Day 2｜马兰红色军工主轴
- 和硕县城 → 马兰红山军博园
- 展陈馆 → 马兰主题雕塑 / 园区观景点 → 防空洞体系
- 红山核武器试爆指挥中心旧址
- 将军楼旧址群 → 司令部机关楼旧址 → 政治部大楼旧址 → 第三招待所旧址
- 夜宿和硕县城

#### Day 3｜曲惠—乌什塔拉—兰城—博湖 / 焉耆衔接线
- 早上先到曲惠古城
- 接曲惠葡萄园观景线
- 预约酒庄参观 / 品鉴
- 沿乌什塔拉乡与沙梁湾方向看灌区、葡萄园与乡野公路线
- 午后到兰城遗址
- 视时间补塔哈其烽火台 / 四十里大墩烽火台
- 晚上根据下一站方向：
  - **往焉耆走**：去焉耆县城住，第二天继续七个星或巴州东线
  - **往博湖走**：去博湖或大河口周边住，第二天继续西岸 / 南岸湖区
  - **往和静走**：建议先在焉耆补给后再北上，不要把长途转场硬塞在古迹日最后

### 可直接执行的联游版：和硕—博湖—焉耆小闭环

#### 走法一｜焉耆起落脚（最省心）
**Day 1 焉耆**
- 焉耆县城落脚
- 七个星佛寺遗址或县城博物馆线
- 晚上在焉耆吃饭、补给

**Day 2 博湖**
- 焉耆出发去博湖
- 大河口 / 西海渔村 / 阿洪口选 1–2 个成熟点
- 中午吃鱼宴
- 下午不过度刷点，傍晚转和硕或继续住博湖

**Day 3 和硕**
- 上午到和硕跑金沙滩 / 东段湖区
- 下午如果体力足够补半天马兰；更推荐住和硕把马兰留到第 4 天

#### 走法二｜乌鲁木齐 / 吐鲁番南下（和硕先开）
**Day 1 和硕**
- 先住和硕
- 下午跑金沙滩、天然浴场、东段湖区

**Day 2 和硕**
- 白天完整给马兰
- 傍晚回县城
- 晚上住和硕

**Day 3 和硕 → 曲惠 / 乌什塔拉 / 兰城 → 焉耆或博湖**
- 上午曲惠古城 + 葡萄园 / 酒庄
- 午后兰城遗址
- 傍晚沿公路节点转焉耆或博湖

#### 走法三｜家庭游 2 晚 3 天最省力版本
- **第 1 晚、第 2 晚都住和硕县城**
- **Day 1**：金沙滩主区 + 日落
- **Day 2**：马兰完整半天到一天
- **Day 3**：曲惠古城 + 葡萄园 + 兰城择重点，不硬拼所有烽燧

这个版本对老人、小孩最友好：白天出去玩，晚上都回县城，不每天拖箱子。

## 景点清单（28 个）

> 说明：以下门票为公开资料、常见票务页、地方介绍、游客经验与文保点常见口径综合整理的**门票参考**，只用于行前估算。和硕存在明显旺淡季差、项目另计、免费文保点配套较弱等情况，请以现场公示为准。

### 1. 金沙滩旅游度假区
- **门票参考**：常见约 ¥30–60；快艇、摩托艇、脚踏船等水上项目另计
- **建议时长**：3–5 小时
- **玩法**：和硕湖区最核心入口。第一次来最适合在这里建立“博斯腾湖东段 / 东北岸”的旅行印象：大湖视野、沙滩、天然浴场、轻度假氛围都集中在这里。
- **交通**：县城自驾最方便，也可包车或打车单点往返
- **预约要点**：旺季、节假日和大风天先确认景区开放状态及水上项目营业情况
- **来源链接**：
  - https://www.xjhs.gov.cn/
  - https://baike.baidu.com/item/和硕县/4814833

### 2. 天然浴场（金沙滩片区）
- **门票参考**：通常含在金沙滩门票体系内
- **建议时长**：1–2 小时
- **玩法**：夏季最典型的亲水内容，适合带孩子踩水、玩沙、拍“新疆湖滨海边感”照片。
- **交通**：位于金沙滩内部，步行可达

- **玩法**：适合出发前做背景补课，帮助理解危须文化、和硕特部、县域发展与地方文化线索。
- **交通**：位于县城，打车或步行方便
- **预约要点**：留意周一闭馆、临时活动和身份证登记要求
- **来源链接**：
  - https://baike.baidu.com/item/和硕县/4814833
  - https://www.xjhs.gov.cn/

### 32. 县城夜间生活面（夜市 / 烧烤街区）
- **门票参考**：无门票，餐饮消费自理
- **建议时长**：1–2 小时
- **玩法**：虽然不是标准“景点”，但对旅行体验很重要。它决定了和硕为什么适合家庭游客和自驾者回县城住，而不是硬住景区边。
- **交通**：县城内步行 / 打车可达
- **预约要点**：旺季热门馆子建议早点去；注意停车秩序
- **来源链接**：
  - https://www.xjhs.gov.cn/
  - https://baike.baidu.com/item/和硕县/4814833

## 美食

和硕的吃饭结构比很多“路过型县城”完整得多：一边靠近博斯腾湖东段，可以补湖鲜；一边又是焉耆盆地绿洲县城，日常新疆馆子稳定；再加上葡萄酒产区加成，可以把餐饮和伴手礼串成一条顺手的线。

### 到和硕优先吃什么
- **烤羊肉串、烤肉拼盘**：最稳的通用答案
- **抓饭**：赶路日效率最高
- **拌面 / 过油肉拌面 / 碎肉拌面**：县城午餐常规强项
- **大盘鸡 / 椒麻鸡**：适合多人拼桌
- **丸子汤、粉汤、烤包子、薄皮包子**：适合早起去马兰前补能量
- **博斯腾湖鱼宴**：当天走金沙滩或接博湖线时值得安排，重点先问清鱼价
- **馕坑肉、夜市烧烤、回民馆子热菜**：夜宵和晚餐都好用
- **葡萄、葡萄干、葡萄酒**：和硕标志性伴手礼

### 最实用的用餐逻辑
- **金沙滩日**：中午不要完全押在景区核心商铺，很多时候县城吃好再去更划算
- **马兰日**：一定提前吃或备好补给，不要默认园区附近随时有稳定餐食
- **曲惠 / 乌什塔拉 / 酒庄日**：酒庄适合品鉴，不一定适合解决正餐，正式吃饭仍建议回县城或转焉耆方向
- **联游时**：
  - 焉耆负责夜间餐饮与补给
  - 博湖负责湖鲜大餐
  - 和硕负责完整一日行程中的稳定用餐和伴手礼采购

### 点鱼和酒庄避坑原则
- 先确认：**斤数、单价、做法费、配菜费、总价**
- 旅游旺季临湖餐厅价格通常明显高于县城
- 如果还要继续开车，司机别碰酒庄品鉴酒

## 住宿

### 1. 和硕县城住宿
**第一次来最推荐。**

优点：
- 去金沙滩、马兰、曲惠、乌什塔拉、兰城都相对居中
- 吃饭、加油、补水、买零食、停车都最方便
- 晚上还能补滨河景区和县城生活面
- 对家庭游客最友好，孩子闹腾、老人休息、临时买药都更省心

适合：第一次来和硕、2 天经典版、3 天完整线、家庭游。

### 2. 金沙滩 / 湖边度假型住宿
如果当季有正常营业的湖边酒店、度假区或民宿，可以住一晚。

优点：
- 湖景和度假感更直接
- 适合亲子、情侣、想放慢节奏的人

缺点：
- 第二天去马兰、曲惠未必最顺
- 餐饮与补给稳定性通常不如县城
- 风大、旺季价格波动更明显

### 3. 联游型住宿安排
- **住和硕**：适合跑“东段湖区 + 马兰 + 曲惠 / 兰城”完整结构
- **住焉耆**：适合把和硕并入焉耆—博湖—和硕联游，夜间餐饮和中转效率更强
- **住博湖 / 大河口周边**：适合第二天主打博湖西岸与南岸，不适合把和硕当主轴的人

### 4. 最实用的换酒店建议
- **2 天行程**：最好只住 1 个地方，优先和硕县城
- **3 天联游**：推荐“焉耆 1 晚 + 和硕 1 晚”或“和硕 2 晚 + 最后一晚转焉耆 / 博湖”
- **不建议**：焉耆、博湖、和硕三地每天换一次，体力和整理成本都会很高

## 交通

### 外部交通
- **乌鲁木齐 / 吐鲁番 → 和硕**：适合作为巴州东线切入口，先玩和硕再往焉耆盆地方向收
- **库尔勒 → 和硕**：适合巴州内部短途联游
- **焉耆 / 博湖 → 和硕**：非常适合做 2–3 天小环线
- **铁路 / 干线公路**：和硕有铁路与干线公路通行条件，是“巴州东大门”这一定位的现实基础

### 县内交通策略
- **县城内**：打车基本够用
- **县域景点之间**：优先自驾 / 包车
- **无车游客**：一天只保留一条主线，不要硬拼湖区 + 马兰 + 古迹三线

### 和硕内部串线顺序
最不容易翻车的排序是：

1. **金沙滩主区**（轻松，适合下午和家庭游）
2. **东段 / 北岸湖景补点**（看时间加减）
3. **马兰整段**（最好单独给半天到一天）
4. **曲惠古城 + 葡萄园 / 酒庄**（适合半天）
5. **乌什塔拉乡田园公路线 + 兰城遗址 + 烽燧补点**（适合半天）

### 公路线节点怎么理解才顺
- **和硕县城**：住宿、加油、买水、补货、吃饭的总控点
- **金沙滩方向**：适合做单独半天或一天，不建议和马兰硬拼同一天
- **乃仁克尔乡 / 马兰方向**：是独立人文主线，跑这一段时要预留充分时间
- **曲惠镇方向**：是危须古城和葡萄园入口，适合和酒庄绑定
- **乌什塔拉乡—沙梁湾方向**：是从田园灌区过渡到兰城遗址的关键公路线，沿途更像“结构阅读线”而不是单点刷景区
- **往焉耆去**：适合晚上转住，解决次日古焉耆与县城功能线
- **往博湖去**：适合把东段湖区看完后，第二天继续补西岸成熟入口和南岸补完线

### 三城衔接的最省力思路
- **和硕 → 博湖**：重点是东段 / 东北岸 vs 西岸 / 南岸互补，不要把两个地方当重复湖边点
- **和硕 → 焉耆**：重点是从和硕的“结构型景点”切到焉耆的“县城枢纽 + 古焉耆文明”
- **焉耆 → 和硕**：最适合上午在焉耆补博物馆 / 古迹，下午到和硕玩金沙滩
- **博湖 → 和硕**：最适合上午大河口 / 阿洪口 / 白鹭洲，下午转和硕东段湖区看日落，第二天再上马兰
- **和硕 → 和静**：不建议古迹日傍晚直接硬拉长途，最好先在焉耆补给，再北上进山

## 避坑

### 1. 不要把和硕理解成“只有一个金沙滩”
如果只去金沙滩拍照打卡就走，等于只看了和硕最浅的一层。和硕真正的完整度来自：**东段湖区 + 马兰 + 曲惠 / 兰城 + 葡萄酒庄**。

### 2. 金沙滩和博湖大河口不是替代关系
- **大河口**更偏博斯腾湖西岸主入口与成熟湖区体验
- **南岸补完线**更偏湿地、水利、湖盆结构补课
- **金沙滩**更偏东段 / 东北岸亲水度假与天然浴场

时间够的话，值得都去，但别重复刷太多同质湖边点。

### 3. 马兰别临时起意就冲
马兰线是和硕最值钱的一段，但也最吃开放安排。出发前最好确认：
- 是否正常开放
- 是否需要实名预约
- 是否有讲解
- 当天开放哪些区
- 防空洞、旧址楼体是否开放

### 4. 没车还想 1 天跑完整个和硕，基本会翻车
和硕不是“下车走走就能把核心景点都刷完”的地方。没有车就老老实实保留一条主线。

### 5. 古城遗址不等于成熟景区
曲惠古城、兰城遗址、烽火台很多时候就是**文保点 + 野外环境**，不一定有厕所、商店、咖啡馆、完整导览。

### 6. 酒庄线别和开车混着来
和硕葡萄酒值得看，但真正的坑很简单：**司机别喝。**如果要认真品鉴，最好当天少开车，或者把回程交给同行人。

### 7. 夏季暴晒比很多人预想得更强
尤其在马兰、兰城、烽燧点位，遮阴几乎没有。不要因为“靠湖”就低估紫外线和脱水风险。

### 8. 乌什塔拉乡道和农路别乱停
兰城和田园线很好拍，但本质仍是乡路和生产道路。别停在转弯、渠边、农机通道和村口主路上。

### 9. 湖边消费先问清二次收费
常见二次收费项包括：
- 水上项目
- 快艇 / 游船
- 停车
- 更衣 / 寄存

### 10. 点湖鱼一定先谈价格
这不是和硕独有问题，但在湖区与旅游旺季尤其要注意，别默认“时价”就透明。

### 11. 家庭游别每天拖着行李转场
第一次来和硕及周边，优先做成**和硕连住 2 晚**，或者**焉耆 1 晚 + 和硕 1 晚**。每天换酒店只会让节奏变差。

### 12. 烽燧和古城别为了拍照踩文保本体
夯土遗址看着“土”，但其实很脆。不要爬城墙、踩马面、滑下护城河边坡。
