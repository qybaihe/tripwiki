# 凉山彝族自治州（四川）

- 更新时间：2026-03-12
- 推荐指数（整体）：⭐4.7（四川省最适合做“湖泊度假 + 彝族风情 + 山地高原 + 古城人文”组合旅行的州域目的地之一；西昌适合短住，盐源泸沽湖适合度假，普格螺髻山适合自然线，会理古城适合人文慢游）

## 城市概览

**四川省凉山彝族自治州**位于**四川省西南部**，州府为**西昌市**，是中国最大的彝族聚居区之一，也是川滇交通与安宁河谷城市带的重要节点。全州不是一个“单点城市景区”，而是一个典型的**州域旅行目的地**：既有西昌这种配套成熟、节奏舒服的中心城市，也有盐源县泸沽湖这种全国知名高原湖泊度假区，还有普格螺髻山、雷波马湖与黄琅、会理古城、冕宁彝海等风格完全不同的分区目的地。

凉山旅行最大的特点有三点：
1. **自然跨度大**：从西昌邛海、泸山这种城市边上的湖山，到海拔 4000 米以上的螺髻山、木里高山地带，再到盐源泸沽湖与雷波马湖，风格差异很大；
2. **民族文化鲜明**：彝族文化是凉山的核心底色，火把节、彝寨、服饰、饮食和礼俗都不是“表演型点缀”，而是这里真实的生活组成；
3. **适合分区玩，不适合贪大求全**：凉山州幅员广，景点之间并非城市步行距离。地图上看似都在一个州里，实际车程常常以小时计。

第一次来，最稳妥的思路不是把全州一网打尽，而是按主线选：
- **西昌城市度假线**：邛海 + 泸山 + 建昌古城 + 周边博物馆/湿地，适合 1-2 天；
- **普格自然线**：螺髻山 + 螺髻九十九里 + 温泉瀑布，适合 1-2 天；
- **盐源湖泊度假线**：泸沽湖四川段 + 草海 + 走婚桥 + 摩梭文化，适合 2-3 天；
- **会理人文古城线**：会理古城 + 绿陶/铜火锅 + 城市慢游，适合 1 天；
- **雷波山水秘境线**：马湖 + 黄琅 + 小众湖山公路，适合 1-2 天；
- **冕宁红色人文线**：彝海结盟遗址 + 灵山寺/灵山景区，可与西昌串联。

如果只给 2-3 天，建议以**西昌为中心做一条短线**，或者单独去**泸沽湖四川段**深度住两晚。凉山不怕景点少，怕的是行程排太满、每天都在赶路。

## 行前速览

### 适合谁
- 想把四川玩出“高原湖泊 + 民族风情 + 山地自然”差异感的人；
- 喜欢湖边慢生活、日落、骑行、轻户外的人；
- 想看彝族文化，不只满足于打卡大景区的人；
- 有自驾能力，愿意接受州域型旅行节奏的人。

### 不太适合谁
- 只想在一个步行可达的小城区刷完所有景点的人；
- 完全不能坐山路车、容易晕车的人；
- 对海拔、强日晒和昼夜温差完全没准备的人。

### 建议天数
- **1 天**：西昌城市精华 / 会理古城 / 雷波马湖单点；
- **2 天**：西昌 + 螺髻山，或西昌 + 冕宁，或会理 + 西昌；
- **3 天**：西昌 + 普格螺髻山 + 建昌古城；或泸沽湖四川段 2 晚深度；
- **4-5 天**：西昌 + 泸沽湖 + 会理，体验较完整；
- **6 天以上**：可把雷波马湖、冕宁、德昌、木里等分区加入。

### 最佳季节
- **3-5 月**：天气稳定，西昌邛海很舒服，春季出片率高；
- **6-8 月**：西昌、螺髻山、泸沽湖都是热门避暑方向，但暑期和火把节附近价格明显上涨；
- **9-11 月**：凉山整体最稳妥的旅行季，泸沽湖、西昌古城、会理古城都很舒服；
- **12-2 月**：西昌仍适合冬季康养，白天体感友好，但高海拔景区和山区路况要更谨慎。

### 气候与穿衣
- 西昌所在安宁河谷冬暖夏不极端，**防晒和昼夜温差**比“纯防寒”更重要；
- 泸沽湖、螺髻山、木里等地早晚明显偏冷，哪怕夏天也建议带抓绒/冲锋衣；
- 雨季山里常见局地阵雨，鞋子尽量防滑。

### 交通到达
- **飞机**：西昌青山机场是进入四川省凉山彝族自治州最方便的航空口岸；
- **铁路**：西昌西站、西昌站是州内核心交通节点；
- **自驾**：凉山非常适合自驾，但很多州内景点分散，山路多、车程长，不要低估耗时；
- **班车/包车**：去泸沽湖、普格、雷波、会理等地，包车或拼车通常更省时间。

### 住宿建议
- **西昌首次到访**：住邛海湿地 / 航天大道 / 建昌古城周边最方便；
- **泸沽湖四川段**：住泸沽湖镇、草海周边或洛洼/达祖等临湖区域；
- **螺髻山线**：住西昌市区或普格县城，若赶早索道建议前一晚靠近景区；
- **会理古城**：住古城内或古城步行可达范围；
- **雷波马湖**：住黄琅镇/马湖周边民宿，条件不如西昌和泸沽湖成熟。

### 预算感受
- 西昌城市段整体消费不算离谱；
- 泸沽湖和暑期热门民宿价格波动明显；
- 螺髻山、泸沽湖等景区的门票、索道、船票、景交叠加后，人均预算要预留；
- 州内跨区包车成本通常比门票更显著。

### 核心提醒
1. **对象必须明确是四川省凉山彝族自治州**，别把它当“西昌周边附属景区”；
2. 凉山适合按分区深度玩，不适合景点列表式“集邮”；
3. 夏季日晒强，高原紫外线和湖边反射都很实在；
4. 泸沽湖四川段与云南段彼此相连，订房和安排行程时要分清所在省份与码头位置；
5. 火把节、暑期、国庆等节点，房价和路况波动都很大。

## 1-3 天行程

### 1 天快览版：西昌城市精华线
- 上午：**建昌古城** → **西昌大通门/四牌楼片区**步行
- 中午：古城或邛海边吃火盆烧烤、醉虾、坨坨肉、土鸡
- 下午：**邛海国家湿地公园**骑行/散步 → **月色风情小镇**
- 傍晚：**泸山**上山看邛海全景，或在邛海边看日落
- 晚上：住西昌市区或邛海边

适合第一次来四川省凉山彝族自治州、时间很紧的人。

### 2 天经典版：西昌 + 螺髻山
- **D1**：建昌古城 → 邛海 → 泸山 → 夜宿西昌
- **D2**：早起前往**螺髻山景区** → 视时间补**螺髻九十九里温泉瀑布** → 返回西昌

适合想把“西昌城市度假感”和“凉山高山景观”一起打包的人。

### 3 天舒适版：西昌 + 冕宁/会理 二选一

#### 方案 A：西昌 + 冕宁人文自然线
- **D1**：西昌建昌古城 + 邛海 + 泸山
- **D2**：螺髻山或邛海深度环湖
- **D3**：前往冕宁 → **彝海结盟遗址** → **灵山寺/灵山景区** → 返回

#### 方案 B：西昌 + 会理古城线
- **D1**：西昌建昌古城 + 邛海湿地
- **D2**：螺髻山
- **D3**：前往**会理古城**慢游，晚上返西昌或直接住会理

### 3 天度假版：泸沽湖四川段
- **D1**：抵达盐源县泸沽湖镇 → 环湖西四川段观景 → 住湖边
- **D2**：**草海** + **走婚桥** + 猪槽船/环湖骑行 + 摩梭文化体验
- **D3**：女神湾/四川湖岸线日出或日落点 → 返程

适合不想赶路、重视湖景和住宿氛围的人。

## 景点

> 说明：以下按四川省凉山彝族自治州州域范围挑选，覆盖西昌、普格、盐源、会理、冕宁、雷波等方向。门票与开放政策会随淡旺季、活动、索道检修、景区升级而变化，以下以公开资料和常见出行口径整理，**务必以景区官方购票页/公众号/现场公示为准**。

### 1. 邛海旅游度假区（西昌市）
- **门票参考**：大景区开放式区域多为免费，个别园区/项目单独收费
- **建议时长**：半天-1 天
- **怎么玩**：邛海是西昌的城市名片，适合骑行、散步、看日落、住湖景酒店。第一次来西昌，先去邛海基本不会错。
- **交通**：距西昌城区很近，打车、自驾、公交都方便
- **预约要点**：开放式区域一般无需预约；节假日湖边停车紧张，骑行道与热门拍照点傍晚人多
- **来源链接**：
  - <https://baike.baidu.com/item/邛海>
  - <https://www.mct.gov.cn/>

### 2. 邛海国家湿地公园（西昌市）
- **门票参考**：多数片区免费，部分观光车/游船/体验项目另计
- **建议时长**：2-4 小时
- **怎么玩**：适合轻松环湖、观鸟、慢跑、租车骑行，是西昌最舒服的低门槛玩法之一。
- **交通**：从市区到梦寻花海、观鸟岛、月色风情小镇等片区都不远
- **预约要点**：旺季和傍晚人最多；不要把湿地公园理解成单一入口景区，它是分片区展开的
- **来源链接**：
  - <https://baike.baidu.com/item/邛海>

### 3. 泸山景区（西昌市）
- **门票参考**：景区通常免费或低门槛，个别寺庙/展馆单独收费
- **建议时长**：2-4 小时
- **怎么玩**：主看邛海全景、山中古刹、古树与猴群，适合和邛海安排在同一天。山上视野好，是西昌经典制高点。
- **交通**：打车、自驾到山脚后步行或换乘；也可与邛海串联
- **预约要点**：山上猴子活跃，别外露食物；雨后山路湿滑；日落前后观景效果最好
- **来源链接**：
  - <https://baike.baidu.com/item/泸山>

### 4. 光福寺（西昌市泸山）
- **门票参考**：通常包含在泸山游览范围内或现场低价开放
- **建议时长**：30 分钟-1 小时
- **怎么玩**：泸山最有代表性的古刹之一，可结合望海楼看邛海，适合对古建和宗教建筑有兴趣的人。
- **交通**：位于泸山游线内，步行抵达
- **预约要点**：文明参观，部分殿宇香火时段较集中
- **来源链接**：
  - <https://baike.baidu.com/item/泸山>

### 5. 凉山彝族奴隶社会博物馆（西昌市泸山片区）
- **门票参考**：常设展多为免费或低价，具体以馆方为准
- **建议时长**：1-2 小时
- **怎么玩**：这是理解凉山社会历史与彝族文化背景的重要补充点，不只是“顺便看看”的小馆。
- **交通**：通常与泸山同游最顺路
- **预约要点**：节假日建议先看馆方开放安排；对历史议题敏感的人建议留足阅读时间
- **来源链接**：
  - <https://baike.baidu.com/item/泸山>

### 6. 建昌古城（西昌市）
- **门票参考**：古城街区免费，个别城楼/展陈/演出项目单独收费
- **建议时长**：2-4 小时，夜游可更长
- **怎么玩**：看明代城池格局、四门、街巷和夜景灯光。相比很多“新仿古街”，建昌古城的历史底子更扎实，适合慢慢走。
- **交通**：位于西昌城区，打车公交都方便
- **预约要点**：街区不用预约；夜景和节庆活动更热闹，节假日停车不太友好
- **来源链接**：
  - <https://baike.baidu.com/item/建昌古城>

### 7. 大通楼 / 大通门片区（西昌市建昌古城）
- **门票参考**：古城公共区域免费
- **建议时长**：30 分钟-1 小时
- **怎么玩**：建昌古城标志性城楼，白天适合看城墙细节，夜里适合拍灯光与街景。
- **交通**：古城步行即可抵达
- **预约要点**：节假日晚间人流大，想拍空镜需要早去
- **来源链接**：
  - <https://baike.baidu.com/item/建昌古城>

### 8. 月色风情小镇（西昌市邛海畔）
- **门票参考**：免费进入，餐饮娱乐消费自理
- **建议时长**：1-2 小时
- **怎么玩**：更偏休闲商业街区，适合在邛海散步后吃饭、喝咖啡、等夜景，不算硬核景点，但很适合游客落脚。
- **交通**：邛海边，打车和自驾方便
- **预约要点**：旺季晚间人多，想要安静建议工作日前往
- **来源链接**：
  - <https://baike.baidu.com/item/邛海>

### 9. 螺髻山景区（普格县）
- **门票参考**：门票常见约 90 元；索道、景交通常另计，整体人均常见在 150-250 元区间
- **建议时长**：4-8 小时
- **怎么玩**：看高山杜鹃、冰蚀地貌、黑龙潭、角峰刃脊、云海和高山湖泊，是四川省凉山彝族自治州最有代表性的高山自然景区之一。
- **交通**：距西昌约 1 小时多车程，适合西昌出发一日往返
- **预约要点**：索道检修、天气封山、雷雨大风都可能影响开放；高海拔温差大，别穿城市休闲鞋硬上山
- **来源链接**：
  - <https://baike.baidu.com/item/螺髻山>

### 10. 螺髻山黑龙潭 / 高山湖群（普格县）
- **门票参考**：通常含于螺髻山景区体系内
- **建议时长**：1-2 小时
- **怎么玩**：这是螺髻山里最出片的精华段之一，高山湖泊、云海和山体线条组合很好，适合慢拍。
- **交通**：随景区索道与游线进入
- **预约要点**：天气决定观感上限；浓雾时拍摄效果会大幅下降
- **来源链接**：
  - <https://baike.baidu.com/item/螺髻山>

### 11. 螺髻九十九里温泉瀑布（普格县）
- **门票参考**：常见约 108-168 元，旺季和套票可能浮动
- **建议时长**：2-4 小时
- **怎么玩**：主打温泉 + 瀑布景观，是凉山辨识度很高的休闲体验点，适合和螺髻山一并安排，或者单独泡汤放松。
- **交通**：通常从西昌或普格出发，自驾/包车最稳妥
- **预约要点**：周末与假期人多；需带泳衣拖鞋；温泉瀑布区地面湿滑
- **来源链接**：
  - <https://baike.baidu.com/item/螺髻山>

### 12. 泸沽湖四川景区（盐源县）
- **门票参考**：四川景区门票常见约 70 元左右，湖上项目另计
- **建议时长**：1-3 天
- **怎么玩**：看高原深水湖、清晨湖面、摩梭村落、环湖公路和慢节奏度假氛围。第一次去建议至少住一晚。
- **交通**：从西昌、攀枝花、丽江等地都可进入；四川段一般从盐源县方向到达
- **预约要点**：旺季民宿很抢手；订房时务必看清是四川段还是云南段；环湖切勿低估车程
- **来源链接**：
  - <https://baike.baidu.com/item/泸沽湖>

### 13. 草海（盐源县泸沽湖镇）
- **门票参考**：通常包含在泸沽湖大景区范围或作为开放式观景区域
- **建议时长**：1-2 小时
- **怎么玩**：草海是泸沽湖最适合看湿地栈道与晨雾意象的区域，适合清晨和傍晚前往。
- **交通**：位于泸沽湖四川段核心游览区域，环湖途中即可到达
- **预约要点**：早晚温差大，晨拍注意保暖；雨天木栈道湿滑
- **来源链接**：
  - <https://baike.baidu.com/item/泸沽湖>

### 14. 走婚桥（盐源县泸沽湖镇）
- **门票参考**：通常免费步行通过或含于景区公共区域
- **建议时长**：30 分钟-1 小时
- **怎么玩**：这是泸沽湖文化辨识度最高的打卡点之一，适合串联草海与摩梭文化理解，但别只停留在“拍照桥”。
- **交通**：位于草海边，步行可达
- **预约要点**：游客密度很高，想拍空镜建议清晨去；尊重当地居民生活空间
- **来源链接**：
  - <https://baike.baidu.com/item/泸沽湖>

### 15. 摩梭村落文化体验点（盐源县泸沽湖周边）
- **门票参考**：多为开放式村落，家访、表演、博物馆体验另计
- **建议时长**：1-3 小时
- **怎么玩**：看摩梭民居、服饰、家访文化与湖畔生活，重点是理解地方文化，而不是猎奇式打卡。
- **交通**：围绕泸沽湖四川岸线分布，自驾/包车最方便
- **预约要点**：参加家访或晚会前先确认是否正规、是否明码标价；尊重当地礼俗和拍照边界
- **来源链接**：
  - <https://baike.baidu.com/item/泸沽湖>

### 16. 会理古城（会理市）
- **门票参考**：古城街区免费，钟鼓楼等个别点位可能单独收费
- **建议时长**：3-5 小时
- **怎么玩**：看棋盘式街巷格局、钟鼓楼、科甲巷、西成巷与明清老街，属于四川被低估的人文古城之一。
- **交通**：可由西昌自驾或乘车前往，会理市区内步行体验较好
- **预约要点**：古城适合早晚逛；白天适合看街巷细部，夜里适合吃饭和散步
- **来源链接**：
  - <https://baike.baidu.com/item/会理古城>

### 17. 会理钟鼓楼（会理古城）
- **门票参考**：以现场公示为准，常见为低价或联票方式
- **建议时长**：30 分钟-1 小时
- **怎么玩**：会理古城核心地标，适合作为古城游览起点，帮助理解四街三关的城市格局。
- **交通**：古城步行即达
- **预约要点**：若开放登楼，留意具体开放时段；节庆与晚间灯光更有氛围
- **来源链接**：
  - <https://baike.baidu.com/item/会理古城>

### 18. 科甲巷（会理古城）
- **门票参考**：免费
- **建议时长**：30 分钟-1 小时
- **怎么玩**：看会理古城内最有传统生活质感的一段街巷，适合搭配西成巷、瀛洲公园一起慢走。
- **交通**：会理古城内步行可达
- **预约要点**：更适合慢慢看门楼、院落和街巷纹理，不适合“拍一张就走”式打卡
- **来源链接**：
  - <https://baike.baidu.com/item/会理古城>

### 19. 瀛洲公园（会理古城西门侧）
- **门票参考**：免费或低价开放
- **建议时长**：30 分钟-1 小时
- **怎么玩**：古城旁的园林小景，适合休息、补节奏，也能看古城生活感而非纯游客感。
- **交通**：位于古城附近，步行方便
- **预约要点**：通常无需预约，适合与古城半日游打包
- **来源链接**：
  - <https://baike.baidu.com/item/会理古城>

### 20. 马湖景区（雷波县）
- **门票参考**：大景区常见低门槛或分项目收费，游船、环湖点位另计
- **建议时长**：半天-1 天
- **怎么玩**：马湖是四川省内很有存在感的深水湖，湖形狭长、山体环绕，风格比泸沽湖更野、更安静，适合想去小众湖景的人。
- **交通**：位于雷波县黄琅镇附近，适合自驾或包车前往
- **预约要点**：当地接待成熟度不如西昌、泸沽湖，住宿和餐饮要提前确认；雨季山区道路需留意
- **来源链接**：
  - <https://baike.baidu.com/item/马湖>

### 21. 黄琅镇（雷波县，马湖门户）
- **门票参考**：免费
- **建议时长**：30 分钟-1 小时，更多作为中转与落脚点
- **怎么玩**：去马湖通常会经过黄琅镇，可作为补给、住宿和出发点，不是硬核景区，但对线路组织很关键。
- **交通**：雷波县方向公路进入
- **预约要点**：节假日住宿选择有限，旺季建议先订；夜间山路不建议盲开
- **来源链接**：
  - <https://baike.baidu.com/item/马湖>
  - <https://baike.baidu.com/item/凉山彝族自治州>

### 22. 彝海结盟纪念地（冕宁县）
- **门票参考**：通常免费开放或低价参观
- **建议时长**：1-2 小时
- **怎么玩**：这是凉山重要的红色历史点位，适合和冕宁线路同走，帮助理解凉山在近现代史中的位置。
- **交通**：由西昌前往冕宁较方便，适合自驾/包车
- **预约要点**：展馆类点位注意开放时段；团队或研学高峰日可能更拥挤
- **来源链接**：
  - <https://baike.baidu.com/item/凉山彝族自治州>

### 23. 灵山寺 / 灵山景区（冕宁县）
- **门票参考**：景区常见约 30-60 元区间，索道/摆渡另计（以现场为准）
- **建议时长**：2-4 小时
- **怎么玩**：适合寺院祈福、山林步道和俯瞰山景，和冕宁人文线搭配较自然。
- **交通**：从冕宁县城或西昌方向前往，自驾最省时
- **预约要点**：佛教活动日香客较多；山上气温和风力通常比县城强
- **来源链接**：
  - <https://baike.baidu.com/item/凉山彝族自治州>

### 24. 安哈彝寨仙人洞景区（西昌市安哈镇）
- **门票参考**：常见约 40-60 元区间
- **建议时长**：2-3 小时
- **怎么玩**：主打彝寨风情、峡谷与溶洞体验，距离西昌不算远，适合做西昌近郊半日游补充。
- **交通**：西昌出发自驾/打车较方便
- **预约要点**：景区体量不算巨大，更适合作为西昌周边补点；雨天洞口与山路注意防滑
- **来源链接**：
  - <https://baike.baidu.com/item/凉山彝族自治州>

### 25. 西昌卫星发射中心旅游区（西昌市近郊）
- **门票参考**：常见约 100 元上下，讲解和摆渡规则以官方为准
- **建议时长**：2-4 小时

- **交通**：通常需按旅游区规定前往，建议提前查官方开放安排
- **预约要点**：预约制色彩较强，遇到任务期、管制期或临时安排可能关闭或调整参观范围

### 26. 西昌民族风情园 / 火把广场片区（西昌市）

- **建议时长**：1-2 小时
- **怎么玩**：如果刚好赶上火把节前后或民族演出活动，这里很值得逛；平时更适合作为西昌夜游补充点。
- **交通**：西昌市区内方便抵达
- **预约要点**：节庆活动需关注当地文旅公告，人流量会非常大

## 美食

### 凉山值得专门吃的
- **西昌火盆烧烤**：凉山最出圈的美食名片，五花肉、小肠、土豆、包浆豆腐、牛肉都值得点；
- **坨坨肉**：彝族传统大块肉食，适合多人分食；
- **苦荞系列**：苦荞粑、苦荞茶、苦荞凉粉，属于凉山地方性很强的食材体系；
- **砣砣鸡/土鸡汤锅**：很多西昌和会理馆子都做得稳；
- **酸菜洋芋汤、荞面、包谷饭**：更接近日常彝家风味；
- **醉虾、湖鲜、烧鱼**：西昌邛海与泸沽湖沿线餐馆常见；
- **会理铜火锅**：会理古城非常值得专门安排一顿；
- **会理鸡火丝饵块、撒莲凉粉、羊肉米线**：会理本地味道明显；
- **盐源苹果及果脯**：季节对了可以顺带买；
- **雷波腊肉、马湖鱼**：适合在马湖线尝鲜。

### 用餐建议
- 西昌夜宵和烧烤很强，安排一顿夜宵基本是标配；
- 泸沽湖景区餐饮价格普遍高于西昌市区，湖景餐厅适合为环境买单；
- 彝餐不少店分量偏大，人少别点过量；
- 火盆烧烤偏重口，带老人或小孩时记得搭配清淡菜和汤。

## 住宿

### 西昌怎么住
- **邛海边**：适合度假、看日落、慢节奏，价格通常更高；
- **建昌古城/老城区**：适合步行逛城、吃夜宵、交通方便；
- **航天大道/市中心商圈**：性价比高，适合作为中转住宿。

### 泸沽湖怎么住
- **四川段临湖民宿**：安静、适合看湖，但旺季价格高；
- **草海附近**：适合次日一早拍晨景；
- **订房前一定核对省份和具体村落**，别以为都叫泸沽湖就无差别。

### 会理怎么住
- 古城内民宿氛围最好，适合夜游；
- 自驾可住古城外围，停车更轻松。

### 雷波/马湖怎么住
- 以实用型住宿为主，对住宿品质要求高的话不建议临时找；
- 山区夜间到店不稳定，最好白天抵达。

## 交通

### 对外交通
- **飞机到西昌**：效率最高，适合 2-3 天短假；
- **高铁/火车到西昌**：性价比高；
- **自驾入凉山**：灵活，但要接受州内长车程现实。

### 州内交通逻辑
- **西昌是最强中转枢纽**：去普格、会理、冕宁都适合从西昌出发；
- **泸沽湖单独成线**：如果主要去泸沽湖，最好不要再硬塞太多西昌周边点；
- **雷波马湖偏小众**：公共交通不如成熟热门线方便，自驾体验更好；
- **木里、深山区**：不建议第一次来就临时冲。

### 自驾提醒
- 山路弯多，凉山不是适合“连续高强度赶夜路”的地方；
- 雨季要关注落石、塌方、临时管制；
- 湖区和古城周边节假日停车压力很明显。

## 避坑

1. **不要把四川省凉山彝族自治州当成“西昌一个城市景点合集”**。凉山州很大，跨区移动成本高。
2. **泸沽湖订房前必须确认是四川段还是云南段**。很多人行程混乱就卡在这一步。
3. **别把螺髻山当天气无所谓的普通山景区**。天气差时体验会断崖式下降，索道和云海观景都受影响。
4. **西昌火盆烧烤别无脑点太多**。分量很实在，而且越烤越容易加单失控。
5. **邛海更适合慢游，不适合“10 分钟打卡完成”**。如果只去一个点拍照，很难感受到西昌的舒服。
6. **节庆别临时订房**。火把节、暑期、国庆、寒暑假，西昌和泸沽湖住宿价格波动明显。
7. **不要低估紫外线**。西昌、泸沽湖、螺髻山都晒，阴天也会晒。
8. **古城类景点更适合早晚**。中午暴晒时会误判它“没意思”。
9. **雷波马湖和小众线不要套用热门景区服务预期**。要接受它更原生态、也更不标准化。
10. **去民族村寨先尊重生活边界**。不要把当地居民当“免费民俗背景板”。
