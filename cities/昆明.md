## 城市概览
昆明是很多人进入云南的第一站，也是最适合“先落地缓冲，再慢慢展开云南旅程”的城市。它最大的优势不是某一个超级单点，而是整体体验均衡：气候友好、城市节奏松、吃住行门槛低，既能轻松逛老城、公园、博物馆，也能在 1 天内切到石林、九乡、西山龙门、滇池观鸥这些更有辨识度的地貌和景观。

如果你第一次来云南，昆明很适合做首站；如果你已经去过大理、丽江、香格里拉，再回头看昆明，也会发现它真正的优点在于“适合住、适合逛、适合收放节奏”。正确打开方式不是暴力刷点，而是把市区烟火、滇池线、石林/九乡分开安排，旅行质量会明显更高。

## 行前速览
- 建议旅行天数：2–4 天
- 适合人群：第一次来云南、情侣周末、家庭亲子、长辈慢游、摄影和美食玩家
- 最佳季节：
  - 11 月—次年 3 月：观鸥季，体感舒服
  - 2—3 月：圆通山樱花季，拍照氛围很好
  - 5—10 月：绿意最盛，但午后阵雨概率较高
- 气候特点：四季温差整体不极端，但早晚温差常比想象中大
- 海拔与体感：昆明海拔较高，多数人问题不大，但初到别安排太满
- 防晒提醒：紫外线偏强，阴天也建议防晒
- 到达方式：
  - 飞机：昆明长水国际机场
  - 高铁：昆明南站为主
  - 火车：昆明站离老城区更近
- 市内交通：地铁覆盖主城区不错，去滇池、西山、古镇、部分公园常需配合公交或打车
- 预约原则：热门博物馆、讲武堂、部分景区建议提前查看官方公众号或票务页
- 总体花费：昆明市区游览成本不高，但石林、九乡、西山索道等项目叠加后预算会上升

## 1-3天行程

### 1 天版：第一次到昆明，先感受城市气质
- 上午：翠湖公园 → 云南陆军讲武堂历史博物馆
- 中午：文林街或翠湖周边吃小锅米线、烤豆腐
- 下午：云南省博物馆 / 昆明老街
- 傍晚：金马碧鸡坊 → 南屏街 → 南强街夜逛
- 晚上：想看城市夜生活，可追加斗南花市夜场

### 2 天版：市区精华 + 滇池西山线
- D1：翠湖公园 → 讲武堂 → 昆明老街 → 金马碧鸡坊
- D2：西山森林公园 → 龙门景区 → 云南民族村 → 海埂大坝看日落

### 3 天版：加入昆明王牌自然景观
- D1：翠湖片区 + 老街 + 省博
- D2：石林风景区一日游
- D3：西山龙门 + 海埂/民族村

### 3 天进阶版：更偏地貌控
- D1：市区慢逛（翠湖、讲武堂、老街）
- D2：石林风景区
- D3：九乡风景区

- 门票参考：免费
- 建议时长：1–2.5 小时
- 玩法：适合清晨或傍晚慢走，看湖景、水鸟、城市日常；可与讲武堂、云南大学、文林街串联成半天轻松线。
- 交通：地铁或公交到翠湖片区后步行可达，打车定位“翠湖公园”也方便。
- 预约要点：通常无需预约；开放时间以现场/官方公告为准。

## 美食
昆明适合吃的，不只是“过桥米线”这一个答案。真正更容易吃舒服的，往往是更接近日常的本地口味：小锅米线、豆花米线、烧饵块、官渡粑粑、汽锅鸡、烤豆腐、木瓜水、菌子类菜肴、鲜花饼，都比只盯着游客榜单更稳。

### 推荐优先吃
- 小锅米线：更像昆明人日常，高概率比你想象中更好吃
- 过桥米线：第一次来仍值得试，但别把期待拉太满
- 豆花米线：更清爽，适合早餐或连续吃重口后换口味
- 烧饵块：很适合作为早餐或夜宵
- 烤豆腐：街边高频小吃，适合边走边吃
- 官渡粑粑：去官渡古镇时顺路试最合适
- 汽锅鸡：更适合 2 人以上分享
- 菌子火锅/菌子炒饭：季节性强，建议选择正规口碑店
- 木瓜水、玫瑰冰粉、鲜花饼：适合加餐和伴手礼

### 觅食区域
- 翠湖 / 文林街：学生感、咖啡馆、小吃店都比较集中
- 老街 / 南屏街 / 南强街：第一次来最省事，选择多
- 篆新农贸市场：想高效率感受本地饮食，这里很值得去
- 官渡古镇：适合顺路吃，不建议单独为吃专程跑一趟

### 点单建议
- 两人以上再冲汽锅鸡、菌子锅更划算
- 早餐尽量留给米线和饵块，不要浪费在全国连锁标准早餐上
- 对“野生菌”保持敬畏，来源不明、价格异常便宜的店谨慎选择

## 住宿
昆明住宿核心不是“住最贵”，而是住在适合你动线的位置。

### 翠湖 / 文林街片区
- 适合：第一次来、喜欢步行和城市气质的人
- 优点：去翠湖、讲武堂、云南大学、圆通片区方便
- 缺点：部分老酒店设施较旧

### 南屏街 / 昆明老街 / 金马碧鸡坊商圈
- 适合：第一次来、想吃喝购物都方便的人
- 优点：餐饮密度高、夜间活动方便、出行容错率高
- 缺点：更热闹，临街房间可能偏吵

### 滇池 / 海埂片区
- 适合：重点玩西山、民族村、海埂和滇池线的人
- 优点：景观和酒店新度往往更好
- 缺点：离老城烟火气略远

### 昆明南站 / 呈贡片区
- 适合：第二天一早要高铁转大理、丽江、版纳的人
- 优点：赶车方便、酒店更新
- 缺点：如果主玩老城和滇池，每天通勤时间更长

### 订房建议
- 第一次来昆明，优先考虑翠湖片区或南屏街/老街片区
- 观鸥季、樱花季、长假和大型会展期间，热门酒店价格会明显上涨
- 如果只是短住，不必为“滇池景观房”支付过高溢价

## 交通

### 大交通
- 昆明长水国际机场：云南最核心空中门户
- 昆明南站：高铁枢纽，去省内热门城市很方便
- 昆明站：离老城区近，部分车次依旧很实用

### 市内交通
- 地铁：适合主城区移动，稳定且便宜
- 公交：覆盖广，但游客使用效率一般
- 打车 / 网约车：在昆明非常实用，尤其景点间转场时
- 自驾 / 包车：如果要去石林、九乡、抚仙湖等近郊，舒适度会明显更高

### 常见出行判断
- 只玩市区：地铁 + 打车足够
- 西山 / 海埂 / 民族村：优先打车或地铁加短打车
- 石林 / 九乡：优先一日游、包车或高铁转接驳
- 斗南花市：地铁可达，夜间返程也比较友好

## 避坑
- 不要把昆明当成“每个单点都震撼”的目的地，它强在综合体验和组合线路。
- 石林、九乡、西山步行量都不小，鞋不舒服会直接影响整天状态。
- 昆明紫外线很强，别被凉爽体感骗了，防晒必须做。
- 观鸥季去海埂，别抱着“到了随便拍都没人”的幻想，热门时段人会很多。
- 不要在节假日把西山、民族村、海埂、老街全塞同一天，转场会很累。
- 菌子类餐饮不要乱试低价猎奇店，正规店更重要。
- 官渡古镇、部分老街区商业化不低，建议抱着“顺路逛逛”的心态。
- 博物馆、讲武堂等别默认现场一定能进，周末和节假日先看预约更稳。
- 昆明早晚温差常被低估，中午短袖、晚上发抖是典型新手错误。
