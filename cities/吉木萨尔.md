# 吉木萨尔

- 更新时间：2026-03-16
- 推荐指数（整体）：⭐⭐⭐⭐☆（非常适合新疆东天山北麓的人文+公路+荒野混合线；如果主人只追求成熟大景区舒适游，优先级不如阜康天池或奇台江布拉克；但如果想把丝路北道、北庭故城、车师古道和东线公路逻辑真正看懂，吉木萨尔反而是昌吉州最值得深挖的县城之一）

## 城市概览

**吉木萨尔**位于**新疆昌吉回族自治州东部、天山北麓、准噶尔盆地南缘**，是一个非常容易被游客低估的地方。很多人路过这里只会记住两个标签：**北庭故城**和**五彩湾**。但如果真正把地图摊开，就会发现它的价值远不止“顺手看个遗址”。

它真正的旅行含金量，在于下面四件事同时成立：

1. **这里是北庭故城所在地**：也就是唐代北庭都护府、后续高昌回鹘夏都的重要空间载体，属于丝绸之路“长安—天山廊道的路网”世界遗产点之一；
2. **这里能把“北庭政治中心”与“佛教艺术传播”一起看懂**：北庭故城不是孤零零一圈土城墙，旁边的**北庭西大寺/高昌回鹘佛寺遗址**，就是很多人口中的“北庭千佛洞”“吉木萨尔千佛洞”认知来源；
3. **这里卡在天山北麓东线公路节点上**：向西能接**阜康—天山天池**，向东能接**奇台—江布拉克—木垒**，向南还能延伸到**车师古道**这一条真正贯通天山南北的历史通道；
4. **这里是很实用的县城补给点**：住宿、加油、吃饭、采购、修整都比在古道口、五彩湾或小乡镇更稳，适合做 1—2 晚基地。

如果主人第一次来吉木萨尔，最容易犯的错，是把它理解成“一个只有遗址、半天就看完的地方”。其实更合理的理解应该是：

- **北庭故城 = 历史骨架**
- **北庭西大寺/千佛洞认知 = 宗教与艺术层次**
- **车师古道 = 天山南北交通逻辑**
- **五彩湾 = 荒漠地貌补层次**
- **县城补给 = 全线落地执行的后勤中枢**

一句话结论：**吉木萨尔不是单点打卡型目的地，而是“读懂昌吉州东线”和“把阜康—吉木萨尔—奇台—天池东线串顺”的关键节点城市。**

## 行前速览

- **城市定位**：昌吉州东部县城型枢纽；北庭故城所在地；车师古道北口补给地；五彩湾与东天山历史公路线的重要中转点。
- **最适合谁**：
  - 对丝路史、唐代西域治理、高昌回鹘、佛教遗址有兴趣的人；
  - 喜欢自驾、公路线、荒野与县城混合旅行的人；
  - 想把**阜康—吉木萨尔—奇台**串成一条顺逻辑东线的人；
  - 不满足于只看天池、还想看新疆“历史骨架”的旅行者。
- **不太适合谁**：
  - 只喜欢超级成熟景区和密集娱乐项目的人；
  - 不做功课就想看遗址的人；
  - 完全不自驾、也不愿包车的人。
- **最建议停留**：
  - **0.5 天**：只看北庭故城，偏赶；
  - **1 天**：北庭故城 + 北庭西大寺/北庭园片区 + 县城；
  - **2 天**：北庭人文主线 + 五彩湾/古海温泉 或 车师古道北口；
  - **3 天**：吉木萨尔为中枢，做**北庭 + 车师古道 + 奇台/江布拉克**或**北庭 + 五彩湾 + 阜康/天池东线联动**。
- **最佳季节**：
  - **4—6 月**：气温更友好，适合北庭遗址、古道北口和公路线；
  - **7—9 月**：车师古道、山口方向、五彩湾和东线公路最有执行价值，但中午暴晒明显；
  - **9—10 月**：东线自驾体感最好，连奇台、江布拉克也更顺；
  - **冬季**：县城可住，但五彩湾、古道、山区线更看天气与道路。
- **交通结论**：
  - **自驾/包车明显最优**；
  - 县城之间通达不算难，但景点分散，真正影响体验的是“景点最后 20—50 公里”。
- **补给结论**：
  - **吉木萨尔县城是东线非常稳的补给点**：吃饭、超市、加油、住宿都比五彩湾、古道口和乡镇靠谱；
  - 去车师古道、五彩湾前，建议在县城先补齐水、零食、防晒与现金/备用充电设备。
- **门票与预约结论**：
  - 北庭故城、五彩湾、温泉等点以现场和官方售票页为准；
  - 遗址类景点更建议关注**开放时间、讲解服务、是否需要区间车**，而不是只盯票价；
  - 车师古道不同段落开放、徒步、骑马、越野体验差异很大，出发前必须再次核实。
- **一句话路线逻辑**：
  - **纯人文**：北庭故城 → 北庭西大寺 → 县城博物馆/北庭园片区；
  - **人文 + 荒野**：北庭故城 → 五彩湾 → 古海温泉；
  - **人文 + 山口古道**：北庭故城 → 泉子街 → 车师古道北口；
  - **东线联动**：阜康/天池 → 吉木萨尔 → 奇台/江布拉克。

## 1—3 天行程

### 1 天快用版：北庭人文主线
**适合：** 第一次来吉木萨尔、只停留一天、对历史感兴趣的人  
**路线：** 吉木萨尔县城 → 北庭故城遗址 → 北庭西大寺 / 高昌回鹘佛寺遗址 → 北庭园/遗址公园展示区 → 回县城吃饭休息 → 晚上县城补给

- 这条线核心不是刷点，而是把“**北庭故城是政治军事中心，西大寺是宗教艺术中心**”这层关系看明白。
- 如果主人以前只在吐鲁番看过交河、高昌、柏孜克里克千佛洞，那到吉木萨尔看北庭，会更容易理解**天山南北两侧其实是一个连续的丝路系统**。
- 建议保留讲解或提前做历史功课，否则很容易只看到“土墙”和“保护棚”。

### 2 天经典版：北庭 + 五彩湾地貌线
**D1：北庭人文线**  
吉木萨尔县城 → 北庭故城 → 北庭西大寺 / 北庭园片区 → 县城住宿

**D2：北部荒漠线**  
县城出发 → 五彩湾风景区 → 古海温泉（可选）→ 傍晚返县城或转住奇台/五彩湾片区

- 这是吉木萨尔最实用的一版。第一天把历史层次看懂，第二天补一个完全不同的荒漠彩丘地貌，内容反差很大。
- 若主人不想赶路，D2 只保五彩湾也够；若旅伴想放松，可把古海温泉作为收尾。

### 3 天增强版：北庭 + 车师古道 + 奇台联动
**D1：吉木萨尔核心人文线**  
县城 → 北庭故城 → 北庭西大寺 → 北庭园片区 → 县城休息

**D2：车师古道北口认知线**  
县城 → 泉子街镇 → 车师古道景区/野狼谷国家森林公园可开放段 → 头道桥—六道桥方向择一浅体验 → 返县城或住泉子街/县城

**D3：东线继续向奇台推进**  
吉木萨尔 → 奇台县城 → 江布拉克景区 / 怪坡 / 半截沟镇 → 住奇台或返程

- 这是最能体现吉木萨尔价值的一版：**历史中心 + 古道山口 + 东线草原麦田**，层次完整。
- 如果主人是从乌鲁木齐或阜康一路东进，这版会比“来回折返”顺很多。

### 3 天公路版：阜康—吉木萨尔—奇台—天池东线联动
**适合：** 自驾、想把昌吉州东线串顺的人

**D1：阜康 / 天山天池**  
乌鲁木齐/昌吉 → 阜康 → 天山天池 → 住阜康或直接东进吉木萨尔

**D2：吉木萨尔人文线**  
吉木萨尔县城 → 北庭故城 → 北庭西大寺 → 县城补给 → 晚住吉木萨尔

**D3：奇台风景线**  
吉木萨尔 → 奇台 → 江布拉克 → 视时间前往木垒或返程

这条线的逻辑比很多人随手拼的路线强得多，因为它遵循的是：

- **阜康负责成熟自然大景**；
- **吉木萨尔负责丝路北道历史与古道转场认知**；
- **奇台负责东天山麦田草原风光**。

如果要一句话记忆：**不是“天池旁边顺手去个吉木萨尔”，而是“阜康—吉木萨尔—奇台本来就该是一条从西到东渐进展开的线”。**

### 1. 北庭故城遗址
- **门票参考：** 常见约 **30—40 元**，也可能存在阶段性免费、优惠票或联票政策。
- **建议时长：** 2—3 小时
- **玩法：** 吉木萨尔第一核心景点，没有之一。它不是普通古城遗址，而是从**汉代车师后国金满城**到**唐代北庭都护府/北庭大都护府**、再到**高昌回鹘夏都别失八里**的连续空间遗存。这里最值得看的不是“城墙漂不漂亮”，而是它在新疆史上的地位：天山北麓政治、军事、文化中心，中央王朝治理西域的重要节点。
- **交通：** 位于县城北偏东约 12 公里，适合自驾、包车或从县城打车前往。
- **预约要点：** 遗址面积大、遮阴少，夏季务必早去或傍晚去；如有讲解服务，强烈建议用讲解，否则很容易看不出门道。

### 2. 北庭西大寺遗址（高昌回鹘佛寺遗址）
- **门票参考：** 常与北庭片区联动参观，常见为片区票务体系内或单独低门槛收费，具体以现场为准。
- **建议时长：** 1—2 小时
- **玩法：** 这是理解“吉木萨尔千佛洞”最关键的一站。严格说，它不是吐鲁番那种典型崖壁密集石窟群，但因遗址内有佛窟、壁画、塑像、题记与回鹘佛教艺术遗存，很多人会把它放进“千佛洞/佛寺遗址”认知体系里。它和北庭故城必须连看：前者讲政治中心，后者讲宗教艺术传播。
- **交通：** 位于北庭故城西侧不远处，通常与北庭故城同线安排最顺。
- **预约要点：** 保护棚、保护措施、开放范围可能影响观感；它更适合“看知识点”，不适合期待视觉冲击型石窟盛景。
- **来源链接：**

### 7. 车师古道景区 / 车师古道国家森林公园
- **门票参考：** 常见有基础门票或景区管理收费，价格多为几十元级；徒步、骑马、越野等项目另计。
- **建议时长：** 半天—1 天
- **玩法：** 这是吉木萨尔除北庭外最值得深挖的点。车师古道是连接**吉木萨尔北庭**与**吐鲁番交河故城**的天山南北捷径，历史上是车师前后国、唐代北庭与安西之间的重要通道。今天来这里，重点不是“景区打卡”，而是理解**天山并不是历史上的绝对屏障，而是被古道穿透的交通山脉**。
- **交通：** 位于泉子街镇方向，自驾最方便；无车会很被动。
- **预约要点：** 不同季节、不同开放段落差异很大；出发前必须确认道路、天气和景区是否开放到你想去的那一段。

### 10. 琼达坂（冰达坂）方向高山古道视角点
- **门票参考：** 通常属于古道深度体验范畴，是否收费取决于进入方式和服务项目。
- **建议时长：** 半天—1 天
- **玩法：** 适合真正想理解“翻越天山”意味着什么的人。这里代表车师古道穿越的高点，是古道纵深感最强的认知位。
- **交通：** 不建议普通游客贸然自行深穿；更适合成熟徒步/越野团队、熟路向导或明确开放的线路组织。
- **预约要点：** 海拔、天气、道路、安全强相关；普通旅行者把这里当“概念认知点”即可，不必硬冲。
- **来源链接：**
  - <https://baike.baidu.com/item/车师古道>

### 11. 泉子街镇
- **门票参考：** 免费（镇区）
- **建议时长：** 0.5—1 小时
- **玩法：** 它不是传统意义上的旅游景点，但对执行层面非常重要。这里是车师古道北口的重要集散节点，也是理解古道从县城向山里过渡的现实地理支点。
- **交通：** 从吉木萨尔县城自驾前往方便。
- **预约要点：** 适合作为古道前的补给和中转，不适合作为“专程来玩”的目的地本身。
- **来源链接：**
  - <https://baike.baidu.com/item/车师古道>

### 12. 五彩湾风景区
- **门票参考：** 常见约 **30—60 元**，不同观景区和组合票可能浮动。
- **建议时长：** 2—4 小时
- **玩法：** 吉木萨尔北部荒漠彩丘代表作。和北庭故城的历史厚重感完全不同，这里主要看彩色丘陵、戈壁、风蚀地貌和傍晚光线下的层次变化。适合给主人做“人文之后补地貌”的第二刀。
- **交通：** 位于县境北部，自驾最方便，路程不要低估。
- **预约要点：** 正午拍照容易发白，最好在傍晚去；风沙大、阴天或灰霾时表现会打折。
- **来源链接：**
  - <https://baike.baidu.com/item/五彩湾风景区>
  - <https://you.ctrip.com/sight/jimsar1446373.html>

### 13. 五彩湾古海温泉
- **门票参考：** 单次泡汤常见几十元到百元以上，住宿套餐波动更大。
- **建议时长：** 2 小时—1 晚
- **玩法：** 如果主人想把五彩湾从“看一眼荒漠就走”升级成轻度假线，古海温泉很实用。尤其适合秋冬、长途自驾后做放松收尾。
- **交通：** 通常与五彩湾联动，自驾前往最便捷。
- **预约要点：** 周末和节假日建议提前看房态和票型；如果只想看景，不一定非得住温泉区。
- **来源链接：**
  - <https://you.ctrip.com/sight/jimsar1446373/1836456.html>
  - <https://baike.baidu.com/item/五彩湾风景区>

### 14. 吉木萨尔县博物馆 / 地方历史文化展示空间
- **门票参考：** 常见为免费或低门槛开放，以当地公告为准。
- **建议时长：** 1—1.5 小时
- **玩法：** 如果主人想把北庭故城看得更扎实，县级博物馆或历史陈列空间是值得去的。尤其适合看完遗址后回头补文物、时间线和地方考古背景。
- **交通：** 县城内。
- **预约要点：** 开放时间可能比大城市馆更保守，午间和闭馆日要提前确认。
- **来源链接：**
  - <https://www.jimsar.gov.cn/>
  - <https://www.cj.gov.cn/>

### 15. 吉木萨尔县城老街区 / 县城生活圈
- **门票参考：** 免费
- **建议时长：** 1—2 小时
- **玩法：** 这也不是“硬核景点”，但非常值得写进攻略。吉木萨尔真正的旅行体验，不只是遗址和古道，还有县城这种天山北麓小城的生活质感：吃拌面、买补给、逛夜市、看本地通勤与农牧县城节奏。
- **交通：** 县城步行 / 打车。
- **预约要点：** 更适合傍晚和晚上；如果第二天跑古道或五彩湾，晚上就别熬太晚。
- **来源链接：**
  - <https://www.jimsar.gov.cn/>

### 16. 北庭镇
- **门票参考：** 免费（镇区）
- **建议时长：** 0.5—1 小时
- **玩法：** 适合把北庭从“景区名词”落回真实地理。很多人看完遗址就走，其实停一下周边聚落环境，更容易理解为什么这里能成为天山北麓长期中心。
- **交通：** 去北庭故城途中顺路。
- **预约要点：** 不值得专程停很久，但顺路很有价值。
- **来源链接：**
  - <https://baike.baidu.com/item/北庭故城遗址>

### 17. 大龙口 / 大龙沟方向山口景观带
- **门票参考：** 视进入段落和具体经营点而定，部分为自然公路沿线观景性质。
- **建议时长：** 1—3 小时
- **玩法：** 这是把“车师古道不是抽象名词”具体化的重要区域。沿北口山谷向南看，可以感受到古道进入天山山口前的地形收束感。
- **交通：** 建议自驾前往，结合泉子街与古道线安排。
- **预约要点：** 山区天气变化快；不要在路况不明时盲目深进。
- **来源链接：**
  - <https://baike.baidu.com/item/车师古道>

### 18. 东线公路观景线（吉木萨尔—奇台方向）
- **门票参考：** 免费（公路观景）
- **建议时长：** 1—3 小时，可作为转场体验
- **玩法：** 这条线本身就值得写进攻略，因为它决定了吉木萨尔的真正联动价值。一路能看到县城、农田、天山北麓地貌、东线聚落过渡，是把吉木萨尔接入奇台、江布拉克的重要纽带。
- **交通：** 自驾最佳。
- **预约要点：** 这是功能型观景线，不是封闭景区；重点在转场效率和节奏，不是每个点都下车。
- **来源链接：**
  - <https://www.cj.gov.cn/>
  - <https://www.jimsar.gov.cn/>

## 美食

吉木萨尔的美食重点，不是“必须有全国闻名的一道菜”，而是**县城作为新疆东线补给点时，吃得够稳、够实在、够恢复体力**。

### 1. 新疆拌面 / 过油肉拌面
- 县城最稳的主食补给之一，跑完北庭或准备第二天进古道、去五彩湾前都很好用。
- 建议选本地人多、翻台快的馆子，通常不容易踩空。

### 2. 烤羊肉串 / 烤包子
- 新疆县城基础盘。适合晚上回县城简单补能量。
- 烤包子适合当第二天赶路前的顺手早餐或加餐。

### 3. 丸子汤 / 牛肉汤类早餐
- 如果第二天要早起跑车师古道或五彩湾，热汤早餐比纯面包咖啡更扛用。

### 4. 抓饭
- 中午不想吃太撑但又要补体力，抓饭是很好的平衡项。

### 5. 大盘鸡 / 椒麻鸡
- 适合多人同行晚餐，尤其在吉木萨尔—奇台东线转场夜里，大家一起吃比较舒服。

### 6. 馕、酸奶、奶茶、奶制品
- 做公路线时非常实用。可以在县城顺手买，放车上当备用补给。

### 7. 县城夜市 / 烧烤摊
- 适合体验本地小城夜生活。重点不是“全国级美食地标”，而是节奏轻松、恢复状态快。

**吃饭建议：**
- **跑遗址当天**：午饭别吃太拖，防止下午暴晒疲劳；
- **跑古道当天**：早饭一定吃热的；
- **去五彩湾当天**：多带水和水果，别指望景区内补给质量；
- **县城补给逻辑**：晚饭时顺手把第二天水、零食、纸巾、防晒一起买了。

## 住宿

吉木萨尔住宿最关键的判断，不是“哪家最豪华”，而是**你第二天想怎么跑线**。

### 1. 住吉木萨尔县城（最推荐）
**适合：** 北庭故城、车师古道、五彩湾都想沾的人。  
**优点：**
- 吃饭、加油、补给、停车最稳；
- 去北庭近，去古道和五彩湾也能接受；
- 适合作为 1—2 晚基地，不用频繁换酒店。  
**缺点：**
- 不是景区型度假住法；
- 酒店整体成熟度一般会低于乌鲁木齐、昌吉、阜康成熟商圈。

### 2. 住北庭片区周边（只适合特别偏人文的人）
**适合：** 只想专注北庭、不想往返的人。  
**优点：** 离遗址近。  
**缺点：** 配套远不如县城成熟，选择面窄；大多数人其实没必要住这里。

### 3. 住五彩湾 / 古海温泉片区
**适合：** 想把第二天变成“荒漠 + 泡汤”轻度假玩法的人。  
**优点：** 看完日落直接泡温泉，节奏轻松。  
**缺点：** 第二天若还要回北庭或去古道，会折返；不适合第一次来全都想顾上的人。

### 4. 住奇台县城（适合继续东进）
**适合：** 吉木萨尔只是中途站，第二天重点玩江布拉克或继续木垒。  
**优点：** 为奇台线做准备更顺。  
**缺点：** 如果主人第二天还想补吉木萨尔内容，就会回拉。

### 5. 住阜康（适合把天池放前一天）
**适合：** 前一日重点玩天山天池，第二天再东进吉木萨尔。  
**优点：** 天池执行更轻松。  
**缺点：** 这样会把吉木萨尔压缩成第二天主任务，需要早出发。

**一句话建议：**
- **第一次来吉木萨尔，默认住县城。**
- 只有明确做“温泉轻度假”或“继续奇台东进”时，再考虑改住五彩湾或奇台。

## 交通

## 外部到达

### 1. 乌鲁木齐 / 昌吉出发
- 自驾最稳，整体路网成熟；
- 适合做成**乌鲁木齐/昌吉 → 阜康 → 吉木萨尔 → 奇台**的西向东推进线；
- 如果主人从乌鲁木齐当天直奔吉木萨尔，通常也能做，但少了阜康缓冲层。

### 2. 阜康过来
- 这是最实用的成熟联动方向之一；
- 很多人玩完天池后直接回乌鲁木齐，其实如果还有 2—3 天，继续东进吉木萨尔会更完整。

### 3. 奇台过来
- 东线倒着玩也可以：奇台 / 江布拉克 → 吉木萨尔 → 阜康 / 乌鲁木齐；
- 但如果主人更重视“从成熟自然大景逐步进入人文深层”，还是推荐先阜康后吉木萨尔再奇台。

## 县内交通

### 1. 自驾 / 包车
- **最推荐，没有之一。**
- 原因不是县城不好到，而是**北庭、车师古道、五彩湾彼此分散，且都不是标准化城市步行景点**。

### 2. 打车
- 县城到北庭故城这种短距离点位可以尝试打车；
- 但去五彩湾、古道口、泉子街深一点区域，返程不确定性明显增大。

### 3. 公共交通
- 县城之间通达不是完全没有，但旅游执行体验会打折很多；
- 特别是古道和五彩湾，**没有车几乎玩不舒服**。

## 关键公路线节点理解

这部分是吉木萨尔攻略最该补但最常被省略的：

### 1. 乌鲁木齐 / 昌吉—阜康—吉木萨尔
- 这是昌吉州东线最稳的一段西向东推进线；
- 适合把**州府/机场补给**、**天池自然景观**、**北庭人文遗址**串成连续逻辑。

### 2. 吉木萨尔—奇台
- 这不是简单县城跳转，而是从**历史中心**进入**草原麦田与东天山风景线**的过渡段；
- 很适合做次日东进。

### 3. 吉木萨尔—泉子街—车师古道北口
- 这是从“县城生活世界”切换到“山口古道世界”的路线；
- 一定要在县城先完成补给，再往里走。

### 4. 吉木萨尔—五彩湾
- 属于北向荒漠线；
- 看着地图不一定觉得远，但实际是典型的“出去容易、回程疲劳会叠加”的路，别把它跟古道硬塞同一天。

## 阜康—奇台—天池东线联动逻辑

主人如果想把这条线玩得顺，建议直接记下面的原则：

1. **阜康解决“新疆经典自然大景”**：天山天池成熟、稳定、第一次最容易获得正反馈；
2. **吉木萨尔解决“东线历史骨架”**：北庭故城、北庭西大寺、车师古道认知都在这里；
3. **奇台解决“东线风景抬升”**：江布拉克把整个路线从遗址和戈壁，推向草原、麦田、林地与天山北坡风光；
4. **不要把天池和北庭挤在同一半天里**：地图看着都是昌吉州，现实体验完全不是一回事；
5. **吉木萨尔最适合作为承上启下的一晚**：前接阜康，后接奇台，这是它在整个东线里最强的价值。

## 避坑

### 1. 不要把北庭故城看成“拍两张城墙就结束”的点
北庭真正的价值在历史地位，不在视觉奇观。没做功课、没讲解，体验会严重打折。

### 2. 不要把“千佛洞”理解成吐鲁番柏孜克里克那种标准石窟群体验
吉木萨尔这条线更准确的理解是：**北庭西大寺/高昌回鹘佛寺遗址构成了当地佛教遗存与“千佛洞认知”层次**。它的价值偏考古、宗教艺术和历史连接，不是纯视觉型石窟打卡。

### 3. 不要把车师古道当普通公园步道
它本质上是古道、山口和探险线路，不同人能走到哪里差很多。普通游客更适合做北口浅体验，不要为了“穿越”硬上。

### 4. 不要把北庭、车师古道、五彩湾、奇台塞进同一天
这是典型地图错觉。每条线都值得单独留时间，硬塞只会一路赶车，谁都玩不好。

### 5. 不要忽略县城补给价值
吉木萨尔县城不是“只是睡一晚的地方”，它是整个东线非常关键的恢复站和补给站。古道、五彩湾之前不在县城补齐东西，后面会很被动。

### 6. 夏季别在正午暴晒时长时间逛遗址
北庭故城遮阴很少，风大、干燥、晒伤快。帽子、防晒、太阳镜和水别省。

### 7. 五彩湾不是任何天气都出片
阴天、灰霾、正午硬光都可能让彩丘“失色”。最好把它放到傍晚。

### 8. 不要对县城夜生活抱大城市期待
吉木萨尔的强项是实用，不是花哨。把它当“休整好、吃好、第二天更能打”的地方，会更满意。

### 9. 冬季和雨后更要看道路与开放状态
特别是古道、山口方向、五彩湾北部线，天气与路况对体验影响很大。

### 10. 如果完全不自驾，行程一定做减法
无车情况下，建议只保**北庭故城 + 北庭西大寺 + 县城**这一套，不要再贪古道和五彩湾。
