# 和田旅游攻略（快速可用版）

- 更新时间：2026-03-15
- 适用范围：新疆维吾尔自治区和田地区，覆盖和田市、和田县、墨玉县、洛浦县、于田县、民丰县、皮山县等常见旅行范围
- 推荐指数：⭐4.4（更适合喜欢丝路文明、和田玉、沙漠绿洲、古遗址、非遗与南疆生活感的人）

---

## 城市概览

和田地区位于新疆最南部、昆仑山北麓、塔克拉玛干沙漠南缘，古称“于阗”，是丝绸之路南道上的核心绿洲之一，也是和田玉文化、佛教东传遗存、绿洲农业与南疆民族生活方式交汇最鲜明的区域之一。这里不是靠一两个“超级网红景区”取胜的目的地，真正的魅力在于：**玉龙喀什河与喀拉喀什河、和田玉、古于阗历史、团城老街、湿地胡杨、沙漠公路、桑皮纸与艾德莱斯等非遗、县城巴扎与夜市烟火气**。

和田旅行通常有三种打开方式：

- **和田市核心圈**：博物馆、团城、夜市、玉石市场，适合 1 天快速了解和田。
- **和田—墨玉—洛浦近郊圈**：老城、湿地、沙漠、非遗工坊，可做 2 天轻深度行程。
- **于田—民丰—皮山东西延伸线**：更偏古遗址、胡杨、沙漠绿洲、尼雅文化和公路旅行，适合 3 天以上。

如果你想看的是一个真实、厚重、仍保留强烈地方纹理的南疆绿洲社会，和田很值得；如果你期待的是高密度成熟商业景区，和田会显得“慢”和“散”。

## 行前速览

### 最佳旅行时间

- **4-6 月**：温度相对友好，适合城市步行、湿地、公园、近郊遗址与县域串联。
- **9-10 月**：更推荐，白天不至于过分暴晒，适合胡杨、沙漠、公路和摄影。
- **7-8 月**：瓜果和夜市体验好，但白天炎热、紫外线很强，行程要避开正午。
- **冬季**：游客少、价格低，适合城市和县城慢游，但户外观感和舒适度一般。

### 建议玩几天

- **1 天**：和田博物馆 + 团城 + 夜市
- **2 天**：增加约特干故城、玉龙喀什河、墨玉或洛浦一线
- **3 天**：可以把和田近郊 + 洛浦/于田串起来
- **4-5 天**：适合延伸民丰尼雅文化、安迪尔遗址、沙漠公路与皮山方向

### 气候与穿衣

- 和田属于典型的**暖温带极端干旱荒漠气候**，日照很强、降水极少、空气干燥、昼夜温差大。
- 必带：**防晒帽、墨镜、防晒霜、润唇膏、保湿霜、补水壶、面巾或口罩**。
- 春秋建议带一件轻薄外套；夏季建议穿透气长袖；去沙地、遗址建议穿运动鞋或轻徒步鞋。

### 交通方式建议

- **只玩市区和近郊**：打车 / 网约车基本够用。
- **跨县串联**：优先自驾或包车，效率明显更高。
- **深度遗址 / 沙漠线**：不要临时起意往里走，最好找熟悉线路的司机。

### 适合谁去

- 丝路历史、佛教遗址、和田玉文化爱好者
- 喜欢真实地方感、慢节奏与县域旅行的人
- 摄影、自驾、公路旅行玩家
- 想系统补齐南疆旅行版图的人

### 不太适合谁

- 只想刷高密度成熟景区的人
- 对干燥、风沙、长距离移动容忍度低的人
- 对县域景点票务和开放不稳定完全不能接受的人

## 1-3天行程

### 1 天标准版：和田初见

**和田博物馆 → 团城景区 → 玉石巴扎 / 玉文化街区 → 和田夜市**

这是第一次到和田最稳的路线。先用博物馆建立“于阗—丝路南道—和田玉”的理解框架，再去团城和玉石街区感受城市肌理，晚上把夜市收尾。

### 2 天进阶版：市区 + 近郊

**D1：和田博物馆 → 团城景区 → 和田夜市**  
**D2：玉龙喀什河观景 / 采玉体验点 → 约特干故城 → 墨玉老城景区 或 拉里昆国家湿地公园**

第二天把“玉、古城、湿地”三个和田代表性元素集中体验，适合第一次来但不想只看市区的人。

### 3 天完整体验版：市区 + 墨玉 / 洛浦 / 于田三选一

**D1：和田市区人文线**：和田博物馆 → 团城 → 夜市  
**D2：和田县 / 墨玉线**：玉龙喀什河 → 约特干故城 → 墨玉老城 → 桑皮纸体验点  
**D3：二选一**  
- **洛浦线**：洛浦县博物馆 → 玉龙湾国家沙漠公园 → 艾德莱斯 / 地毯工坊  
- **于田线**：于田玫瑰产业带（花期）→ 克里雅河下游胡杨林 → 于田县城返程

如果你有 4-5 天，可以继续东进民丰，看尼雅文化与沙漠古遗址；如果更喜欢西线人文，可改去皮山。

## 景点清单（18+）

> 说明：和田地区不少景点属于县域文旅项目、开放式点位、文保遗址或季节性景观，票务和开放信息存在变动。以下为公开资料与旅行平台可见信息整理出的**门票参考**，以景区现场公示和出发前复核为准。

### 1. 和田博物馆
- **门票参考**：通常免费，凭身份证入馆
- **建议时长**：1.5-2.5 小时
- **玩法**：看于阗历史、丝路南道文物、佛教遗存、和田玉文化，是理解和田最重要的入门站
- **交通**：位于和田市区，打车前往方便
- **预约要点**：注意周一闭馆或实名入馆要求，节假日建议早到
- **来源链接**：
  - https://baike.baidu.com/item/和田地区
  - https://www.xinjiang.gov.cn/

### 2. 团城景区（和田团城）
- **门票参考**：常见为免费开放，个别展陈或体验项目另收费
- **建议时长**：1-2 小时
- **玩法**：适合看和田老城改造后的街巷空间、民居外立面和市井生活感，拍照也很出片
- **交通**：和田市区内打车直达
- **预约要点**：晚上氛围更好；尊重居民生活，不要闯入院落或贴脸拍摄
- **来源链接**：
  - https://www.xinjiang.gov.cn/
  - https://baike.baidu.com/item/和田市

### 3. 和田夜市
- **门票参考**：免费进入，按实际餐饮消费计
- **建议时长**：1.5-2 小时
- **玩法**：吃烤包子、烤肉、抓饭、面肺子、酸奶、石榴汁等南疆代表性小吃
- **交通**：位于和田市区，晚上打车方便
- **预约要点**：适合晚饭时段去；下单前先问清分量和价格
- **来源链接**：
  - https://www.xinjiang.gov.cn/
  - https://baike.baidu.com/item/和田市

### 4. 约特干故城
- **门票参考**：常见参考约 ¥58-120，夜场 / 演艺联票可能更高
- **建议时长**：2-4 小时
- **玩法**：偏沉浸式文旅项目，主打古于阗文化场景、建筑氛围、夜游灯光和演艺，不是传统考古遗址公园
- **交通**：位于和田县方向，建议打车或包车前往
- **预约要点**：旺季夜场更值得；出发前确认营业日、演出时间、是否需实名购票
- **来源链接**：
  - https://www.trip.com/
  - https://you.ctrip.com/

### 5. 玉龙喀什河观景 / 采玉体验点
- **门票参考**：观景多免费；体验型采玉项目几十至数百元不等
- **建议时长**：1-2 小时
- **玩法**：感受“和田玉母亲河”意象，了解籽料、河床、冲刷与采玉文化，重在体验不是捡宝
- **交通**：和田市、和田县、洛浦县周边多可短途抵达，适合包车半日
- **预约要点**：不要相信“保证捡到好玉”式营销；把它当文化体验更稳
- **来源链接**：
  - https://baike.baidu.com/item/和田地区
  - https://baike.baidu.com/item/洛浦县

### 6. 和田玉文化街区 / 玉石巴扎
- **门票参考**：免费
- **建议时长**：1-2 小时
- **玩法**：看本地玉石交易氛围、籽料与成品工艺，适合把“和田玉”从概念拉回现实市场
- **交通**：和田市区内可打车前往
- **预约要点**：不懂玉就只逛不买；真要消费，务必走正规店铺、索要票据、谨慎高价交易
- **来源链接**：
  - https://baike.baidu.com/item/和田地区
  - https://www.xinjiang.gov.cn/

### 7. 墨玉老城景区
- **门票参考**：常见为免费或低价开放
- **建议时长**：1.5-2 小时
- **玩法**：看县城街巷、传统民居风格和比较朴素的生活感，适合慢逛而不是“打卡冲刺”
- **交通**：距和田市不远，打车或包车半日可达
- **预约要点**：适合与墨玉其他点位打包；配套较简，别抱成熟景区预期
- **来源链接**：
  - https://baike.baidu.com/item/墨玉县
  - https://www.xinjiang.gov.cn/

### 8. 拉里昆国家湿地公园
- **门票参考**：常见参考约 ¥20-40，观光车或园内项目另计
- **建议时长**：2-3 小时
- **玩法**：看沙漠边缘湿地、湖泊、芦苇、候鸟和荒漠中的水系反差，是和田自然类景点中比较好用的一处
- **交通**：位于墨玉县雅瓦乡、乌尔其乡一带，建议自驾或包车
- **预约要点**：春秋更舒服；夏季注意防晒，蚊虫季注意驱蚊
- **来源链接**：
  - https://baike.baidu.com/item/墨玉县
  - https://you.ctrip.com/

### 9. 桑皮纸传统制作技艺体验点（墨玉）
- **门票参考**：参观点多为免费或低消费体验，手作另收费
- **建议时长**：40-90 分钟
- **玩法**：看桑皮纸制作流程，是和田地区代表性的非遗体验之一，适合买点轻伴手礼
- **交通**：通常作为墨玉文化体验线路中的顺路点，建议包车安排
- **预约要点**：最好提前确认是否开门、当天是否有师傅演示
- **来源链接**：
  - https://baike.baidu.com/item/墨玉县
  - https://www.mct.gov.cn/

### 10. 麻扎塔格戍堡遗址
- **门票参考**：多为开放遗址或无统一票务，以现场为准
- **建议时长**：1-2 小时
- **玩法**：适合汉唐边防史、屯戍遗址和丝路考古爱好者，不适合把它当成熟景区游
- **交通**：位于墨玉县北部偏远地带，建议专业包车
- **预约要点**：偏远、配套弱，出发前确认路线、天气和返程时间
- **来源链接**：
  - https://baike.baidu.com/item/墨玉县
  - https://baike.baidu.com/item/和田地区

### 11. 库木拉巴特佛寺遗址
- **门票参考**：通常无标准商业票务，以文保开放情况为准
- **建议时长**：1-1.5 小时
- **玩法**：看和田佛教遗址残存信息，适合和于阗佛教传播史结合理解
- **交通**：位于墨玉县扎瓦镇西侧沙漠地带，建议包车前往
- **预约要点**：不要期待完整寺院式观光，重点是遗址价值；注意文保边界
- **来源链接**：
  - https://baike.baidu.com/item/墨玉县
  - https://www.mct.gov.cn/

### 12. 夏合勒克庄园
- **门票参考**：低价或免费，具体看现场管理情况
- **建议时长**：40-60 分钟
- **玩法**：看典型南疆乡土庄园与果园式聚落痕迹，适合对民居和地方社会史感兴趣的人
- **交通**：位于墨玉县扎瓦方向，可与佛寺遗址或老城顺路串联
- **预约要点**：点位较小众，更适合作为专题补充而非专程主行程
- **来源链接**：
  - https://baike.baidu.com/item/墨玉县
  - https://www.xinjiang.gov.cn/

### 13. 洛浦县博物馆
- **门票参考**：通常免费或低价开放
- **建议时长**：1-1.5 小时
- **玩法**：适合看地方文物、纺织、陶器、木器与县域历史，是理解洛浦与于阗腹心区关系的好补充
- **交通**：位于洛浦县城，可与玉龙湾国家沙漠公园同日安排
- **预约要点**：出发前确认开馆时间；县级博物馆节奏慢，适合安静细看
- **来源链接**：
  - https://baike.baidu.com/item/洛浦县
  - https://www.xinjiang.gov.cn/

### 14. 新疆洛浦玉龙湾国家沙漠公园
- **门票参考**：常见参考约 ¥30-60，滑沙、骑骆驼、沙漠摩托等项目另收费
- **建议时长**：2-3 小时
- **玩法**：可玩滑沙、骑骆驼、看沙海日落，是距离县城较近、可操作性较强的沙漠体验点
- **交通**：位于洛浦县多鲁镇方向，自驾或包车方便
- **预约要点**：项目型消费容易叠加，建议先问清总价；傍晚光线更好
- **来源链接**：
  - https://baike.baidu.com/item/洛浦县
  - https://piao.qunar.com/

### 15. 洛浦县丝路滑雪场
- **门票参考**：雪季常见参考约 ¥60-180，视雪具和套票而定
- **建议时长**：2-4 小时
- **玩法**：冬季到和田时的少见娱乐点，适合轻戏雪、初级滑雪体验
- **交通**：位于洛浦县工业园区方向，自驾 / 打车可达
- **预约要点**：强季节性，仅雪季有意义；出发前先核实是否开放
- **来源链接**：
  - https://baike.baidu.com/item/洛浦县
  - https://you.ctrip.com/

### 16. 艾德莱斯绸 / 地毯织造非遗体验点（洛浦）
- **门票参考**：参观多免费，购买成品另计
- **建议时长**：40-90 分钟
- **玩法**：看艾德莱斯绸织染、地毯织造和家庭式手工业，是理解南疆日常审美和手工体系的好入口
- **交通**：通常位于洛浦县城或周边工坊线路内，适合包车串联
- **预约要点**：如想拍制作过程，先征得同意；购买成品要比较工艺与价格
- **来源链接**：
  - https://baike.baidu.com/item/洛浦县
  - https://www.mct.gov.cn/

### 17. 于田玫瑰产业带 / 玫瑰花田（季节性）
- **门票参考**：多数开放花田免费或低价参观，体验项目另计
- **建议时长**：1-2 小时
- **玩法**：于田玫瑰是当地特色产业，花期可看采摘、晾晒、花茶和精油相关体验
- **交通**：位于于田县周边乡镇，建议包车前往
- **预约要点**：花期通常比“是否有景区”更重要；淡季不建议专程跑远
- **来源链接**：
  - https://baike.baidu.com/item/于田县
  - https://www.xinjiang.gov.cn/

### 18. 克里雅河下游胡杨林（于田方向）
- **门票参考**：多为开放观景或临时收费点，以现场为准
- **建议时长**：2-3 小时
- **玩法**：看荒漠胡杨、河道绿洲与戈壁的过渡景观，秋季摄影表现最好
- **交通**：位于于田县北部 / 下游区域，建议自驾或包车
- **预约要点**：秋季最优；远离城市，要提前准备补水和返程时间
- **来源链接**：
  - https://baike.baidu.com/item/于田县
  - https://baike.baidu.com/item/和田地区

### 19. 于田县博物馆 / 县域展陈点
- **门票参考**：通常免费或低价
- **建议时长**：1 小时
- **玩法**：补充扜弥—于阗—现代于田的历史脉络，适合作为东线的室内缓冲站
- **交通**：于田县城内可达
- **预约要点**：开馆节奏较县城化，出发前确认时间；适合顺路进，不必单独远赴
- **来源链接**：
  - https://baike.baidu.com/item/于田县
  - https://www.xinjiang.gov.cn/

### 20. 民丰县尼雅博物馆 / 尼雅文化展示点
- **门票参考**：通常免费或低价
- **建议时长**：1-1.5 小时
- **玩法**：如果你对精绝古国、尼雅遗址、丝路古国史感兴趣，这是民丰最重要的理解入口
- **交通**：位于民丰县城，适合东线自驾停留
- **预约要点**：很多人直奔遗址，但先看馆更容易理解尼雅文化体系
- **来源链接**：
  - https://baike.baidu.com/item/民丰县
  - https://www.xinjiang.gov.cn/

### 21. 安迪尔遗址
- **门票参考**：多无标准票务，以文保管理和现场状况为准
- **建议时长**：2-3 小时
- **玩法**：国家重点文物保护单位方向的重要遗存，适合古国遗址、佛寺遗址、丝路南道考古爱好者
- **交通**：位于民丰县安迪尔乡方向，通常需包车或深度自驾
- **预约要点**：偏远、天气和路况影响大，不适合作为“普通游客随手半日游”
- **来源链接**：
  - https://baike.baidu.com/item/民丰县
  - https://www.mct.gov.cn/

### 22. 夏羊塔克古城
- **门票参考**：多无统一票务，以文保情况为准
- **建议时长**：1-2 小时
- **玩法**：看古城与遗址组合，是民丰一带古国遗迹的补充点
- **交通**：位于民丰县偏远区域，建议与安迪尔遗址打包安排
- **预约要点**：更适合遗址爱好者；普通游客若时间有限可以不强求
- **来源链接**：
  - https://baike.baidu.com/item/民丰县
  - https://www.xinjiang.gov.cn/

### 23. 唐兰城遗址
- **门票参考**：多为无标准商业票务
- **建议时长**：1-2 小时
- **玩法**：看唐代屯戍和古城遗痕，适合把精绝—于阗—唐代经营西域这条线串起来理解
- **交通**：民丰县安迪尔方向，建议深度包车线路前往
- **预约要点**：历史价值大于观光价值，注意尊重文保边界
- **来源链接**：
  - https://baike.baidu.com/item/民丰县
  - https://www.mct.gov.cn/

### 24. 亚瓦通古孜乡 / 沙漠边缘村落体验点
- **门票参考**：多免费或低消费进入
- **建议时长**：1-1.5 小时
- **玩法**：看沙漠边缘村落的人居方式、绿洲农业和县域生活节奏，适合公路旅行停靠
- **交通**：位于民丰县亚瓦通古孜乡方向，自驾 / 包车更方便
- **预约要点**：更适合顺路看生活方式，不适合抱着大型景区预期去
- **来源链接**：
  - https://baike.baidu.com/item/民丰县
  - https://www.xinjiang.gov.cn/

### 25. 皮山古国文化线（皮山县城及周边）
- **门票参考**：多数为开放式县域点位，个别展陈或园区低价收费
- **建议时长**：半天
- **玩法**：适合感受“西域三十六国之皮山国”历史脉络，补齐和田西线人文版图
- **交通**：和田西线可包车或先到皮山县城再转当地交通
- **预约要点**：更偏区域性历史体验，出发前最好确定具体点位组合
- **来源链接**：
  - https://baike.baidu.com/item/皮山县
  - https://www.xinjiang.gov.cn/

### 26. 桑株岩画
- **门票参考**：多无统一票务，以现场为准
- **建议时长**：1-1.5 小时
- **玩法**：看古代狩猎、生活题材岩画，是皮山较小众但有考古意味的点
- **交通**：位于皮山县康克尔柯尔克孜民族乡一带，建议包车前往
- **预约要点**：偏远且小众，不建议无准备前往；不要触碰岩画面
- **来源链接**：
  - https://baike.baidu.com/item/皮山县
  - https://www.mct.gov.cn/

### 27. 吐尔地阿吉庄园（皮山农场方向）
- **门票参考**：低价或免费，以现场为准
- **建议时长**：40-60 分钟
- **玩法**：看近代庄园遗迹和地方社会结构痕迹，适合对近现代地方史有兴趣的人
- **交通**：位于皮山农场方向，可包车顺访
- **预约要点**：更适合主题性深度旅行，不是大众必去点
- **来源链接**：
  - https://baike.baidu.com/item/皮山县
  - https://www.xinjiang.gov.cn/

### 28. 县域巴扎 / 乡村集市体验点（和田周边通用）
- **门票参考**：免费
- **建议时长**：1 小时
- **玩法**：如果碰上巴扎日，非常值得去看本地农产品、工具、服饰、家居和真实交易场景
- **交通**：需结合住地与巴扎日安排，通常包车最灵活
- **预约要点**：拍人像前先示意或征得同意；巴扎比“景区”更真实，也更需要礼貌
- **来源链接**：
  - https://www.xinjiang.gov.cn/
  - https://baike.baidu.com/item/和田地区

## 美食

和田的吃不是精致路线，而是**扎实、重碳水、重肉香、重烟火气**。如果只在和田待一天，至少保证吃到夜市和一家本地清餐馆。

### 必吃清单

- **烤包子**：刚出炉最好吃，外皮酥，羊肉洋葱馅最经典。
- **抓饭**：羊肉、胡萝卜、葡萄干的搭配最常见，油香足，饱腹感强。
- **烤肉 / 红柳烤肉**：南疆标准答案，羊排、羊肝、羊肉串都值得点。
- **缸子肉 / 手抓肉**：适合晚餐，肉香直接。
- **拉条子 / 拌面**：便宜、顶饱、稳妥。
- **面肺子 / 米肠子**：典型新疆风味，接受度因人而异。
- **烤鸽子 / 鸽子汤**：有些店做得很出色，适合补体力。
- **酸奶、奶皮子、冰淇淋类乳制品**：适合解腻。
- **石榴汁、杏皮茶、玫瑰花饮品**：夏季很实用。
- **核桃、红枣、巴旦木、葡萄干**：适合买作伴手礼。

### 觅食建议

- 夜市适合一次多尝几样，但别贪多，南疆摊位分量往往不小。
- 想吃得稳，去本地人多的清餐馆通常不容易踩雷。
- 团城、夜市、主城区主干道一带，最容易解决一日三餐。

## 住宿

### 住哪里最方便

#### 1. 和田市中心 / 团城周边
- 适合第一次到和田的人
- 吃饭、打车、逛夜市都方便
- 缺点是旺季房源紧张，热门酒店波动大

#### 2. 和田火车站 / 交通枢纽周边
- 适合中转型行程
- 进出方便，价格通常比市中心更友好
- 但夜间氛围和步行便利性一般

#### 3. 墨玉 / 洛浦县城
- 适合第二天还要继续跑县域线路的人
- 能节省来回折返时间
- 配套不如和田市成熟，但对深度游更实用

#### 4. 民丰县城
- 适合东线深度游和要去尼雅 / 安迪尔方向的人
- 选择不算多，尽量提前预订
- 作为沙漠公路节点很实用

### 住宿档次建议

- **舒适型**：优先选连锁或本地口碑商务酒店，卫生和热水更稳定。
- **经济型**：选择不少，但务必看近半年真实评价。
- **自驾深度线**：不要太晚抵达偏远县城，尽量白天入住，减少临时找房风险。

## 交通

### 如何到达和田

#### 飞机
- 可飞抵**和田昆冈机场**。
- 对时间敏感的旅行者最省事，通常与乌鲁木齐、喀什等地联动。

#### 火车
- **和田站**是和田地区重要铁路节点。
- 和若铁路开通后，和田与洛浦、策勒、于田、民丰、若羌之间的串联价值更高。

#### 自驾
- 南疆自驾常把和田作为关键一站。
- 可与喀什、阿克苏、若羌、库尔勒等方向连成环线，但地图距离和真实耗时往往不是一回事。

### 区域内交通

#### 市区
- 打车 / 网约车效率最高。
- 公交能用，但对游客而言效率不如打车。

#### 跨县
- **包车最实用**：特别适合墨玉、洛浦、于田、民丰一线。
- **自驾最自由**：但要考虑补给、限速、长距离疲劳和天气。
- **火车适合东线串联**：尤其是洛浦—策勒—于田—民丰方向。

#### 沙漠 / 遗址线
- 不建议只靠临时打车深入偏远遗址。
- 最好提前确认：**往返、等候、油费、是否熟路、能否进入具体点位**。

## 避坑

### 1. 不要把和田当成“网红快闪城市”
和田真正值得看的，是区域感和人文厚度，而不是一下午刷完所有“名场面”。

### 2. 玉石消费一定克制
“和田玉”三个字很容易让人冲动消费。**不懂就别做高价决策**，把玉石市场当参观场景最稳。

### 3. 别低估干燥、暴晒和风沙
南疆的热不是普通意义上的“有点晒”，补水、电解质、防晒和润唇膏都很重要。

### 4. 远郊遗址不是成熟景区
安迪尔遗址、麻扎塔格戍堡、部分佛寺遗址更偏考古探访，不是“游客中心 + 咖啡店 + 完整步道”的玩法。

### 5. 尊重拍摄边界
老城、乡村、巴扎都很出片，但先示意、别贴脸、别闯民宅、别把他人真实生活当猎奇素材。

### 6. 票务和营业信息会变
和田不少县域景点线上更新并不及时，**出发前一天最好再复核一次**。

### 7. 夜市点餐别过量
很多摊位分量大、油重，一次点太多容易浪费，也影响后续继续逛。

### 8. 长线尽量白天跑
尤其去于田、民丰、皮山或沙漠边缘线路，尽量别把陌生县道和荒漠公路压到深夜。
