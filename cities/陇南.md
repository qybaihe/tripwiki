# 陇南｜秦巴山地与峡谷湖泊并存的甘肃南部门户

## 1. 城市概览
陇南位于甘肃最南端，连接甘、陕、川三地，整体气质和很多人印象中的“西北荒漠感”完全不同。这里更像秦巴山地与川北过渡地带：山高谷深、森林密、溪水多，既有溶洞、峡谷、高山湖泊，也有三国文化、红色遗址、古镇与乡村慢游资源。

对旅行者来说，陇南最大特点不是“某一个超级城市景点”，而是**县域分散、片区差异很大**。武都适合作为交通落点，宕昌看官鹅沟，成县兼顾汉碑与山景，康县偏森林避暑与乡村度假，文县适合天池和碧口古镇，两当则更适合自然峡谷和红色主题联游。

如果是第一次来，最实用的思路不是“大而全”，而是按片区拆开玩：
- **武都—宕昌线**：万象洞、朝阳洞、官鹅沟、哈达铺，适合第一次入门。
- **成县—康县线**：西狭颂、鸡峰山、杜甫草堂、阳坝梅园沟、花桥村，适合人文+山水慢游。
- **文县—两当—礼县线**：文县天池、碧口古镇、云屏三峡、两当兵变纪念馆、祁山武侯祠，适合主题深度游。

总的判断很简单：陇南值得来，但要按路线玩，不要试图把它当成“一个城区+周边景点”的城市来刷。

## 行前速览
- **推荐季节**：4-6 月、9-11 月最稳；7-8 月适合避暑，但需重点关注暴雨、山洪、落石预警。
- **适合人群**：自驾游客、峡谷山水爱好者、轻徒步人群、喜欢三国文化和红色历史的人。
- **不太适合**：只想靠市内公共交通高效串多个县区的人。
- **城市节奏**：跨县移动时间往往比想象中长，路线安排宁可少一个点，也不要硬塞。
- **穿着建议**：夏季城区偏热，但峡谷、溶洞、天池一带早晚较凉，建议短袖+薄外套组合。
- **鞋子建议**：防滑运动鞋优先，很多景区都有栈道、台阶、湿滑路面。
- **门票习惯**：门票不一定高，但景交、摆渡、索道、游船常常另算。
- **住宿策略**：优先住在第二天主景区附近，减少清晨跨县折返。
- **饮食关键词**：洋芋搅团、荞面、土鸡、腊肉、山野菜、核桃制品、花椒风味、文县热凉面。
- **信息核验**：门票、开放、预约、道路情况发布前后都可能变化，临行前务必再核实一次。

## 1-3天行程

### 1日快刷版：武都入门
适合高铁/飞机刚落地、只想先看一条轻量路线的人。

**推荐路线**：万象洞 → 武都午餐 → 朝阳洞 / 姚寨沟（二选一）
- 上午先去万象洞，溶洞内温差更舒服，也相对适合避开午后人流。
- 中午回武都吃本地面食、搅团或农家菜。
- 下午如果想看人文崖寺，去朝阳洞；如果更想轻松走溪谷步道，去姚寨沟。
- 夜宿武都，方便第二天转宕昌、成县或继续停留。

### 2日经典版：武都 + 宕昌
适合第一次来陇南、想看最有代表性山水景区的人。

**Day 1：武都**
- 万象洞
- 朝阳洞 / 姚寨沟（二选一）
- 傍晚前往宕昌住宿，或者住武都次日早起出发

**Day 2：宕昌**
- 全天官鹅沟景区
- 若返程时间合适、对红色历史感兴趣，可补看哈达铺红军长征纪念馆

这个组合是第一次来陇南最省心的一条线，既有溶洞，也有峡谷湖泊，完成度很高。

### 3日深度版：成县 + 康县 + 文县/两当
适合不想只看一个爆款景区、想把陇南层次玩出来的人。

**Day 1：成县人文山水线**
- 西狭颂风景区
- 鸡峰山国家森林公园
- 杜甫草堂
- 夜宿成县

**Day 2：康县森林乡村线**
- 阳坝梅园沟
- 阳坝片区慢游 / 花桥村
- 夜宿康县或阳坝民宿

**Day 3：二选一**
- **自然线**：文县天池 → 碧口古镇
- **峡谷红色线**：云屏三峡 → 两当兵变纪念馆

### 3日主题版：三国与历史文化线
适合本身就对三国、先秦考古和地方历史更感兴趣的人。

**Day 1**：礼县祁山武侯祠 → 祁山堡周边点位 → 礼县住宿  
**Day 2**：大堡子山遗址相关点位 → 成县西狭颂 → 成县住宿  
**Day 3**：鸡峰山或杜甫草堂 → 返回武都/陇南站

如果你只有一次来陇南的机会、时间又不多，优先级建议是：**官鹅沟 + 万象洞**；有 3 天以上，再把成县、康县、文县这些片区补进来。

## 2. 景点/点位
> 说明：以下信息以公开资料和常见票务口径整理，景区门票、开放时间、预约要求、道路通行情况可能随季节、天气、活动和运营调整变化。若无明确官方统一信息，均按“**以现场/官方公告为准**”处理。

### 2.1 官鹅沟景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1天。
- **玩法**：这是陇南自然景观的头部代表，主打峡谷、溪流、森林、湖泊和多段栈道组合，适合第一次来陇南的游客作为核心目的地。时间充足建议整天慢走，不要只匆匆打卡。
- **交通**：通常从宕昌县城自驾或包车前往最方便；从武都当天往返也可行，但整体会比较赶。
- **预约要点**：旺季、节假日及恶劣天气下可能出现限流、分时入园或部分路段关闭，以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/官鹅沟>

### 2.2 官珠沟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：官鹅沟核心游线之一，优势是亲水感强、瀑布溪谷密集、步道成熟，适合轻徒步和拍照。第一次去建议把它作为官鹅沟内部优先级很高的一段。
- **交通**：进入官鹅沟景区后，结合景区内部接驳与步道游览。
- **预约要点**：雨后水量更大，但也更容易遇到临时管制或湿滑风险，以现场/官方公告为准。
- **来源链接**：<https://www.dangchang.gov.cn/>

### 2.3 鹅嫚沟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：3-4小时。
- **玩法**：相较官珠沟更偏高山湖泊和开阔视野，适合看层叠山体、湖水倒影和秋季色彩。如果时间够，官珠沟与鹅嫚沟组合才是完整的一天。
- **交通**：通常依赖景区内部接驳或景区既定游览线路。
- **预约要点**：景区内部转运会占时间，建议早到；晚段返程高峰可能排队，以现场/官方公告为准。
- **来源链接**：<https://www.dangchang.gov.cn/>

### 2.4 万象洞
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1.5-2小时。
- **玩法**：大型喀斯特溶洞，是武都最稳定、最适合入门的景点之一。洞内钟乳石、石幔、石柱景观较丰富，夏季还有明显避暑优势。
- **交通**：距武都城区不算远，自驾、出租车或网约车通常都比较方便。
- **预约要点**：洞内地面可能湿滑，穿防滑鞋更稳；开放时段、讲解安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/万象洞>

### 2.5 朝阳洞
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1-2小时。
- **玩法**：偏人文向的崖壁寺观型景点，适合和万象洞搭配成武都一日游。它的价值在山崖环境、洞窟氛围和寺观遗存，不在大体量景区感。
- **交通**：通常从武都城区自驾前往更方便，包车也比较合适。
- **预约要点**：台阶和山路较多，雨天体验会下降；门票和开放安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/武都朝阳洞>

### 2.6 姚寨沟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：适合做武都周边的轻量自然补充，溪谷、林木和步道体验更偏轻松，不想爬太多山的人更容易接受。
- **交通**：从武都出发建议自驾或包车，公共交通衔接通常不算高效。
- **预约要点**：若遇强降雨或山区道路异常，可能影响开放和通行，以现场/官方公告为准。
- **来源链接**：<https://www.lnswhlyj.cn/>

### 2.7 哈达铺红军长征纪念馆
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1.5-2小时。
- **玩法**：如果想让陇南之行不止是自然风景，这里很适合作为官鹅沟同线补充。它在长征史中的地位很高，适合配合讲解一起看。
- **交通**：位于宕昌县哈达铺镇方向，通常自驾最顺，包车也可操作。
- **预约要点**：免费开放类纪念馆也可能受周一闭馆、团队接待、讲解时段限制，以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/哈达铺红军长征纪念馆>

### 2.8 西狭颂风景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：这里的核心不是单纯看自然，而是看汉代著名摩崖石刻《西狭颂》与峡谷环境结合。对历史、书法、摩崖石刻感兴趣的人会觉得非常值。
- **交通**：从成县县城出发自驾或打车都比较方便。
- **预约要点**：文保区域参观规则、拍摄限制、开放时段以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/西狭颂风景区>

### 2.9 鸡峰山国家森林公园
- **门票参考**：以现场/官方公告为准。
- **建议时长**：3-4小时。
- **玩法**：属于成县一带偏“体力型”的主景点，适合登高看群山云雾、森林景观和山地视野。天气好时观景体验明显更强。
- **交通**：成县出发以自驾、包车最省心。
- **预约要点**：如涉及索道、区间车等项目，票务和运营情况以现场/官方公告为准；雾天可见度一般。
- **来源链接**：<https://baike.baidu.com/item/鸡峰山国家森林公园>

### 2.10 杜甫草堂（杜公祠）
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1小时。
- **玩法**：适合文学和历史爱好者，重点看杜甫在同谷时期相关纪念遗存。单独远道而来未必必要，但和成县其他点串起来很顺。
- **交通**：通常位于成县城郊，打车或自驾都比较方便。
- **预约要点**：开放时间、临时维护安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/杜公祠>

### 2.11 阳坝梅园沟
- **门票参考**：以现场/官方公告为准。
- **建议时长**：半天到1天。
- **玩法**：康县很有代表性的森林峡谷景区，夏季避暑体验很好，适合慢慢走，不适合打卡式压缩。溪瀑、林木、清凉感是它的核心卖点。
- **交通**：通常从康县或阳坝镇方向自驾进入最方便。
- **预约要点**：暴雨、山洪、落石预警时需格外谨慎，开放与否以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/梅园沟>

### 2.12 阳坝亚热带生态旅游风景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-4小时。
- **玩法**：更适合偏康养、休闲、呼吸森林空气的人，不是那种一路高潮的强刺激型景区。适合住在阳坝，把节奏放慢。
- **交通**：通常从康县县城或阳坝镇转入，自驾最方便。
- **预约要点**：旺季住宿与景区承载压力都会增加，联票、开放、预约政策以现场/官方公告为准。
- **来源链接**：<https://www.lnswhlyj.cn/>

### 2.13 花桥村
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1-2小时。
- **玩法**：更偏乡村休闲和民宿氛围体验，适合在山水景区之后放慢节奏，看看村落环境、街巷和秦巴山区生活感。
- **交通**：康县片区内自驾串联最方便。
- **预约要点**：村域开放区和个别体验项目规则可能不同，住宿与活动安排以现场/官方公告为准。
- **来源链接**：<https://www.lnswhlyj.cn/>

### 2.14 文县天池国家森林公园
- **门票参考**：以现场/官方公告为准。
- **建议时长**：半天到1天。
- **玩法**：主角是高山湖泊，适合喜欢山中大水面、清凉气候和相对静态景观的人。天气好的时候湖景最出彩，雨雾天气则更偏氛围感。
- **交通**：从文县县城前往通常需要较长山路车程，自驾最稳，包车次之。
- **预约要点**：高海拔区域天气变化快，开放状态、道路情况、景交安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/文县天池国家森林公园>

### 2.15 碧口古镇
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：更适合当作旅途中的古镇停留点，而不是期待它是大型商业古城。可以看看白龙江边的老街、生活气息，顺便吃饭、住一晚。
- **交通**：文县南部片区，自驾最方便。
- **预约要点**：节假日停车和住宿会偏紧张，古镇开放、展馆和活动安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/碧口古镇>

### 2.16 碧口水运文化老街片区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1小时。
- **玩法**：适合作为碧口古镇内部的延伸步行段，看老街、水运贸易痕迹和沿江氛围。它更适合顺路体验，不值得单独奔赴。
- **交通**：通常在碧口古镇片区步行游览。
- **预约要点**：街区开放和个别馆舍开放规则以现场/官方公告为准。
- **来源链接**：<https://www.lnswhlyj.cn/>

### 2.17 云屏三峡自然风景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：3-4小时。
- **玩法**：是两当片区很有代表性的自然景区，峡谷、山水、栈道层次感都不错，适合轻徒步和拍照，完成度较高。
- **交通**：一般从两当县城或周边自驾前往最方便。
- **预约要点**：雨后虽然出片，但也伴随湿滑和安全限制，开放及游览方式以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/云屏三峡自然风景区>

### 2.18 两当兵变纪念馆
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1-1.5小时。
- **玩法**：适合和云屏三峡放在同一天，形成“自然 + 红色历史”的组合。馆内更建议配讲解看，理解会完整很多。
- **交通**：通常位于两当县城一带，打车、自驾都方便。
- **预约要点**：免费场馆不代表随时可进，开放、讲解、团队接待安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/两当兵变纪念馆>

### 2.19 祁山武侯祠
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1-2小时。
- **玩法**：三国迷会比较有感觉，适合和祁山堡、周边古战场点位联游。普通游客如果对三国无感，感受会更依赖讲解和前置知识。
- **交通**：礼县祁山镇方向，自驾最为方便。
- **预约要点**：开放、讲解、周边点位联动安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/武侯祠>

### 2.20 祁山堡及周边三国遗址点位
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1-2小时。
- **玩法**：这类点位的吸引力主要在历史联想和地理位置，不是大体量视觉型景区。适合做三国线补充，不适合期待强烈娱乐性。
- **交通**：礼县片区通常建议自驾串联多个点位。
- **预约要点**：部分点位信息化和导览程度一般，开放、位置和可进入性以现场/官方公告为准。
- **来源链接**：<https://www.gswb.gov.cn/>

### 2.21 大堡子山遗址相关点位
- **门票参考**：以现场/官方公告为准。
- **建议时长**：1.5-2小时。
- **玩法**：适合对秦早期文化、先秦考古有明确兴趣的人。它的价值更多来自考古史意义，而不是单纯“好拍”“热闹”。
- **交通**：礼县片区建议自驾或包车前往。
- **预约要点**：遗址展示区、展馆、临展开放情况会变化，参观规则以现场/官方公告为准。
- **来源链接**：<https://www.gswb.gov.cn/>

### 2.22 晚霞湖水利风景区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：适合自驾途中做半日休闲，重点是湖景、傍晚光线和较轻松的游览节奏。如果天气配合，日落时段会比白天更有感觉。
- **交通**：通常从西和县城前往较方便，自驾和打车都可考虑。
- **预约要点**：若涉及游船等项目，运营和票务安排以现场/官方公告为准；天气不好时观感会明显打折。
- **来源链接**：<https://baike.baidu.com/item/晚霞湖>

### 2.23 仇池山
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：更偏地方历史和山地景观结合，适合对仇池国、地方传说和区域历史感兴趣的人。它不一定是最热门，但有自己的主题性。
- **交通**：西和县域内一般以自驾前往最省心。
- **预约要点**：景区开放、道路通行、配套项目安排以现场/官方公告为准。
- **来源链接**：<https://baike.baidu.com/item/仇池山>

### 2.24 青龙山旅游度假区
- **门票参考**：以现场/官方公告为准。
- **建议时长**：2-3小时。
- **玩法**：更偏度假、亲子、休闲和康养，不是典型“硬核自然景区”。如果你本来就准备在康县放慢节奏，这类点位反而合适。
- **交通**：康县片区自驾前往更方便。
- **预约要点**：园区型项目可能分基础入园和单独收费体验，具体以现场/官方公告为准。
- **来源链接**：<https://www.lnswhlyj.cn/>

### 2.25 白龙江沿线观景点
- **门票参考**：以现场/官方公告为准。
- **建议时长**：0.5-1小时。
- **玩法**：适合作为武都—文县等自驾线路上的临时停靠补景点，看江谷、山体、桥梁和道路穿行感。属于“路上风景”，不是专程目的地。
- **交通**：仅建议在正规观景点或安全停车带临停，自驾最适合。
- **预约要点**：不建议在弯道、隧道口、无停车区随意停车拍照；通行与临停规则以现场/官方公告为准。
- **来源链接**：<https://www.lnswhlyj.cn/>

## 美食
陇南的吃不是那种靠单一网红名菜出圈的城市，而是很典型的山区食材 + 西北主食 + 川味边缘感的组合。真正稳定好吃的，很多都在县城家常馆子、农家乐和民宿餐桌上。

### 推荐重点吃
- **洋芋搅团**：本地很稳的代表性主食，顶饱，也很有地方感。
- **荞面、杂粮面、浆水面**：适合做早餐或简餐，山区风味明显。
- **文县热凉面**：偏酸辣开胃，名字有记忆点，夏天吃很合适。
- **土鸡、腊肉、山野菜**：康县、文县、宕昌一带尤其值得试，农家菜通常比景区快餐更可靠。
- **核桃制品**：陇南核桃很有名，核桃仁、核桃饼、核桃油都适合买回去。
- **花椒风味菜**：麻香明显，但通常没有部分川菜那么猛烈，多数人都能接受。

### 点菜建议
- 景区周边优先选土鸡、腊肉、时令蔬菜、面食，踩雷率低。
- 不要迷信“网红榜单”，陇南很多靠谱馆子看起来都很普通。
- 想买特产时，核桃、木耳、蜂蜜、花椒比常规旅游纪念品更实用。

## 住宿

### 武都区
适合作为抵达日、离开日、武都周边景点游玩的落脚点。优点是餐饮和补给最全，打车方便；缺点是如果第二天主玩宕昌、文县、两当，清晨转场会比较累。

### 宕昌县城 / 官鹅沟景区附近
如果官鹅沟是主目标，建议优先住这一区域。最大优势是省体力，不用一大早跨县赶山路；缺点是夜生活和酒店成熟度通常不如武都。

### 成县县城
适合西狭颂、鸡峰山、杜甫草堂一线，也方便衔接成县机场。住得相对舒服，去康县也顺；但若第二天主玩文县或两当，就不是最优落点。

### 康县阳坝 / 花桥村民宿
适合想要避暑、度假、慢游的人。环境通常更好，住起来更有松弛感；缺点是旺季房源紧张，标准参差不齐，要尽量提前订。

### 文县县城 / 碧口镇
如果第二天主玩文县天池或碧口古镇，直接住下通常比当天来回折返更合理。整体便利性不如武都，但能明显节省路上时间。

### 两当县城
适合云屏三峡、两当兵变纪念馆路线的短住需求，走宝鸡、汉中方向时衔接也方便。缺点是度假氛围一般，更偏功能型住宿。

## 交通

### 大交通怎么进陇南
- **飞机**：可关注陇南成县机场，但航线和班期可能有季节性变化，订票前务必确认实际执飞情况。
- **铁路**：常见落点包括陇南站、哈达铺站等，具体选哪个站，取决于你重点玩武都、宕昌还是其他县区。
- **自驾进城**：从天水、广元、汉中方向进入都比较常见，山区高速整体不算难开，但依然要给山路和天气留余量。

### 到了之后怎么跑
- **首选自驾**：这是陇南最适合的旅行方式，尤其适合跨县串景点。
- **次选包车/拼车**：如果不想自己开山路，这通常比纯公共交通舒适得多。
- **县城打车**：大多县城可以解决，但景区返程未必稳定，最好提前约好司机。
- **公共交通**：能坐，但游客体验通常一般，换乘、等待、下车后二次转运都比较耗时。

### 驾驶提醒
- 山区夜路尽量少赶。
- 雨季重点关注落石、塌方、山洪和临时交通管制。
- 地图上看着不远的跨县移动，实际常常要比预期多花很多时间。

## 避坑
- **不要把陇南当成“市区景点群”来玩**：它本质上是多个山地片区组成的地级市。
- **不要一天硬塞多个大景区**：官鹅沟、文县天池、云屏三峡这类点位，本身就值得半天到一天。
- **不要只看门票预算**：景交、索道、游船、停车费叠加后，才是更接近真实的旅行成本。
- **雨季少冒险**：峡谷、栈道、山路都容易受天气影响，遇到恶劣天气及时降级路线更明智。
- **热门季节住宿要提前锁**：避暑季、红叶季、节假日的景区附近房源很容易紧张。
- **对古镇和乡村的预期要放对**：陇南很多地方的看点是真实生活感，不是重商业包装。
- **文博馆和纪念馆先查开放时间**：免费不等于随时开放，周一、维护日、团队接待都可能影响参观。
- **别轻视体力消耗**：台阶、步道和长距离转场叠加，会比想象中更累。
