# 甘孜藏族自治州旅游攻略

> 适用模式：mode2｜适合第一次去川西、准备自驾/包车、希望在 1—3 天内快速建立甘孜旅行框架的旅行者。
>
> 说明：甘孜藏族自治州是四川西部的超大面积自治州，玩法本质上是“州域旅行”而不是单一县城打卡。点位之间距离远、海拔起伏大、天气变化快，做攻略时一定先看路线，再看景点。文中涉及门票、开放、预约等信息若存在调整，统一以现场/官方公告为准。

## 1. 城市概览

甘孜藏族自治州位于四川省西部、青藏高原东南缘，是川西高原旅行的核心目的地之一，州府为康定市。这里最典型的旅行关键词是：**雪山、草原、海子、冰川、寺庙、藏寨、公路风景、高海拔摄影与自驾环线**。

对大多数游客来说，甘孜不是“逛一个城”，而是沿着几条经典走廊移动：

- **康定—新都桥—塔公—八美**：适合 1—2 天入门，风景密度高。
- **康定—雅江—理塘—稻城亚丁**：适合 3 天及以上，偏雪山海子线。
- **泸定—磨西—海螺沟**：适合加入冰川、温泉和红色历史内容。
- **丹巴—甲居藏寨—党岭/莫斯卡**：适合人文、村落和轻野线。

如果是第一次去，最稳妥的理解方式很简单：

1. **康定负责过渡适应海拔**；
2. **新都桥/塔公负责草原、雪山和光影**；
3. **稻城亚丁负责“川西封面级”风景**；
4. **丹巴、海螺沟、理塘、巴塘等地负责把线路做深。**

甘孜最迷人的地方，是一路不断变化：上午还在峡谷与河谷之间，下午就进入草原雪山；昨天住在 2500 米上下的康定，第二天可能已经在 4000 米左右的观景台吹风看日落。所以它非常值得去，但也非常依赖节奏、天气和身体状态管理。

## 2. 行前速览

### 2.0 一眼看懂甘孜

- **推荐季节**：5—6 月、9—10 月中旬最稳。
- **玩法关键词**：自驾、公路风景、轻徒步、摄影、藏地人文。
- **海拔特点**：康定相对友好；新都桥、塔公、理塘、稻城亚丁普遍偏高。
- **适合人群**：能接受长途车程、早晚温差、天气不确定性的旅行者。
- **不太适合**：把行程排得很满、怕颠簸、完全不能接受高海拔的人。

### 2.0.1 最佳旅行时间

- **5—6 月**：草原返绿，雪山仍有积雪，整体清爽，适合初次入门。
- **7—8 月**：植被旺盛，但雨季明显，塌方、修路、云雾遮山概率更高。
- **9—10 月中旬**：公认黄金期，秋色、雪山能见度、摄影表现通常更好。
- **11 月—次年 3 月**：雪景强，但高海拔道路、观景台、支线乡道受冰雪影响的概率上升。

### 2.0.2 海拔与高反提醒

- 康定城区通常在 2500 米上下，相对适合做第一晚。
- 新都桥、塔公、理塘、鱼子西、墨石公园、稻城亚丁等地普遍在 3300—4500 米区间。
- 第一天不建议直接冲高强度徒步或连续看多个高点日落。
- 如果出现明显头痛、恶心、胸闷、持续失眠等情况，优先减缓节奏，必要时下撤。

### 2.0.3 交通方式建议

- **最佳方式：自驾或包车。** 甘孜景点分散，州内很多精彩点位并不适合完全依赖公共交通。
- **不想开车**：可先到康定，再参加木格措、新都桥、塔公等周边拼车/包车；去稻城亚丁则更适合整段包车或跟团。
- **山路驾驶原则**：少夜驾、少赶路、雨雪天保守开，不用把导航预估时间当成真实到达时间。

### 2.0.4 装备建议

- 四季都建议带：冲锋衣/保暖外套、防晒霜、墨镜、帽子、润唇膏。
- 去鱼子西、冷嘎措、子梅垭口、亚丁长线等点位，建议多带：登山杖、防滑鞋、保温杯。
- 高原紫外线很强，即使阴天也容易晒伤。

### 2.0.5 证件与预约

- 身份证必须随身携带。
- 自驾需备驾驶证、行驶证。
- 稻城亚丁、海螺沟、木格措、墨石公园、甲居藏寨等正规景区，建议出发前查看官方购票入口。
- 非标准景区型观景点，很多是现场收费或村集体运营，规则常有变化，**以现场/官方公告为准**。

## 3. 1—3天行程

### 3.1 1 天快闪：康定—木格措—新都桥/塔公

**适合谁**：从成都周末出发，想快速感受甘孜风景变化的人。  
**路线建议**：成都/雅安 → 泸定（可短停）→ 康定 → 木格措 → 折多山沿线 → 新都桥/塔公 → 住新都桥或返程

**行程重点**：

- 上午先抵康定，不建议一上来就连续冲多个高海拔野点。
- 木格措作为正规景区，风景完成度高，适合第一次进甘孜的人建立“高原湖泊+雪山+草甸”的直观印象。
- 下午继续往新都桥或塔公方向走，重点看沿途光影、草原、公路与雪山背景。
- 体力和天气都不错时，再考虑鱼子西日落；否则住新都桥更稳。

### 3.2 2 天经典：康定—新都桥—塔公—墨石公园

**D1** 成都/雅安 → 泸定桥 → 康定 → 木格措 → 新都桥住  
**D2** 新都桥晨景 → 塔公草原/塔公寺 → 木雅圣地或墨石公园 → 返康定/成都

**这条线的优点**：

- 景色变化快，适合第一次去甘孜。
- 海拔爬升相对平缓，比直接冲理塘、稻城更友好。
- 摄影和休闲兼顾，不需要太强体能。
- 住宿和补给条件相对成熟。

### 3.3 3 天进阶：理塘—海子山—稻城亚丁

**D1** 成都 → 康定/新都桥 → 雅江住  
**D2** 雅江 → 理塘 → 勒通古镇 → 海子山/姊妹湖沿线 → 稻城/香格里拉镇住  
**D3** 稻城亚丁一日游（优先短线）→ 稻城或返程中转

**建议这样玩**：

- 第一天不要贪多，重点是适应海拔。
- 第二天在理塘一带注意放慢节奏，县城海拔高，不适合刚到就剧烈活动。
- 第三天去亚丁，第一次更建议走冲古寺—珍珠海或洛绒牛场方向的相对轻量玩法。
- 如果想冲牛奶海/五色海，最好额外加 1 晚，别把 3 天行程拉成纯赶路。

## 4. 景点/点位

> 本章节为 mode2 可发布结构，以下点位按“好上手、常见、代表性强”优先排序。所有门票、开放、预约等不确定信息，统一按要求写为：**以现场/官方公告为准**。

### 2.1 泸定桥

- **门票参考**：门票参考以现场/官方公告为准。
- **建议时长**：1—1.5 小时。
- **玩法**：适合当作进甘孜或返程路上的历史地标停留点，重点看铁索桥本体、大渡河峡谷和长征文化背景；如果时间紧，停留拍照+简单参观即可。
- **交通**：位于泸定县城，走雅康高速进出都比较方便，自驾最省心。
- **预约要点**：节假日人流和停车压力会明显上升，是否需要线上预约、联票或分时进入以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/泸定桥

### 2.2 海螺沟景区

- **门票参考**：门票、景交车、索道及开放范围以现场/官方公告为准。
- **建议时长**：半天—1 天。
- **玩法**：核心看点是冰川、森林、温泉和贡嘎山视角，适合和泸定、磨西古镇安排在同一段行程里；如果恢复开放区域较完整，可以把它作为甘孜“冰川线”的代表体验。
- **交通**：通常从成都/雅安经泸定到磨西镇，再进入景区，自驾或包车最常见。
- **预约要点**：景区开放情况可能受地质灾害修复、季节和维护影响，是否开放、开放到哪一级景点、是否需提前预约，均以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://www.hailuogou.com/

### 2.3 磨西古镇

- **门票参考**：古镇公共区域通常为开放式，具体收费项目以现场/官方公告为准。
- **建议时长**：1—2 小时。
- **玩法**：适合在海螺沟前后补给、散步、拍老街和山地小镇氛围，也可以作为休整点，不需要安排过长参观时间。
- **交通**：位于海螺沟方向的重要集散地，可自驾直达，也可作为班车/包车落脚点。
- **预约要点**：古镇本身一般不复杂，但住宿、停车、周边开放点位如有调整，以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/磨西镇

### 2.4 木格措

- **门票参考**：门票、观光车及相关套票以现场/官方公告为准。
- **建议时长**：4—6 小时。
- **玩法**：这是康定周边最适合第一次去的正规景区之一，可以看高原湖泊、草甸、雪山背景、红石滩和药池沸泉，整体完成度高，适合半日到一日游。
- **交通**：从康定市区出发较方便，自驾、包车都成熟。
- **预约要点**：旺季是否需要提前线上购票、观光车是否限流、部分点位开放状态如何，以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/康定情歌（木格措）风景区/15204784

### 2.5 康定情歌城/康定城区

- **门票参考**：城区公共空间多为免费，个别景点收费以现场/官方公告为准。
- **建议时长**：2—4 小时。
- **玩法**：如果第一晚住康定，可以把城区当作适应高原的缓冲站，逛折多河沿线、看夜景、找餐馆、补给，比一上来就冲高海拔观景点更稳妥。
- **交通**：雅康高速进出方便，作为甘孜东部门户节点非常合适。
- **预约要点**：城区一般不需要特别预约，若涉及个别收费景点、停车或活动安排，以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/康定市

### 2.6 折多山垭口

- **门票参考**：公路观景点常见为开放式，具体管理和收费以现场/官方公告为准。
- **建议时长**：20—40 分钟。
- **玩法**：它更像“进川西的海拔分界线”和公路仪式感节点，适合短暂停车看山口、经幡和远处山势，不适合久跑久跳。
- **交通**：位于康定往新都桥方向公路沿线，自驾经过即可到达。
- **预约要点**：是否允许长时间停留、道路是否限行、是否临时交通管制，以现场/官方公告为准；冬季和恶劣天气尤其要保守。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/折多山

### 2.7 新都桥

- **门票参考**：镇区和多数公路风景为开放式，收费项目以现场/官方公告为准。
- **建议时长**：半天—1 天。
- **玩法**：新都桥不是靠单一景区取胜，而是靠晨雾、草原、藏居、溪流和公路光影形成整体氛围，适合慢拍、发呆和作为中转住宿点。
- **交通**：位于川藏南线重要节点，自驾最方便，也常见拼车游客在此落脚。
- **预约要点**：景点本身通常不复杂，但旺季住宿紧张，若打算住一晚，建议提前锁房；具体周边点位开放情况以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/新都桥

### 2.8 鱼子西

- **门票参考**：观景平台、中转车或管理费以现场/官方公告为准。
- **建议时长**：2—3 小时，日落最常见。
- **玩法**：川西热门观景点之一，天气好时可以看贡嘎群峰、日落层次和星空；真正决定体验的是能见度，不是“到了就一定有大片”。
- **交通**：多从新都桥或甲根坝方向换乘当地车辆上山，普通游客更适合包车或跟车。
- **预约要点**：是否可自驾直上、是否需要换乘、收费方式和开放时段，均以现场/官方公告为准；山顶风大温低，务必备保暖层。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/鱼子西

### 2.9 塔公草原

- **门票参考**：开放草原和观景点收费方式以现场/官方公告为准。
- **建议时长**：2—4 小时。
- **玩法**：重点是草原延展感、村镇氛围和雅拉雪山背景，很适合第一次接触“藏区草原+雪山”组合的游客，节奏可以放慢一些。
- **交通**：从新都桥前往塔公较方便，自驾、包车都成熟。
- **预约要点**：草原区域通常较开放，但核心观景地、停车和骑马等项目规则以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/塔公草原

### 2.10 塔公寺

- **门票参考**：参观收费及开放范围以现场/官方公告为准。
- **建议时长**：0.5—1 小时。
- **玩法**：适合把草原自然风景和人文寺庙体验放在一起看，重点不是久逛，而是感受寺院、白塔、转经氛围和藏地日常秩序。
- **交通**：位于塔公镇内，和塔公草原通常同线安排。
- **预约要点**：是否开放内部区域、是否限制拍摄、法会期间参观规则如何，以现场/官方公告为准；进入寺院请尊重宗教礼仪。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/塔公寺

### 2.11 木雅圣地

- **门票参考**：门票、景交及项目收费以现场/官方公告为准。
- **建议时长**：2—4 小时。
- **玩法**：如果你想要一个相对成体系、便于初次游客理解藏地景观与文化意象的景区，木雅圣地是塔公方向比较省心的选择，适合轻松逛拍。
- **交通**：通常与新都桥、塔公、墨石公园串联，自驾较方便。
- **预约要点**：购票、园区开放区域、演出或项目安排是否调整，以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/木雅圣地景区

### 2.12 墨石公园

- **门票参考**：门票、景交车及园内项目以现场/官方公告为准。
- **建议时长**：2—3 小时。
- **玩法**：以独特的灰黑色石林峡谷地貌取胜，很适合拍人像、公路大片和“异星感”照片；如果天气晴朗、穿搭颜色对比明显，会更出片。
- **交通**：多从八美方向进入，常与塔公、新都桥组成一条成熟线路。
- **预约要点**：是否需要提前购票、景交运行情况、可进入区域等以现场/官方公告为准；高原紫外线强，帽子墨镜要备齐。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/墨石公园

### 2.13 雅拉雪山观景区

- **门票参考**：观景区、摆渡车或管理费以现场/官方公告为准。
- **建议时长**：2—4 小时。
- **玩法**：适合把雅拉雪山作为核心主角来拍，常见玩法是草甸轻走、找前景拍雪山、等光线变化，不需要把节奏拉得太赶。
- **交通**：通常从塔公镇周边前往最方便，自驾或包车均可。
- **预约要点**：是否能自驾进入、是否需换乘、具体开放点位和收费方式，以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/雅拉雪山

### 2.14 甲居藏寨

- **门票参考**：门票、观光车、停车等以现场/官方公告为准。
- **建议时长**：3—4 小时；想拍晨雾建议住 1 晚。
- **玩法**：这是丹巴最有代表性的嘉绒藏寨景观区，层叠民居和山谷田园感很强；最好的玩法往往不是急着刷点，而是在寨内住一晚，看傍晚和清晨光线变化。
- **交通**：从丹巴县城出发较近，自驾和包车都方便。
- **预约要点**：是否需提前购票、观光车运行方式、是否能自驾到部分观景点，以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/甲居藏寨

### 2.15 梭坡古碉群

- **门票参考**：村落参观、讲解和开放碉楼收费以现场/官方公告为准。
- **建议时长**：1—2 小时。
- **玩法**：适合对嘉绒藏族建筑、石碉和村落历史更感兴趣的人，不一定是“必打卡爆点”，但对丰富丹巴线路层次很有帮助。
- **交通**：位于丹巴周边，自驾最方便，可与甲居藏寨同日安排。
- **预约要点**：部分碉楼或村寨区域可能不对外完全开放，具体参观范围和讲解安排以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/丹巴碉楼

### 2.16 党岭

- **门票参考**：环保费、摆渡车、徒步线路相关费用以现场/官方公告为准。
- **建议时长**：1 天；如深度徒步建议预留更多时间。
- **玩法**：党岭更偏“轻野线+徒步”属性，看点在雪山、海子、彩林和自然感，不适合完全没有高原徒步准备的人临时起意冲进去。
- **交通**：通常从丹巴方向进入，后续还可能涉及换乘当地车辆。
- **预约要点**：道路通行、换乘安排、徒步开放状态和天气影响很大，出发前与现场信息同步最重要，一切以现场/官方公告为准。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/党岭

### 2.17 莫斯卡

- **门票参考**：村落进入、换乘车、住宿及相关管理费用以现场/官方公告为准。
- **建议时长**：半天—1 天；深度体验可住 1 晚。
- **玩法**：莫斯卡吸引人的地方在于高原村落、草甸、环山地貌和相对偏野的氛围；比起“打卡大景区”，它更适合喜欢慢节奏和非标准化体验的人。
- **交通**：多从丹巴方向包车或越野车进入，普通轿车不建议盲目硬闯。
- **预约要点**：是否需要换乘、村内管理规则、是否限制游客活动区域，以现场/官方公告为准；尊重村民生活，不打扰、不乱投喂动物。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/莫斯卡/25902

### 2.18 理塘勒通古镇·千户藏寨

- **门票参考**：古镇主体及附属展馆收费以现场/官方公告为准。
- **建议时长**：2—3 小时。
- **玩法**：理塘是“世界高城”气质很强的节点，勒通古镇适合作为人文停留点，看看街区、藏式民居和高海拔县城的日常氛围，不必用景区思路硬刷。
- **交通**：位于理塘县城，主公路可达性较好，自驾顺路停留很方便。
- **预约要点**：若遇活动、演艺或特定展馆开放调整，以现场/官方公告为准；理塘海拔较高，刚到别久逛暴走。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/勒通古镇·千户藏寨

### 2.19 姊妹湖

- **门票参考**：公路观景点通常开放式，收费规则以现场/官方公告为准。
- **建议时长**：20—40 分钟。
- **玩法**：这是理塘往巴塘方向很经典的路拍点，适合短暂停车看双湖、高海拔公路和空旷地貌，不需要安排太久。
- **交通**：位于沿线公路旁，自驾体验最好。
- **预约要点**：是否允许停车、观景区域是否调整、道路状况如何，以现场/官方公告为准；注意拍照时不要占道。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/姊妹湖

### 2.20 海子山自然保护区

- **门票参考**：沿线观景点及保护区相关收费以现场/官方公告为准。
- **建议时长**：1—2 小时。
- **玩法**：海子山的魅力在于荒凉、辽阔、冰蚀遗迹感和成片高原海子群，适合作为去稻城路上的“地貌型景观停留点”。
- **交通**：位于理塘—稻城公路沿线，自驾最方便。
- **预约要点**：具体可停留观景点、临时交通管理和开放范围，以现场/官方公告为准；高海拔风大，停留时间别太久。
- **来源链接**：
  - http://www.gzz.gov.cn/
  - https://baike.baidu.com/item/海子山

### 2.21 稻城亚丁景区

- **门票参考**：门票、景交车、电瓶车及其他项目以现场/官方公告为准。
- **建议时长**：1—2 天。
- **玩法**：这是甘孜最具代表性的封面级景区，第一次去建议优先短线或中等强度玩法，先看冲古寺、珍珠海、洛绒牛场，再根据身体状态决定是否继续深入。
- **交通**：通常先到稻城县或香格里拉镇，再换乘景区体系进入；自驾和包车最常见。
- **预约要点**：旺季住宿和门票都建议提前关注；是否限流、当日开放线路、骑马/电瓶车安排等，以现场/官方公告为准。

### 2.22 冲古寺

- **玩法**：它是亚丁景区的重要分流节点，适合短暂停留、看寺院与雪山背景，也是去珍珠海和洛绒牛场的关键中转位置。

- **预约要点**：进入方式、步道开放情况和整体景区安排，以现场/官方公告为准；高海拔环境下别走太急。

## 5. 美食

甘孜的饮食并不是“网红美食城市”路线，更像高原环境下的实用型风味组合：藏餐、川菜、牦牛肉、汤锅、面食、热饮。真正适合大多数游客的吃法，是**别追求花样，优先吃热的、稳的、能补给的**。

### 5.1 值得试的食物

- **牦牛肉汤锅/牦牛肉火锅**：高原地区很常见，适合多人分食，暖身效果直接。
- **酥油茶**：咸香型为主，不一定每个人都一口爱上，但在高海拔环境下很有地方感。
- **牦牛酸奶**：很多游客接受度比酥油茶更高，口感浓厚。
- **糌粑、青稞饼、奶渣**：更偏体验型，适合尝试，不一定适合顿顿吃。
- **康定凉粉**：如果在康定城区停留，可以试试这类更偏川味的小吃。
- **藏式家常菜**：在塔公、理塘、丹巴、稻城一带民宿常见，风味会比景区餐更自然。

### 5.2 哪些地方更适合吃饭

- **康定**：配套最好，选择最多，最适合第一晚补给。
- **新都桥**：游客餐厅多，适合中转，味道稳定性看店。
- **塔公**：适合边看风景边吃简餐或藏餐，别对“精致餐饮”期待过高。
- **理塘/稻城/香格里拉镇**：以家常炒菜、汤锅、面食为主，主打实用。
- **丹巴**：适合住民宿顺带体验嘉绒藏式土菜。

### 5.3 点单建议

- 高反明显时少吃太油太辣。
- 高原环境更适合热汤、面条、粥类，而不是上来就烧烤重口。
- 旺季很多地方上菜慢，不用拿城市效率要求山里餐厅。

## 6. 住宿

### 6.1 第一次去，住哪里最合理

**康定市区**  
适合第一晚过渡，海拔相对更友好，餐饮和补给条件最好。

**新都桥**  
适合摄影和环线中转，去塔公、鱼子西、甲根坝都方便，但旺季价格波动大。

**塔公**  
适合想看草原、白塔、雪山氛围的人，夜里更冷，整体设施平均略弱于新都桥。

**理塘县城**  
适合作为去稻城前的中转，但海拔较高，睡眠质量可能不如康定、新都桥。

**稻城县城/香格里拉镇**  
去亚丁建议优先在这两个点住：
- 想配套更稳，住稻城县城；
- 想第二天进景区更省时间，住香格里拉镇。

**丹巴县城/甲居藏寨**  
想图方便住县城；想看村落清晨和傍晚氛围，直接住寨内更值。

### 6.2 订房建议

- 9—10 月、暑期、节假日一定提前订。
- 看到“雪山房”“观景房”最好问清：是房间直面，还是平台能看到。
- 高海拔住宿最值得确认的不是装修，而是：**供暖、热水、隔音、是否供氧**。

## 7. 交通

### 7.1 如何进入甘孜

- **自驾**：最主流，从成都出发经雅康高速到泸定、康定，再向新都桥、理塘、稻城延伸。
- **包车/拼车**：适合不想自己开山路的人，尤其是鱼子西、亚丁、丹巴支线等区域。
- **班线/客运**：能解决部分城市间位移，但对景点覆盖有限，灵活性一般。
- **飞机**：可关注康定、稻城亚丁相关航线，但要提前评估天气、航班波动和落地接驳成本。

### 7.2 州内移动逻辑

- **康定—新都桥—塔公—八美**：入门最成熟。
- **康定—雅江—理塘—稻城**：去亚丁的主通道。
- **泸定—磨西—海螺沟**：适合加入冰川和温泉线。
- **丹巴—甲居藏寨—党岭/莫斯卡**：更偏支线和轻野体验。

### 7.3 自驾注意事项

- 导航时间常常偏乐观，实际会被停车、修路、堵车、拍照频繁拉长。
- 别在高原把“再开 2 小时”想得太轻松，山路和夜路会放大疲劳。
- 很多观景点真正耗时在“找停车位、等光线、补给和厕所”。
- 如果第二天要冲亚丁、鱼子西日落或冷门支线，前一晚一定别睡太晚。

## 8. 避坑

### 8.1 不要把甘孜当作普通城市游

这是整个州域旅行，不是一个县城 walk。地图上看着不远，实际 2—5 小时山路很常见，一天塞太多核心点位基本等于纯赶路。

### 8.2 第一天别硬冲最猛的高海拔点

第一次进高原就冲鱼子西、理塘、亚丁长线，翻车概率很高。更稳的顺序永远是：**康定过渡 → 新都桥/塔公适应 → 再决定要不要继续冲更高。**

### 8.3 别把“天气预报晴”当成拍到雪山的保证

川西天气变化非常快，尤其午后云层经常上来。真想看雪山，尽量给晨景和备用时段，而不是只押一个日落窗口。

### 8.4 旺季不要临时订房

新都桥、塔公、稻城、香格里拉镇旺季非常容易出现“只剩贵房或差房”的情况，尤其秋季和国庆。

### 8.5 山路别逞强夜驾

鱼子西、党岭、莫斯卡、子梅垭口、冷嘎措这类支线，不适合没有经验的人硬开夜路。看完日落要评估司机状态和下山风险。

### 8.6 真正贵的往往不是门票，而是综合成本

很多地方贵在景交、换乘、越野车、住宿、时间和天气损耗。做预算时别只盯着“大门票”。

### 8.7 尊重寺庙、村寨和当地生活边界

塔公寺、藏寨、莫斯卡等地，不要把别人生活区当拍照布景。拍人先问，同意再拍；法器、经幡、宗教空间别乱碰。

### 8.8 防晒和保暖一个都别省

高原最容易被低估的不是冷，是晒；最容易被高估的不是白天，是夜里。白天晒伤、晚上冻着，都是川西游客经典翻车点。
