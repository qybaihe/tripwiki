# 黔东南苗族侗族自治州（贵州）

- 更新时间：2026-03-12
- 推荐指数（整体）：⭐⭐⭐⭐⭐（贵州最适合做“民族村寨 + 古城河谷 + 梯田田园 + 山地漂流 + 深度县域旅行”组合的一州，第一次去贵州但不想只看贵阳黄果树的人，来这里通常不会空手而归）

## 城市概览

**贵州省黔东南苗族侗族自治州**位于**贵州省东南部**，州府为**凯里市**。这不是一个“到州府逛一圈就算来过”的普通城市型目的地，而是典型的**州域旅行目的地**：从凯里出发，向东有镇远古城、舞阳河、青龙洞这样的历史与山水组合；向南有西江千户苗寨、朗德苗寨、雷公山等苗族文化核心区；向东南有肇兴侗寨、堂安梯田、从江占里侗寨、岜沙苗寨这样的侗乡与苗乡腹地；向北和东北还有施秉云台山、杉木河、丹寨、剑河温泉、榕江侗寨群等风格完全不同的线路。

黔东南苗族侗族自治州最大的魅力，不是“某一个网红景区特别火”，而是这里**民族文化不是摆拍背景，而是日常生活本身**。鼓楼、风雨桥、吊脚楼、寨门、银饰、侗族大歌、苗年、酸汤、腌鱼、糯食、赶集和村寨公共空间，很多都还保留着真实生活气息。也正因为如此，黔东南更适合：

1. **别贪多**：一次选 1 条主线，体验会远好于“每天换一个县”；
2. **别只追网红**：西江和肇兴固然值得去，但朗德、岜沙、堂安、隆里、大利、占里这类次级点位，往往才是这州真正有记忆点的部分；
3. **别只看夜景**：很多村寨和梯田真正好看的时间，是清晨、午后和傍晚，而不是纯夜游；
4. **别把它当演出景区**：这里最值得看的往往是建筑肌理、人与公共空间、山地聚落格局，以及围绕村寨形成的生活方式。

如果你是第一次来**贵州省黔东南苗族侗族自治州**，最稳妥的思路通常有三种：
- **经典首刷线**：凯里 → 西江千户苗寨 → 镇远古城
- **侗乡深度线**：黎平肇兴侗寨 → 堂安梯田 → 从江占里/岜沙 → 榕江侗寨群
- **自然人文混合线**：凯里 → 雷公山/下司古镇 → 镇远 → 施秉云台山/杉木河

一句话总结：**黔东南苗族侗族自治州不适合暴走式集邮，适合放慢节奏、以县域为单位做深度体验。**

## 行前速览

### 适合谁
- 第一次来贵州，想看民族村寨、古城和山地人文的人；
- 喜欢摄影、建筑、田园、古镇、慢旅行的人；
- 对苗族、侗族文化有兴趣，不满足于“看个演出就走”的人；
- 自驾、包车或愿意接受县域转车的人；
- 想避开一线热门景区套路感，寻找更有地方感旅行的人。

### 不太适合谁
- 只想住在一个中心城区步行玩遍所有景点的人；
- 完全不接受山路、县城转车、村寨接驳的人；
- 对非标准化住宿、山地天气变化、村寨夜间安静环境不适应的人。

### 建议天数
- **1 天**：只能玩西江千户苗寨 / 镇远古城 / 肇兴侗寨中的一个核心点；
- **2 天**：适合西江 + 镇远，或肇兴 + 堂安；
- **3 天**：比较像样，可做凯里—西江—镇远，或黎平—肇兴—从江线；
- **4-5 天**：适合做州域精华线，把苗寨、侗寨、古城、梯田和自然线拼起来；
- **6 天以上**：可以把榕江、丹寨、剑河、施秉、雷山、从江等分区玩得更完整。

### 最佳季节
- **3-5 月**：山地转绿、气温舒服，适合古城和村寨慢游；
- **6-8 月**：避暑属性明显，梯田和山地景观好，但雨水较多、周末旺季游客也多；
- **9-10 月**：天气通常更稳，摄影体验更好，堂安、加榜等田园线更舒服；
- **11 月-次年 2 月**：古城、温泉、侗寨可玩，但山里早晚偏冷，部分梯田和山地景观观感弱一些。

### 交通进入
- **高铁**：凯里南站是进入州内最方便的高铁口岸之一，镇远、榕江、从江、三穗、施秉等也有高铁或铁路节点；
- **飞机**：常见从贵阳龙洞堡机场转高铁/大巴进入，也可关注黎平机场、凯里黄平机场等航线情况；
- **自驾**：非常适合做黔东南深度线，但山路、县道、夜路都不建议硬赶；
- **公共交通**：州内跨县以高铁 + 客运 + 打车/景区接驳组合最常见，能玩，但灵活性比自驾差。

### 住宿建议
- **首刷经典线**：凯里住 1 晚 + 西江住 1 晚 + 镇远住 1 晚，是最舒服的节奏；
- **西江千户苗寨**：建议住景区内半山或观景位，夜景和清晨体验差别很大；
- **镇远古城**：优先住舞阳河沿岸老城步行区，夜景和散步都方便；
- **肇兴侗寨**：建议住寨内鼓楼区附近或风雨桥附近，方便清晨和夜晚慢逛；
- **从江 / 榕江 / 丹寨 / 剑河**：若做县域深度，更适合住县城或成熟村寨民宿，不要指望全程都像成熟景区那样标准化。

### 预算感受
- 州内大部分县城和村寨消费比一线旅游城市友好；
- 但西江、镇远、肇兴等热门点位节假日房价波动明显；
- 真正拉高预算的，往往不是门票，而是**多段交通、包车和旺季住宿**；
- 深度玩侗寨/苗寨时，住一晚通常比“匆匆打卡离开”更值。

### 核心提醒
1. 本文对象明确为**贵州省黔东南苗族侗族自治州**，不是“黔东南某一县城攻略”；
2. 村寨类景点更适合白天和清晨慢走，不要只夜里进寨拍完灯光就走；
3. 雨季山地常有雾、阵雨、道路湿滑，鞋子和雨具要靠谱；
4. 很多村寨的价值在“整体空间与生活状态”，不是看完一个表演就结束；
5. 若想拍清晨梯田、寨雾、河谷晨光，尽量住进景区或附近；
6. 自驾别高估自己夜间跑山路的能力，州域范围比地图上看起来更大。

## 1-3 天行程

### 1 天快览版：西江千户苗寨线
- 上午：凯里出发 → **西江千户苗寨**入寨，白天先逛观景台下方街巷、风雨桥、博物馆
- 中午：苗寨内吃酸汤鱼、糯米饭、腊肉、米酒
- 下午：深走吊脚楼区和观景步道，避开纯主街游客流
- 傍晚：上观景台看千户灯火亮起
- 晚上：住西江景区内，第二天一早拍晨景再走

适合第一次到贵州省黔东南苗族侗族自治州、时间最紧的人。

### 2 天经典版：西江 + 镇远
- **D1**：凯里 → 西江千户苗寨（白天深走 + 夜景）→ 住西江
- **D2**：西江出发 → 镇远古城 → 青龙洞 / 祝圣桥 / 舞阳河沿岸 → 夜宿镇远或返程

这是首刷黔东南最稳的组合：一个看苗寨山地聚落，一个看古城河谷历史。

### 3 天经典版：凯里 + 西江 + 镇远
- **D1**：凯里市区 → 黔东南州民族博物馆 → 下司古镇 / 凯里酸汤体验 → 住凯里
- **D2**：凯里 → 西江千户苗寨 → 朗德苗寨（二选一或串联）→ 住西江/雷山
- **D3**：前往镇远古城 → 青龙洞 → 祝圣桥 → 夜景后返程

适合想先建立州域认知，再进入热门核心景区的人。

### 3 天侗乡深度版：肇兴 + 堂安 + 从江
- **D1**：黎平/从江到达 → **肇兴侗寨** → 五团鼓楼群、风雨桥夜景 → 住肇兴
- **D2**：清晨前往**堂安梯田**步道 → 下午去**从江岜沙苗寨**或**占里侗寨** → 住从江
- **D3**：补**大利侗寨**或榕江方向侗寨群 → 返程

适合喜欢侗族村寨、梯田、建筑和慢游的人。

## 景点清单（精选 20 个）

> 说明：以下景点均位于**贵州省黔东南苗族侗族自治州**州域范围内。门票、景交、演出、索道、漂流、停车等价格会因旺淡季和政策调整而波动，以下为 2026 年前后常见公开口径/旅游市场参考值，出发前请以官方购票页、景区公众号、现场公示为准。

### 1. 西江千户苗寨（雷山县）
- **门票参考**：成人票常见约 90 元，4 程景区观光车常见约 20 元左右
- **建议时长**：半天-1 天，推荐住 1 晚
- **玩法**：看世界级规模苗族吊脚楼聚落、观景台夜景、风雨桥、苗寨街巷、刺绣银饰与博物馆。第一次来黔东南，优先级通常就是它。
- **交通**：从凯里南站/凯里市区出发最方便，自驾、景区直通车、包车都常见
- **预约要点**：节假日和暑期房价、客流都很夸张，务必提前订票订房；如果想拍晨景和空街，最好住景区内而不是当天往返
- **来源链接**：
  - <https://www.xjqhmz.com/>
  - <https://baike.baidu.com/item/西江千户苗寨>

### 2. 西江千户苗寨观景台（雷山县）
- **门票参考**：通常包含在西江千户苗寨门票体系内
- **建议时长**：30 分钟-1.5 小时
- **玩法**：傍晚和夜晚看千户灯火，清晨看寨雾与层层吊脚楼，是西江最经典的机位之一
- **交通**：景区内步行或乘景交到达后再步行
- **预约要点**：傍晚人最多，想拍空镜建议一早上去；雨雾天反而可能比大晴天更有层次
- **来源链接**：
  - <https://www.xjqhmz.com/>
  - <https://baike.baidu.com/item/西江千户苗寨>

### 3. 西江苗族博物馆（雷山县）
- **门票参考**：通常包含在景区游览体系内或免费开放
- **建议时长**：40 分钟-1.5 小时
- **玩法**：适合在进寨第一天先看，能把苗族服饰、银饰、民俗、聚落结构先建立基础认知
- **交通**：位于西江千户苗寨景区内，步行可达
- **预约要点**：以现场开放时间为准，中午或闭馆前不要卡点冲
- **来源链接**：
  - <https://www.xjqhmz.com/>
  - <https://baike.baidu.com/item/西江千户苗寨>

### 4. 朗德苗寨（雷山县）
- **门票参考**：常见约 50 元上下
- **建议时长**：2-3 小时
- **玩法**：相比西江更小而安静，适合看传统木构苗寨、寨门、鼓藏坪和较少商业化的村寨氛围
- **交通**：从凯里或西江出发包车/自驾较方便，也可走一日游串联
- **预约要点**：更适合白天慢逛，不建议只为了夜景去；若遇民俗演出，以现场安排为准
- **来源链接**：
  - <https://baike.baidu.com/item/朗德上寨>
  - <https://www.mct.gov.cn/>

### 5. 雷公山国家级自然保护区（雷山县）
- **门票参考**：景区/保护区不同点位收费不同，常见几十元区间
- **建议时长**：半天-1 天
- **玩法**：看黔东南中部山地森林、云海、峡谷和苗岭高山景观，适合喜欢自然线和徒步的人
- **交通**：通常从雷山、凯里、西江方向转入，自驾/包车最方便
- **预约要点**：天气影响大，雨雾天别低估山路湿滑；并非纯轻松打卡点，出发前确认开放点位
- **来源链接**：
  - <https://baike.baidu.com/item/雷公山>
  - <https://www.mee.gov.cn/>

### 6. 镇远古城（镇远县）
- **门票参考**：古城街区免费，个别展馆/游船单独收费
- **建议时长**：半天-1 天，推荐住 1 晚
- **玩法**：看舞阳河穿城而过、明清老街、河岸吊脚楼、夜景灯火与山城河谷格局。它和一般平地古镇很不一样，最大的看点是“城在山中、河在城中”。
- **交通**：镇远站或高铁/铁路中转后打车进古城方便，自驾也成熟
- **预约要点**：古城公共区域无需预约；节假日核心河岸房价上涨明显，停车也不轻松，建议尽量住步行区
- **来源链接**：
  - <https://baike.baidu.com/item/镇远古城>
  - <https://www.mct.gov.cn/>

### 7. 青龙洞古建筑群（镇远县）
- **门票参考**：常见约 60 元左右
- **建议时长**：1-2 小时
- **玩法**：临崖而建的道佛儒合一古建筑群，是镇远最有辨识度的人文点位之一，适合登高看古城河景
- **交通**：位于镇远古城核心区，步行可达
- **预约要点**：台阶较多，穿舒适鞋；临江和高处拍照注意安全
- **来源链接**：
  - <https://baike.baidu.com/item/青龙洞古建筑群>
  - <https://www.mct.gov.cn/>

### 8. 祝圣桥（镇远县）
- **门票参考**：免费
- **建议时长**：20 分钟-1 小时
- **玩法**：镇远古城标志性古桥，白天看桥体和河景，晚上看灯火倒影都很值
- **交通**：古城内步行串联最方便
- **预约要点**：无需预约；夜景时段游客密度高
- **来源链接**：
  - <https://baike.baidu.com/item/祝圣桥>
  - <https://baike.baidu.com/item/镇远古城>

### 9. 舞阳河景区（镇远县/施秉县）
- **门票参考**：游船类项目常见约 100-160 元区间，具体看航线和码头
- **建议时长**：2-4 小时
- **玩法**：看喀斯特峡谷、碧水河谷、两岸奇峰，适合把镇远古城的人文线延伸为山水线
- **交通**：通常从镇远或施秉方向接驳前往，依码头而定
- **预约要点**：看天气和水位安排，雨季需关注停航或调整公告；游船项目最好提前确认班次
- **来源链接**：
  - <https://baike.baidu.com/item/舞阳河>
  - <https://www.mct.gov.cn/>

### 10. 施秉云台山（施秉县）
- **门票参考**：成人票常见约 80 元上下
- **建议时长**：3-5 小时
- **玩法**：看白云岩喀斯特峰丛、绝壁、天坑和山体台地，是黔东南自然景观里很能打的一张牌
- **交通**：从施秉县城或镇远方向前往较方便，自驾/包车更稳妥
- **预约要点**：山地步道较多，夏季注意防晒补水；雨后云雾状态可能更好，但也更滑
- **来源链接**：
  - <https://baike.baidu.com/item/施秉云台山>
  - <https://whc.unesco.org/en/list/1414/>

### 11. 杉木河漂流（施秉县）
- **门票参考**：漂流票常见约 180-220 元区间
- **建议时长**：半天
- **玩法**：以清水漂流、山谷水体和夏季清凉体验出名，是黔东南热门夏季项目之一
- **交通**：通常从施秉县城接驳，自驾和跟团都多
- **预约要点**：强季节性项目，雨量、水位、气温都会影响开放；要看景区当天公告，带好换洗衣物
- **来源链接**：
  - <https://baike.baidu.com/item/杉木河>
  - <https://www.mct.gov.cn/>

### 12. 下司古镇（凯里市）
- **门票参考**：古镇公共区域多为免费或低门槛，个别项目另计
- **建议时长**：2-3 小时
- **玩法**：看清水江沿岸码头古镇格局、桥梁、老街与凯里周边较成熟的休闲古镇氛围，适合作为凯里到西江前后的补位点
- **交通**：距凯里市区不远，打车、自驾方便
- **预约要点**：更适合闲逛和吃饭，不是硬核历史遗址型古镇；周末人比工作日多
- **来源链接**：
  - <https://baike.baidu.com/item/下司古镇>
  - <https://www.mct.gov.cn/>

### 13. 黔东南州民族博物馆（凯里市）
- **门票参考**：常设展通常免费开放
- **建议时长**：1-2 小时
- **玩法**：这是进入州域旅行前非常值得看的知识补充点，能把苗族、侗族、服饰、乐器、建筑、节庆先看明白
- **交通**：位于凯里市区，适合与市区住宿日同游
- **预约要点**：以馆方开放时间为准，部分时段可能需要实名登记
- **来源链接**：
  - <https://baike.baidu.com/item/黔东南州民族博物馆>
  - <https://www.mct.gov.cn/>

### 14. 肇兴侗寨（黎平县）
- **门票参考**：成人票常见约 80 元，景区接驳/摆渡视购票方式而定
- **建议时长**：半天-1 天，推荐住 1 晚
- **玩法**：看“仁义礼智信”五团鼓楼群、侗族风雨桥、寨中河道与夜间鼓楼灯光，是贵州侗寨旅行的代表性目的地之一
- **交通**：从从江站、黎平机场、黎平县城等地前往都较常见，自驾/包车最灵活
- **预约要点**：住寨内体验明显优于匆匆往返；旺季鼓楼周边住宿会涨价，尽量提前订
- **来源链接**：
  - <https://baike.baidu.com/item/肇兴侗寨>
  - <https://www.mct.gov.cn/>

### 15. 堂安梯田（黎平县）
- **门票参考**：常见与肇兴联动套票或接驳收费，整体门槛不高
- **建议时长**：2-4 小时
- **玩法**：看山地梯田、侗寨村落、传统步道和田园景观，适合从肇兴徒步或乘车上去再走下来
- **交通**：通常从肇兴侗寨接驳前往，景区车或包车最常见
- **预约要点**：早晨和傍晚更好看；下雨后步道湿滑，但云雾层次也往往更漂亮
- **来源链接**：
  - <https://baike.baidu.com/item/堂安侗寨>
  - <https://baike.baidu.com/item/肇兴侗寨>

### 16. 岜沙苗寨（从江县）
- **门票参考**：成人票常见约 40-60 元区间
- **建议时长**：2-3 小时
- **玩法**：看“最后的枪手部落”形象背后的苗寨聚落、发式、服饰和民俗展示，是从江方向最有辨识度的点位之一
- **交通**：从从江县城或从江高铁站出发较方便，自驾/打车/包车均可
- **预约要点**：民俗展示时间以现场安排为准，不要只冲着表演去；更值得看的是寨子本身和村寨氛围
- **来源链接**：
  - <https://baike.baidu.com/item/岜沙苗寨>
  - <https://www.mct.gov.cn/>

### 17. 占里侗寨（从江县）
- **门票参考**：常见为免费或低门槛进入，个别讲解/停车另计
- **建议时长**：1.5-3 小时
- **玩法**：看侗族村寨格局、水系、鼓楼和较生活化的寨居环境，适合和从江/肇兴串联做深度线
- **交通**：从从江县城出发包车/自驾较方便
- **预约要点**：这里更偏生活型村寨，不是高强度商业景区；尊重村民日常，不要把每个角落都当摆拍道具
- **来源链接**：
  - <https://baike.baidu.com/item/占里侗寨>
  - <https://www.mct.gov.cn/>

### 18. 大利侗寨（榕江县）
- **门票参考**：多为免费或低门槛进入
- **建议时长**：2-3 小时
- **玩法**：木构鼓楼、古井、水系与侗寨肌理保存较好，是榕江方向很值得慢看的村寨点位
- **交通**：从榕江县城出发自驾/包车更方便
- **预约要点**：适合白天慢走；如果想拍村寨空间与建筑细节，阴天反而更稳定
- **来源链接**：
  - <https://baike.baidu.com/item/大利侗寨>
  - <https://www.mct.gov.cn/>

### 19. 隆里古城（锦屏县）
- **门票参考**：古城街区多为免费或低门槛，个别展陈另计
- **建议时长**：2-3 小时
- **玩法**：看明代军事屯堡色彩的古城格局、街巷和城墙遗存，和镇远的河谷古城风格完全不同
- **交通**：从锦屏县或黎平、天柱方向串联较常见，自驾最灵活
- **预约要点**：更适合对古城肌理和历史背景有兴趣的人；商业化不算重，配套也不要期待太高
- **来源链接**：
  - <https://baike.baidu.com/item/隆里古城>
  - <https://www.mct.gov.cn/>

### 20. 丹寨万达小镇（丹寨县）
- **门票参考**：小镇开放区免费，演艺、非遗体验项目单独收费
- **建议时长**：2-4 小时，可住 1 晚
- **玩法**：适合作为相对轻松、配套成熟的丹寨入口，看非遗工坊、夜景、美食和周边苗侗文化展示
- **交通**：从凯里、都匀、贵阳方向自驾/大巴都较常见
- **预约要点**：它更像旅游综合体和州域分流入口，不要把它当成“原生古寨”；如果想看更真实村寨，可与附近乡村点位联动
- **来源链接**：
  - <https://baike.baidu.com/item/丹寨万达小镇>
  - <https://www.mct.gov.cn/>

## 美食

**贵州省黔东南苗族侗族自治州**的吃，核心不是“大菜排场”，而是非常鲜明的地方风味：酸、糯、发酵、烟熏、山野、米食和河鲜。

### 必吃清单
- **凯里酸汤鱼**：最稳的州域名片，红酸汤更常见，适合第一次来的人先吃一顿正宗版本；
- **酸汤牛肉 / 酸汤鸡**：不吃鱼的人也有替代项；
- **侗家腌鱼、腌肉**：肇兴、从江、榕江一带更常见，带发酵风味；
- **糯米饭、糍粑、灰碱粽、油茶**：早餐和小吃系统很强；
- **牛瘪、羊瘪**：重口地方菜，能接受再试，不建议无脑挑战；
- **米豆腐、米粉、粉皮、卷粉**：各县城早餐体系都不错；
- **鼓藏肉、腊肉、熏肉、苗家土鸡**：村寨民宿常见；
- **侗家合拢宴 / 长桌宴**：偏体验型，用来感受氛围可以，但不必对菜品精致度期待过高；
- **米酒、甜酒、糯米酒**：很多苗寨侗寨都会遇到，能喝再喝；
- **从江香猪、榕江小黄牛相关菜式**：往南部县域玩时可重点留意。

### 去哪吃更稳
- **凯里市区**：酸汤体系最成熟，第一次来在凯里吃一顿，比在纯景区里盲点更稳；
- **西江千户苗寨**：适合体验苗家菜和夜晚吃饭氛围，但价格通常高于凯里；
- **镇远古城**：适合河边吃晚饭，偏“古城旅游餐饮 + 黔东南本地菜”的组合；
- **肇兴 / 从江 / 榕江**：适合吃侗家、腌鱼、糯食、小锅菜，但标准化程度没大城市高。

### 点菜建议
- 第一次来：**酸汤鱼 + 糯米饭 + 腊肉拼盘 + 时令野菜**，基本不会出错；
- 如果住寨内民宿：可以问有没有**家常版苗家/侗家菜**，往往比旅游套餐更值得；
- 对重口发酵类食物谨慎一点，先少量尝试。

## 住宿建议

### 凯里
- **适合谁**：高铁进出、想住得方便、想先逛博物馆和市区吃饭的人；
- **优点**：交通最好、吃饭方便、价格选择多；
- **缺点**：旅行氛围不如景区和古城强。

### 西江千户苗寨
- **适合谁**：第一次来黔东南，想看夜景和清晨苗寨的人；
- **优点**：夜景、晨景、寨内慢游体验完整；
- **缺点**：旺季房价浮动大，山地步行和搬行李比较折腾。

### 镇远古城
- **适合谁**：喜欢古城夜景、河边散步、拍照和慢节奏的人；
- **优点**：夜里非常适合闲逛；
- **缺点**：自驾停车不算特别友好，旺季临河房价格高。

### 肇兴侗寨
- **适合谁**：偏爱侗族建筑、鼓楼风雨桥和清晨村寨氛围的人；
- **优点**：住寨内体验感强，适合发呆；
- **缺点**：配套没有成熟城市那么标准化，旺季热门房型要抢。

### 县城住宿（从江、榕江、丹寨、剑河等）
- **适合谁**：做深度县域线、重视第二天赶路效率的人；
- **优点**：价格通常更友好，停车方便；
- **缺点**：旅行氛围不如住在核心村寨里。

## 交通指南

### 外部到达
- **贵阳中转**：最稳妥的方式。先到贵阳，再高铁到凯里南、榕江、从江、镇远等节点；
- **高铁直达县域**：现在进入黔东南很多核心县更方便了，像榕江、从江等可直接作为起点；
- **自驾**：适合串联多个县，但务必留足车程，不要把地图直线距离当成实际耗时。

### 州内移动
- **凯里 → 西江**：常见景区直通车、班车、打车/包车；
- **凯里 → 镇远**：铁路/高铁 + 打车都方便；
- **肇兴 → 堂安**：景区接驳或包车最实用；
- **从江/榕江侗寨群**：更适合包车或自驾；
- **施秉云台山 / 杉木河**：县城出发的接驳和包车都比较常见。

### 是否适合自驾
- **适合**，尤其是 3 天以上深度线；
- 但不建议为了省住宿费天天夜间跨县赶路；
- 村寨道路、山路、雨天和雾天都要保守驾驶。

## 避坑提示

1. **不要把贵州省黔东南苗族侗族自治州玩成“西江一日游”**。西江值得，但它只代表黔东南的一部分。
2. **不要只为夜景冲进西江或镇远当天来回**。这两个地方都更适合住一晚。
3. **不要高估公共交通串小村寨的效率**。深度玩侗寨、苗寨，包车往往更省时间。
4. **不要把所有村寨都想成同一种风格**。西江偏规模与夜景，朗德偏传统苗寨，肇兴偏侗寨鼓楼系统，岜沙更偏民俗辨识度，占里/大利更生活化。
5. **不要只看表演时间表**。真正有价值的常常是早晨的寨门、鼓楼边的老人、木楼之间的坡道、水边洗衣和村寨整体空间。
6. **雨季别穿平底光滑鞋**。古城石板路、村寨木梯、山地步道都挺容易滑。
7. **在热门景区吃饭别盲点“长桌宴套餐”**。想吃得稳，可以先看本地口碑馆子或回凯里吃酸汤。
8. **拍照要尊重居民生活**。别把镜头怼到老人、小孩和屋内空间上，更不要擅自拍摄祭祀或私密活动。
9. **别迷信所有“官方看起来像官网”的站点**。州内不少旅游信息以公众号、政府文旅号、景区现场公告为准，第三方网页经常过时。
10. **节假日尽早订房**。西江、镇远、肇兴都是典型旺季一房难求区域。
