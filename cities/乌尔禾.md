# 乌尔禾

- 更新时间：2026-03-16
- 适合人群：北疆自驾玩家、首次看雅丹地貌的人、世界魔鬼城摄影党、亲子地质科普家庭、克拉玛依延展2—3天旅行者
- 推荐停留：1—2天；若联动克拉玛依主城区可安排2—3天
- 城市关键词：世界魔鬼城、雅丹主场、荒野公路线、艾里克湖、胡杨林、影视城、百口泉、克拉玛依北翼、自驾节点

## 城市概览
乌尔禾最容易被低估的地方，是很多人只把它理解成“去世界魔鬼城看一眼”的单点景区。实际上，**乌尔禾真正成立的旅行逻辑，不是一个景区，而是一整套“雅丹主场 + 荒野公路线 + 湖泊/胡杨补充 + 克拉玛依联动”的小型目的地结构**。

先把定位说清楚：如果克拉玛依整座城市是“石油文明 + 公路旅行”的骨架，那么乌尔禾就是这套骨架里最具视觉冲击力、最像北疆荒野名片的那一段。它最强的不是城市漫游感，而是**世界魔鬼城这一国家级、世界级辨识度的雅丹地貌主场**。很多新疆旅行者会在不同地方见到“魔鬼城”或雅丹，但乌尔禾的优势在于：景区成熟度高、可达性强、地貌形态有代表性、与公路旅行的结合度极高，而且第一次看雅丹的人很容易在这里建立清晰认知。

乌尔禾的第二层价值，在景区之外。真正会玩的人，不会把乌尔禾压缩成“进景区坐观光车、拍几张土丘、转身就走”。因为乌尔禾精彩的地方，还包括**从克拉玛依主城区北上到乌尔禾的荒野公路线、风城地貌、开阔戈壁、低饱和色带、湖面与风车的过渡，以及秋季胡杨作为色彩补丁的节奏变化**。也就是说，乌尔禾不是“只有一个爆点、其余都凑数”的地方；它更像一条以世界魔鬼城为核心、周边多个中小型点位为支撑的荒野体验线。

第三层才是旅行执行。乌尔禾适合和克拉玛依联动，但不能被克拉玛依吞掉。最实用的思路通常是：**克拉玛依主城区负责住宿补给、城市理解和亲子舒适度；乌尔禾负责雅丹主场、摄影时段和荒野感；如果时间再多，再看是否接独山子或阿勒泰方向。** 这样排，逻辑顺、体验也稳定。

我对乌尔禾的定位很明确：**它不是“大而全”的县区型城市目的地，而是北疆少数非常适合专门为一片雅丹与一段公路停下来的地方。** 第一次来这里，关键不是景点越多越好，而是要把“世界魔鬼城为什么值得单独来、乌尔禾公路线怎么跑、是否住一晚、怎么兼顾摄影和亲子”这几个执行问题处理对。

## 行前速览

### 最佳季节
- **5—6月：** 气温相对友好，风大但还算适合长时间户外，适合第一次刷世界魔鬼城。
- **7—8月：** 日照最强，白天热、晒、干，但公路条件与整体可玩性高，适合自驾党；中午体验会明显下降。
- **9—10月：** 全年最推荐。魔鬼城傍晚光线更柔和，乌尔禾胡杨林更有价值，摄影与亲子都更好安排。
- **11月—次年4月：** 可以去，但更多偏“荒凉感”和路过型打卡；部分户外点位观感、舒适度和服务配套下降。

### 这地方最正确的打开方式
- **第一次来：一定把世界魔鬼城放在下午到日落时段。** 这是乌尔禾体验成败的第一关键。
- **不要只看景区，要看公路。** 从克拉玛依北上乌尔禾，本身就是一条很适合建立地貌认知的路。
- **想拍照：优先住乌尔禾。** 日落后再回克拉玛依主城区，会多出一段疲劳返程。
- **带孩子：主打“景区主线 + 一个轻松补充点”即可。** 不要一天硬塞太多野点。

### 推荐停留时长
- **半天：** 只能做世界魔鬼城快刷版，且必须卡下午光线。
- **1天：** 最实用，可完成“世界魔鬼城 + 影视城/艾里克湖/公路停靠点”中的2—3项。
- **2天：** 最适合摄影党、亲子家庭和不想太赶的人，可加入胡杨林、百口泉、露营小镇等。
- **2—3天联动版：** 与克拉玛依主城区组合最完整，建议“主城区1天 + 乌尔禾1天 + 机动半天/1天”。

### 气候与装备
- 风大、干燥、紫外线硬，是乌尔禾最稳定的旅行背景。
- 墨镜、帽子、防晒霜、润唇膏、保湿喷雾基本全年建议带。
- 景区和荒野公路停靠点尽量穿包脚鞋，不建议拖鞋。
- 摄影设备注意防沙；换镜头尽量在车里完成。
- 带娃建议额外准备：湿巾、遮阳帽、足量饮水、轻零食、防风薄外套。

### 交通结论
- **最优解是自驾。** 乌尔禾真正的体验很依赖公路线和多个分散点位。
- **不会开车也能来，但更适合“包车/拼车 + 魔鬼城主线”。**
- **如果只靠临时打车硬串多个点，体验通常会比较差。**

### 住宿结论
- **只想稳：** 住克拉玛依主城区，配套强、吃饭方便。
- **想拍世界魔鬼城日落/次日早起继续跑线：** 住乌尔禾更合理。
- **亲子家庭：** 低龄儿童更适合住克拉玛依；中大童家庭可考虑在乌尔禾住1晚减少折返。

### 预约与信息判断
- 世界魔鬼城旺季、节假日建议提前购票。
- 博物馆、纪念馆类点位留意闭馆日与实名要求。
- 荒野型、开放型点位不等于可以乱走，很多地方真正要看的不是“下道探险”，而是“在合法安全的停靠点看大地貌”。

## 1—3天行程

### 1天快刷版｜第一次来乌尔禾的最优解
**适合：** 从克拉玛依出发的一日游、自驾过路、第一次看雅丹

**路线：**
克拉玛依出发 → 乌尔禾荒野公路线沿途停靠拍摄 → 世界魔鬼城（下午—日落主时段）→ 乌尔禾影视城/西部乌镇（二选一）→ 夜宿乌尔禾或返克拉玛依

**执行重点：**
- 上午别把体力耗在碎点位，真正高潮要留给魔鬼城。
- 如果天气通透、风不大，可以在进乌尔禾前后安排1—2个公路停靠点拍环境照。
- 带娃家庭更建议“魔鬼城 + 晚餐 + 夜宿/返程”，不要再强塞胡杨林和太多野外点。

### 2天经典版｜乌尔禾本地深度线
**D1：世界魔鬼城主场日**
- 中午前后抵达乌尔禾
- 先在城区午餐、补水、休整
- 下午进入世界魔鬼城
- 日落后视体力安排影视城或乌尔禾夜间休闲区
- 夜宿乌尔禾

**D2：公路线 + 湖泊/胡杨补充线**
艾里克湖 → 乌尔禾胡杨林（秋季优先）→ 百口泉知青村/百口泉公园 → 房车露营公园或荒野公路观景停靠点 → 返回克拉玛依

这个版本最好的一点，是把“乌尔禾真正主菜”和“乌尔禾周边辅助层”拆开，节奏更顺，也更适合亲子。

### 3天联动版｜乌尔禾 + 克拉玛依完整玩法
**D1：克拉玛依主城区**
黑油山景区 → 克拉玛依博物馆 → 克拉玛依河夜景

**D2：乌尔禾魔鬼城核心线**
克拉玛依出发 → 乌尔禾公路线 → 世界魔鬼城 → 乌尔禾影视城 → 夜宿乌尔禾

**D3：乌尔禾补充线后返程**
艾里克湖 → 胡杨林/百口泉（二选一或组合）→ 返克拉玛依；若后续继续自驾，可顺接独山子、阿勒泰或福海方向

### 摄影版执行逻辑
- **第一原则：魔鬼城一定卡傍晚侧光。** 正午只能做踩点，真正出片在下午后段。
- **第二原则：乌尔禾不只拍“土丘”，还要拍“公路—戈壁—风车—低矮雅丹”的空间关系。**
- **第三原则：住乌尔禾比回克拉玛依更利于日落后、蓝调时刻和第二天晨间补拍。**
- **镜头建议：** 广角拍大场景，中焦拍雅丹轮廓层次，长焦压缩风车/道路与地貌关系。

### 亲子版执行逻辑
- **低龄儿童：** 最稳是住克拉玛依，白天只冲魔鬼城主线，晚上回城休息。
- **中大童：** 可以住乌尔禾1晚，做“魔鬼城 + 影视城 + 艾里克湖/胡杨林”两段式安排。
- **亲子讲解重点：** 岩石风蚀、雅丹形成、荒漠植被、石油城市与戈壁环境关系，比硬刷景点更有价值。
- **不建议：** 中午长时间暴晒、把泥火山或无护栏荒野边缘当成儿童放风区。

## 景点

> 下面收录 18 个乌尔禾及强相关点位。考虑到乌尔禾大量体验价值来自“景区 + 公路 + 开放式地貌/休闲节点”的组合，以下既包含标准景区，也包含在实际行程里很有价值的功能型点位。门票、区间车、开放时间、道路条件会有季节波动，出发前请再核对景区售票页、政府公告与现场公示。

### 1. 世界魔鬼城（乌尔禾世界魔鬼城景区）
- **门票参考：** 成人票常见约42—60元；含区间车/联票常见约70—100元，以当日售票页为准
- **建议时长：** 3—5小时
- **玩法：** 乌尔禾旅行绝对核心。这里不是简单“看几堆土”，而是看典型雅丹残丘、风蚀城堡、孤峰、沟槽与光线塑形。第一次来乌尔禾，主场必须给它。
- **交通：** 位于乌尔禾区，自驾最方便；从克拉玛依主城区往返通常需半天以上
- **预约要点：** 旺季建议提前购票；尽量下午入园；风大时注意防沙，部分项目/区域以景区当天安排为准
- **来源链接：**
  - https://www.mct.gov.cn/
  - https://www.xinjiang.gov.cn/
  - https://you.ctrip.com/sight/kelamayi335/15124.html

### 2. 乌尔禾荒野雅丹公路线（克拉玛依—乌尔禾沿线风城地貌停靠带）
- **门票参考：** 免费
- **建议时长：** 1—2小时（通常作为沿途停靠）
- **玩法：** 这是乌尔禾攻略里最值得强化的一条线。真正的“乌尔禾感”不只在景区里，也在一路开过去时不断出现的风蚀地貌、砾石戈壁、长直公路与风车阵列。适合拍公路旅行感和大地景关系。
- **交通：** 自驾最佳
- **预约要点：** 只在合法、安全的停车带或观景点停靠；严禁在主干道随意下车拍照
- **来源链接：**
  - https://www.xinjiang.gov.cn/
  - https://www.kelamayi.com.cn/

### 3. 乌尔禾影视城
- **门票参考：** 常见低价门票或与周边点位组合票，具体以现场为准
- **建议时长：** 1—2小时
- **玩法：** 西部废土感很强，适合作为魔鬼城前后的拍照型补充点。它不是乌尔禾的第一主菜，但非常适合增强“荒野片场感”。
- **交通：** 乌尔禾区内，自驾串联方便
- **预约要点：** 一般无需复杂预约；如遇活动、拍摄或临时维护，以现场公告为准
- **来源链接：**
  - https://you.ctrip.com/sight/kelamayi335/15135.html
  - https://www.kelamayi.com.cn/

### 4. 艾里克湖
- **门票参考：** 常见免费或无固定门票；营地/娱乐项目另计
- **建议时长：** 1—2.5小时
- **玩法：** 荒漠里的水域调节器。很适合在魔鬼城高强度风沙与土色之后，切换到开阔湖面、风车、天际线和轻松停留节奏。适合亲子放松、日落前后散步、轻摄影。
- **交通：** 自驾最方便
- **预约要点：** 野外点位，建议自备饮水和零食；大风天气体验下降明显
- **来源链接：**
  - https://you.ctrip.com/place/kelamayi335.html
  - https://www.kelamayi.com.cn/

### 5. 乌尔禾胡杨林
- **门票参考：** 旺季常见约20—40元，摆渡/观光车另计
- **建议时长：** 1.5—3小时
- **玩法：** 不是乌尔禾全年第一优先级，但在9—10月很有价值。它的作用更像给整体荒野线路加一层金黄色调，让乌尔禾不只是灰黄戈壁。
- **交通：** 通常与艾里克湖、魔鬼城、自驾公路线组合
- **预约要点：** 强季节型景点，出发前最好确认当年变色进度；非秋季不要抱过高预期
- **来源链接：**
  - https://you.ctrip.com/sight/kelamayi335/5717379.html
  - https://www.kelamayi.com.cn/

### 6. 百口泉知青村
- **门票参考：** 常见免费或低价参观，以现场公示为准
- **建议时长：** 1—1.5小时
- **玩法：** 如果说魔鬼城负责地貌冲击，百口泉知青村更像乌尔禾的人文补丁。适合看知青历史、屯垦开发记忆与石油边疆生活线索，能让行程不只剩风景。
- **交通：** 乌尔禾区内或周边，自驾更方便
- **预约要点：** 留意场馆开放时段；部分展陈区可能午间休息或淡季缩时
- **来源链接：**
  - https://www.kelamayi.com.cn/
  - https://you.ctrip.com/place/kelamayi335.html

### 7. 百口泉公园
- **门票参考：** 免费
- **建议时长：** 0.5—1小时
- **玩法：** 更偏本地休闲型点位，适合住在乌尔禾时傍晚散步、带孩子活动或作为知青村后的轻松收尾。
- **交通：** 乌尔禾城区内可达
- **预约要点：** 公共开放空间，无需预约；夜间活动留意风力与温差
- **来源链接：**
  - https://www.kelamayi.com.cn/
  - https://you.ctrip.com/place/kelamayi335.html

### 8. 乌尔禾国际房车露营公园
- **门票参考：** 按房型、营位或项目收费，浮动较大
- **建议时长：** 1—2小时体验或住1晚
- **玩法：** 对自驾客非常友好。重点不是“景点密度”，而是把乌尔禾住舒服、把魔鬼城日落和第二天早起公路线衔接顺。适合亲子、房车、自驾深度玩家。
- **交通：** 乌尔禾区内，自驾最佳
- **预约要点：** 旺季周末、节假日建议提前订；大风天露营体验会明显受影响
- **来源链接：**
  - https://www.kelamayi.com.cn/
  - https://you.ctrip.com/place/kelamayi335.html

### 9. 西部乌镇（乌尔禾特色休闲小镇）
- **门票参考：** 免费进入，餐饮与演艺消费自理
- **建议时长：** 1—2小时
- **玩法：** 适合作为夜间补给点，不是硬核景点，但对住乌尔禾的人很实用：晚饭、补水、轻散步、带娃消磨一小时都合适。
- **交通：** 乌尔禾区内，步行/打车/自驾均可
- **预约要点：** 通常无需预约；淡季部分商铺营业时间会缩短
- **来源链接：**
  - https://www.kelamayi.com.cn/
  - https://you.ctrip.com/place/kelamayi335.html

### 10. 乌尔禾镇城区观景与补给带
- **门票参考：** 免费
- **建议时长：** 0.5—1小时
- **玩法：** 它不是传统景点，但对真正执行行程很重要。这里适合做午餐、加油、补水、整理器材和夜宿落脚。乌尔禾线路能否轻松，很大程度靠这里打底。
- **交通：** 乌尔禾城区内
- **预约要点：** 无需预约；旺季餐饮高峰尽量避开最晚时段
- **来源链接：**
  - https://www.kelamayi.com.cn/

### 11. 白杨河大峡谷
- **门票参考：** 常见免费或低价管理性收费，以当地公示为准
- **建议时长：** 2—3小时
- **玩法：** 比世界魔鬼城更小众、更偏“知道的人才会去”的补充型地貌点。适合二刷乌尔禾、喜欢人少和轻徒步的自驾玩家。
- **交通：** 建议自驾或包车
- **预约要点：** 出发前确认道路与天气；不建议天黑后进入野外峡谷地带
- **来源链接：**
  - https://www.kelamayi.com.cn/
  - https://baike.baidu.com/search/word?word=%E7%99%BD%E6%9D%A8%E6%B2%B3%E5%A4%A7%E5%B3%A1%E8%B0%B7

### 12. 乌尔禾泥火山群
- **门票参考：** 多为开放式地貌点，常见免费或现场管理性收费
- **建议时长：** 1—2小时
- **玩法：** 适合对地质感兴趣的人。看泥浆喷溢、干裂地表和不同于雅丹的地貌质感，能补足乌尔禾不只一种“荒野长相”。
- **交通：** 自驾或包车更合理
- **预约要点：** 不要离开既有道路乱走，局部地表可能松软；雨后或湿地状态下更要谨慎
- **来源链接：**
  - https://www.kelamayi.com.cn/

### 13. 乌尔禾雅丹地貌外围观景点
- **门票参考：** 免费
- **建议时长：** 0.5—1.5小时
- **玩法：** 适合不进深度景区、但想在安全范围内看乌尔禾雅丹外围轮廓的人。更偏“停靠型观景”，而非完整游览。
- **交通：** 自驾最佳
- **预约要点：** 别误入封闭或无护栏区域；风沙大时体感会迅速变差
- **来源链接：**

  - https://www.kelamayi.com.cn/

### 14. 风车戈壁摄影带
- **门票参考：** 免费
- **建议时长：** 0.5—1小时
- **玩法：** 乌尔禾线路里非常适合拍“公路 + 风机 + 大地色块”的地方。它的价值在于增强旅行叙事，不在于单点观光浓度。
- **交通：** 自驾沿线停靠
- **预约要点：** 严格按安全停靠规则操作，切勿翻越围栏或进入作业区域
- **来源链接：**

### 15. 乌尔禾日落观景点
- **门票参考：** 免费或依附所在景区/营地收费
- **建议时长：** 0.5—1.5小时
- **玩法：** 适合不想把日落完全压在魔鬼城里的游客，或摄影党第二晚补拍。看低角度光线扫过戈壁、风机和雅丹边缘，氛围很强。
- **交通：** 通常依赖自驾
- **预约要点：** 注意返程天黑后的驾驶疲劳；别为了机位进入危险边坡或无人管控区
- **来源链接：**
  - https://www.kelamayi.com.cn/

### 16. 乌尔禾星空观测点
- **门票参考：** 免费或营地收费
- **建议时长：** 1—2小时
- **玩法：** 晴朗、少云、远离主城区时，乌尔禾夜空条件不错。对住乌尔禾的摄影党和亲子家庭来说，星空是住一晚的重要加分项。
- **交通：** 自驾或住营地/露营公园更方便
- **预约要点：** 夜间风大、温差明显，三脚架和保暖外套要备好；不要单独深入陌生野外区域
- **来源链接：**
  - https://www.kelamayi.com.cn/

### 17. 克拉玛依—乌尔禾石油荒野联动线
- **门票参考：** 免费
- **建议时长：** 半天到1天（作为联动线路）
- **玩法：** 这不是单一景点，而是非常重要的行程结构：主城区先看黑油山/博物馆，理解石油城市底色，再来乌尔禾看魔鬼城和荒野，整趟旅行会完整很多。
- **交通：** 自驾最顺；包车也适合
- **预约要点：** 别把主城区和乌尔禾硬塞成“当天全都看完还想轻松”的错觉行程，合理做法通常是至少分成1.5—2天
- **来源链接：**
  - https://www.kelamayi.com.cn/
  - https://www.xinjiang.gov.cn/

### 18. 乌尔禾亲子地质科普线（魔鬼城—胡杨—湖泊组合）
- **门票参考：** 依实际进入景点分别计费
- **建议时长：** 1天
- **玩法：** 这是最适合家庭的“组合型景点”。上午轻松补给，中段进魔鬼城看雅丹和风蚀，后段去艾里克湖或胡杨林做舒缓，孩子不容易审美疲劳。
- **交通：** 自驾或包车
- **预约要点：** 本质是路线玩法，不是独立售票点；关键在节奏控制和防晒补水
- **来源链接：**

## 美食
乌尔禾不是那种以“专门来吃”为核心吸引力的地方，但它是**非常典型的荒野自驾补给型节点**。真正重要的不是网红店，而是：能不能在去魔鬼城前后把饭吃稳、把水和零食备齐。

### 优先吃什么
- **烤肉 / 红柳烤串：** 最稳的新疆旅行晚餐选项，适合日落后补充热量。
- **抓饭：** 最适合白天出发前吃，抗饿、出餐快。
- **拌面 / 拉条子：** 自驾午餐很合适，效率高。
- **大盘鸡 / 辣子鸡：** 适合多人同行，分量通常不小。
- **奶茶、酸奶、馕、烤包子：** 很适合作为车上补给。

### 吃饭逻辑
- **主城区更稳，乌尔禾够用。** 如果你对口味和环境要求高，正餐尽量放在克拉玛依解决；乌尔禾更适合功能型补给。
- **拍魔鬼城日落前别空腹硬扛。** 最好午后在乌尔禾镇先吃点主食，不然傍晚风大、体力掉得很快。
- **带孩子建议提前备零食和饮料。** 不要把希望全部寄托给景区里临时购买。

## 住宿

### 1. 住克拉玛依主城区：第一次来最稳
- **适合：** 低龄亲子、对酒店品质和餐饮选择要求高的人
- **优点：** 住宿条件和补给能力强，晚上也更舒服
- **缺点：** 去乌尔禾看日落后返程较累，摄影时段不占优

### 2. 住乌尔禾：为魔鬼城和公路线服务
- **适合：** 摄影、自驾、中大童家庭、想把乌尔禾玩完整的人
- **优点：** 看完魔鬼城日落不用夜间长途回撤；第二天还能从容加艾里克湖/胡杨林/百口泉
- **缺点：** 酒店数量和餐饮丰富度不如克拉玛依主城区

### 3. 房车/露营型住法
- **适合：** 深度自驾、房车旅行、想拍星空的人
- **优点：** 沉浸感强，和乌尔禾环境更搭
- **缺点：** 大风天气、温差、卫生舒适度会直接影响体验

### 亲子住宿怎么选
- **低龄家庭：** 更推荐住克拉玛依，白天来乌尔禾快刷主线
- **中大童家庭：** 住乌尔禾1晚往往更合理，能减少来回折返造成的情绪和体力消耗
- **选酒店重点：** 停车、热水、空调/供暖、早餐、周边是否方便买水，比装修风格重要得多

## 交通

### 外部交通
- **飞机/火车直达乌尔禾的便利性有限，通常以克拉玛依为主要交通入口。**
- 多数游客实际走法是：先到克拉玛依，再自驾/包车前往乌尔禾。

### 与克拉玛依怎么联动最合理
- **第一次来新疆北疆：** 建议把乌尔禾作为克拉玛依2—3天行程里的核心外线。
- **只想看魔鬼城：** 可从克拉玛依一日往返，但会比较赶。
- **想兼顾摄影和舒适：** 建议克拉玛依住1晚 + 乌尔禾住1晚。
- **继续向外延展：** 乌尔禾适合接福海、阿勒泰、和布克赛尔等方向的公路旅行节奏。

### 市内与周边移动
- **乌尔禾镇内：** 打车勉强可解决基础需求
- **世界魔鬼城、艾里克湖、胡杨林、泥火山、白杨河等：** 强烈建议有车
- **包车适合人群：** 不开车、带老人孩子、但又希望高效串线的人

### 自驾提醒
- 出发前把油、水、零食、充电设备准备好。
- 不要低估风和晒对驾驶疲劳的放大作用。
- 傍晚返程克拉玛依时要特别注意长直路疲劳驾驶。
- 在新疆拍公路，安全比任何机位都重要。

## 避坑
1. **不要把乌尔禾当成“魔鬼城一张门票”的地方。** 真正值钱的是景区 + 公路 + 周边补充点的组合。
2. **世界魔鬼城别中午冲刺式打卡。** 它太吃光线，正午进只会觉得“怎么全是土”。
3. **如果你很在意摄影，别贪方便住回克拉玛依。** 日落后长途回撤很伤体验。
4. **胡杨林别全年都按秋景期待。** 非秋季就是补充型点位。
5. **别把野外开放点位当越野探险场。** 泥火山、峡谷边缘、无人路肩都有风险。
6. **不要在主干道随意停车拍“公路大片”。** 乌尔禾最容易出片，也最容易因为想出片而出事。
7. **亲子行程别硬塞太多野点。** 一个主景区 + 一个轻松点，通常比4个碎点更适合孩子。
8. **不要等到进景区才想起买水。** 乌尔禾旅行最影响体感的往往不是门票，而是补给是否到位。
9. **别忽视乌尔禾与克拉玛依主城区的关系。** 先理解石油城市，再看雅丹地貌，旅行会更完整。
10. **不要把乌尔禾排成“顺路看看”的纯过路停靠。** 如果只给它1—2小时，它很可能发挥不出真实价值。
