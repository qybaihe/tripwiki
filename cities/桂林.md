# 桂林

- 更新时间：2026-03-15
- 适用场景：第一次去桂林、想快速做出 1–3 天可落地行程的人
- 阅读提醒：桂林很多项目分成“免费开放区 / 单独收费点 / 游船竹筏演出项目”，而且旺季、节假日、枯水期和天气都会影响价格、航线与放票规则。本文统一给“常见参考区间 + 官方/权威入口”，出发前请再核一次。

## 城市概览

桂林的强项很明确：市区景观密度高，漓江和阳朔是全国辨识度最高的山水名片，龙脊梯田又能补上山地风光和少数民族村寨体验。第一次来如果时间不多，最稳的思路不是把点位撒太开，而是按 **桂林市区 / 漓江-阳朔 / 龙脊梯田** 三条线来做取舍：1 天玩市区，2 天加阳朔，3 天再上龙脊，就已经很完整。

- 推荐季节：4–6 月水量足、山体颜色漂亮；9–11 月体感舒适、适合徒步和骑行；7–8 月偏热且旺季排队更明显。
- 建议天数：
  - 1 天：桂林市区地标 + 夜游
  - 2 天：市区 + 漓江/阳朔
  - 3 天：再加龙脊梯田或阳朔深度
- 适合人群：第一次去广西的人、自然山水控、摄影爱好者、轻徒步和骑行玩家、亲子家庭
- 不太适合：只想“纯躺酒店不折腾”、却又不愿接受景点之间通勤的人

## 行前速览

### 天气与穿着
- 春季和初夏多雨，山水状态通常更好，但鞋子要防滑。
- 夏天湿热，阳朔骑行和白天爬山会比较晒，防晒、防蚊和补水要带够。
- 秋季综合体验最好；冬季不会特别冷，但阴雨天体感会偏湿冷。

### 到达与大交通
- 机场：桂林两江国际机场
- 火车：桂林站、桂林北站、桂林西站；多数游客到桂林北站或桂林站更方便
- 去阳朔：可高铁到阳朔站，再换乘到县城；也可直接从桂林乘大巴/包车
- 去龙脊：常见是桂林市区报一日团、乘专线车或包车前往

### 住宿怎么选
- **桂林市区（象山 / 秀峰 / 两江四湖周边）**：适合首晚、末晚和市区夜游，吃饭方便。
- **阳朔县城 / 西街周边**：适合第一次去阳朔，晚上逛街、吃饭、租车都方便。
- **遇龙河 / 十里画廊沿线民宿**：适合拍田园风光、想住得安静一点的人。
- **龙脊平安 / 金坑大寨**：适合住一晚看日出或晨雾，但要接受山路和搬运行李。

### 预约与购票原则
- 漓江游船、遇龙河竹筏、印象刘三姐、龙脊旺季住宿，建议提前锁定。
- 看到“低价拼团 + 多个项目打包”时，先确认是否包含接送、景交、保险、二消和实际码头。
- 天气和水位会直接影响游船、竹筏和部分山地项目，行程不要排太死。

## 1-3天行程

### 1 天快玩版：桂林市区地标
- 上午：象鼻山
- 中午：正阳步行街 / 东西巷吃饭
- 下午：靖江王城·独秀峰 或 芦笛岩
- 傍晚：伏波山 / 叠彩山二选一看城景
- 晚上：两江四湖夜游

适合：高铁周末游、第一次来桂林但时间很紧的人。

### 2 天经典版：市区 + 阳朔

**Day 1：桂林市区**
- 象鼻山
- 靖江王城·独秀峰
- 东西巷 / 正阳步行街
- 两江四湖夜游

**Day 2：漓江 + 阳朔**
- 早上：桂林出发坐漓江游船到阳朔，或直接前往兴坪 / 阳朔
- 下午：十里画廊骑行、月亮山、遇龙河沿线慢玩
- 晚上：西街逛吃，住阳朔

适合：第一次来，想把“桂林山水甲天下”的核心体验拿到手的人。

### 3 天均衡版：市区 + 阳朔 + 龙脊 / 深度阳朔

**Day 1：桂林市区**
- 象鼻山
- 靖江王城·独秀峰
- 两江四湖

**Day 2：阳朔精华**
- 遇龙河竹筏（尽量早场）
- 十里画廊骑行
- 银子岩
- 西街

**Day 3：二选一**
- 方案 A：龙脊梯田一日往返
- 方案 B：兴坪古镇 + 20 元背景打卡 + 相公山 / 九马画山深度

如果带长辈或孩子，第三天更推荐龙脊或兴坪，不一定要再上强度高的徒步。

## 景点清单（≥18 个）

> 统一说明：以下门票均为 **常见参考区间**。景区免费开放、联票、船票、套票、旺季涨价、优惠政策、临时限流都很常见；预约、票价和开放时间请以景区官方公众号、政府文旅公告、景区官网或当日票务页为准。

### 1. 象鼻山景区
- 门票参考：常见口径为免费开放；如遇实名预约或分时段管理，以当期公告为准
- 建议时长：1–2 小时
- 玩法：拍桂林城市地标、沿江步道散步、黄昏和夜景出片率高
- 交通：位于桂林市区核心，打车、公交或步行串联正阳步行街都方便
- 预约要点：节假日和高峰期建议提前看景区官方公众号是否需要预约
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://gl.bendibao.com/tour/2023126/15065.shtm

### 2. 两江四湖景区（夜游为主）
- 门票参考：夜游船常见约 ¥160–¥220+；不同码头、船型、航线和节假日会浮动
- 建议时长：1.5–2.5 小时
- 玩法：夜游环城水系、看桥梁灯光和城市山水夜景，适合轻松玩
- 交通：按船票指定码头集合，常见码头在市区核心区
- 预约要点：实名出票常见；节假日尽量提前订黄金时段；受水位影响有时会调整为一江四湖或四湖路线
- 来源链接：
  - https://www.citsguilin.com/menpiao/liangjiangsihu.htm
  - https://wglj.guilin.gov.cn/

### 3. 漓江精华游船（桂林—阳朔）
- 门票参考：三星 / 四星游船常见约 ¥200–¥360+，舱位、餐食和船型不同差价较大
- 建议时长：4–6 小时（含集合登船）
- 玩法：第一次来最经典的山水体验，重点看黄布倒影、九马画山等漓江精华段
- 交通：从桂林指定码头登船，到阳朔下船；也可反向搭配高铁 / 包车
- 预约要点：旺季建议提前锁票；枯水期、涨水期或恶劣天气可能调整航线与码头
- 来源链接：
  - https://www.liriver.com.cn/mobile/article/zxlj.tzgg/137
  - https://wglj.guilin.gov.cn/

### 4. 遇龙河竹筏漂流
- 门票参考：按航段和筏型计价，常见约 ¥200–¥400+/筏；热门航段节假日更高
- 建议时长：1.5–3 小时（含排队）
- 玩法：比漓江更贴近田园风光，适合拍喀斯特峰林、稻田、古桥和村落
- 交通：前往阳朔遇龙河各上筏点，打车最省事
- 预约要点：旺季非常建议提前预约；不同航段体验差异明显，优先选早场避开排队和暴晒
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://hk.trip.com/travel-guide/attraction/yangshuo/yulong-river-rafting-10758988/

### 5. 十里画廊
- 门票参考：公路和沿线田园景观免费；个别单独景点另收费
- 建议时长：3–6 小时
- 玩法：阳朔最适合骑行、电动车和边走边停的一条风景线，可串联月亮山、工农桥、遇龙河周边
- 交通：从阳朔县城租自行车、电动车或打车前往
- 预约要点：不需要预约；夏季中午很晒，建议上午或傍晚骑
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://you.ctrip.com/sight/yangshuo702/10579.html

### 6. 阳朔西街
- 门票参考：免费
- 建议时长：1–3 小时
- 玩法：夜间逛街、吃饭、喝酒、小店闲逛，是阳朔最成熟的游客集散区
- 交通：位于阳朔县城中心，步行最方便
- 预约要点：不用预约；旺季人挤人，住宿尽量不要选最吵的临街房间
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://cn.tripadvisor.com/Attraction_Review-g303712-d1834698-Reviews-West_Street-Yangshuo_County_Guangxi.html

### 7. 龙脊梯田景区
- 门票参考：成人票常见 ¥80；景区专线车常见约 ¥40/人；金坑索道、演出等另计
- 建议时长：1 天—2 天
- 玩法：看梯田线条、村寨风貌、晨雾日出；平安和金坑大寨是第一次去最常见的选择
- 交通：桂林市区专线车、一日团或包车前往
- 预约要点：节假日和秋收季住宿紧张；住山上更容易拍到清晨景色，但要预留爬坡和搬运行李时间
- 来源链接：
  - http://www.longjitour.com/spots/show_1.html
  - https://wglj.guilin.gov.cn/

### 8. 银子岩
- 门票参考：常见约 ¥65–¥90
- 建议时长：1.5–2.5 小时
- 玩法：看大型喀斯特溶洞钟乳石，是阳朔雨天和炎热天气的稳妥备选
- 交通：从阳朔县城打车约 20–40 分钟
- 预约要点：通常不用太早预约，但团队游客多，尽量早去或错峰
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://you.ctrip.com/sight/yangshuo702/10623.html

### 9. 兴坪古镇
- 门票参考：古镇步行区域免费；乘船、竹筏、观景项目另计
- 建议时长：2–4 小时
- 玩法：逛老街、拍江边山水、作为 20 元人民币背景打卡的中转地
- 交通：阳朔县城乘班车、网约车或包车前往
- 预约要点：如果要叠加漓江小船或竹筏，旺季尽量早到
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://you.ctrip.com/sight/yangshuo702/10609.html

### 10. 黄布倒影 / 20 元人民币背景打卡点
- 门票参考：通常和兴坪漓江项目联动消费，岸边观景免费或低成本可达，乘船项目另计
- 建议时长：1–2 小时
- 玩法：打卡“20 元人民币背景”，晴天和有层次云雾时更出片
- 交通：一般从兴坪镇周边前往，常见方式是步行到观景处或乘当地小船
- 预约要点：如果想避开人流，尽量上午早点去；是否乘船看个人预算和天气
- 来源链接：
  - https://www.liriver.com.cn/
  - https://you.ctrip.com/sight/yangshuo702/10609.html

### 11. 相公山
- 门票参考：常见约 ¥60–¥80
- 建议时长：2–4 小时
- 玩法：桂林摄影党热门日出点，可俯瞰漓江大弯与峰林层次
- 交通：多从兴坪或阳朔拼车、包车前往
- 预约要点：如果看日出，凌晨就要出发；雨雾天有概率白跑，但云海条件也往往更好
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://you.ctrip.com/sight/yangshuo702/1841660.html

### 12. 芦笛岩
- 门票参考：常见约 ¥90–¥100
- 建议时长：1.5–2 小时
- 玩法：市区最成熟的溶洞之一，适合雨天、老人和孩子
- 交通：桂林市区打车约 20–30 分钟
- 预约要点：通常无需复杂预约，但旅行团多，上午早点到更舒服
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/10572.html

### 13. 靖江王城·独秀峰
- 门票参考：常见约 ¥90–¥120
- 建议时长：2–3 小时
- 玩法：看明代藩王府遗址、桂林历史文脉，登独秀峰俯瞰市区
- 交通：位于桂林市中心，步行或打车都方便
- 预约要点：节假日排队较明显；如果只想轻松看历史文化，这里比纯商业街更值得留时间
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/10570.html

### 14. 叠彩山
- 门票参考：常见约 ¥25–¥35
- 建议时长：1–2 小时
- 玩法：短时间登高看桂林城景和漓江线条，强度比长线徒步低
- 交通：桂林市区内可达，适合与伏波山二选一
- 预约要点：台阶较多，夏天中午不建议去硬爬
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/10575.html

### 15. 伏波山
- 门票参考：常见约 ¥20–¥30
- 建议时长：1–1.5 小时
- 玩法：俯瞰漓江、看还珠洞和山体石刻，属于市区内的快速登高点
- 交通：市区打车或公交方便
- 预约要点：不用专门预留太多时间，适合和东西巷 / 王城串联
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/10568.html

### 16. 七星景区
- 门票参考：常见约 ¥55–¥70；园内部分项目另计
- 建议时长：2–4 小时
- 玩法：综合型城市公园，可看七星岩、骆驼山、花桥，适合亲子和慢逛
- 交通：位于桂林市区东侧，打车和公交都方便
- 预约要点：景区面积较大，夏天注意补水；若只想抓核心景点，可提前决定走短线
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/10571.html

### 17. 东西巷
- 门票参考：免费
- 建议时长：1–2 小时
- 玩法：桂林市中心比较成熟的步行街区，适合买伴手礼、吃本地小吃、晚上散步
- 交通：市中心步行可达
- 预约要点：不是纯古街，商业化比较明显，更适合作为吃饭和休息点，不必抱太高“古镇感”期待
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/1418536.html

### 18. 正阳步行街
- 门票参考：免费
- 建议时长：1–2 小时
- 玩法：市区最方便的夜间逛吃区，可与东西巷、象鼻山、王城连成一片
- 交通：位于桂林市中心
- 预约要点：主要价值在吃饭和夜间氛围，不是必须专程打卡的“景区型”景点
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/1418535.html

### 19. 世外桃源
- 门票参考：常见约 ¥100–¥120
- 建议时长：2–4 小时
- 玩法：乘船进景区、看田园和民俗演出，适合亲子和不想自己折腾路线的人
- 交通：位于桂林前往阳朔方向，包车或跟团更方便
- 预约要点：看清楚票种是否含演出 / 接驳；高峰期团队较多
- 来源链接：
  - https://you.ctrip.com/sight/guilin28/10607.html
  - https://wglj.guilin.gov.cn/

### 20. 月亮山
- 门票参考：常见约 ¥20–¥35
- 建议时长：1–2 小时
- 玩法：看天然拱洞，是十里画廊沿线最经典的辨识点之一
- 交通：阳朔十里画廊沿线骑行 / 电动车可到
- 预约要点：如果只是打卡拍照，不一定非要爬到很高；夏天台阶暴晒明显
- 来源链接：
  - http://www.yangshuo.gov.cn/
  - https://you.ctrip.com/sight/yangshuo702/10611.html

### 21. 古东瀑布
- 门票参考：常见约 ¥60–¥90
- 建议时长：3–5 小时
- 玩法：玩水、溯溪、爬瀑布，夏天尤其受欢迎
- 交通：位于桂林郊区，包车、跟团或自驾更方便
- 预约要点：雨季关注安全提示；最好带换洗衣物、防滑鞋或按景区要求租用装备
- 来源链接：
  - https://wglj.guilin.gov.cn/
  - https://you.ctrip.com/sight/guilin28/10577.html

### 22. 印象刘三姐
- 门票参考：常见约 ¥198–¥320+
- 建议时长：2–3 小时
- 玩法：看张艺谋团队打造的山水实景演出，晚上安排很合适
- 交通：位于阳朔，可打车前往演出地
- 预约要点：旺季热门场次建议提前订；雨天通常也会按景区安排演出或调整
- 来源链接：
  - http://www.yangshuo.gov.cn/

## 美食

桂林吃饭不要只盯“桂林米粉”四个字。真正的思路是：**早上吃粉，中午吃家常，晚上在阳朔或市区找啤酒鱼、田螺酿、荔浦芋扣肉这类更成体系的菜。**

### 必吃清单
- **桂林米粉**：卤菜粉、锅烧粉最常见；本地人很多是干拌后再配汤。
- **阳朔啤酒鱼**：常见用剑骨鱼、毛骨鱼等现点现做，味道偏鲜辣。
- **田螺酿**：广西风味代表之一，适合配啤酒。
- **荔浦芋扣肉**：偏家常、偏下饭，适合多人分食。
- **恭城油茶**：接受度看人，但值得试一次。
- **马蹄糕、桂花糕、松糕、粽子类小吃**：适合买伴手礼。
- **竹筒饭 / 竹筒鸡 / 农家菜**：在龙脊、阳朔郊外常见。

### 吃饭建议
- 市区想稳妥吃粉，优先找本地连锁或排队但明码标价的老店。
- 阳朔西街和景区口餐厅环境方便，但溢价常常更高。
- 点啤酒鱼前先问清楚：鱼的品种、斤数、做法、总价怎么记。
- 看见“现捞现杀”不等于一定贵，但一定要先把单价和加工方式问明白。

## 住宿

### 第一次去怎么选
- **只住 1 晚**：住桂林市区两江四湖 / 象山周边，通勤最省脑子。
- **住 2 晚**：1 晚桂林 + 1 晚阳朔，是最稳的经典搭配。
- **住 3 晚及以上**：可考虑 1 晚桂林 + 1–2 晚阳朔 + 1 晚龙脊。

### 分区建议
- **桂林市区**：适合夜游、逛街、第二天赶高铁 / 机场。
- **阳朔县城**：适合第一次来，生活便利、晚饭和租车方便。
- **遇龙河 / 十里画廊**：适合想住风景里，但通勤和外卖便利度不如县城。
- **龙脊梯田山上民宿**：景色强，但一定先确认“车能到哪、还要不要爬多久、行李怎么送”。

### 订房提醒
- 阳朔景观房差价能很大，先看窗外到底是不是“真景观”而不是“部分可见”。
- 龙脊民宿冬天保暖和热水稳定性比“装修照片”更重要。
- 自驾去龙脊要注意停车点与接驳规则，别默认能把车直接开到民宿门口。

## 交通

### 外部交通
- **飞机**：到桂林两江国际机场，适合跨省直飞。
- **高铁**：华南、中部很多城市到桂林很方便；去阳朔也可以直接坐高铁。

### 城内与跨区交通
- **桂林市区**：景点较集中，打车 + 步行最省心，公交也够用。
- **桂林 → 阳朔**：高铁、汽车、包车都行；如果本身买了漓江游船，就等于把交通和游览合并了。
- **阳朔县内**：骑行、电动车、打车都常见；十里画廊一带自己掌控节奏最好。
- **桂林 / 阳朔 → 龙脊**：专线车和包车最常见，一日游适合省心，住一晚更适合拍清晨。

### 实操建议
- 桂林和阳朔不是“下楼全靠步行就能串完”的关系，不要低估换乘时间。
- 节假日从阳朔返桂林、去高铁站、去热门码头，最好多预留 30–60 分钟。
- 如果只安排 2 天，不建议把龙脊也硬塞进去，会很赶。

## 避坑

- **不要把“桂林”和“阳朔”当成同一个步行圈。** 两地核心玩法不同，通勤也要算时间。
- **漓江游船、遇龙河竹筏、印象刘三姐不要全挤在一天。** 体验会变成赶场。
- **遇龙河最怕中午去。** 暴晒、排队、拍照逆光，体验明显变差。
- **阳朔西街不是桂林最值得专程打卡的“景点”，它更像夜间补充。** 真正核心还是山水和田园。
- **龙脊一日游能去，但别指望“一天看尽全部片区”。** 第一次去选平安或金坑一个主片区就够。
- **看日出上相公山有赌天气成分。** 不是每天都能出大片，别因此把整趟行程心态搞崩。
- **景区门口低价揽客项目先问完整价格。** 尤其竹筏、小交通、旅拍、包车、演出票升级位。
- **桂林米粉不一定越网红越好。** 很多好吃的是本地人常去的普通店，关键看翻台和卤水状态。
- **行李安排很重要。** 如果是“桂林进、阳朔住、龙脊住、再回桂林”的路线，尽量轻装，不要带太大箱子上梯田。
