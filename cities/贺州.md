## 0. 城市概览
贺州位于广西东北部，处在桂、粤、湘三地交界带，是一座很适合做“2-3 天轻度假”的低密度山水城市。它不是那种靠单一超级景区吃饭的旅游目的地，而是把**古镇、瀑布、森林、石林、溶洞、温泉、古村与瑶乡风情**组合在一起，形成一条节奏很舒服的城市休闲线。

第一次来贺州，最常见的玩法有两种：
- **经典入门线**：黄姚古镇 + 姑婆山 + 玉石林/紫云仙境 + 温泉
- **人文乡野线**：黄姚古镇 + 富川古明城 + 秀水状元村 + 岔山村/凤溪古瑶寨

这座城市最适合的人，不是“必须连续刷爆点”的特种兵，而是愿意把一座城市拆成几段来感受的人：白天看山水、傍晚逛古镇、晚上泡温泉，第二天再去古村或石林，整体体验会比单点打卡更完整。

## 1. 行前速览

### 1.1 建议停留天数
- **1 天**：黄姚古镇 / 姑婆山 / 玉石林-紫云仙境三选一主线快刷
- **2 天**：黄姚古镇 + 姑婆山，或黄姚古镇 + 玉石林/紫云仙境
- **3 天**：黄姚古镇 + 姑婆山/十八水 + 玉石林/紫云仙境 + 温泉
- **4 天及以上**：加富川古城古村、钟山田园、公路风景、自驾环线

### 1.2 最佳旅行时间
- **3-5 月**：山林、瀑布、古镇状态较好，适合踏青与摄影
- **6-8 月**：山里避暑不错，但雨水偏多，瀑布景区要防滑
- **9-11 月**：气候舒适，适合古镇慢游、自驾、轻徒步
- **12-2 月**：适合温泉度假，山景偏清冷但更安静

### 1.3 城市气质一句话
- **黄姚看人文氛围**
- **姑婆山、十八水看山林瀑布**
- **玉石林、紫云仙境看喀斯特奇观**
- **西溪/路花温泉看放松度假**
- **富川古村古城看历史与瑶乡**

### 1.4 交通判断
- **最优解：自驾或包车**。贺州适合串点，但很多点不在一条城市轨道式动线上。
- **高铁到达**：优先看贺州站、钟山西站。
- **公共交通可玩，但效率一般**：市区近郊景点可打车，跨县区景点衔接不算丝滑。

### 1.5 门票与预约总提醒
贺州很多景点在旅游平台、公众号、线下窗口会出现不同价格；本文涉及门票、开放、预约信息仅作出行参考，**如无权威统一口径，一律以现场/官方公告为准**。

### 1.6 适合怎么住
- 想看古镇夜景与清晨：**住黄姚古镇 1 晚**
- 想高效串景区：**住八步区/贺州站附近**
- 想放松收尾：**住温泉度假酒店**

### 2.1 黄姚古镇
- **门票参考**：成人票常见区间约 ¥75-95；具体以现场/官方公告为准。
- **建议时长**：4-8 小时；如住一晚体验更完整。
- **玩法**：核心玩法是逛石板街、古戏台、带龙桥、龙爪榕、老宅巷道与夜景灯火。傍晚和次日清晨最适合拍照，也最能感受到古镇的生活气。
- **交通**：位于昭平县黄姚镇，通常从贺州城区/贺州站自驾或打车前往更方便；公共交通可到昭平后再转乘。
- **预约要点**：节假日建议提前在线查看门票与景区限流情况；古镇内民宿热门房型建议提前预订；其余以现场/官方公告为准。
- **来源链接**：
  - https://you.ctrip.com/sight/hezhou707/136423.html
  - https://baike.baidu.com/item/黄姚古镇

- **玩法**：这是黄姚古镇最经典的桥景点位之一，适合拍石桥、流水、古宅与倒影，清晨人少时更容易出片。
- **交通**：位于黄姚古镇景区内，步行可达。
- **预约要点**：一般无需单独预约；雨天石板路、桥面周边较滑；以现场/官方公告为准。

- **建议时长**：20-30 分钟。
- **玩法**：适合看黄姚古镇的公共空间结构与古镇中心氛围，也是游客和本地生活气息交汇较明显的节点。
- **交通**：位于黄姚古镇核心步行区。
- **预约要点**：通常无需单独预约；节假日该区域游客密度较高，建议错峰拍照；以现场/官方公告为准。

- **交通**：位于黄姚古镇景区内部，步行串联即可。

### 2.5 黄姚诗画姚江 / 姚江竹筏点位
- **门票参考**：项目票常见区间约 ¥35-50；具体以现场/官方公告为准。

- **玩法**：从水上角度看古镇外围、村落、山体与河道风光，适合想把黄姚玩得更松弛的人，也适合亲子。
- **交通**：通常位于黄姚古镇周边，可与古镇同日串联。
- **预约要点**：水位、天气、运营安排可能影响是否开放或发班；建议当天确认；以现场/官方公告为准。

### 2.6 姑婆山旅游区
- **门票参考**：成人票常见区间约 ¥66-80；具体以现场/官方公告为准。
- **建议时长**：4-6 小时。
- **玩法**：贺州自然风景代表之一，适合看山林、溪流、瀑布、茶园、影视取景地，夏天避暑感明显。第一次来贺州，如果只选一个“山水景区”，姑婆山通常是优先项。
- **交通**：位于八步区北部方向，距贺州市区约二十多公里，自驾或打车最方便。
- **预约要点**：节假日可提前查询门票与观光车安排；雨后景色更好，但步道更滑；以现场/官方公告为准。
- **来源链接**：
  - https://you.ctrip.com/sight/hezhou707/10758974.html
  - https://baike.baidu.com/item/姑婆山

### 2.7 仙姑瀑布
- **门票参考**：通常包含在姑婆山景区门票内；以现场/官方公告为准。
- **建议时长**：30-60 分钟。
- **玩法**：适合拍瀑布、感受水汽与林间清凉，是姑婆山里最直观、最容易获得“山水值回票价”感觉的点位之一。
- **交通**：位于姑婆山景区内，通常通过步行或景区内部接驳到达。
- **预约要点**：暴雨、强降雨后可能有临时管控或步道调整；以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/姑婆山
  - https://you.ctrip.com/sight/hezhou707/10758974.html

### 2.8 姑婆山茶园 / 方家茶园片区
- **门票参考**：通常包含在景区联动游览范围内；以现场/官方公告为准。
- **建议时长**：30-60 分钟。
- **玩法**：看层叠茶园、公路与山雾，适合拍清新感构图；如果天气带一点薄雾，观感通常更好。
- **交通**：位于姑婆山景区或周边联动区域，自驾或景区内部交通衔接更方便。
- **预约要点**：一般无需单独预约，按景区当天开放范围游览即可；以现场/官方公告为准。
- **来源链接**：
  - https://baike.baidu.com/item/姑婆山
  - https://you.ctrip.com/sight/hezhou707/10758974.html

### 2.9 玉石林风景区
- **门票参考**：成人票常见区间约 ¥55-60；具体以现场/官方公告为准。
- **建议时长**：2-3 小时。
- **玩法**：以洁白石林、石芽、峰丛地貌见长，和常见的青灰色石林视觉差异明显，适合地貌爱好者和摄影爱好者。
- **交通**：位于平桂区方向，距市区不远，自驾、打车都比较方便。
- **预约要点**：通常不属于强预约景区，但暑热天气要注意防晒、补水；以现场/官方公告为准。
- **来源链接**：
  - https://you.ctrip.com/sight/hezhou707/15480.html
  - https://baike.baidu.com/item/玉石林

### 2.10 紫云仙境景区
- **门票参考**：成人票常见区间约 ¥40-50；具体以现场/官方公告为准。
- **建议时长**：1.5-2.5 小时。
- **玩法**：核心看点是溶洞内部钟乳石、地下空间氛围和喀斯特洞景，适合与玉石林组成“地貌双拼”路线，下雨天也相对不影响体验。
- **交通**：通常与市区、玉石林一线联动较方便，打车或自驾可达。
- **预约要点**：一般无需复杂预约；洞内潮湿、地面湿滑，建议穿防滑鞋；以现场/官方公告为准。
- **来源链接**：
  - https://you.ctrip.com/sight/hezhou707/1816947.html
  - https://baike.baidu.com/item/紫云仙境

### 2.11 十八水原生态景区
- **门票参考**：成人票常见区间约 ¥48-60；具体以现场/官方公告为准。
- **建议时长**：3-4 小时。
- **玩法**：适合看叠瀑、溪谷、原始山林与徒步步道。如果你比起“到此一游式观景”更喜欢在山里走一走，十八水通常会比普通观光点更有参与感。
- **交通**：位于市区外山林方向，自驾、包车更稳妥。
- **预约要点**：雨天观瀑效果更好，但也更滑；部分天气情况下可能调整开放区域；以现场/官方公告为准。
- **来源链接**：

  - https://baike.baidu.com/item/十八水原生态景区

### 2.12 西溪森林温泉度假村
- **门票参考**：温泉票常见区间约 ¥198-228；房型、套餐价格以现场/官方公告为准。
- **建议时长**：2-5 小时；住一晚会更舒服。
- **玩法**：主打森林温泉、度假放松与酒店休闲，适合把它放在行程后段做“收尾”，也适合情侣、亲子、长辈。
- **交通**：通常建议自驾或包车前往，往返更省心。
- **预约要点**：周末、节假日房态与价格波动会比较明显，住宿建议提前订；营业与温泉开放安排以现场/官方公告为准。
- **来源链接**：
  - https://you.ctrip.com/sight/hezhou707.html

### 2.13 贺州温泉 / 路花温泉景区
- **门票参考**：温泉票常见区间约 ¥60-120；套餐与住宿价格以现场/官方公告为准。
- **建议时长**：2-4 小时。
- **玩法**：适合安排在山水行程之后泡汤放松，整体门槛比高端度假温泉更低，是比较常见的“当天去当天回”选项。
- **交通**：位于八步区黄田镇方向，从市区出发相对方便。
- **预约要点**：节假日建议提前查看夜场、套票、可用池区等信息；其余以现场/官方公告为准。
- **来源链接**：

- **建议时长**：1-2 小时。
- **玩法**：相比黄姚古镇，它更偏日常生活感与老街气质，不是那种标准景区型古镇，适合喜欢看旧街肌理和市井氛围的人。
- **交通**：位于八步区贺街镇方向，自驾或打车可达。
- **预约要点**：通常无需预约；更适合白天去看建筑与街巷；以现场/官方公告为准。

### 2.19 岔山村
- **门票参考**：多见免费开放或低价维护性收费口径；具体以现场/官方公告为准。

## 3. 1-3 天行程建议

### 3.1 1 天快刷版：第一次到贺州
**适合**：高铁到达、时间紧、只想抓核心印象的人。

**路线 A：玉石林 + 紫云仙境 + 市区晚餐**
- 上午去玉石林
- 下午去紫云仙境
- 晚上回八步区吃饭

**路线 B：黄姚古镇半日深逛**
- 中午前后抵达黄姚
- 下午看石板街、带龙桥、古戏台
- 傍晚等古镇亮灯后返程

**路线 C：姑婆山 + 温泉**
- 白天走姑婆山主线
- 晚上去路花温泉/伴山温泉

### 3.2 2 天经典版：古镇 + 山水
**Day 1：黄姚古镇住一晚**
- 下午进入黄姚古镇
- 依次看古戏台、带龙桥、龙爪榕、老街巷道
- 晚上在古镇吃饭、看夜景
- 次日清晨再拍一轮人少的古镇

**Day 2：姑婆山 / 玉石林二选一**
- 喜欢森林瀑布：去姑婆山
- 喜欢地貌奇观：去玉石林 + 紫云仙境
- 晚上返回市区或直接返程

### 3.3 3 天舒适版：最稳妥的首次路线
**Day 1：黄姚古镇**
- 下午入镇，慢逛 + 夜宿

**Day 2：姑婆山 / 十八水 + 温泉**
- 白天走山水线
- 晚上泡温泉，降低赶路疲劳

**Day 3：玉石林 + 紫云仙境**
- 走一条相对轻松的喀斯特地貌线
- 返程前在市区补本地菜

### 3.4 3 天人文乡野版：适合自驾
**Day 1**：黄姚古镇  
**Day 2**：富川古明城 + 秀水状元村 + 岔山村  
**Day 3**：凤溪古瑶寨 + 钟山百里水墨画廊 / 龟石水库

这条线更安静，也更吃自驾与天气。

## 4. 美食
贺州的饮食不是那种一眼“全国网红”的爆款型，而是**桂东北 + 粤边 + 瑶乡风味混合**，越是本地馆子、乡镇馆子，越容易吃出特色。

### 4.1 值得留意的本地味道
- **黄姚豆豉菜**：黄姚古镇代表味型，常见有豆豉蒸鱼、豆豉排骨、豆豉炒菜等。
- **油茶**：带明显桂北风格，适合早餐或下午点心式尝试。
- **酿菜**：酿豆腐、酿辣椒、酿苦瓜都很常见。
- **瑶家烟熏腊味**：富川、瑶寨周边更容易吃到。
- **山菌、竹笋、山坑鱼、土鸡**：山地农家菜常见搭配。
- **牛肠酸、酸嘢类小吃**：偏开胃，适合路边尝鲜。

### 4.2 去哪吃更稳
- **八步区市区**：本地馆子多，适合正餐、粉面、夜宵。
- **黄姚古镇内**：适合吃豆豉菜、古镇民宿餐。
- **富川/钟山乡镇**：适合吃农家菜，但更依赖自驾。

### 4.3 点菜建议
- 两人出行别点太满，贺州很多馆子分量不小。
- 古镇主街餐馆别只看门头热闹，拐进支巷往往体验更稳。
- 想吃本地味，优先找**豆豉、酿菜、山货、土菜**这些关键词。

## 5. 住宿

### 5.1 黄姚古镇住宿
**适合**：第一次来、喜欢夜景和清晨古镇氛围的人。  
**优点**：晚上和第二天早上的黄姚，往往比白天更值得。  
**注意**：老房改造民宿隔音、热水、搬运行李体验差异很大，订前重点看评价。

### 5.2 八步区 / 贺州站附近住宿
**适合**：高铁落地、想高效串联玉石林、紫云仙境、姑婆山的人。  
**优点**：交通、吃饭、补给方便。  
**缺点**：度假感一般。

### 5.3 温泉度假型住宿
**适合**：情侣、亲子、长辈、想把节奏放慢的人。  
**建议**：安排在最后一晚最舒服，整体行程会更有“度假收尾感”。

### 5.4 订房避坑点
- 黄姚古镇民宿重点看：**停车位置、拖箱距离、是否临街、隔音、热水稳定性**。
- 温泉酒店重点看：**是否含温泉票、可用汤池数量、夜场时间、儿童政策**。
- 节假日别只看最低价，很多低价房不一定在你以为的核心位置。

## 6. 交通

### 6.1 外部交通
- **高铁**：贺州站是最常用的进出点，珠三角方向进入尤其方便。
- **自驾**：如果计划玩黄姚、富川、钟山、乡村点位，自驾体验会比纯公共交通高不少。
- **客运**：可以到县城后再转车，但效率与灵活度一般。

### 6.2 市内交通
- **市区到玉石林/紫云仙境**：打车通常能解决。
- **市区到姑婆山/十八水/温泉**：建议自驾、包车或提前安排往返车。
- **市区到黄姚古镇**：跨县区，临时打车成本可能较高，建议提前做计划。
- **黄姚到富川/钟山古村线**：自驾几乎是最佳解。

### 6.3 自驾提醒
- 山区道路弯多，雨天谨慎驾驶。
- 乡村景点导航终点不一定就是停车点，建议出发前查最近停车场。
- 节假日黄姚古镇周边停车可能紧张，尽量早到。

---

## 7. 避坑
- **不要把黄姚古镇只当成“拍一小时就走”的点**：它真正值钱的是傍晚、夜晚和清晨。
- **不要低估点位之间距离**：贺州不是所有景点都在一个景区群里，跨县区会花时间。
- **不要对公共交通抱太高预期**：能玩，但不一定舒服。
- **不要雨天穿普通平底鞋去瀑布/溶洞景区**：十八水、姑婆山、紫云仙境都可能湿滑。
- **不要忽略温泉价格波动**：周末、节假日和普通工作日差价可能很大。
- **不要错季硬冲花海、荷塘类点位**：这些地方很看季节状态。
- **不要只盯平台最低票价**：很多低价票可能附带时间、场次或使用限制。
- **不要在古镇最热闹主街随便盲选餐馆**：往支巷走几步，常常更稳。
