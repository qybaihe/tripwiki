## 0. 城市概览

防城港在广西最南端，是中国大陆海岸线最西南端的海滨城市之一，也是广西少数能把**海港、边境、口岸、海岛村落、滨海公路、红树林、山地雨林**放进同一趟行程里的目的地。

它不是那种靠单一超级景点撑起来的城市，更像一条由不同气质片区拼起来的旅行带：
- **港口区 / 西湾**：城市海湾、夜景、住宿和餐饮最方便。
- **江山半岛**：白浪滩、怪石滩、白沙湾、月亮湾、白龙炮台等，适合海岸线一日游。
- **东兴边境线**：东兴口岸、国门、竹山村、大清国一号界碑、京族三岛，适合感受边境与海边叠加的氛围。
- **上思生态线**：十万大山、野人谷、温泉，更适合做第三天的山地休闲收尾。

如果你想找的是：
- 一个比北海更松弛、比纯口岸城市更好玩、比纯海岛城市更多样的广西海边目的地；
- 一趟旅行里既看海，也看边境，还能顺手吃海鲜和越南风味；
- 有车之后会明显更舒服、但没车也能分段完成的 2-3 日短途；

那防城港是很值得安排的。

## 1. 行前速览

### 1.1 最适合谁来
- 喜欢海边，但不想去过度拥挤的热门海滨城市的人。
- 想看国门、界碑、口岸贸易和边境城市气质的人。
- 自驾党、摄影党、情侣、亲子、小团队周末出行。
- 去过北海、涠洲岛之后，想换一种广西沿海体验的人。

### 1.2 建议停留天数
- **1 天**：只能刷江山半岛或东兴边境线其中一条。
- **2 天**：最经典，刚好做出“海岸线 + 边境线”组合。
- **3 天**：最舒服，可加十万大山 / 野人谷 / 温泉。
- **4 天及以上**：适合慢游、拍日落、住海边、分散安排多个点。

### 1.3 最佳旅行时间
- **10 月—次年 4 月**：天气相对舒服，适合海边步行和城市漫游。
- **5—9 月**：海边更有夏日感，但更晒、更闷，也更要注意雷雨和台风。
- **节假日 / 暑期 / 周末傍晚**：白浪滩、金滩、东兴口岸周边人流会明显上涨。

### 1.4 穿衣与装备
- 海边紫外线强，建议带：防晒霜、帽子、墨镜、薄外套、凉鞋 / 防滑鞋。
- 礁石区和海边湿滑地带不要穿过滑鞋底。
- 山地线建议带轻便徒步鞋、驱蚊液、备用衣物。
- 口岸 / 边境村落活动建议随身带身份证。

### 1.5 到达方式
- **高铁**：通常可到防城港北站、东兴市站；南宁方向进入最方便。
- **自驾**：非常推荐，江山半岛、东兴、上思三条线串起来会轻松很多。
- **大巴 / 城际**：也能到，但景点分散，落地后还需要打车 / 包车解决区间移动。

### 1.6 城市内怎么走
- **港口区 / 西湾 / 市区短途**：打车够用。
- **江山半岛一线**：最好自驾或包车。
- **东兴 + 竹山村 + 京岛 / 金滩**：自驾效率最高。
- **上思方向**：更适合单独拿半天到一天安排。

### 1.7 出发前要知道
- 海边礁石、潮汐和湿滑是真风险，不要为拍照冒险。
- 口岸、国门、界碑等区域要遵守边境管理要求。
- 海鲜消费要先问清单价、称重、做法费、加工费。
- 文中门票、开放、预约信息仅作出行整理参考；**对不确定内容统一以现场/官方公告为准**。

### 2.1 白浪滩
- **门票参考**：常见为收费海滩，具体票价、停车费、冲洗费等以现场/官方公告为准。
- **建议时长**：2-4 小时。
- **玩法**：防城港最有代表性的海滩之一，适合踏浪、看海、拍日落、带娃玩沙。沙滩开阔，第一次来江山半岛基本都会把这里放进主行程。
- **交通**：位于江山半岛，适合自驾或打车前往，也可与月亮湾、白沙湾、怪石滩同日串联。
- **预约要点**：一般按现场购票 / 官方渠道安排；节假日停车紧张，建议早到；天气不好时以现场/官方公告为准。

### 2.7 江山半岛旅游度假区
- **门票参考**：片区型度假区，本体通常不单独计票，内部各景点收费以现场/官方公告为准。

- **玩法**：如果不想把每个点单独拆开，可以把江山半岛整体当成一条滨海自驾线来走，核心是海岸、公路、海滩和礁石景观。
- **交通**：距离防城港市区不算太远，自驾最佳。
- **预约要点**：片区开放以现场/官方公告为准；建议避开正午，把主要拍照时间留给上午晚些时候或傍晚。

### 2.16 京岛风景名胜区
- **门票参考**：片区联动管理，票务及内部收费项目以现场/官方公告为准。

- **玩法**：这里的价值不只是看海，更在于京族文化、海边村落、渔家生活和边境海岛氛围，是防城港辨识度很高的一条线。

- **预约要点**：预约、开放、活动安排以现场/官方公告为准；若赶上节庆活动，人流会明显增加。

- **预约要点**：是否收费、停车、娱乐项目开放等以现场/官方公告为准；旺季傍晚车流集中，建议早点进场。

### 2.18 巫头海滩

## 3. 1-3 天行程安排

### 3.1 1 天快刷版：第一次到防城港
**方案 A｜海岸线版**
- 防城港市区出发
- 白浪滩 → 月亮湾 → 白沙湾 → 怪石滩 → 西湾夜景
- 适合第一次来、主要想看海的人。

**方案 B｜边境版**
- 东兴口岸 → 东兴国门景区 → 中越友谊大桥外景 → 竹山村 → 大清国一号界碑 → 万尾金滩日落
- 适合想快速体验边境气质的人。

### 3.2 2 天经典版：海岸线 + 边境线
**D1｜江山半岛一整天**
- 白浪滩
- 月亮湾
- 白沙湾
- 怪石滩
- 白龙古炮台
- 晚上回港口区 / 西湾吃海鲜、住市区

**D2｜东兴边境 + 京族三岛**
- 东兴口岸
- 东兴国门景区
- 中越友谊大桥外景
- 竹山村
- 大清国一号界碑
- 京岛风景名胜区 / 万尾金滩看日落

这是第一次去防城港最稳的一套：一天看海，一天看边境，内容完全不重样。

### 3.3 3 天进阶版：加一条山地休闲线
**D1｜江山半岛海岸线**
- 白浪滩 → 白沙湾 → 怪石滩 → 西湾夜景

**D2｜东兴边境线**
- 东兴口岸 → 国门 → 竹山村 → 一号界碑 → 万尾金滩

**D3｜上思生态放松线**
- 十万大山国家森林公园 / 野人谷 二选一
- 时间宽松的话，下午或傍晚去布透温泉做收尾

### 3.4 自驾党推荐节奏
- 第一天住港口区，方便吃饭和补给。
- 第二天如重点玩东兴，可直接住东兴，少折返。
- 第三天若去上思，再往山地方向走，避免来回穿城。

## 4. 美食

防城港的吃，核心不是“必须打卡某家超级名店”，而是吃对类型：**海鲜、京族风味、越南风味、广西沿海家常小吃**。

### 4.1 海鲜怎么吃
常见品类：
- 生蚝、虾、蟹、花甲、鱿鱼、海螺、石斑、沙虫等。

常见吃法：
- 白灼、清蒸、蒜蓉、椒盐、香辣、海鲜粥。

点菜建议：
- 先问清**单价、称重方式、做法费、加工费、纸巾 / 茶位是否另收**。
- 景区海边大排档别默认“看着普通就不贵”。
- 想吃稳一点，优先看本地人多、明码标价清楚的店。

### 4.2 京族风味
可以留意：
- 京族米粉
- 风吹饼
- 虾饼
- 海鲜粥
- 烤海产

这类风味在京岛、万尾、东兴方向更容易遇到，更有在地感。

### 4.3 越南风味
东兴边境线非常适合吃这一口：
- 越南鸡粉
- 法棍
- 春卷
- 卷粉
- 滴漏咖啡
- 酸辣口小吃

如果你没打算出境，在东兴其实也能感受到比较明显的边境饮食气质。

### 4.4 推荐觅食区域
- **港口区 / 西湾**：适合住市区的人，选择最多。
- **东兴口岸周边**：越南风味、小吃、边贸感更强。
- **万尾金滩 / 京岛一带**：适合边看海边吃，但价格一定先问清。

## 5. 住宿

### 5.1 港口区 / 西湾片区
**适合**：第一次来、防城港市区中转、想住方便的人。

优点：
- 酒店选择多。
- 吃饭方便。
- 晚上还能散步看海湾夜景。

缺点：
- 去东兴和上思都还要跑一段。

### 5.2 江山半岛 / 白浪滩周边
**适合**：想把海边体验拉满的人。

优点：
- 离海近，方便看晨昏海景。
- 节奏慢，适合情侣和摄影党。

缺点：
- 配套不如市区成熟。
- 淡季部分民宿品质波动较大。

### 5.3 东兴市区 / 口岸周边
**适合**：第二天重点玩口岸、竹山村、京族三岛的人。

优点：
- 去东兴口岸、国门和金滩都更顺手。
- 吃夜宵方便，边境城市氛围更足。

缺点：
- 节假日价格波动明显。

### 5.4 上思 / 温泉度假区
**适合**：3 天以上行程，主打山地休闲、温泉放松的人。

优点：
- 安静，适合疗休养式慢游。

缺点：
- 如果主要玩海边和东兴，住这边会增加通勤成本。

### 5.5 订房建议
- 周末、节假日、暑期、春节前后价格上涨明显。
- 海边民宿重点看：卫生、返潮、热水、隔音、停车、空调效果。
- 自驾党尽量优先选停车方便的住宿。

## 6. 交通

### 6.1 外部到达
- **高铁**：南宁、钦州等方向进入最方便，适合周末短途。
- **飞机**：通常可先飞南宁，再转高铁或自驾进入防城港。
- **自驾**：如果你想完整玩江山半岛 + 东兴 + 上思，自驾体验明显最好。

### 6.2 市内交通建议

**打车**
- 适合市区、东兴城区的点对点短途。
- 跨区多点连线时成本会上去。

**自驾 / 租车**
- 最推荐。
- 原因很简单：景点分散，且很多最好看的点位就在路上和沿线。

**包车 / 拼车一日游**
- 适合不会开车、又想多点连刷的人。
- 但要提前确认路线，避免购物点或过度赶路。

### 6.3 常见区间理解
- **市区 → 江山半岛**：适合半天到一天。
- **市区 → 东兴口岸**：适合半天到一天。
- **东兴 → 竹山村 / 万尾金滩**：适合半天联动。
- **防城港 / 东兴 → 上思生态线**：更适合整天安排。

## 7. 避坑

### 7.1 不要贪多
防城港不是那种所有景点都挤在一起的城市。把白浪滩、东兴口岸、竹山村、金滩、十万大山硬塞进一天，只会一直赶路。

### 7.2 礁石区别为拍照冒险
怪石滩这类地方很出片，但礁石真的滑，海浪也不是摆设。不要踩到边缘，也不要背对海浪摆太久。

### 7.3 海鲜先问价再下单
重点确认：
- 是按斤还是按份；
- 做法费怎么算；
- 配菜、茶位、纸巾是否另收；
- 最终结账前再核一次菜单。

### 7.4 边境区域守规矩
口岸、界碑、边检设施、桥梁周边别乱飞无人机、别越线、别在限制区域久留拍摄。

### 7.5 不要低估防晒
海边“有风”不代表“不晒”。特别是白浪滩、金滩、竹山村这类无遮挡区域，很容易晒伤。

### 7.6 海边住宿别只看便宜
便宜海边民宿常见问题是返潮、隔音差、卫生一般、热水不稳、停车麻烦。宁可多花一点，也别把睡眠和洗澡体验赌掉。

### 7.7 天气不好就及时换计划
遇到雷雨、台风、强对流天气时，海边和山地都可能受影响。与其硬冲，不如改成市区散步、东兴吃东西、压缩点位数量。
