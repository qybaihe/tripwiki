# 龙岩旅游攻略（福建）

> 版本：快速可用版  
> 适合：周末 2-3 天、亲子、客家文化、山水徒步、土楼摄影、红色旅游  
> 交通总提示：外地进入龙岩，通常先到龙岩站/龙岩冠豸山机场，再按区县分线去永定、连城、长汀、上杭、武平、漳平等地。龙岩市域跨度较大，跨县景点更适合自驾或包车串联。  
> 说明：本版优先做“可用完整稿”，门票以景区官方/购票平台实时页面为准；对免费或信息波动大的点位，已明确标注“以现场公示为准”。

## 龙岩速览

龙岩是福建西部的旅行大市，玩法非常分散，但也正因为分散，适合按主题拆开玩：
- **永定线**：土楼群、客家文化、乡村摄影
- **连城线**：冠豸山、石门湖、龙硿洞、培田古村，偏山水与古村
- **长汀线**：古城、店头街、济川门、卧龙书院，适合慢逛
- **上杭线**：古田会议旧址、梅花山、中国虎园，适合红色+生态
- **武平/漳平线**：梁野山、中山河湿地、九鹏溪，适合休闲度假

## 行程建议

### 2天经典版
- **D1 永定**：福建土楼永定景区 → 初溪古村落景区 → 夜宿永定/龙岩市区
- **D2 连城**：冠豸山景区 → 石门湖 → 龙硿洞

### 3天深度版
- **D1 上杭**：古田旅游区 → 梅花山中国虎园
- **D2 长汀**：汀州古城 → 济川门 → 店头街 → 卧龙书院
- **D3 连城/永定二选一**：
  - 山水线：冠豸山景区 → 石门湖 → 培田古民居
  - 客家线：福建土楼永定景区 → 福建土楼王景区 → 初溪古村落景区

## 1. 福建土楼永定景区
- **地点**：龙岩市永定区，通常指永定土楼核心游览区
- **门票**：常见为联票/分景区票，价格浮动较大，**以官方售票页和现场公示为准**
- **推荐指数**：★★★★★
- **建议时长**：4-8 小时
- **玩法**：看客家土楼群落、拍建筑几何感、了解土楼居住文化，适合第一次来龙岩的人
- **预约要点**：旺季、节假日建议提前在线购票；若想看多个土楼群，建议先确认联票是否覆盖目标点
- **交通**：从龙岩市区或永定站自驾/包车最省心；公共交通可先到永定城区再换乘景区班线

## 美食建议
- **长汀豆腐干**：长汀最稳的伴手礼之一
- **连城白鸭汤**：连城代表味道，适合山水线用餐补给
- **客家酿豆腐**：永定、长汀都常见
- **簸箕粄、芋子包、炒九门头**：适合在古城/老街边走边吃
- **牛肉汤、河田鸡相关菜式**：适合在长汀线安排正餐

## 住宿怎么选
- **第一次去、想省折腾**：住龙岩市区
- **重点玩土楼**：住永定区或土楼景区周边民宿
- **重点玩山水**：住连城县城
- **重点玩古城夜景**：住长汀古城附近
- **重点玩红色+生态**：住上杭县城或古田周边

## 避坑提示
- 龙岩不是“一个市区打车全搞定”的城市，**景点分散是最大特征**，行程别排太满。
- 土楼、冠豸山、石门湖、龙硿洞分属不同方向，**别误以为当天能随便串**。
- 门票与联票波动比较大，尤其是土楼和山水组合产品，**下单前一定看票型说明**。
- 山区景点天气变化快，夏天防晒防蚊，雨天注意道路湿滑。
- 古田、博物馆、红色旧址类点位，可能遇到周一闭馆或临时预约规则调整，出发前再查一遍。
