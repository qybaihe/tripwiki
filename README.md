## 仓库定位

`TripWiki` 不是景点资料堆砌库，而是面向真实出行决策的城市攻略库。

每个城市攻略目标包含：
- 城市概览
- 行前速览 / 行前准备
- 1–3 天（部分城市扩展到 4 天）行程建议
- 景点清单（尽量达到 18+）
- 门票参考 / 建议时长 / 玩法 / 交通 / 预约要点 / 来源链接
- 美食
- 住宿
- 交通
- 避坑
- 参考来源

当前整体内容以 **快速可用版** 为主：
- 先完成全国覆盖
- 再逐步补齐高质量缺口
- 对无法稳定坐实的信息，保留“待复核”而不硬编

---

## 目录结构

- `cities/`：城市攻略 Markdown 文件
- `data/progress.json`：城市覆盖进度状态
- `BACKLOG.md`：早期队列/说明文件
- `SOURCES.md`：来源与引用口径
- `ISSUES.md`：已完成覆盖后，仍需补偿的质量缺口清单
- `controller/`：主控状态与协调规则
- `scripts/`：自动化脚本
