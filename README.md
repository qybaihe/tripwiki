# TripWiki

一个持续增长的旅游城市攻略仓库：按城市沉淀「景点 + 门票/价格区间 + 推荐指数 + 建议游玩时长 + 玩法/亮点 + 开放时间/预约要点 + 交通/地址 + 来源链接 + 更新时间」。

## 字段口径
- **推荐指数**：5 分制（⭐1–⭐5），结合“代表性/体验/性价比/拥挤度/可替代性”。
- **门票价格**：优先官方/景区/文旅局，其次主流平台；价格可能变化，必须标注来源与更新时间。
- **游玩时间**：建议区间（例如 1–2h / 3–4h / 半天 / 全天）。

## 目录
- `cities/`：每个城市一份 Markdown
- `cities/_TEMPLATE.md`：写作模板
- `BACKLOG.md`：城市队列与进度
- `SOURCES.md`：常用来源与引用规范

## 更新方式（自动化）
- 主控脚本每 30 分钟运行一次：生成下一批城市内容 → 收口提交 → push。
- 写入过程中，若关键信息（价格/开放/预约）无法坐实，必须进“待复核”。
